# AWS::D ynamoDB::라이브 사양에 대한 테이블 시간

지정된 테이블에 대해 TTL(Time to Live)을 사용하거나 사용하지 않도록 설정하는 데 사용되는 설정을 나타냅니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AttributeName" : String,
  "Enabled" : Boolean
}
```

## 속성

`AttributeName` 테이블에 있는 항목의 만료 시간을 저장하는 데 사용되는 TTL 속성의 이름입니다.

- `AttributeName` TTL을 사용하도록 설정하거나 TTL이 이미 사용하도록 설정된 경우 이 속성이 필요합니다.
- 이 특성을 업데이트하려면 먼저 TTL을 사용 안함으로 설정한 다음 새 속성 이름으로 TTL을 사용 가능하게 해야 합니다.
  _필수_: 조건부  
  _유형_: 문자열  
  _업데이트 필요 사항_: 중단 없음

`Enabled`  
테이블에서 TTL을 활성화(true) 또는 비활성화(false)할지 여부를 나타냅니다.  
_필수 항목 여부_: 예  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
