# AWS::KinesisFirehose::D eliveryStream AmazonopensearchserviceBufferingHints

Amazon OpenSearch Service 대상으로 데이터를 전송하기 전에 수행할 버퍼링에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "IntervalInSeconds" : Integer,
  "SizeInMBs" : Integer
}
```

## 속성

`IntervalInSeconds`  
수신 데이터를 대상으로 전달하기 전에 지정된 기간(초) 동안 버퍼링합니다. 기본값은 300(5분)입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `60`  
_최대_: `900`  
_업데이트 필요 사항_: 중단 없음

`SizeInMBs`  
들어오는 데이터를 대상으로 전달하기 전에 지정된 크기(MB)로 버퍼링합니다. 기본값은 5입니다. 이 파라미터는 일반적으로 10초 이내에 전송 스트림으로 수집하는 데이터의 양보다 큰 값으로 설정하는 것이 좋습니다. 예를 들어 일반적으로 1MB/초의 속도로 데이터를 수집하는 경우 값은 10MB 이상이어야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `1`  
_최대_: `100`  
_업데이트 필요 사항_: 중단 없음
