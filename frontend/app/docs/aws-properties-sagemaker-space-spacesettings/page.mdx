# AWS::SageMaker::공간 공간 설정

공간 설정 모음입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "JupyterServerAppSettings" : JupyterServerAppSettings,
  "KernelGatewayAppSettings" : KernelGatewayAppSettings
}
```

## 속성

`JupyterServerAppSettings`  
JupyterServer 앱 설정.  
*필수 항목 *여부: 아니요  
_형식_: JupyterServerAppSettings  
_업데이트 필요 사항_: 중단 없음

`KernelGatewayAppSettings`  
KernelGateway 앱 설정.  
*필수 항목 *여부: 아니요  
_형식_: KernelGatewayAppSettings  
_업데이트 필요 사항_: 중단 없음
