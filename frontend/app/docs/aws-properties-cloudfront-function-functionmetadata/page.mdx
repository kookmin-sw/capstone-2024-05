# AWS::CloudFront::함수 함수 메타데이터

CloudFront 함수에 대한 메타데이터를 포함합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "FunctionARN" : String
}
```

## 속성

`FunctionARN`  
함수의 Amazon 리소스 이름(ARN)입니다. ARN은 함수를 고유하게 식별합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
