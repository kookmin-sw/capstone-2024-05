# AWS::람다::함수 DeadLetterConfig

실패한 비동기 호출에 대한 배달 못 한 편지 대기열입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "TargetArn" : String
}
```

## 속성

`TargetArn`  
Amazon SQS 대기열 또는 Amazon SNS 주제의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_패턴_: `(arn:(aws[a-zA-Z-]*)?:[a-z0-9-.]+:.*)|()`  
_업데이트 필요 사항_: 중단 없음

## 예제

### 배달 못 한 편지 대기열

배달 못한 편지 대기열을 함수에 추가합니다.
