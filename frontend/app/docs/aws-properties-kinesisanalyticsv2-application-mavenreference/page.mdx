# AWS::KinesisAnalyticsV2::애플리케이션 MavenReference

Maven 참조를 지정하는 데 필요한 정보입니다. Maven 참조를 사용하여 종속성 JAR 파일을 지정할 수 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ArtifactId" : String,
  "GroupId" : String,
  "Version" : String
}
```

## 속성

`ArtifactId`  
Maven 참조의 아티팩트 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `256`  
_패턴_: `[a-zA-Z0-9_.-]+`  
_업데이트 필요 사항_: 중단 없음

`GroupId`  
Maven 참조의 그룹 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `256`  
_패턴_: `[a-zA-Z0-9_.-]+`  
_업데이트 필요 사항_: 중단 없음

`Version`  
Maven 참조의 버전입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `256`  
_패턴_: `[a-zA-Z0-9_.-]+`  
_업데이트 필요 사항_: 중단 없음
