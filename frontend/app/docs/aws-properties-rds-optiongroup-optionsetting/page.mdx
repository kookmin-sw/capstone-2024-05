# AWS::RDS::OptionGroup 옵션 설정

`OptionSetting` 속성 형식은 속성 내의 옵션 값을 지정합니다 `OptionSetting` .

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Value" : String
}
```

## 속성

`Name`  
설정할 수 있는 설정이 있는 옵션의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Value`  
옵션 설정의 현재 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
