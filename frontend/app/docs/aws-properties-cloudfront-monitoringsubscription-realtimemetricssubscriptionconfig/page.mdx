# AWS::CloudFront::MonitoringSubscription RealtimeMetrics구독구성

추가 CloudWatch 지표에 대한 구독 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "RealtimeMetricsSubscriptionStatus" : String
}
```

## 속성

`RealtimeMetricsSubscriptionStatus`  
지정된 CloudFront 배포에 대해 추가 CloudWatch 지표가 활성화되었는지 여부를 나타내는 플래그입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `Disabled | Enabled`  
_업데이트 필요 사항_: 중단 없음
