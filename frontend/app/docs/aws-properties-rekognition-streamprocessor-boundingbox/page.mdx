# AWS::Rekognition::StreamProcessor 바운딩박스

레이블, 얼굴, 텍스트 또는 개인 보호 장비 주위의 경계 상자를 식별합니다. `left` (x 좌표)와 `top` (y 좌표)는 경계 상자의 위쪽과 왼쪽을 나타내는 좌표입니다. 이미지의 왼쪽 위 모서리는 원점(0,0)입니다.

반환되는 `top` `left` 및 값은 전체 이미지 크기의 비율입니다. 예를 들어 입력 이미지가 700x200픽셀이고 경계 상자의 왼쪽 위 좌표가 350x50픽셀인 경우 API는 `left` 값 0.5(350/700)와 `top` 값 0.25(50/200)를 반환합니다.

`width` 및 `height` 값은 경계 상자의 크기를 전체 이미지 차원의 비율로 나타냅니다. 예를 들어 입력 이미지가 700x200픽셀이고 경계 상자 너비가 70픽셀인 경우 반환되는 너비는 0.1입니다. 자세한 내용은 BoundingBox를 참조하십시오.

**메모**  
 경계 상자 좌표는 음수 값을 가질 수 있습니다. 예를 들어, Amazon Rekognition이 이미지 엣지에 있고 부분적으로만 보이는 얼굴을 감지할 수 있는 경우 서비스는 이미지 경계를 벗어난 좌표를 반환할 수 있으며, 이미지 엣지에 따라 or 값에 대해 음수 값 또는 1보다 큰 값을 얻을 수 있습니다 `left` `top` .

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Height" : Double,
  "Left" : Double,
  "Top" : Double,
  "Width" : Double
}
```

## 속성

`Height`  
전체 이미지 높이의 비율로 나타낸 경계 상자의 높이입니다.  
_필수 항목 여부_: 예  
_유형_: 더블  
_업데이트 필요 사항_: 바꿔 놓음

`Left`  
전체 이미지 너비의 비율로 나타낸 경계 상자의 왼쪽 좌표입니다.  
_필수 항목 여부_: 예  
_유형_: 더블  
_업데이트 필요 사항_: 바꿔 놓음

`Top`  
전체 이미지 높이의 비율로 나타낸 경계 상자의 위쪽 좌표입니다.  
_필수 항목 여부_: 예  
_유형_: 더블  
_업데이트 필요 사항_: 바꿔 놓음

`Width`  
전체 이미지 너비의 비율로 표시되는 경계 상자의 너비입니다.  
_필수 항목 여부_: 예  
_유형_: 더블  
_업데이트 필요 사항_: 바꿔 놓음
