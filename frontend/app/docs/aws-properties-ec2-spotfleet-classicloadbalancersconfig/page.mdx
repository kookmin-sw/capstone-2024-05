# AWS::EC2::SpotFleet ClassicLoadBalancersConfig

스팟 집합에 연결할 Classic Load Balancer를 지정합니다. 스팟 집합은 실행 중인 스팟 인스턴스를 이러한 Classic Load Balancer에 등록합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ClassicLoadBalancers" : [ ClassicLoadBalancer, ... ]
}
```

## 속성

`ClassicLoadBalancers`  
하나 이상의 Classic Load Balancer.  
_필수 항목 여부_: 예  
_유형_: ClassicLoadBalancer 목록  
_최대_: `5`  
_업데이트 필요 사항_: 바꿔 놓음
