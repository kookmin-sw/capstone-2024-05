# AWS::SageMaker::엔드포인트

`AWS::SageMaker::Endpoint` 리소스를 사용하여 요청에 지정된 구성을 사용하여 엔드포인트를 만듭니다. Amazon SageMaker는 엔드포인트를 사용하여 리소스를 프로비저닝하고 모델을 배포합니다. AWS::SageMaker::EndpointConfig 리소스를 사용하여 엔드포인트 구성을 생성합니다. 자세한 내용은 Amazon SageMaker 개발자 안내서의 Amazon SageMaker 호스팅 서비스에 모델 배포를 참조하세요\*\*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::SageMaker::Endpoint",
  "Properties" : {
      "DeploymentConfig" : DeploymentConfig,
      "EndpointConfigName" : String,
      "EndpointName" : String,
      "ExcludeRetainedVariantProperties" : [ VariantProperty, ... ],
      "RetainAllVariantProperties" : Boolean,
      "RetainDeploymentConfig" : Boolean,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`DeploymentConfig`  
끝점에 대한 배포 구성으로, 원하는 배포 전략과 롤백 구성이 포함되어 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: DeploymentConfig  
_업데이트 필요 사항_: 중단 없음

`EndpointConfigName`  
엔드포인트에 대한 구성을 지정하는 AWS::SageMaker::EndpointConfig 리소스의 이름입니다. 자세한 내용은 CreateEndpointConfig 단원을 참조하십시오.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}`  
_업데이트 필요 사항_: 중단 없음

`EndpointName`  
끝점의 이름입니다. 이름은 AWS 계정의 AWS 리전 내에서 고유해야 합니다. 이름은 에서 대소문자를 구분하지 않지만 `CreateEndpoint`대소문자는 유지되며 일부 주소에서 일치해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}`  
_업데이트 필요 사항_: 바꿔 놓음

`ExcludeRetainedVariantProperties`  
값이 로 설정된 RetainAllVariantProperties 를 사용하여 끝점 리소스를 업데이트하는 경우 `true` `ExcludeRetainedVariantProperties` 에서 제공하는 값으로 재정의할 VariantProperty 유형의 목록을 지정합니다`EndpointConfig`. 에 대한 값을 지정하지 않으면 `ExcludeAllVariantProperties`variant 속성이 재정의되지 않습니다. 새 엔드포인트 리소스를 만들 때 또는 로 설정된 경우 이 속성을 사용하지 마세요 `RetainAllVariantProperties` `false`.  
*필수 항목 *여부: 아니요  
_형식_: VariantProperty 목록  
_최대_: `3`  
_업데이트 필요 사항_: 중단 없음

`RetainAllVariantProperties`  
엔드포인트 리소스를 업데이트할 때 인스턴스 수 또는 variant 가중치와 같은 variant 속성의 보존을 활성화하거나 비활성화합니다. 끝점을 업데이트할 때 variant 속성을 유지하려면 로 설정합니다 `RetainAllVariantProperties` `true`. `EndpointConfig` 끝점을 업데이트할 때 새 호출 에 지정된 variant 속성을 사용하려면 `RetainAllVariantProperties` 로 `false` 설정합니다. 이 속성은 끝점 리소스를 업데이트할 때만 사용하고 새 끝점 리소스를 만들 때는 사용하지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`RetainDeploymentConfig`  
마지막 배포 구성을 다시 사용할지 여부를 지정합니다. 기본값은 false(구성이 재사용되지 않음)입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`Tags`  
이 리소스에 적용할 키-값 쌍 목록입니다.  
자세한 내용은 * AWS Billing and Cost Management 사용 설명서*의 리소스 태그 및 비용 할당 태그 사용을 참조하십시오.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_최대_: `50`  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`엔드포인트의 Amazon 리소스 이름(ARN)을 반환합니다 `arn:aws:sagemaker:us-west-2:012345678901:endpoint/myendpoint`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

`Fn::GetAtt` 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

####

`EndpointName`  
끝점의 이름(예: `MyEndpoint`.
