# AWS::Lambda::함수 EphemeralStorage

함수 디렉터리의 크기( `/tmp` MB)입니다. 기본값은 512이지만 512MB에서 10,240MB 사이의 정수일 수 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Size" : Integer
}
```

## 속성

`Size`  
함수 디렉터리의 크기입니다 `/tmp` .  
_필수 항목 여부_: 예  
_유형_: Integer  
_최소_: `512`  
_최대_: `10240`  
_업데이트 필요 사항_: 중단 없음
