# AWS::EFS::액세스 포인트 PosixUser

액세스 포인트를 사용하여 NFS 클라이언트가 수행하는 모든 파일 시스템 작업에 사용되는 액세스 포인트의 사용자 ID, 그룹 ID 및 보조 그룹 ID를 포함한 전체 POSIX ID입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Gid" : String,
  "SecondaryGids" : [ String, ... ],
  "Uid" : String
}
```

## 속성

`Gid`  
이 액세스 포인트를 사용하는 모든 파일 시스템 작업에 사용되는 POSIX 그룹 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`SecondaryGids`  
이 액세스 포인트를 사용하는 모든 파일 시스템 작업에 사용되는 보조 POSIX 그룹 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_최대_: `16`  
_업데이트 필요 사항_: 바꿔 놓음

`Uid`  
이 액세스 포인트를 사용하는 모든 파일 시스템 작업에 사용되는 POSIX 사용자 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
