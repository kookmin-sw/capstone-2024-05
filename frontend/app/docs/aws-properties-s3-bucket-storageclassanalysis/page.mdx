# AWS::S3::버킷 StorageClassAnalysis

Amazon S3 버킷에 대한 서로 다른 스토리지 클래스 간의 장단점을 분석하기 위해 수집하고 사용할 수 있는 액세스 패턴과 관련된 데이터를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DataExport" : DataExport
}
```

## 속성

`DataExport`  
Amazon S3 버킷에 대한 스토리지 클래스 분석과 관련된 데이터를 내보내는 방법을 지정합니다.  
*필수 항목 *여부: 아니요  
_형식_: DataExport  
_업데이트 필요 사항_: 중단 없음
