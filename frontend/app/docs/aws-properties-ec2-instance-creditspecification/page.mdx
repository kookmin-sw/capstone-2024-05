# AWS::EC2::인스턴스 크레딧 사양

T 인스턴스의 CPU 사용량에 대한 크레딧 옵션을 지정합니다.

`CreditSpecification` 은 AWS::EC2::Instance 리소스의 속성입니다.

자세한 내용은 Amazon EC2 사용 설명서*의 버스트 가능한 성능 인스턴스를 참조하세요*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CPUCredits" : String
}
```

## 속성

`CPUCredits`  
인스턴스의 CPU 사용량에 대한 크레딧 옵션입니다.  
유효한 값: `standard` | `unlimited`  
`host` 테넌시가 있는 T3 인스턴스는 `unlimited` CPU 크레딧 옵션을 지원하지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
