# AWS::EC2::캐리어 게이트웨이

이동통신사 게이트웨이를 만듭니다. 통신 사업자 게이트웨이에 대한 자세한 내용은 * AWS Wavelength 개발자 안내서*의 통신 사업자 게이트웨이를 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::CarrierGateway",
  "Properties" : {
      "Tags" : [Tag, ... ],
      "VpcId" : String
    }
}
```

## 속성

`Tags`  
통신 사업자 게이트웨이에 할당된 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

`VpcId`  
통신 사업자 게이트웨이와 연결된 VPC의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`통신 사업자 게이트웨이 ID를 반환합니다. 예를 들면 다음과 같습니다 `cagw-05a8da9a199afb1c7`.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`CarrierGatewayId` 이동통신사 게이트웨이의 ID입니다.

`OwnerId` 통신 사업자 게이트웨이 소유자의 AWS 계정 ID입니다.

`State` 통신 사업자 게이트웨이의 상태입니다.
