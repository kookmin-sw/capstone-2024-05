# AWS::CloudFront::D istribution S3OriginConfig

Amazon S3 오리진에 대한 정보를 포함하는 복합 형식입니다. 오리진이 사용자 지정 오리진 또는 웹 사이트 엔드포인트로 구성된 S3 버킷인 경우 `CustomOriginConfig` 요소를 대신 사용합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "OriginAccessIdentity" : String
}
```

## 속성

`OriginAccessIdentity`  
오리진과 연결할 CloudFront 오리진 액세스 ID입니다. 최종 사용자가 ** CloudFront를 통해 Amazon S3 버킷의 객체에만 액세스할 수 있도록 원본 액세스 ID를 사용하여 오리진을 구성합니다. 값의 형식은 다음과 같습니다.  
origin-access-identity/cloudfront/_ID-of-origin-access-identity_  
여기서 `ID-of-origin-access-identity` 는 `ID` 원본 액세스 ID를 생성할 때 CloudFront가 요소에 반환한 값입니다.  
최종 사용자가 CloudFront URL 또는 Amazon S3 URL을 사용하여 객체에 액세스할 수 있도록 하려면 빈 요소를 지정합니다 `OriginAccessIdentity` .  
기존 배포에서 원본 액세스 ID를 삭제하려면 배포 구성을 업데이트하고 빈 요소를 포함합니다 `OriginAccessIdentity` .  
원본 액세스 ID를 바꾸려면 배포 구성을 업데이트하고 새 원본 액세스 ID를 지정합니다.  
원본 액세스 ID에 대한 자세한 내용은 Amazon CloudFront 개발자 안내서의 CloudFront를 통해 프라이빗 콘텐츠 제공을 참조하십시오**.
