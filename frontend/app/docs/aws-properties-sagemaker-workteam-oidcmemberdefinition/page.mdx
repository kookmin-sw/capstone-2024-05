# AWS::SageMaker::Workteam OidcMemberDefinition

OIDC ID 공급자(IdP)에 있는 사용자 그룹 목록입니다. 1개에서 10개의 그룹을 사용하여 단일 개인 작업 팀을 만들 수 있습니다. 에 있는 목록에 사용자 그룹을 추가하면 `Groups`해당 사용자 그룹을 하나 이상의 비공개 작업 팀에 추가할 수 있습니다. 비공개 작업 팀에 사용자 그룹을 추가하면 해당 사용자 그룹의 모든 작업자가 작업 팀에 추가됩니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "OidcGroups" : [ String, ... ]
}
```

## 속성

`OidcGroups`  
속성 설명을 사용할 수 없습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음
