# AWS::ApiGateway::D ocumentationPart 위치

이 `Location` 속성은 설명서가 적용되는 Amazon API Gateway API 엔터티의 위치를 지정합니다. `Location` 은 AWS::ApiGateway::D ocumentationPart 리소스의 속성입니다.

**메모**  
제약 조건 및 유효한 값을 포함하여 각 속성에 대한 자세한 내용은 _Amazon API Gateway REST API Reference의 DocumentationPart를 참조하십시오_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Method" : String,
  "Name" : String,
  "Path" : String,
  "StatusCode" : String,
  "Type" : String
}
```

## 속성

`Method`  
메서드의 HTTP 동사입니다. , `METHOD` `PATH_PARAMETER` `QUERY_PARAMETER` `REQUEST_HEADER` `REQUEST_BODY` `RESPONSE`, , `RESPONSE_HEADER`, `RESPONSE_BODY` 및 의 API 엔터티 유형에 유효한 필드입니다. 기본값은 `*` 모든 메서드에 대한 것입니다. 적용 가능한 자식 엔터티가 다른 특성의 보다 일반적인 사양을 사용하여 동일한 유형의 엔터티의 콘텐츠를 상속하는 경우 `location` 자식 엔터티의 특성은 `method` 부모 엔터티의 특성과 정확히 일치해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Name`  
대상 API 엔터티의 이름입니다. , , 및 의 API 엔터티 유형에 유효한 필수 필드입니다 `AUTHORIZER` `MODEL` `PATH_PARAMETER` `QUERY_PARAMETER` `REQUEST_HEADER` `REQUEST_BODY` `RESPONSE_HEADER`. 다른 엔터티 형식에 대해 유효하지 않은 필드입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Path`  
대상의 URL 경로입니다. `RESOURCE` `METHOD` `PATH_PARAMETER` `QUERY_PARAMETER` `REQUEST_HEADER` `REQUEST_BODY`, `RESPONSE`, `RESPONSE_HEADER`, `RESPONSE_BODY` 및 의 API 엔터티 유형에 유효한 필드입니다. 기본값은 `/` 루트 리소스에 대한 것입니다. 적용 가능한 자식 엔터티가 다른 특성의 보다 일반적인 사양을 사용하여 동일한 유형의 다른 엔터티의 콘텐츠를 상속하는 경우 `location` 자식 엔터티의 특성은 `path` 접두사로서 부모 엔터티의 특성과 일치해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`StatusCode`  
응답의 HTTP 상태 코드입니다. , , 및 의 API 엔터티 유형에 유효한 필드입니다 `RESPONSE` `RESPONSE_HEADER` `RESPONSE_BODY`. 기본값은 `*` 모든 상태 코드에 대한 것입니다. 적용 가능한 자식 엔터티가 다른 특성의 보다 일반적인 사양을 사용하여 동일한 유형의 엔터티의 콘텐츠를 상속하는 경우 `location` 자식 엔터티의 특성은 `statusCode` 부모 엔터티의 특성과 정확히 일치해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_패턴_: `^([1-5]dd|*|s*)$`  
*업데이트 필요 *사항: 바꿔 놓음

`Type`  
문서 콘텐츠가 적용되는 API 엔터티의 유형입니다. 유효한 값은 `API`, `AUTHORIZER`, `MODEL`, `RESOURCE` `METHOD` `PATH_PARAMETER` `QUERY_PARAMETER` `REQUEST_HEADER` `REQUEST_BODY` `RESPONSE` `RESPONSE_HEADER` `RESPONSE_BODY` 및 입니다. 콘텐츠 상속은 `API`, `AUTHORIZER`, `METHOD`, `MODEL`, 또는 `REQUEST_BODY` 형식의 엔터티에는 적용되지 않습니다 `RESOURCE` .  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `API | AUTHORIZER | METHOD | MODEL | PATH_PARAMETER | QUERY_PARAMETER | REQUEST_BODY | REQUEST_HEADER | RESOURCE | RESPONSE | RESPONSE_BODY | RESPONSE_HEADER`  
_업데이트 필요 사항_: 바꿔 놓음
