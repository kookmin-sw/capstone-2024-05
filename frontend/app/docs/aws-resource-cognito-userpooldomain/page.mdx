# AWS::Cognito::UserPoolDomain

AWS::Cognito::UserPoolDomain 리소스는 사용자 풀에 대한 새 도메인을 생성합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Cognito::UserPoolDomain",
  "Properties" : {
      "CustomDomainConfig" : CustomDomainConfigType,
      "Domain" : String,
      "UserPoolId" : String
    }
}
```

## 속성

`CustomDomainConfig`  
응용 프로그램에 대한 등록 및 로그인 페이지를 호스트하는 사용자 지정 도메인에 대한 구성입니다. 이 객체를 사용하여 ACM에서 관리하는 SSL 인증서를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: CustomDomainConfigType  
_업데이트 필요 사항_: 중단 없음

`Domain`  
응용 프로그램에 대한 등록 및 로그인 페이지를 호스트하는 도메인의 도메인 이름입니다. 예를 들면 다음과 같습니다 `auth.example.com`. 접두사 도메인을 사용하는 경우 이 필드는 앞에 있는 도메인의 첫 번째 부분을 나타냅니다 `.auth.[region].amazoncognito.com`.  
이 문자열은 소문자, 숫자 및 하이픈만 포함할 수 있습니다. 첫 번째 또는 마지막 문자에 하이픈을 사용하지 마세요. 마침표를 사용하여 하위 도메인 이름을 구분합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `63`  
_패턴_: `^a-z0-9?$`  
_업데이트 필요 사항_: 바꿔 놓음

`UserPoolId`  
사용자 풀 도메인을 연결하려는 사용자 풀의 사용자 풀 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `55`  
_패턴_: `[w-]+_[0-9a-zA-Z]+`  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`"Domain"인 physicalResourceId를 반환합니다. 예를 들어:

`{ "Ref": "your-test-domain" }`

Amazon Cognito 사용자 풀 도메인의 경우 `your-test-domain`Ref는 사용자 풀 도메인의 이름을 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

## 예제

### 사용자 풀에 대한 새 사용자 지정 도메인 생성

다음 예제에서는 참조된 사용자 풀에 사용자 지정 도메인 "my-test-user-pool-domain"을 생성합니다.

#### JSON (영문)

```
{
   "UserPoolDomain":{
      "Type":"AWS::Cognito::UserPoolDomain",
      "Properties":{
         "UserPoolId":{
            "Ref":"UserPool"
         },
         "Domain":"my-test-user-pool-domain.myapplication.com",
         "CustomDomainConfig":{
            "CertificateArn":{
               "Ref":"CertificateArn"
            }
         }
      }
   }
}
```

### 사용자 풀에 대한 새 기본 도메인 생성

다음 예제에서는 참조된 사용자 풀에 새 기본 도메인인 "my-test-user-pool-domain"을 생성합니다.

#### JSON (영문)

```
{
   "UserPoolDomain":{
      "Type":"AWS::Cognito::UserPoolDomain",
      "Properties":{
         "UserPoolId":{
            "Ref":"UserPool"
         },
         "Domain":"my-test-user-pool-domain"
      }
   }
}
```
