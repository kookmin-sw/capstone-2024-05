# AWS::S3::MultiRegionAccessPoint 리전

다중 리전 액세스 포인트를 생성할 때 특정 리전과 연결된 버킷입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Bucket" : String,
  "BucketAccountId" : String
}
```

## 속성

`Bucket`  
리전에 연결된 버킷의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`BucketAccountId`  
이 다중 리전 액세스 포인트와 연결된 Amazon S3 버킷을 소유하는 AWS 계정 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
