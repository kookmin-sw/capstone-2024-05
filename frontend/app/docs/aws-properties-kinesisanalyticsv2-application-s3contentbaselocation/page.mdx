# AWS::KinesisAnalyticsV2::애플리케이션 S3ContentBaseLocation

Amazon Data Analytics 애플리케이션의 기본 위치입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BasePath" : String,
  "BucketARN" : String
}
```

## 속성

`BasePath`  
S3 버킷의 기본 경로입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `1024`  
_패턴_: `[a-zA-Z0-9/!-_.*'()]+`  
_업데이트 필요 사항_: 중단 없음

`BucketARN`  
S3 버킷의 Amazon 리소스 이름(ARN)입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `2048`  
_패턴_: `arn:.*`  
_업데이트 필요 사항_: 중단 없음
