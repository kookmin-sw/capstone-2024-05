# AWS::SageMaker::모델 추론 실행 구성

다중 컨테이너 엔드포인트의 컨테이너가 실행되는 방법에 대한 세부 정보를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Mode" : String
}
```

## 속성

`Mode` 다중 컨테이너의 컨테이너가 실행되는 방식입니다. 유효한 값은 다음과 같습니다.

- `Serial` - 컨테이너는 직렬 파이프라인으로 실행됩니다.
- `Direct` - 지정한 개별 컨테이너만 실행됩니다.
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _업데이트 필요 사항_: 바꿔 놓음
