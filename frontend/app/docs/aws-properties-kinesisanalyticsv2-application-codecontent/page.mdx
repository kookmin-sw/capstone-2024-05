# AWS::KinesisAnalyticsV2::애플리케이션 코드콘텐츠

Managed Service for Apache Flink 애플리케이션에 대한 애플리케이션 코드 또는 애플리케이션 코드의 위치를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "S3ContentLocation" : S3ContentLocation,
  "TextContent" : String,
  "ZipFileContent" : String
}
```

## 속성

`S3ContentLocation`  
애플리케이션 코드가 포함된 Amazon S3 버킷에 대한 정보입니다.  
*필수 항목 *여부: 아니요  
_형식_: S3ContentLocation  
_업데이트 필요 사항_: 중단 없음

`TextContent`  
Managed Service for Apache Flink 애플리케이션의 텍스트 형식 코드입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `102400`  
_업데이트 필요 사항_: 중단 없음

`ZipFileContent`  
Managed Service for Apache Flink 애플리케이션의 zip 형식 코드입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
