# AWS::Route53Resolver::ResolverRule 대상 주소

CreateResolverRule 요청에서 DNS 쿼리를 전달하려는 IP의 배열입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Ip" : String,
  "Ipv6" : String,
  "Port" : String
}
```

## 속성

`Ip`  
DNS 쿼리를 전달할 IPv4 주소 1개.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `7`  
_최대_: `36`  
_업데이트 필요 사항_: 중단 없음

`Ipv6`  
 DNS 쿼리를 전달할 IPv6 주소 1개.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `7`  
_최대_: `39`  
_업데이트 필요 사항_: 중단 없음

`Port`  
`Ip` DNS 쿼리를 전달하려는 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `65535`  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

- AWS::Route53Resolver::ResolverRule 주제의 반환 값
- Amazon Route 53 API Reference\* \*의 TargetAddress
