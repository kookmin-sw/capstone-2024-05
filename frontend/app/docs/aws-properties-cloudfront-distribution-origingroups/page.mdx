# AWS::CloudFront::D istribution 오리진 그룹

배포에 대해 지정된 원본 그룹에 대한 복합 데이터 형식입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Items" : [ OriginGroup, ... ],
  "Quantity" : Integer
}
```

## 속성

`Items`  
배포의 항목(원본 그룹)입니다.  
*필수 항목 *여부: 아니요  
_유형_: OriginGroup 목록  
_업데이트 필요 사항_: 중단 없음

`Quantity`  
원본 그룹의 수입니다.  
_필수 항목 여부_: 예  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
