# AWS::EC2::IPAMPool 프로비저닝된 Cidr

IPAM 풀에 프로비저닝된 CIDR입니다. CIDR은 IP 주소 및 관련 네트워크 마스크(또는 넷마스크)를 나타내며 IP 주소 범위를 나타냅니다. IPv4 CIDR의 예는 다음과 같습니다 `10.24.34.0/23`. IPv6 CIDR의 예는 입니다 `2001:DB8::/32`.

**메모**  
이 리소스 유형에서는 넷마스크 길이를 사용하여 CIDR을 프로비저닝할 수 없습니다. 넷마스크 길이를 사용하여 CIDR을 프로비저닝하려면 AWS::EC2::IPAMPoolCidr을 사용합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Cidr" : String
}
```
