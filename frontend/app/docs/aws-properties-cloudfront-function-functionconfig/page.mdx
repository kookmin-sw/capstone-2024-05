# AWS::CloudFront::함수 함수 구성

CloudFront 함수에 대한 구성 정보를 포함합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Comment" : String,
  "Runtime" : String
}
```

## 속성

`Comment`  
함수를 설명하는 주석입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Runtime`  
함수의 런타임 환경입니다. 유일하게 유효한 값은 `cloudfront-js-1.0`입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `cloudfront-js-1.0`  
_업데이트 필요 사항_: 중단 없음
