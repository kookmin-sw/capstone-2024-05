# AWS::SageMaker::Code리포지토리

Git 리포지토리를 SageMaker 계정의 리소스로 생성합니다. 생성한 노트북에 Git 소스 제어를 사용할 수 있도록 리포지토리를 노트북 인스턴스와 연결할 수 있습니다. Git 리포지토리는 SageMaker 계정의 리소스이므로 둘 이상의 노트북 인스턴스와 연결할 수 있으며, 연결된 노트북 인스턴스의 수명 주기와 독립적으로 유지됩니다.

리포지토리는 AWS CodeCommit 또는 다른 Git 리포지토리에서 호스팅할 수 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::SageMaker::CodeRepository",
  "Properties" : {
      "CodeRepositoryName" : String,
      "GitConfig" : GitConfig,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`CodeRepositoryName`  
Git 리포지토리의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}$`  
_업데이트 필요 사항_: 바꿔 놓음

`GitConfig`  
Git 리포지토리가 있는 URL과 리포지토리에 액세스하는 데 사용되는 자격 증명이 포함된 AWS Secrets Manager 보안 암호의 ARN을 포함하여 Git 리포지토리에 대한 구성 세부 정보입니다.  
_필수 항목 여부_: 예  
_형식_: GitConfig  
_업데이트 필요 사항_: 중단 없음

`Tags`  
코드 리포지토리에 대한 태그 목록입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`코드 리포지토리의 Amazon 리소스 이름(ARN)을 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.
