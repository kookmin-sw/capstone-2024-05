# AWS::RDS::D BInstance 엔드포인트

이 데이터 형식은 Amazon RDS DB 인스턴스에 연결하는 데 필요한 정보를 나타냅니다. 이 데이터 형식은 다음 작업에서 응답 요소로 사용됩니다.

- `CreateDBInstance`
- `DescribeDBInstances`
- `DeleteDBInstance`

Amazon Aurora DB 클러스터 엔드포인트를 나타내는 데이터 구조는 단원을 참조하십시오 `DBClusterEndpoint`.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Address" : String,
  "HostedZoneId" : String,
  "Port" : String
}
```

## 속성

`Address`  
DB 인스턴스의 DNS 주소를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`HostedZoneId`  
호스팅 영역을 생성할 때 Amazon Route 53이 할당하는 ID를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Port`  
데이터베이스 엔진이 수신 대기하는 포트를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
