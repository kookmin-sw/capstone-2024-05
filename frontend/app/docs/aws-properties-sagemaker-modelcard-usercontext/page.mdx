# AWS::SageMaker::모델 카드 사용자 컨텍스트

실험, 평가판, 평가판 구성 요소, 계보 그룹, 프로젝트 또는 모델 카드를 만들거나 수정한 사용자에 대한 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DomainId" : String,
  "UserProfileArn" : String,
  "UserProfileName" : String
}
```

## 속성

`DomainId`  
사용자와 연결된 도메인입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`UserProfileArn`  
사용자 프로필의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`UserProfileName`  
사용자 프로필의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
