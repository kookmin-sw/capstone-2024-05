# AWS::S3Object람다::액세스포인트 AwsLambda

`AwsLambda` 속성 유형은 속성 설명을 사용할 수 없습니다. AWS::S3ObjectLambda::AccessPoint의 경우.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "FunctionArn" : String,
  "FunctionPayload" : String
}
```

## 속성

`FunctionArn`  
속성 설명을 사용할 수 없습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`FunctionPayload`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
