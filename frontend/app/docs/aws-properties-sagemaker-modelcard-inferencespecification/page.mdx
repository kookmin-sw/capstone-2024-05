# AWS::SageMaker::ModelCard 추론 사양

학습 작업이 실행된 후 추론 생성을 수행하는 방법을 정의합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Containers" : [ Container, ... ]
}
```

## 속성

`Containers`  
추론 코드가 포함된 도커 이미지의 Amazon ECR 레지스트리 경로입니다.  
_필수 항목 여부_: 예  
_유형_: 컨테이너 목록  
_최대_: `15`  
_업데이트 필요 사항_: 중단 없음
