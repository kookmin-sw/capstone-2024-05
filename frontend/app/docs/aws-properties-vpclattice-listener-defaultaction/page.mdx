# AWS::VpcLattice::리스너 DefaultAction

기본 규칙에 대한 작업입니다. 각 수신기에는 기본 규칙이 있습니다. 일치하는 다른 규칙이 없는 경우 기본 규칙이 사용됩니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "FixedResponse" : FixedResponse,
  "Forward" : Forward
}
```

## 속성

`FixedResponse`  
사용자 지정 HTTP 응답을 반환하는 작업에 대해 설명합니다.  
*필수 항목 *여부: 아니요  
_유형_: FixedResponse  
_업데이트 필요 사항_: 중단 없음

`Forward`  
전달 작업에 대해 설명합니다. 전달 작업을 사용하여 하나 이상의 대상 그룹에 요청을 라우팅할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 앞으로  
_업데이트 필요 사항_: 중단 없음
