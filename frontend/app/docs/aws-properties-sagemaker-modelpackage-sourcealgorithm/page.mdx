# AWS::SageMaker::ModelPackage 소스 알고리즘

모델 패키지를 만드는 데 사용된 알고리즘을 지정합니다. 알고리즘은 SageMaker 계정의 알고리즘 리소스이거나 구독하는 AWS Marketplace의 알고리즘이어야 합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AlgorithmName" : String,
  "ModelDataUrl" : String
}
```

## 속성

`AlgorithmName`  
모델 패키지를 만드는 데 사용된 알고리즘의 이름입니다. 알고리즘은 SageMaker 계정의 알고리즘 리소스이거나 구독하는 AWS Marketplace의 알고리즘이어야 합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `170`  
_패턴_: `(arn:aws[a-z-]*:sagemaker:[a-z0-9-]*:[0-9]{12}:[a-z-]*/)?(a-zA-Z0-9{0,62})(?<!-)$`  
_업데이트 필요 사항_: 바꿔 놓음

`ModelDataUrl`  
모델 교육의 결과인 모델 아티팩트가 저장되는 Amazon S3 경로입니다. 이 경로는 압축된 단일 `gzip` tar 아카이브(`.tar.gz` 접미사)를 가리켜야 합니다.  
모델 아티팩트는 알고리즘과 동일한 AWS 리전에 있는 S3 버킷에 있어야 합니다.
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `1024`  
_패턴_: `^(https|s3)://([^/]+)/?(.*)$`  
_업데이트 필요 사항_: 바꿔 놓음
