# AWS::S3::버킷 복제 시간

S3 RTC 활성화 여부 및 모든 객체 및 객체에 대한 작업을 복제해야 하는 시간을 포함하여 S3 Replication Time Control(S3 RTC) 관련 정보를 지정하는 컨테이너입니다. 블록과 함께 지정해야 합니다 `Metrics` .

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Status" : String,
  "Time" : ReplicationTimeValue
}
```

## 속성

`Status`  
 복제 시간을 사용할 수 있는지 여부를 지정합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `Disabled | Enabled`  
_업데이트 필요 사항_: 중단 없음

`Time`  
 모든 개체 및 개체에 대한 작업에 대해 복제가 완료되어야 하는 시간을 지정하는 컨테이너입니다.  
_필수 항목 여부_: 예  
_유형_: ReplicationTimeValue  
_업데이트 필요 사항_: 중단 없음
