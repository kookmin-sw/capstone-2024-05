# AWS::SageMaker::ModelExplainabilityJobDefinition ModelExplainabilityApp사양

모델 설명 작업에 대한 Docker 컨테이너 이미지 구성 개체입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ConfigUri" : String,
  "Environment" : {Key: Value, ...},
  "ImageUri" : String
}
```

## 속성

`ConfigUri`  
설명 가능성 파라미터를 정의하는 JSON 형식의 S3 파일입니다. 이 JSON 구성 파일에 대한 자세한 내용은 모델 설명 가능성 파라미터 구성을 참조하세요.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `1024`  
_패턴_: `^(https|s3)://([^/]+)/?(.*)$`  
_업데이트 필요 사항_: 바꿔 놓음

`Environment`  
Docker 컨테이너에서 환경 변수를 설정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 맵  
_업데이트 필요 사항_: 바꿔 놓음

`ImageUri`  
모델 설명 가능성 작업에서 실행할 컨테이너 이미지입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `255`  
_패턴_: `.*`  
_업데이트 필요 사항_: 바꿔 놓음
