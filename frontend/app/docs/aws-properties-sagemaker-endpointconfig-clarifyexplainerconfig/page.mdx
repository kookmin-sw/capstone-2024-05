# AWS::SageMaker::EndpointConfig ClarifyExplainerConfig

`ClarifyExplainerConfig` 속성 유형은 속성 설명을 사용할 수 없음을 지정합니다. AWS::SageMaker::EndpointConfig의 경우.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "EnableExplanations" : String,
  "InferenceConfig" : ClarifyInferenceConfig,
  "ShapConfig" : ClarifyShapConfig
}
```

## 속성

`EnableExplanations`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`InferenceConfig`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: ClarifyInferenceConfig  
_업데이트 필요 사항_: 바꿔 놓음

`ShapConfig`  
속성 설명을 사용할 수 없습니다.  
_필수 항목 여부_: 예  
_형식_: ClarifyShapConfig  
_업데이트 필요 사항_: 바꿔 놓음
