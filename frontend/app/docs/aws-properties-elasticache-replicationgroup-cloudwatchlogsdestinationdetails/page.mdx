# AWS::ElastiCache::복제 그룹 CloudWatchLogsDestination세부 정보

CloudWatch Logs 대상의 구성 세부 정보입니다. 이 필드는 필수로 표시되지만 CloudWatch Logs가 대상으로 선택된 경우에만 표시됩니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "LogGroup" : String
}
```

## 속성

`LogGroup`  
CloudWatch Logs 로그 그룹의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
