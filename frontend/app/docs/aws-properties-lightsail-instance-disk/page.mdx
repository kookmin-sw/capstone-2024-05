# AWS::Lightsail::인스턴스 디스크

`Disk` 는 Hardware 속성의 속성입니다. 인스턴스에 연결된 디스크를 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AttachedTo" : String,
  "AttachmentState" : String,
  "DiskName" : String,
  "IOPS" : Integer,
  "IsSystemDisk" : Boolean,
  "Path" : String,
  "SizeInGb" : String
}
```

## 속성

`AttachedTo`  
디스크가 연결된 리소스입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 업데이트가 지원되지 않습니다.

`AttachmentState`  
(더 이상 사용되지 않음) 디스크의 연결 상태입니다.  
2017년 11월 14일 이전 릴리스에서는 이 매개 변수가 `attached` API 응답의 시스템 디스크에 대해 반환되었습니다. 이제 더 이상 사용되지 않지만 여전히 응답에 포함되어 있습니다. 대신 사용하십시오 `isAttached` .
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 업데이트가 지원되지 않습니다.

`DiskName`  
디스크의 고유 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`IOPS`  
디스크의 IOPS(초당 입출력 작업 수)입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 업데이트가 지원되지 않습니다.

`IsSystemDisk`  
이 디스크가 시스템 디스크(운영 체제가 로드되어 있음)인지 여부를 나타내는 부울 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 업데이트가 지원되지 않습니다.

`Path`  
디스크 경로입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 업데이트가 지원되지 않습니다.

`SizeInGb`  
디스크의 크기(GB)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 업데이트가 지원되지 않습니다.
