# AWS::SageMaker::InferenceExperiment RealTimeInferenceConfig

실시간 추론 엔드포인트에 모델을 배포하기 위한 인프라 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "InstanceCount" : Integer,
  "InstanceType" : String
}
```

## 속성

`InstanceCount`  
로 지정된 형식의 인스턴스 수 `InstanceType`입니다.  
_필수 항목 여부_: 예  
_유형_: Integer  
_최소_: `0`  
_업데이트 필요 사항_: 중단 없음

`InstanceType`  
모델이 배포되는 인스턴스 유형입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열
