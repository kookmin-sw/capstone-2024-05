# AWS::Lambda::EventSource매핑 SelfManagedEventSource

이벤트 원본에 대한 자체 관리형 Apache Kafka 클러스터입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Endpoints" : Endpoints
}
```

## 속성

`Endpoints`  
Kafka 브로커에 대한 부트스트랩 서버 목록(형식): `"KafkaBootstrapServers": ["abc.xyz.com:xxxx","abc2.xyz.com:xxxx"]`.  
*필수 항목 *여부: 아니요  
_유형_: 끝점  
_업데이트 필요 사항_: 바꿔 놓음
