# AWS::Route53RecoveryReadiness::ResourceSet NLBResource

DNS 대상 리소스가 가리키는 Network Load Balancer 리소스입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Arn" : String
}
```

## 속성

`Arn`  
Network Load Balancer 리소스 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
