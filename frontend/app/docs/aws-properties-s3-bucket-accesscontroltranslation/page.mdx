# AWS::S3::버킷 AccessControlTranslation

교차 계정 시나리오(원본 버킷과 대상 버킷 소유자가 동일하지 않은 경우)에서만 이를 지정하고 복제본 소유권을 대상 버킷을 소유한 AWS 계정으로 변경하려는 경우에만 지정합니다. 복제 구성에 지정되지 않은 경우 원본 객체를 소유한 동일한 AWS 계정에서 복제본을 소유합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Owner" : String
}
```

## 속성

`Owner`  
복제본 소유권을 지정합니다. 기본값 및 유효한 값은 _Amazon S3 API Reference의 PUT 버킷 복제를 참조하세요_.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `Destination`  
_업데이트 필요 사항_: 중단 없음
