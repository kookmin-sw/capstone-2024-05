# AWS CloudFormation StackSets 샘플 템플릿

이 단원에는 엔터프라이즈에서 AWS CloudFormation StackSets를 사용하는 데 도움이 될 수 있는 몇 가지 샘플 AWS CloudFormation 템플릿에 대한 링크가 포함되어 있습니다. 이 섹션에 나열된 템플릿은 AWS CloudTrail 또는 AWS Config 및 그 안의 규칙을 활성화합니다.

**샘플 템플릿**

| 묘사                                                               | S3 링크   |
| ------------------------------------------------------------------ | --------- |
| AWS CloudTrail 활성화                                              | 일부 주소 |
| AWS Config 활성화                                                  | 일부 주소 |
| 중앙 로깅으로 AWS Config 활성화                                    | 일부 주소 |
| CloudTrail이 활성화되었는지 여부를 결정하도록 AWS Config 규칙 구성 | 일부 주소 |
| 루트 MFA가 활성화되었는지 확인하도록 AWS Config 규칙 구성          | 일부 주소 |
| EIP가 연결되어 있는지 확인하도록 AWS Config 규칙 구성              | 일부 주소 |
| EBS 볼륨이 암호화되었는지 여부를 결정하도록 AWS Config 규칙 구성   | 일부 주소 |
