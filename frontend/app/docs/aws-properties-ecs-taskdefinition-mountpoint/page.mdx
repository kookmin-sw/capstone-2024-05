# AWS::ECS::태스크 정의 마운트 포인트

컨테이너 정의에 사용되는 볼륨 탑재 지점에 대한 세부 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ContainerPath" : String,
  "ReadOnly" : Boolean,
  "SourceVolume" : String
}
```

## 속성

`ContainerPath`  
호스트 볼륨을 탑재할 컨테이너의 경로입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`ReadOnly`  
이 값이 인 경우 `true`컨테이너는 볼륨에 대한 읽기 전용 액세스 권한을 갖습니다. 이 값이 `false`인 경우 컨테이너는 볼륨에 쓸 수 있습니다. 기본값은 `false`입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 바꿔 놓음

`SourceVolume`  
마운트할 볼륨의 이름입니다. `name` 작업 정의의 매개 변수 에서 참조되는 볼륨 이름이어야 합니다`volume`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
