# AWS::RDS::D B표준 프로세서기능

`ProcessorFeature` 속성 유형은 DB 인스턴스 클래스 상태의 프로세서 기능을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Value" : String
}
```

## 속성

`Name`  
프로세서 기능의 이름입니다. 유효한 이름은 `coreCount` 및 `threadsPerCore`입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Value`  
프로세서 기능 이름의 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
