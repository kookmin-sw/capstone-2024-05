# AWS::Lightsail::인스턴스 월별 전송

`MonthlyTransfer` 는 Networking 속성의 속성입니다. 인스턴스에 할당된 월별 데이터 전송량(GB)을 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "GbPerMonthAllocated" : String
}
```

## 속성

`GbPerMonthAllocated`  
인스턴스에 할당된 월별 데이터 전송량(GB)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
