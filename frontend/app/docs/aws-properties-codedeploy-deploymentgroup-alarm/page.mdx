# AWS::CodeDeploy::D eploymentGroup 경보

`Alarm` 속성 유형은 AWS CodeDeploy 배포 그룹에 사용할 CloudWatch 경보를 지정합니다. `Alarm` CodeDeploy DeploymentGroup AlarmConfiguration 속성의 속성에는 속성 유형 목록이 포함되어 `Alarm` 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String
}
```

## 속성

`Name`  
경보의 이름입니다. 최대 길이는 255자입니다. 각 경보 이름은 경보 목록에서 한 번만 사용할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
