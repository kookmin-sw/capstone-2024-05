# AWS::EC2::NetworkInsightsAnalysis AlternatePath힌트

실현 가능한 경로의 잠재적인 중간 구성요소에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ComponentArn" : String,
  "ComponentId" : String
}
```

## 속성

`ComponentArn`  
구성 요소의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ComponentId`  
구성 요소의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
