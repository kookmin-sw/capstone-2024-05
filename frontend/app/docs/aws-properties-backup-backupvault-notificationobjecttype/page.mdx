# AWS::백업::BackupVault NotificationObjectType

대상 백업 볼트에 대한 SNS 이벤트 알림 속성을 포함하는 객체를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BackupVaultEvents" : [ String, ... ],
  "SNSTopicArn" : String
}
```

## 속성

`BackupVaultEvents`  
백업 자격 증명 모음에 리소스를 백업하는 작업의 상태를 나타내는 이벤트 배열입니다. 유효한 이벤트는 _AWS Backup API 안내서의 BackupVaultEvents를 참조하세요_.  
_필수 항목 여부_: 예  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`SNSTopicArn`  
Amazon Simple Notification Service(Amazon SNS) 주제를 고유하게 식별하는 ARN 예를 들어, `arn:aws:sns:us-west-2:111122223333:MyTopic`.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
