# AWS::EFS::액세스 포인트 액세스 포인트 태그

태그는 파일 시스템에 연결된 키-값 페어입니다. 및 속성에 허용되는 문자 `Key` `Value` 는 UTF-8로 표현할 수 있는 문자, 공백 및 숫자와 다음 문자입니다.` + - = . _ : /`

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Key" : String,
  "Value" : String
}
```

## 속성

`Key`  
태그 키(String)입니다. 키는 로 시작할 수 없습니다`aws:`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `128`  
_패턴_: `^(?![aA]{1}[wW]{1}[sS]{1}:)([p{L}p{Z}p{N}_.:/=+-@]+)$`  
_업데이트 필요 사항_: 중단 없음

`Value`  
태그 키의 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `256`  
_패턴_: `^([p{L}p{Z}p{N}_.:/=+-@]*)$`  
_업데이트 필요 사항_: 중단 없음
