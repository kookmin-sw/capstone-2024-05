# AWS::Route53::HealthCheck HealthCheck태그

이 `HealthCheckTag` 속성은 리소스와 연결된 하나의 키-값 쌍을 설명합니다 `AWS::Route53::HealthCheck` .

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Key" : String,
  "Value" : String
}
```

## 속성

`Key`  
의 값은 `Key` 수행하려는 작업에 따라 달라집니다.

- **상태 확인 또는 호스팅 영역에 태그 추가**: `Key` 은 새 태그에 지정할 이름입니다.
- **Edit a tag**: `Key` 은 변경하려는 태그의 이름입니다 `Value` .
- ** 키 삭제**: `Key` 은 제거하려는 태그의 이름입니다.
- **상태 확인에 이름 지정**: 기본 태그를 편집합니다 `Name` . Amazon Route 53 콘솔의 상태 확인 목록 \**\*\* 에는 각 상태 확인에 지정한 이름을 볼 수 있는 [Name] 열이 포함되어 있습니다.
  *필수 항목 여부*: 예  
  *유형*: 문자열  
  *최대*: `128`  
  *업데이트 필요 사항\*: 중단 없음

`Value`  
의 값은 `Value` 수행하려는 작업에 따라 달라집니다.

- **상태 확인 또는 호스팅 영역에 태그 추가**: `Value` 은 새 태그에 부여할 값입니다.
- **태그 편집**: `Value` 은 태그를 할당하려는 새 값입니다.
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _최대_: `256`  
  _업데이트 필요 사항_: 중단 없음

## 참고 항목
