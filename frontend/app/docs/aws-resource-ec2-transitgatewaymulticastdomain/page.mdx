# AWS::EC2::TransitGatewayMulticastDomain

지정된 transit gateway를 사용하여 멀티캐스트 도메인을 생성합니다.

도메인을 생성하기 전에 transit gateway가 available 상태여야 합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::TransitGatewayMulticastDomain",
  "Properties" : {
      "Options" : Options,
      "Tags" : [Tag, ... ],
      "TransitGatewayId" : String
    }
}
```

## 속성

`Options` transit gateway 멀티캐스트 도메인에 대한 옵션입니다.

- AutoAcceptSharedAssociations(사용 | 사용 안 함)
- Igmpv2Support(활성화 | 비활성화)
- StaticSourcesSupport(활성화 | 비활성화)
  *필수 항목 *여부: 아니요  
  _유형_: 옵션  
  _업데이트 필요 사항_: 중단 없음

`Tags`  
transit gateway 멀티캐스트 도메인에 대한 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

`TransitGatewayId`  
전송 게이트웨이의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`transit gateway 멀티캐스트 도메인 ID를 반환합니다. 예를 들면 다음과 같습니다 `tgw-mcast-domain-000fb24d04EXAMPLE`.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`CreationTime` 멀티캐스트 도메인이 생성된 시간입니다.

`State` 멀티캐스트 도메인의 상태입니다.

`TransitGatewayMulticastDomainArn` 멀티캐스트 도메인의 Amazon 리소스 이름(ARN)입니다.

`TransitGatewayMulticastDomainId` 멀티캐스트 도메인의 ID입니다.
