# AWS::SageMaker::엔드포인트 구성 AsyncInferenceConfig

끝점이 비동기 유추를 수행하는 방법에 대한 구성을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ClientConfig" : AsyncInferenceClientConfig,
  "OutputConfig" : AsyncInferenceOutputConfig
}
```

## 속성

`ClientConfig`  
비동기식 추론 중에 SageMaker가 모델 컨테이너와 상호 작용하는 데 사용하는 클라이언트의 동작을 구성합니다.  
*필수 항목 *여부: 아니요  
_형식_: AsyncInferenceClientConfig  
_업데이트 필요 사항_: 바꿔 놓음

`OutputConfig`  
비동기 유추 호출 출력에 대한 구성을 지정합니다.  
_필수 항목 여부_: 예  
_형식_: AsyncInferenceOutputConfig  
_업데이트 필요 사항_: 바꿔 놓음
