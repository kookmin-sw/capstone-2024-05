# AWS::KinesisAnalyticsV2::애플리케이션 GlueDataCatalog구성

애플리케이션에서 작성하는 Apache Flink SQL 쿼리 및 테이블 API 변환에 사용하는 Glue 데이터 카탈로그의 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DatabaseARN" : String
}
```

## 속성

`DatabaseARN`  
데이터베이스의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `2048`  
_패턴_: `arn:.*`  
_업데이트 필요 사항_: 중단 없음
