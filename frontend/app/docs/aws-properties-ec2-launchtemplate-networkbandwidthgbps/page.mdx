# AWS::EC2::Launch템플릿 네트워크 대역폭Gbps

네트워크 대역폭의 최소 및 최대 양(Gbps)입니다.

**메모**  
최소 대역폭을 설정한다고 해서 인스턴스가 최소 대역폭에 도달한다는 보장은 없습니다. Amazon EC2는 지정된 최소 대역폭을 지원하는 인스턴스 유형을 식별하지만, 인스턴스의 실제 대역폭이 지정된 최소 대역폭보다 낮아질 수 있습니다. 자세한 내용은 _Amazon EC2 사용 설명서의 사용 가능한 인스턴스 대역폭을 참조하세요_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Max" : Double,
  "Min" : Double
}
```

## 속성

`Max`  
네트워크 대역폭의 최대 양(Gbps)입니다. 최대 제한을 지정하지 않으려면 이 매개 변수를 생략합니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음

`Min`  
네트워크 대역폭의 최소 양(Gbps)입니다. 이 매개 변수를 지정하지 않으면 최소 제한이 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음
