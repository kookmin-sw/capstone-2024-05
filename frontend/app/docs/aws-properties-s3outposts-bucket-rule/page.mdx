# AWS::S3Outposts::버킷 규칙

Amazon S3 on Outposts 버킷 수명 주기 규칙의 컨테이너입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AbortIncompleteMultipartUpload" : AbortIncompleteMultipartUpload,
  "ExpirationDate" : String,
  "ExpirationInDays" : Integer,
  "Filter" : Filter,
  "Id" : String,
  "Status" : String
}
```

## 속성

`AbortIncompleteMultipartUpload`  
불완전한 멀티파트 업로드 중단 규칙의 컨테이너입니다.  
*필수 항목 *여부: 아니요  
_형식_: AbortIncompleteMultipartUpload  
_업데이트 필요 사항_: 중단 없음

`ExpirationDate`  
만료 날짜를 지정하여 객체의 주기에 대한 만료를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ExpirationInDays`  
객체의 수명 주기에 대한 만료를 객체가 S3 on Outposts 버킷에 있었던 일 수 형식으로 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Filter`  
수명 주기 규칙의 필터에 대한 컨테이너입니다.  
*필수 항목 *여부: 아니요  
_유형_: 필터  
_업데이트 필요 사항_: 중단 없음

`Id`  
 수명 주기 규칙의 고유 식별자입니다. 값은 255자를 초과할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Status`  
인 경우 `Enabled`규칙이 현재 적용되고 있습니다. 인 경우 `Disabled`규칙이 현재 적용되고 있지 않습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
