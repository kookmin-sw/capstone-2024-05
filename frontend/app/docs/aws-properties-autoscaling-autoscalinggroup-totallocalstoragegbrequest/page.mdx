# AWS::AutoScaling::AutoScalingGroup TotalLocalStorageGBRequest

`TotalLocalStorageGBRequest` 은 `InstanceRequirements` 인스턴스 유형의 최소 및 최대 총 로컬 스토리지 크기(GB)를 설명하는 AWS::AutoScaling::AutoScalingGroup LaunchTemplateOverrides 속성 유형의 속성 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Max" : Integer,
  "Min" : Integer
}
```

## 속성

`Max`  
스토리지 최대값(GB)입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Min`  
최소 스토리지 단위(GB)입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
