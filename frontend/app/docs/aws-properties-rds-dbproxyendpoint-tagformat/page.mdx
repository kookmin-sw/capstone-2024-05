# AWS::RDS::D BProxyEndpoint 태그 형식

키-값 페어로 구성된 DB 프록시 엔드포인트에 할당된 메타데이터입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Key" : String,
  "Value" : String
}
```

## 속성

`Key`  
value는 태그의 선택적 값입니다. 문자열 값의 길이는 1-256자의 유니코드 문자일 수 있으며 접두사를 사용할 수 없습니다`aws:`. 문자열에는 유니코드 문자, 숫자, 공백, '_', '.', '/', '=', '+', '-'(Java 정규식: "^(\*)$")의 집합만 포함될 수 있습니다[p{L}p{Z}p{N}_.:/=+-].  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Value`  
키-값 페어로 구성된 DB 인스턴스에 할당된 메타데이터입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
