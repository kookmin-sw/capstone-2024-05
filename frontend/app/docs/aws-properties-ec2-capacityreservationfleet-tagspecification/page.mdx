# AWS::EC2::CapacityReservationFleet 태그 사양

리소스를 만들 때 리소스에 적용할 태그입니다. 태그를 지정할 때 태그를 지정할 리소스 유형을 지정해야 하며, 그렇지 않으면 요청이 실패합니다.

**메모**  
 `Valid Values` 태그를 지정할 수 있는 모든 리소스 종류를 나열합니다. 그러나 사용 중인 작업이 이러한 모든 리소스 유형에 태그를 지정하는 것을 지원하지 않을 수 있습니다. 사용 중인 작업에 대해 지원되지 않는 리소스 종류에 태그를 지정하려고 하면 오류가 발생합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ResourceType" : String,
  "Tags" : [Tag, ... ]
}
```

## 속성

`ResourceType`  
생성 시 태그를 지정할 리소스의 유형입니다. 를 지정합니다`capacity-reservation-fleet`.  
리소스를 만든 후 태그를 지정하려면 CreateTags 단원을 참조하십시오.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
리소스에 적용할 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 바꿔 놓음
