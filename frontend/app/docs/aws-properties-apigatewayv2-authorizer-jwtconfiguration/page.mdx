# AWS::ApiGatewayV2::권한 부여자 JWTConfiguration

이 `JWTConfiguration` 특성은 JWT 권한 부여자의 구성을 지정합니다. `JWT` 권한 부여자 유형에 필요합니다. HTTP API에 대해서만 지원됩니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Audience" : [ String, ... ],
  "Issuer" : String
}
```

## 속성

`Audience`  
JWT의 의도된 수신자 목록입니다. 유효한 JWT는 `aud` 이 목록에 있는 하나 이상의 항목과 일치하는 항목을 제공해야 합니다. RFC 7519를 참조하십시오. `JWT` 권한 부여자 유형에 필요합니다. HTTP API에 대해서만 지원됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Issuer`  
JSON 웹 토큰을 발급하는 ID 공급자의 기본 도메인입니다. 예를 들어, Amazon Cognito 사용자 풀의 형식은 다음과 같습니다`some-address `. `JWT` 권한 부여자 유형에 필요합니다. HTTP API에 대해서만 지원됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
