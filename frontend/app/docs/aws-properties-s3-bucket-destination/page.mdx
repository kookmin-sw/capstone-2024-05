# AWS::S3::버킷 대상

Amazon S3 버킷에 대한 분석 또는 구성 결과를 게시할 위치에 대한 정보를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BucketAccountId" : String,
  "BucketArn" : String,
  "Format" : String,
  "Prefix" : String
}
```

## 속성

`BucketAccountId`  
대상 S3 버킷을 소유하는 계정 ID입니다. 계정 ID를 제공하지 않으면 데이터를 내보내기 전에 소유자의 유효성이 확인되지 않습니다.  
 이 값은 선택 사항이지만 대상 버킷 소유권이 변경되는 경우 문제를 방지하기 위해 설정하는 것이 좋습니다.
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`BucketArn`  
데이터를 내보낼 버킷의 Amazon 리소스 이름(ARN)입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Format`  
데이터를 Amazon S3로 내보낼 때 사용되는 파일 형식을 지정합니다.  
_허용되는 값_: `CSV` | `ORC` | `Parquet`  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Prefix`  
데이터를 내보낼 때 사용할 접두사입니다. 접두사는 모든 결과 앞에 추가됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
