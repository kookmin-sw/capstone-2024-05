# AWS::SageMaker::ModelExplainabilityJobDefinition S3출력

모니터링 작업의 결과가 저장되는 Amazon S3 스토리지 위치입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "LocalPath" : String,
  "S3UploadMode" : String,
  "S3Uri" : String
}
```

## 속성

`LocalPath`  
Amazon SageMaker가 모니터링 작업의 결과를 저장하는 Amazon S3 스토리지 위치의 로컬 경로입니다. LocalPath는 출력 데이터의 절대 경로입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`S3UploadMode`  
모니터링 작업의 결과를 지속적으로 업로드할지 또는 작업이 완료된 후에 업로드할지 여부입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`S3Uri`  
Amazon SageMaker가 모니터링 작업의 결과를 저장하는 Amazon S3 스토리지 위치를 식별하는 URI입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
