# AWS::EC2::LaunchTemplate IamInstanceProfile

인스턴스의 IAM 역할에 대한 컨테이너인 IAM 인스턴스 프로파일을 지정합니다. IAM 역할을 사용하여 AWS 자격 증명을 인스턴스에 배포할 수 있습니다.

Amazon EC2 Auto Scaling 그룹에서 사용할 시작 템플릿을 생성하는 경우 인스턴스 프로파일의 이름 또는 ARN을 지정할 수 있지만 둘 다 지정할 수는 없습니다.

`IamInstanceProfile` 은 AWS::EC2::LaunchTemplate LaunchTemplateData의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Arn" : String,
  "Name" : String
}
```

## 속성

`Arn`  
인스턴스 프로파일의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Name`  
인스턴스 프로파일의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
