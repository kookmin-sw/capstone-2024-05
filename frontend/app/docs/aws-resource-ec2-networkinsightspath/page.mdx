# AWS::EC2::NetworkInsights경로

도달 가능성을 분석할 경로를 지정합니다.

VPC Reachability Analyzer를 사용하면 Virtual Private Cloud(VPC)의 두 리소스 간 네트워크 연결성을 분석하고 디버깅할 수 있습니다. 자세한 내용은 Reachability Analyzer 사용 설명서를 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::NetworkInsightsPath",
  "Properties" : {
      "Destination" : String,
      "DestinationIp" : String,
      "DestinationPort" : Integer,
      "FilterAtDestination" : PathFilter,
      "FilterAtSource" : PathFilter,
      "Protocol" : String,
      "Source" : String,
      "SourceIp" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`Destination`  
대상의 ID 또는 ARN입니다. 리소스가 다른 계정에 있는 경우 ARN을 지정해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`DestinationIp`  
대상의 IP 주소입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `15`  
_패턴_: `^([0-9]{1,3}.){3}[0-9]{1,3}$`  
_업데이트 필요 사항_: 바꿔 놓음

`DestinationPort`  
대상 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `0`  
_최대_: `65535`  
_업데이트 필요 사항_: 바꿔 놓음

`FilterAtDestination`  
대상의 특정 필터와 일치하는 네트워크 경로로 분석 범위를 지정합니다. 이 매개 변수를 지정하면 대상 IP 주소에 대한 매개 변수를 지정할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: PathFilter  
_업데이트 필요 사항_: 바꿔 놓음

`FilterAtSource`  
소스의 특정 필터와 일치하는 네트워크 경로로 분석 범위를 지정합니다. 이 매개 변수를 지정하면 원본 IP 주소 또는 대상 포트에 대한 매개 변수를 지정할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: PathFilter  
_업데이트 필요 사항_: 바꿔 놓음

`Protocol`  
프로토콜입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `tcp | udp`  
_업데이트 필요 사항_: 바꿔 놓음

`Source`  
소스의 ID 또는 ARN입니다. 리소스가 다른 계정에 있는 경우 ARN을 지정해야 합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`SourceIp`  
소스의 IP 주소입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `15`  
_패턴_: `^([0-9]{1,3}.){3}[0-9]{1,3}$`  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
경로에 추가할 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`경로의 ID를 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`CreatedDate` 경로가 만들어진 타임스탬프입니다.

`DestinationArn` 대상의 Amazon 리소스 이름(ARN)입니다.

`NetworkInsightsPathArn` 경로의 Amazon 리소스 이름(ARN)입니다.

`NetworkInsightsPathId` 경로의 ID입니다.

`SourceArn` 소스의 Amazon 리소스 이름(ARN)입니다.
