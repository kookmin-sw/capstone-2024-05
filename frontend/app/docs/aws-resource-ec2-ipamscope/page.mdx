# AWS::EC2::IPAM코프

IPAM에서 범위는 IPAM 내에서 가장 높은 수준의 컨테이너입니다. IPAM에는 두 개의 기본 범위가 포함되어 있습니다. 각 범위는 단일 네트워크의 IP 공간을 나타냅니다. 개인 범위는 모든 개인 IP 주소 공간을 위한 것입니다. 공용 범위는 모든 공용 IP 주소 공간을 위한 것입니다. 범위를 사용하면 IP 주소 겹침이나 충돌 없이 연결되지 않은 여러 네트워크에서 IP 주소를 다시 사용할 수 있습니다.

자세한 내용은 Amazon VPC IPAM 사용 설명서*의 IPAM 작동 방식을 참조하세요*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::IPAMScope",
  "Properties" : {
      "Description" : String,
      "IpamId" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`Description`  
범위에 대한 설명입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`IpamId`  
이 범위를 만드는 IPAM의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
리소스에 할당된 태그의 키/값 조합입니다. 필터 이름에 태그 키를 사용하고 태그 값을 필터 값으로 사용합니다. 예를 들어, 키와 값이 있는 태그가 있는 모든 리소스를 찾으려면 `Owner` `TeamA` `tag:Owner` 필터 이름과 `TeamA` 필터 값을 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`IPAM `Ref`범위 ID가 반환됩니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Arn` 범위의 ARN입니다.

`IpamArn` IPAM의 ARN입니다.

`IpamScopeId` IPAM 범위의 ID입니다.

`IpamScopeType` 범위의 형식입니다.

`IsDefault` 범위가 기본 범위인지 여부를 정의합니다.

`PoolCount` 범위의 풀 수입니다.
