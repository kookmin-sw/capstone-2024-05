# AWS::SageMaker::ModelExplainabilityJobDefinition ModelExplainabilityJobInput

모델 설명 작업에 대한 입력입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BatchTransformInput" : BatchTransformInput,
  "EndpointInput" : EndpointInput
}
```

## 속성

`BatchTransformInput`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: BatchTransformInput  
_업데이트 필요 사항_: 바꿔 놓음

`EndpointInput`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: EndpointInput  
_업데이트 필요 사항_: 바꿔 놓음
