# AWS CloudFormation StackSets 시작하기

첫 번째 스택 세트를 생성하기 전에 스택 세트 작업을 위한 사전 조건의 필수 계정 설정 단계를 완료했는지 확인합니다.

이 연습의 템플릿은 미국 서부(오레곤) 리전(us-west-2) 및 미국 동부(버지니아 북부) 리전(us-east-1) 내의 대상 계정에서 AWS Config를 활성화합니다. **AWS Config 활성화** 템플릿은 다음 S3 버킷에 있습니다. StackSets 콘솔에서 이 샘플 템플릿을 선택할 수도 있습니다.

**항목**

- 스택 세트 생성
- 스택 세트 업데이트
- 스택 세트에 스택 추가
- 스택 인스턴스의 파라미터 재정의Override parameters on stack instances
- 스택 세트에서 스택 인스턴스 삭제
- 스택 세트 삭제
- 서비스 관리형 권한이 있는 스택 세트에 대한 자동 배포 관리
