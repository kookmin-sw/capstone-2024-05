# AWS::EC2::NetworkInsightsPath FilterPortRange

포트 범위를 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "FromPort" : Integer,
  "ToPort" : Integer
}
```

## 속성

`FromPort`  
범위의 첫 번째 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `0`  
_최대_: `65535`  
_업데이트 필요 사항_: 바꿔 놓음

`ToPort`  
범위의 마지막 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `0`  
_최대_: `65535`  
_업데이트 필요 사항_: 바꿔 놓음
