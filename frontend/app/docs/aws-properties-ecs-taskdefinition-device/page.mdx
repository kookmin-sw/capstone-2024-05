# AWS::ECS::TaskDefinition 디바이스

이 `Device` 속성은 컨테이너 인스턴스 호스트 장치를 나타내는 개체를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ContainerPath" : String,
  "HostPath" : String,
  "Permissions" : [ String, ... ]
}
```

## 속성

`ContainerPath`  
호스트 장치를 노출할 컨테이너 내부의 경로입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`HostPath`  
호스트 컨테이너 인스턴스에 있는 장치의 경로입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Permissions`  
장치의 컨테이너에 제공할 명시적 권한입니다. 기본적으로 컨테이너에는 `read` `write` , , 및 `mknod` 장치에 대한 권한이 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 바꿔 놓음
