# AWS::EC2::VerifiedAccessTrustProvider 디바이스 옵션

AWS Verified Access 디바이스 자격 증명 기반 신뢰 공급자에 대한 옵션을 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "TenantId" : String
}
```

## 속성

`TenantId`  
디바이스 ID 공급자가 있는 테넌트 애플리케이션의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
