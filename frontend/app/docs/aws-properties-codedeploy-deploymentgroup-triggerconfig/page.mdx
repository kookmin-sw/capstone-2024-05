# AWS::CodeDeploy::D eploymentGroup 트리거 구성

배포 그룹에 대한 알림 트리거에 대한 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "TriggerEvents" : [ String, ... ],
  "TriggerName" : String,
  "TriggerTargetArn" : String
}
```

## 속성

`TriggerEvents`  
 알림을 트리거하는 이벤트 유형입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`TriggerName`  
알림 트리거의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`TriggerTargetArn`  
배포 또는 인스턴스 이벤트에 대한 알림이 전송되는 Amazon Simple Notification Service 주제의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
