# AWS::EC2::TransitGatewayPeeringAttachment 피어링어태치먼트 상태

Transit Gateway 피어링 연결의 상태입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Code" : String,
  "Message" : String
}
```

## 속성

`Code`  
상태 코드입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Message`  
상태 메시지(해당하는 경우)  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
