# AWS::VpcLattice::규칙 헤더 매치

헤더 일치에 대한 제약 조건에 대해 설명합니다. 규칙 작업을 적용하기 전에 들어오는 요청을 요청 헤더 값을 기반으로 하는 규칙과 일치시킵니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CaseSensitive" : Boolean,
  "Match" : HeaderMatchType,
  "Name" : String
}
```

## 속성

`CaseSensitive`  
일치 항목이 대/소문자를 구분하는지 여부를 나타냅니다. 기본값은 false입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`Match`  
헤더 일치 유형입니다.  
_필수 항목 여부_: 예  
_형식_: HeaderMatchType  
_업데이트 필요 사항_: 중단 없음

`Name`  
헤더의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
