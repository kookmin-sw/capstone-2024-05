# AWS::SageMaker::ModelPackage 메타데이터 속성

추적 엔터티, 평가판 또는 평가판 구성 요소의 메타데이터 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CommitId" : String,
  "GeneratedBy" : String,
  "ProjectId" : String,
  "Repository" : String
}
```

## 속성

`CommitId`  
커밋 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `1024`  
_패턴_: `.*`  
_업데이트 필요 사항_: 바꿔 놓음

`GeneratedBy`  
이 엔터티가 생성된 엔터티입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `1024`  
_패턴_: `.*`  
_업데이트 필요 사항_: 바꿔 놓음

`ProjectId`  
프로젝트 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `1024`  
_패턴_: `.*`  
_업데이트 필요 사항_: 바꿔 놓음

`Repository`  
저장소입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `1024`  
_패턴_: `.*`  
_업데이트 필요 사항_: 바꿔 놓음
