# AWS::ECR::ReplicationConfiguration ReplicationConfiguration

레지스트리에 대한 복제 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Rules" : [ ReplicationRule, ... ]
}
```

## 속성

`Rules`  
복제 구성에 대한 복제 대상과 저장소 필터를 나타내는 개체의 배열입니다.  
_필수 항목 여부_: 예  
_타입_: ReplicationRule 목록  
_최대_: `10`  
_업데이트 필요 사항_: 중단 없음
