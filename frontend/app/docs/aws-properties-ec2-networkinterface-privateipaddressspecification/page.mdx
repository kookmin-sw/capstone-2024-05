# AWS::EC2::NetworkInterface PrivateIpAddress사양

네트워크 인터페이스의 보조 개인 IPv4 주소를 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Primary" : Boolean,
  "PrivateIpAddress" : String
}
```

## 속성

`Primary`  
개인 IP 주소를 기본 개인 주소로 설정합니다. 기본 개인 IP 주소는 하나만 설정할 수 있습니다. 기본 프라이빗 IP 주소를 지정하지 않으면 Amazon EC2가 자동으로 기본 프라이빗 IP 주소를 할당합니다.  
_필수 항목 여부_: 예  
_유형_: Boolean  
_업데이트 필요 사항_: 일부 중단

`PrivateIpAddress`  
네트워크 인터페이스의 개인 IP 주소입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 일부 중단
