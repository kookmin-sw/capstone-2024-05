# AWS::Route53해석자::ResolverConfig

VPC의 해석기 구성에 대한 정보를 포함하는 복합 유형입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Route53Resolver::ResolverConfig",
  "Properties" : {
      "AutodefinedReverseFlag" : String,
      "ResourceId" : String
    }
}
```

## 속성

`AutodefinedReverseFlag`  
의 원하는 상태를 나타냅니다 `AutodefinedReverse`. 생성 시 유일하게 지원되는 값은 `DISABLE`입니다. 이 리소스를 삭제하면 `AutodefinedReverse` 기본값인 `ENABLED`로 돌아갑니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`ResourceId`  
해석기를 구성할 Amazon Virtual Private Cloud VPC의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `64`  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`ID `Ref`를 반환합니다 `ResolverConfiguration` .

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`AutodefinedReverse` Route 53 Resolver가 역방향 DNS 조회를 위한 자동 정의 규칙을 생성할지 여부의 상태입니다. 이 기능은 기본적으로 활성화되어 있습니다.

`Id` Route 53 Resolver 구성의 ID입니다.

`OwnerId` Amazon Virtual Private Cloud VPC의 소유자 계정 ID입니다.
