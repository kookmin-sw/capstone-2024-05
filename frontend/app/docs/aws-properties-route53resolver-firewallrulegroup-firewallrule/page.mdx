# AWS::Route53Resolver::FirewallRuleGroup 방화벽 규칙

규칙 그룹의 단일 방화벽 규칙입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Action" : String,
  "BlockOverrideDnsType" : String,
  "BlockOverrideDomain" : String,
  "BlockOverrideTtl" : Integer,
  "BlockResponse" : String,
  "FirewallDomainListId" : String,
  "Priority" : Integer
}
```

## 속성

`Action` DNS 방화벽이 규칙의 도메인 목록에 있는 도메인 중 하나와 일치할 때 DNS 쿼리에 대해 수행해야 하는 작업은 다음과 같습니다.

- `ALLOW` - 요청이 통과되도록 허용합니다.
- `ALERT` - 요청이 통과하도록 허용하되 로그에 경고를 보냅니다.
- `BLOCK` - 요청을 허용하지 않습니다. 이 값을 지정하면 다음 `BlockResponse` 항목도 지정해야 합니다.

인 `BlockResponse` 경우 `OVERRIDE`다음 `OVERRIDE` 속성을 모두 지정해야 합니다.

- `BlockOverrideDnsType`
- `BlockOverrideDomain`
- `BlockOverrideTtl`
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _허용되는 값_: `ALERT | ALLOW | BLOCK`  
  _업데이트 필요 사항_: 중단 없음

`BlockOverrideDnsType`  
DNS 레코드의 유형입니다. 이 값은 에서 제공한 레코드 값의 형식을 결정합니다`BlockOverrideDomain`. `BLOCK` `BlockResponse` 설정이 있는 규칙 작업에 사용됩니다`OVERRIDE`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `CNAME`  
_업데이트 필요 사항_: 중단 없음

`BlockOverrideDomain`  
쿼리에 대한 응답으로 다시 보낼 사용자 지정 DNS 레코드입니다. `BLOCK` `BlockResponse` 설정이 있는 규칙 작업에 사용됩니다`OVERRIDE`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_업데이트 필요 사항_: 중단 없음

`BlockOverrideTtl`  
DNS 해석기 또는 웹 브라우저가 제공된 재정의 레코드를 캐시하는 데 권장되는 시간(초)입니다. `BLOCK` `BlockResponse` 설정이 있는 규칙 작업에 사용됩니다`OVERRIDE`.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`BlockResponse`  
DNS 방화벽이 요청을 차단하도록 하는 방법입니다. 규칙 작업 설정에 사용됩니다 `BLOCK`.

- `NODATA` - 쿼리가 성공했음을 나타내는 응답이지만 사용할 수 있는 응답이 없습니다.
- `NXDOMAIN` - 쿼리에 있는 도메인 이름이 존재하지 않음을 나타내는 응답입니다.
- `OVERRIDE` - 응답에 사용자 지정 재정의를 제공합니다. 이 옵션을 사용하려면 규칙 설정에서 사용자 지정 처리 세부 정보가 필요합니다 `BlockOverride*` .
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _허용되는 값_: `NODATA | NXDOMAIN | OVERRIDE`  
  _업데이트 필요 사항_: 중단 없음

`FirewallDomainListId`  
규칙에 사용되는 도메인 목록의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `64`  
_업데이트 필요 사항_: 중단 없음

`Priority`  
규칙 그룹에 있는 규칙의 우선 순위입니다. 이 값은 규칙 그룹 내에서 고유해야 합니다. DNS 방화벽은 가장 낮은 설정부터 시작하여 우선 순위에 따라 규칙 그룹의 규칙을 처리합니다.  
_필수 항목 여부_: 예  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
