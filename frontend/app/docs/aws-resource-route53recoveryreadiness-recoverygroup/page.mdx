# AWS::Route53복구 준비::복구 그룹

Amazon Route 53 Application Recovery Controller에서 복구 그룹을 생성합니다. 복구 그룹은 응용 프로그램을 나타냅니다. 일반적으로 리소스 및 기능 측면에서 서로의 복제본인 두 개 이상의 셀로 구성되므로 한 리전에서 다른 리전으로(예: 한 리전에서 다른 리전으로) 장애 조치할 수 있습니다. 복구 그룹을 만들면 준비 검사를 사용하여 응용 프로그램의 리소스를 감사할 수 있습니다.

자세한 내용은 Amazon Route 53 Application Recovery Controller 개발자 안내서의 준비 검사, 리소스 세트 및 준비 범위를 참조하세요.

Route 53 ARC Readiness는 us-east-1 및 us-west-2 AWS 리전만 지원합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Route53RecoveryReadiness::RecoveryGroup",
  "Properties" : {
      "Cells" : [ String, ... ],
      "RecoveryGroupName" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`Cells`  
복구 그룹의 셀 Amazon 리소스 이름(ARN) 목록입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`RecoveryGroupName`  
만들 복구 그룹의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
리소스와 연결된 태그의 컬렉션입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref` `RecoveryGroupName`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`RecoveryGroupArn` 복구 그룹의 Amazon 리소스 이름(ARN)입니다.
