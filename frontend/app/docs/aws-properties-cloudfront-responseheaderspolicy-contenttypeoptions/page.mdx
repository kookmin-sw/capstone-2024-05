# AWS::CloudFront::ResponseHeadersPolicy ContentTypeOptions

CloudFront에 값이 로 설정된 HTTP 응답 헤더가 `X-Content-Type-Options`포함되어 있는지 여부를 결정합니다`nosniff`.

HTTP 응답 헤더에 대한 자세한 `X-Content-Type-Options` 내용은 MDN Web Docs의 X-Content-Type-Options를 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Override" : Boolean
}
```

## 속성

`Override`  
CloudFront가 `X-Content-Type-Options` 오리진에서 수신한 HTTP 응답 헤더를 이 응답 헤더 정책에 지정된 헤더로 재정의할지 여부를 결정하는 부울입니다.  
_필수 항목 여부_: 예  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
