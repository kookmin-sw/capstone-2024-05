# AWS::S3::StorageLens CloudWatch메트릭

이 리소스는 Amazon S3 Storage Lens 지표에 대한 Amazon CloudWatch 게시 옵션을 활성화합니다.

자세한 내용은 Amazon S3 사용 설명서*의 CloudWatch에서 S3 Storage Lens 지표 모니터링을 참조하세요*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "IsEnabled" : Boolean
}
```

## 속성

`IsEnabled`  
이 속성은 S3 Storage Lens에 대한 CloudWatch 게시 옵션이 활성화되어 있는지 여부를 식별합니다.  
_필수 항목 여부_: 예  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
