# AWS::CodeDeploy::D eploymentGroup EC2태그 필터

Amazon EC2 태그 필터에 대한 정보입니다.

태그 및 태그 그룹을 사용하여 Amazon EC2 인스턴스 및 온프레미스 인스턴스를 관리하는 방법에 대한 자세한 내용은 AWS CodeDeploy 사용 설명서의 AWS CodeDeploy에서 배포 그룹에 대한 인스턴스 태깅을 참조하십시오\*\*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Key" : String,
  "Type" : String,
  "Value" : String
}
```

## 속성

`Key`  
태그 필터 키입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Type` 태그 필터 유형:

- `KEY_ONLY`: 키만.
- `VALUE_ONLY`: 값만.
- `KEY_AND_VALUE`: 키 및 값입니다.
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _허용되는 값_: `KEY_AND_VALUE | KEY_ONLY | VALUE_ONLY`  
  _업데이트 필요 사항_: 중단 없음

`Value`  
태그 필터 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
