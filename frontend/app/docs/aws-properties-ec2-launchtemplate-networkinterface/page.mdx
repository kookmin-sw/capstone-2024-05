# AWS::EC2::LaunchTemplate 네트워크 인터페이스

네트워크 인터페이스에 대한 매개 변수를 지정합니다.

`NetworkInterface` 은 AWS::EC2::LaunchTemplate LaunchTemplateData의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AssociateCarrierIpAddress" : Boolean,
  "AssociatePublicIpAddress" : Boolean,
  "DeleteOnTermination" : Boolean,
  "Description" : String,
  "DeviceIndex" : Integer,
  "Groups" : [ String, ... ],
  "InterfaceType" : String,
  "Ipv4PrefixCount" : Integer,
  "Ipv4Prefixes" : [ Ipv4PrefixSpecification, ... ],
  "Ipv6AddressCount" : Integer,
  "Ipv6Addresses" : [ Ipv6Add, ... ],
  "Ipv6PrefixCount" : Integer,
  "Ipv6Prefixes" : [ Ipv6PrefixSpecification, ... ],
  "NetworkCardIndex" : Integer,
  "NetworkInterfaceId" : String,
  "PrimaryIpv6" : Boolean,
  "PrivateIpAddress" : String,
  "PrivateIpAddresses" : [ PrivateIpAdd, ... ],
  "SecondaryPrivateIpAddressCount" : Integer,
  "SubnetId" : String
}
```

## 속성

`AssociateCarrierIpAddress`  
새 네트워크 인터페이스의 이동통신사 IP 주소를 eth0과 연결합니다.  
Wavelength Zone에서 인스턴스를 시작하고 통신 사업자 IP 주소를 네트워크 인터페이스와 연결하려는 경우 이 옵션을 사용합니다. 통신 사업자 IP 주소에 대한 자세한 내용은 * AWS Wavelength 개발자 안내서*의 통신 사업자 IP 주소를 참조하세요.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`AssociatePublicIpAddress`  
새 네트워크 인터페이스의 경우 공용 IPv4 주소를 eth0과 연결합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`DeleteOnTermination`  
인스턴스가 종료될 때 네트워크 인터페이스가 삭제되는지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`Description`  
네트워크 인터페이스에 대한 설명입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`DeviceIndex`  
네트워크 인터페이스 연결에 대한 장치 인덱스입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Groups`  
하나 이상의 보안 그룹의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`InterfaceType`  
네트워크 인터페이스의 유형입니다. Elastic Fabric Adapter(EFA)를 생성하려면 을 지정합니다`efa`. 자세한 내용은 *Amazon Elastic Compute Cloud 사용 설명서*의 Elastic Fabric Adapter를 참조하세요.  
EFA를 생성하지 않는 경우 `interface` 이 파라미터를 지정하거나 생략합니다.  
유효한 값: `interface` | `efa`  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Ipv4PrefixCount`  
네트워크 인터페이스에 자동으로 할당되는 IPv4 접두사 수입니다. 이 옵션을 사용하는 경우 이 옵션을 사용할 수 없습니다 `Ipv4Prefix` .  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Ipv4Prefixes`  
네트워크 인터페이스에 할당할 하나 이상의 IPv4 접두사입니다. 이 옵션을 사용하는 경우 이 옵션을 사용할 수 없습니다 `Ipv4PrefixCount` .  
*필수 항목 *여부: 아니요  
_형식_: Ipv4PrefixSpecification 목록  
_업데이트 필요 사항_: 중단 없음

`Ipv6AddressCount`  
네트워크 인터페이스에 할당할 IPv6 주소의 수입니다. Amazon EC2는 서브넷 범위에서 IPv6 주소를 자동으로 선택합니다. 특정 IPv6 주소를 지정하는 경우에는 이 옵션을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Ipv6Addresses`  
서브넷의 IPv6 CIDR 블록 범위에 있는 하나 이상의 특정 IPv6 주소. 여러 IPv6 주소를 지정하는 경우에는 이 옵션을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: Ipv6Add 목록  
_업데이트 필요 사항_: 중단 없음

`Ipv6PrefixCount`  
네트워크 인터페이스에 자동으로 할당되는 IPv6 접두사 수입니다. 이 옵션을 사용하는 경우 이 옵션을 사용할 수 없습니다 `Ipv6Prefix` .  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Ipv6Prefixes`  
네트워크 인터페이스에 할당할 하나 이상의 IPv6 접두사입니다. 이 옵션을 사용하는 경우 이 옵션을 사용할 수 없습니다 `Ipv6PrefixCount` .  
*필수 항목 *여부: 아니요  
_형식_: Ipv6PrefixSpecification 목록  
_업데이트 필요 사항_: 중단 없음

`NetworkCardIndex`  
네트워크 카드의 인덱스입니다. 일부 인스턴스 유형은 여러 네트워크 카드를 지원합니다. 기본 네트워크 인터페이스는 네트워크 카드 인덱스 0에 할당되어야 합니다. 기본값은 네트워크 카드 인덱스 0입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`NetworkInterfaceId`  
네트워크 인터페이스의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PrimaryIpv6`  
네트워크 인터페이스의 기본 IPv6 주소입니다. IPv6 GUA 주소를 기본 IPv6로 설정하면 인스턴스가 종료되거나 네트워크 인터페이스가 분리될 때까지 첫 번째 IPv6 GUA가 기본 IPv6 주소가 됩니다. 기본 IPv6 주소에 대한 자세한 내용은 RunInstances 단원을 참조하십시오.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`PrivateIpAddress`  
네트워크 인터페이스의 기본 프라이빗 IPv4 주소입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PrivateIpAddresses`  
하나 이상의 프라이빗 IPv4 주소.  
*필수 항목 *여부: 아니요  
_형식_: PrivateIpAdd 목록  
_업데이트 필요 사항_: 중단 없음

`SecondaryPrivateIpAddressCount`  
네트워크 인터페이스에 할당할 보조 프라이빗 IPv4 주소의 수입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`SubnetId`  
네트워크 인터페이스에 대한 서브넷의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
