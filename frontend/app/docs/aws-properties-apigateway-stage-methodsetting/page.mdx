# AWS::ApiGateway::스테이지 메서드 설정

`MethodSetting` 속성 유형은 단계의 모든 메서드에 대한 설정을 구성합니다.

리소스 의 속성`MethodSettings`에는 `AWS::ApiGateway::Stage` 속성 형식 목록이 포함되어 `MethodSetting` 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CacheDataEncrypted" : Boolean,
  "CacheTtlInSeconds" : Integer,
  "CachingEnabled" : Boolean,
  "DataTraceEnabled" : Boolean,
  "HttpMethod" : String,
  "LoggingLevel" : String,
  "MetricsEnabled" : Boolean,
  "ResourcePath" : String,
  "ThrottlingBurstLimit" : Integer,
  "ThrottlingRateLimit" : Double
}
```

## 속성

`CacheDataEncrypted`  
캐시된 응답이 암호화되는지 여부를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`CacheTtlInSeconds`  
캐시된 응답에 대한 TTL(Time to Live)을 초 단위로 지정합니다. TTL이 높을수록 응답이 더 오래 캐시됩니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`CachingEnabled`  
요청에 대해 응답을 캐시하고 반환할지 여부를 지정합니다. 응답을 캐시하려면 스테이지에서 캐시 클러스터를 활성화해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`DataTraceEnabled`  
Amazon CloudWatch Logs에 푸시된 로그 항목에 영향을 주는 이 메서드에 대해 데이터 추적 로깅을 활성화할지 여부를 지정합니다. 이는 API 문제를 해결하는 데 유용할 수 있지만 민감한 데이터를 로깅할 수 있습니다. 프로덕션 API에 대해서는 이 옵션을 활성화하지 않는 것이 좋습니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`HttpMethod`  
HTTP 메서드입니다. 여러 리소스와 메서드에 설정을 적용하려면 및 에 별표(`*`) `HttpMethod` `/*` 를 `ResourcePath` 지정합니다. 이 매개 변수는 를 지정할 때 필요합니다`MethodSetting`.  
_필수_: 조건부  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`LoggingLevel`  
Amazon CloudWatch Logs에 푸시된 로그 항목에 영향을 주는 이 메서드의 로깅 수준을 지정합니다. 유효한 값은 `OFF`, `ERROR`, 및 `INFO`입니다. `ERROR` CloudWatch Logs에 오류 수준 항목만 작성하도록 선택하거나 `INFO` 모든 `ERROR` 이벤트와 추가 정보 이벤트를 포함하도록 선택합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`MetricsEnabled`  
이 메서드에 대해 Amazon CloudWatch 지표를 사용할 수 있는지 여부를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`ResourcePath`  
이 메서드의 리소스 경로입니다. 슬래시(`/`)는 다음과 같이 인코딩되며 `~1` 초기 슬래시에는 슬래시가 포함되어야 합니다. 예를 들어, 경로 값은 로 `/resource/subresource` 인코딩되어야 합니다`/~1resource~1subresource`. 루트 경로를 지정하려면 슬래시()만 사용합니다`/`. 여러 리소스와 메서드에 설정을 적용하려면 및 에 별표(`*`) `HttpMethod` `/*` 를 `ResourcePath` 지정합니다. 이 매개 변수는 를 지정할 때 필요합니다`MethodSetting`.  
_필수_: 조건부  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ThrottlingBurstLimit`  
제한 버스트 제한을 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`ThrottlingRateLimit`  
제한 속도 제한을 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음
