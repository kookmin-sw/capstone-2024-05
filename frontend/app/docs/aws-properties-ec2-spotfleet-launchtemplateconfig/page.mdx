# AWS::EC2::SpotFleet LaunchTemplateConfig

시작 템플릿을 지정하고 재정의합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "LaunchTemplateSpecification" : FleetLaunchTemplateSpecification,
  "Overrides" : [ LaunchTemplateOverrides, ... ]
}
```

## 속성

`LaunchTemplateSpecification`  
사용할 시작 템플릿입니다. `NetworkInterfaceId` 스팟 집합에서 네트워크 인터페이스 ID를 지정할 수 없으므로 시작 템플릿에 파라미터가 포함되어 있지 않은지 확인합니다.  
*필수 항목 *여부: 아니요  
_유형_: FleetLaunchTemplateSpecification  
_업데이트 필요 사항_: 바꿔 놓음

`Overrides`  
지정한 모든 파라미터는 시작 템플릿의 동일한 파라미터를 재정의합니다.  
*필수 항목 *여부: 아니요  
_유형_: LaunchTemplateOverrides 목록  
_업데이트 필요 사항_: 바꿔 놓음
