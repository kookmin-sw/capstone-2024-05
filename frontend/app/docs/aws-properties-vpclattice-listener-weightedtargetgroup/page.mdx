# AWS::VpcLattice::리스너 WeightedTargetGroup

대상 그룹의 가중치를 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "TargetGroupIdentifier" : String,
  "Weight" : Integer
}
```

## 속성

`TargetGroupIdentifier`  
대상 그룹의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Weight`  
전달 작업에 대해 여러 대상 그룹을 지정하는 경우에만 필요합니다. 가중치는 요청이 대상 그룹에 분산되는 방식을 결정합니다. 예를 들어, 각각 가중치가 10인 두 개의 대상 그룹을 지정하면 각 대상 그룹은 요청의 절반을 수신합니다. 가중치가 10인 대상 그룹과 가중치가 20인 대상 그룹 두 개를 지정하면 가중치가 20인 대상 그룹이 다른 대상 그룹보다 두 배 더 많은 요청을 수신합니다. 지정된 대상 그룹이 하나만 있는 경우 기본값은 100입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
