# AWS::ElasticLoadBalancing::LoadBalancer AppCookieStickinessPolicy

Classic Load Balancer에 대한 애플리케이션 제어 세션 고정성에 대한 정책을 지정합니다.

정책을 수신기와 연결하려면 수신기의 PolicyNames 속성을 사용합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CookieName" : String,
  "PolicyName" : String
}
```

## 속성

`CookieName`  
고정에 사용되는 응용 프로그램 쿠키의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PolicyName`  
생성 중인 정책의 니모닉 이름입니다. 이름은 이 로드 밸런서에 대한 정책 집합 내에서 고유해야 합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
