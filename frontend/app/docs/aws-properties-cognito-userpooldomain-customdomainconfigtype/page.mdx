# AWS::Cognito::UserPoolDomain 사용자 지정 도메인구성 유형

응용 프로그램에 대한 등록 및 로그인 웹 페이지를 호스트하는 사용자 지정 도메인에 대한 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CertificateArn" : String
}
```

## 속성

`CertificateArn`  
AWS Certificate Manager SSL 인증서의 Amazon 리소스 이름(ARN)입니다. 사용자 지정 도메인의 하위 도메인에 대해 이 인증서를 사용합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `20`  
_최대_: `2048`  
_패턴_: `arn:[w+=/,.@-]+:[w+=/,.@-]+:([w+=/,.@-]*)?:[0-9]+:[w+=/,.@-]+(:[w+=/,.@-]+)?(:[w+=/,.@-]+)?`  
_업데이트 필요 사항_: 중단 없음
