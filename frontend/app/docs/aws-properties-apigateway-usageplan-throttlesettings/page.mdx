# AWS::ApiGateway::UsagePlan 스로틀 설정

`ThrottleSettings` 는 사용자가 REST API를 호출할 때 전체 요청 빈도(초당 평균 요청) 및 버스트 용량을 지정하는 AWS::ApiGateway::UsagePlan 리소스의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BurstLimit" : Integer,
  "RateLimit" : Double
}
```

## 속성

`BurstLimit`  
API 대상 요청 버스트 속도 제한입니다. 이렇게 하면 일정 기간 동안 목표 속도 제한보다 더 많은 요청을 통과할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`RateLimit`  
API 대상 요청 속도 제한입니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음
