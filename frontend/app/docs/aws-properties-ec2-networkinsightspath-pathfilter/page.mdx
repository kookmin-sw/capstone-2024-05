# AWS::EC2::NetworkInsightsPath 경로 필터

경로 분석을 위한 필터 집합에 대해 설명합니다. 결과 경로가 여러 개 있을 수 있는 경우 경로 필터를 사용하여 분석 범위를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DestinationAddress" : String,
  "DestinationPortRange" : FilterPortRange,
  "SourceAddress" : String,
  "SourcePortRange" : FilterPortRange
}
```

## 속성

`DestinationAddress`  
대상 IPv4 주소입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `15`  
_패턴_: `^([0-9]{1,3}.){3}[0-9]{1,3}$`  
_업데이트 필요 사항_: 바꿔 놓음

`DestinationPortRange`  
대상 포트 범위입니다.  
*필수 항목 *여부: 아니요  
_형식_: FilterPortRange  
_업데이트 필요 사항_: 바꿔 놓음

`SourceAddress`  
소스 IPv4 주소입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `15`  
_패턴_: `^([0-9]{1,3}.){3}[0-9]{1,3}$`  
_업데이트 필요 사항_: 바꿔 놓음

`SourcePortRange`  
원본 포트 범위입니다.  
*필수 항목 *여부: 아니요  
_형식_: FilterPortRange  
_업데이트 필요 사항_: 바꿔 놓음
