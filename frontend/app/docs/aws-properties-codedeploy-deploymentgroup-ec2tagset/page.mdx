# AWS::CodeDeploy::D eploymentGroup EC2태그 세트

`EC2TagSet` 속성 유형은 Amazon EC2 인스턴스에 적용되는 태그 그룹에 대한 정보를 지정합니다. 배포 그룹에는 모든 태그 그룹으로 식별된 Amazon EC2 인스턴스만 포함됩니다. `EC2TagSet` 와 동일한 템플릿에서 사용할 수 없습니다 `EC2TagFilter`.

태그 및 태그 그룹을 사용하여 Amazon EC2 인스턴스 및 온프레미스 인스턴스를 관리하는 방법에 대한 자세한 내용은 AWS CodeDeploy에서 배포 그룹에 인스턴스 태깅 단원을 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Ec2TagSetList" : [ EC2TagSetListObject, ... ]
}
```

## 속성

`Ec2TagSetList`  
 배포 그룹에 포함하려는 Amazon EC2 인스턴스에 이미 적용된 Amazon EC2 태그입니다. CodeDeploy에는 이 배포 그룹에서 지정한 태그로 식별되는 모든 Amazon EC2 인스턴스가 포함됩니다.  
 중복은 허용되지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: EC2TagSetListObject 목록  
_업데이트 필요 사항_: 중단 없음
