# AWS::SageMaker::사용자 프로필 RStudioServerProAppSettings

앱과의 사용자 상호 작용을 구성하는 설정 컬렉션입니다 `RStudioServerPro` .

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AccessStatus" : String,
  "UserGroup" : String
}
```

## 속성

`AccessStatus`  
현재 사용자가 앱에 액세스할 수 있는지 여부를 나타냅니다 `RStudioServerPro` .  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `DISABLED | ENABLED`  
_업데이트 필요 사항_: 바꿔 놓음

`UserGroup`  
사용자가 앱 내에서 갖는 권한 수준입니다 `RStudioServerPro` . 기본값은 입니다`User`. 이 `Admin` 값을 사용하면 사용자가 RStudio 관리 대시보드에 액세스할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `R_STUDIO_ADMIN | R_STUDIO_USER`  
_업데이트 필요 사항_: 바꿔 놓음
