# AWS::SageMaker::AppImageConfig 커널 사양

Jupyter 커널의 사양입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DisplayName" : String,
  "Name" : String
}
```

## 속성

`DisplayName`  
커널의 표시 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `1024`  
_업데이트 필요 사항_: 중단 없음

`Name`  
이미지에 있는 Jupyter 커널의 이름입니다. 이 값은 대소문자를 구분합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `1024`  
_업데이트 필요 사항_: 중단 없음
