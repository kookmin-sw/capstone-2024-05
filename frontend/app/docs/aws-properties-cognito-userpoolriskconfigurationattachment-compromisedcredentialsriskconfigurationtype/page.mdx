# AWS::Cognito::UserPoolRiskConfigurationAttachment CompromisedCredentialsRiskConfigurationType

손상된 자격 증명 위험 구성 유형입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Actions" : CompromisedCredentialsActionsType,
  "EventFilter" : [ String, ... ]
}
```

## 속성

`Actions`  
손상된 자격 증명은 구성 작업의 위험을 감수합니다.  
_필수 항목 여부_: 예  
_유형_: CompromisedCredentialsActionsType  
_업데이트 필요 사항_: 중단 없음

`EventFilter`  
이러한 이벤트에 대한 작업을 수행합니다. 기본값은 이벤트 필터가 지정되지 않은 경우 모든 이벤트를 수행하는 것입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음
