# AWS::ApiGatewayV2::경로 응답

리소스 `AWS::ApiGatewayV2::RouteResponse` 는 WebSocket API에 대한 라우팅 응답을 생성합니다. 자세한 내용은 API Gateway 개발자 안내서의 API Gateway에서 WebSocket API에 대한 라우팅 응답 설정을 참조하십시오\*\*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::ApiGatewayV2::RouteResponse",
  "Properties" : {
      "ApiId" : String,
      "ModelSelectionExpression" : String,
      "ResponseModels" : Json,
      "ResponseParameters" : {Key: Value, ...},
      "RouteId" : String,
      "RouteResponseKey" : String
    }
}
```

## 속성

`ApiId`  
API 식별자입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`ModelSelectionExpression`  
경로 응답에 대한 모델 선택 식입니다. WebSocket API에 대해서만 지원됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ResponseModels`  
경로 응답에 대한 응답 모델입니다.  
*필수 항목 *여부: 아니요  
_형식_: Json  
_업데이트 필요 사항_: 중단 없음

`ResponseParameters`  
경로 응답 매개 변수입니다.  
*필수 항목 *여부: 아니요  
_유형_: ParameterConstraints 맵  
_업데이트 필요 사항_: 중단 없음

`RouteId`  
경로 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`RouteResponseKey`  
경로 응답 키입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`경로 응답 리소스 ID(예: )를 반환합니다 `abc123`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`RouteResponseId` 경로 응답 ID입니다.

## 예제

### 경로 응답 생성 예제

다음 예제에서는 `RouteResponse` 이미 `MyApi` 호출 `integration` 된 및 `MyIntegration` 호출 `route` 된 가 있는 WebSocket API에 대한 리소스를 `MyRoute` 만듭니다.

#### JSON (영문)

```
{
    "MyRouteResponse": {
        "Type": "AWS::ApiGatewayV2::RouteResponse",
        "Properties": {
            "RouteId": {
                "Ref": "MyRoute"
            },
            "ApiId": {
                "Ref": "MyApi"
            },
            "RouteResponseKey": "$default"
        }
    }
}
```
