# AWS::EC2::EC2Fleet 배치

인스턴스의 배치에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Affinity" : String,
  "AvailabilityZone" : String,
  "GroupName" : String,
  "HostId" : String,
  "HostResourceGroupArn" : String,
  "PartitionNumber" : Integer,
  "SpreadDomain" : String,
  "Tenancy" : String
}
```

## 속성

`Affinity`  
전용 호스트의 인스턴스에 대한 선호도 설정입니다.  
이 파라미터는 CreateFleet 또는 ImportInstance에 대해 지원되지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`AvailabilityZone`  
인스턴스의 가용 영역입니다.  
지정하지 않으면 가용 영역이 리전의 로드 밸런싱 기준에 따라 자동으로 선택됩니다.  
이 파라미터는 CreateFleet에서 지원되지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`GroupName`  
인스턴스가 속한 배치 그룹의 이름입니다. 를 지정`GroupName`하면 를 지정할 수 없습니다`GroupId`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`HostId`  
인스턴스가 상주하는 전용 호스트의 ID입니다.  
이 파라미터는 CreateFleet 또는 ImportInstance에 대해 지원되지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`HostResourceGroupArn`  
인스턴스를 시작할 호스트 리소스 그룹의 ARN입니다.  
이 매개 변수를 지정하는 경우 **Tenancy** 매개 변수를 생략하거나 로 설정합니다`host`.  
이 파라미터는 CreateFleet에서 지원되지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`PartitionNumber`  
인스턴스가 있는 파티션의 번호입니다. 배치 그룹 전략이 로 설정된 경우에만 유효합니다`partition`.  
이 파라미터는 CreateFleet에서 지원되지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음

`SpreadDomain`  
나중에 사용할 수 있도록 예약되어 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tenancy`  
인스턴스의 테넌시입니다. 테넌시가 있는 인스턴스는 `dedicated` 단일 테넌트 하드웨어에서 실행됩니다.  
이 파라미터는 CreateFleet에서 지원되지 않습니다. `host` 테넌시는 ImportInstance 또는 CPU 크레딧 옵션에 대해 구성된 T3 인스턴스에 대해 지원되지 않습니다 `unlimited` .  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `dedicated | default | host`  
_업데이트 필요 사항_: 바꿔 놓음
