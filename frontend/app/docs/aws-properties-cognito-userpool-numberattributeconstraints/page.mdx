# AWS::Cognito::UserPool NumberAttribute제약 조건

number 데이터 형식인 특성의 최소값 및 최대값입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "MaxValue" : String,
  "MinValue" : String
}
```

## 속성

`MaxValue`  
number 데이터 형식인 속성의 최대값입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `131072`  
_업데이트 필요 사항_: 중단 없음

`MinValue`  
number 데이터 형식인 속성의 최소값입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `131072`  
_업데이트 필요 사항_: 중단 없음
