# AWS::SageMaker::모델 카드 추가 정보

모델에 대한 추가 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CaveatsAndRecommendations" : String,
  "CustomDetails" : {Key: Value, ...},
  "EthicalConsiderations" : String
}
```

## 속성

`CaveatsAndRecommendations`  
응용 프로그램에서 이 모델을 사용할 수 있는 사용자를 위한 주의 사항 및 권장 사항입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`CustomDetails`  
모델에 대해 문서화할 추가 정보입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 맵  
_업데이트 필요 사항_: 중단 없음

`EthicalConsiderations`  
모델 카드 작성자가 문서화한 모든 윤리적 고려 사항.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
