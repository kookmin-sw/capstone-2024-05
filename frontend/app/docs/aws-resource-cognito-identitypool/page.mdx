# AWS::Cognito::IdentityPool (영문)

리소스 `AWS::Cognito::IdentityPool` 는 Amazon Cognito 자격 증명 풀을 생성합니다.

AWS CloudFormation에서 실수로 리소스를 삭제하지 않으려면 DeletionPolicy 속성 및 UpdateReplacePolicy 속성을 사용하여 삭제 또는 교체 시 리소스를 유지합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Cognito::IdentityPool",
  "Properties" : {
      "AllowClassicFlow" : Boolean,
      "AllowUnauthenticatedIdentities" : Boolean,
      "CognitoEvents" : Json,
      "CognitoIdentityProviders" : [ CognitoIdentityProvider, ... ],
      "CognitoStreams" : CognitoStreams,
      "DeveloperProviderName" : String,
      "IdentityPoolName" : String,
      "OpenIdConnectProviderARNs" : [ String, ... ],
      "PushSync" : PushSync,
      "SamlProviderARNs" : [ String, ... ],
      "SupportedLoginProviders" : Json
    }
}
```

## 속성

`AllowClassicFlow`  
기본(클래식) 인증 흐름을 사용하도록 설정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`AllowUnauthenticatedIdentities`  
자격 증명 풀이 인증되지 않은 로그인을 지원하는지 여부를 지정합니다.  
_필수 항목 여부_: 예  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`CognitoEvents`  
구성할 이벤트입니다.  
*필수 항목 *여부: 아니요  
_형식_: Json  
_업데이트 필요 사항_: 중단 없음

`CognitoIdentityProviders`  
Amazon Cognito 사용자 풀 및 해당 클라이언트 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: CognitoIdentityProvider 목록  
_업데이트 필요 사항_: 중단 없음

`CognitoStreams`  
Amazon Cognito 스트림을 구성하기 위한 구성 옵션입니다.  
*필수 항목 *여부: 아니요  
_유형_: CognitoStreams  
_업데이트 필요 사항_: 중단 없음

`DeveloperProviderName`  
Amazon Cognito가 사용자를 참조할 때 사용하는 "도메인"입니다. 이 이름은 백엔드와 Amazon Cognito 서비스가 개발자 공급자에 대해 통신할 수 있도록 하는 자리 표시자 역할을 합니다. 의 경우 `DeveloperProviderName`문자와 마침표(.), 밑줄(\_) 및 대시(-)를 사용할 수 있습니다.  
_최소 길이_: 1  
_최대 길이_: 100  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`IdentityPoolName`  
Amazon Cognito 자격 증명 풀의 이름입니다.  
_최소 길이_: 1  
_최대 길이_: 128  
_패턴_: `[ws+=,.@-]+`  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`OpenIdConnectProviderARNs`  
OpenID 연결 공급자의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`PushSync`  
자격 증명 풀에 적용할 구성 옵션입니다.  
*필수 항목 *여부: 아니요  
_유형_: PushSync  
_업데이트 필요 사항_: 중단 없음

`SamlProviderARNs`  
SAML(Security Assertion Markup Language) 공급자의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`SupportedLoginProviders`  
제공자 이름을 제공자 앱 ID에 매핑하는 키-값 쌍입니다.  
*필수 항목 *여부: 아니요  
_형식_: Json  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`와 `Ref`같은 를 `IdentityPoolId`반환합니다`us-east-2:0d01f4d7-1305-4408-b437-12345EXAMPLE`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Name` Amazon Cognito 자격 증명 풀의 이름으로, 문자열로 반환됩니다.
