# AWS::Lightsail::컨테이너 PublicDomainName

`PublicDomainName` 은 AWS::Lightsail::Container 리소스의 속성입니다. 컨테이너 서비스와 함께 사용할 공용 도메인 이름(예: 및 )에 대해 설명합니다 `example.com` `www.example.com`. 또한 컨테이너 서비스에 사용할 인증서에 대해서도 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CertificateName" : String,
  "DomainNames" : [ String, ... ]
}
```

## 속성

`CertificateName`  
공용 도메인에 대한 인증서의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`DomainNames`  
컨테이너 서비스에 사용할 퍼블릭 도메인 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음
