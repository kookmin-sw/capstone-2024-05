# AWS::S3ObjectLambda::AccessPoint 별칭

객체 Lambda 액세스 포인트의 별칭입니다. 자세한 내용은 S3 버킷 객체 Lambda 액세스 포인트에 버킷 스타일 별칭을 사용하는 방법을 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Status" : String,
  "Value" : String
}
```

## 속성

`Status`  
객체 Lambda 액세스 포인트 별칭의 상태입니다. 상태가 인 경우 `PROVISIONING`객체 Lambda 액세스 포인트가 별칭을 프로비저닝하고 있으며 별칭을 아직 사용할 준비가 되지 않았습니다. 상태가 인 경우 `READY`객체 Lambda 액세스 포인트 별칭이 성공적으로 프로비저닝되어 사용할 준비가 된 것입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Value`  
객체 Lambda 액세스 포인트의 별칭 값입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
