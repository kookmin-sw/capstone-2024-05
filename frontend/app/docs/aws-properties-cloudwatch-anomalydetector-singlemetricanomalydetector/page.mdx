# AWS::CloudWatch::AnomalyDetector SingleMetricAnomalyDetector

이상 탐지기가 입력으로 사용하는 시계열을 제공하는 CloudWatch 지표 및 통계를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Dimensions" : [ Dimension, ... ],
  "MetricName" : String,
  "Namespace" : String,
  "Stat" : String
}
```

## 속성

`Dimensions`  
변칙 검색 모델을 만들 메트릭 차원입니다.  
*필수 항목 *여부: 아니요  
_유형_: 차원 목록  
_최대_: `30`  
_업데이트 필요 사항_: 바꿔 놓음

`MetricName`  
변칙 검색 모델을 만들 메트릭의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_업데이트 필요 사항_: 바꿔 놓음

`Namespace`  
변칙 검색 모델을 만들 메트릭의 네임스페이스입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_패턴_: `[^:].*`  
_업데이트 필요 사항_: 바꿔 놓음

`Stat`  
메트릭 및 변칙 검색 모델에 사용할 통계입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `50`  
_패턴_: `(SampleCount|Average|Sum|Minimum|Maximum|IQM|(p|tc|tm|ts|wm)(d{1,2}(.d{0,10})?|100)|[ou]d+(.d*)?)(_E|_L|_H)?|(TM|TC|TS|WM)(((((d{1,2})(.d{0,10})?|100(.0{0,10})?)%)?:((d{1,2})(.d{0,10})?|100(.0{0,10})?)%|((d{1,2})(.d{0,10})?|100(.0{0,10})?)%:(((d{1,2})(.d{0,10})?|100(.0{0,10})?)%)?))|(TM|TC|TS|WM|PR)(((d+(.d{0,10})?|(d+(.d{0,10})?[Ee][+-]?d+)):((d+(.d{0,10})?|(d+(.d{0,10})?[Ee][+-]?d+)))?|((d+(.d{0,10})?|(d+(.d{0,10})?[Ee][+-]?d+)))?:(d+(.d{0,10})?|(d+(.d{0,10})?[Ee][+-]?d+))))`  
_업데이트 필요 사항_: 바꿔 놓음
