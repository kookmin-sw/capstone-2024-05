# AWS::SageMaker::모델 카드 평가 세부 정보

모델의 평가 세부 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Datasets" : [ String, ... ],
  "EvaluationJobArn" : String,
  "EvaluationObservation" : String,
  "Metadata" : {Key: Value, ...},
  "MetricGroups" : [ MetricGroup, ... ],
  "Name" : String
}
```

## 속성

`Datasets`  
모델을 평가하는 데 사용되는 데이터 세트의 위치입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`EvaluationJobArn`  
평가 작업의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`EvaluationObservation`  
모델 평가 중에 수행된 모든 관찰.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Metadata`  
평가 결과와 연관된 추가 특성입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 맵  
_업데이트 필요 사항_: 중단 없음

`MetricGroups`  
평가 지표 그룹 개체입니다.  
*필수 항목 *여부: 아니요  
_유형_: MetricGroup 목록  
_업데이트 필요 사항_: 중단 없음

`Name`  
평가 작업 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
