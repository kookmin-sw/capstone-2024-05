# AWS::SageMaker::모델 카드 MetricDataItems

측정항목 데이터. 에 `type` 대해 지정하는 데이터 형식을 결정합니다 `value` `XAxisName` `YAxisName`. 메트릭 값 지정에 대한 자세한 내용은 모델 카드 JSON 스키마를 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Notes" : String,
  "Type" : String,
  "Value" : Json,
  "XAxisName" : [ String, ... ],
  "YAxisName" : [ String, ... ]
}
```

## 속성

`Name`  
메트릭의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Notes`  
메트릭에 추가할 메모입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Type` 다음 데이터 형식 중 하나를 지정해야 합니다.

- 막대 차트 – `bar_char`
- 부울– `boolean`
- 선형 그래프 – `linear_graph`
- 행렬– `matrix`
- 수– `number`
- 문자열– `string`
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _업데이트 필요 사항_: 중단 없음

`Value`  
메트릭의 데이터 유형입니다. 메트릭의 _값은_ 메트릭의 유형과 호환되어야 합니다\**.  
*필수 항목 여부*: 예  
*형식*: Json  
*업데이트 필요 사항\*: 중단 없음

`XAxisName`  
x축의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`YAxisName`  
y축의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음
