# AWS::S3::StorageLens AdvancedCostOptimization메트릭

이 리소스를 사용하면 Amazon S3 Storage Lens 고급 비용 최적화 지표를 사용할 수 있습니다. 고급 비용 최적화 지표는 스토리지 비용을 관리하고 최적화하는 데 사용할 수 있는 인사이트를 제공합니다(예: 전환, 만료 및 불완전한 멀티파트 업로드에 대한 수명 주기 규칙 수).

자세한 내용은 Amazon S3 사용 설명서*의 S3 Storage Lens를 사용한 스토리지 활동 및 사용량 평가를 참조하세요*. 전체 지표 목록은 _Amazon S3 사용 설명서의 S3 Storage Lens 지표 용어집을 참조하세요_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "IsEnabled" : Boolean
}
```

## 속성

`IsEnabled`  
고급 비용 최적화 메트릭을 사용할 수 있는지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
