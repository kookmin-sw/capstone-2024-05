# AWS::Lambda::LayerVersion 콘텐츠

Lambda 계층의 콘텐츠를 포함하는 ZIP 아카이브입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "S3Bucket" : String,
  "S3Key" : String,
  "S3ObjectVersion" : String
}
```

## 속성

`S3Bucket`  
계층 아카이브의 Amazon S3 버킷입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `3`  
_최대_: `63`  
_패턴_: `^[0-9A-Za-z.-_]*(?<!.)$`  
_업데이트 필요 사항_: 바꿔 놓음

`S3Key`  
계층 아카이브의 Amazon S3 키입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `1024`  
_업데이트 필요 사항_: 바꿔 놓음

`S3ObjectVersion`  
버전이 지정된 객체의 경우, 사용할 레이어 아카이브 객체의 버전입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `1024`  
_업데이트 필요 사항_: 바꿔 놓음

## 예제

### 레이어 콘텐츠

Amazon S3에서 계층 아카이브의 위치입니다.

#### JSON (영문)

```
        "Content": {
            "S3Bucket": "my-bucket-us-west-2-123456789012",
            "S3Key": "layer.zip"
        }
```
