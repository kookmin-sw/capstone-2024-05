# AWS::KinesisAnalyticsV2::애플리케이션 애플리케이션 구성

Managed Service for Apache Flink 애플리케이션에 대한 작성 매개변수를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ApplicationCodeConfiguration" : ApplicationCodeConfiguration,
  "ApplicationSnapshotConfiguration" : ApplicationSnapshotConfiguration,
  "EnvironmentProperties" : EnvironmentProperties,
  "FlinkApplicationConfiguration" : FlinkApplicationConfiguration,
  "SqlApplicationConfiguration" : SqlApplicationConfiguration,
  "VpcConfigurations" : [ VpcConfiguration, ... ],
  "ZeppelinApplicationConfiguration" : ZeppelinApplicationConfiguration
}
```

## 속성

`ApplicationCodeConfiguration`  
Managed Service for Apache Flink 애플리케이션의 코드 위치 및 유형 매개변수입니다.  
_필수_: 조건부  
_형식_: ApplicationCodeConfiguration  
_업데이트 필요 사항_: 중단 없음

`ApplicationSnapshotConfiguration`  
Managed Service for Apache Flink 애플리케이션에 대해 스냅샷을 사용할 수 있는지 여부를 설명합니다.  
*필수 항목 *여부: 아니요  
_형식_: ApplicationSnapshotConfiguration  
_업데이트 필요 사항_: 중단 없음

`EnvironmentProperties`  
Managed Service for Apache Flink 애플리케이션의 실행 특성에 대해 설명합니다.  
*필수 항목 *여부: 아니요  
_형식_: EnvironmentProperties  
_업데이트 필요 사항_: 중단 없음

`FlinkApplicationConfiguration`  
Managed Service for Apache Flink 애플리케이션에 대한 작성 및 업데이트 매개변수입니다.  
*필수 항목 *여부: 아니요  
_유형_: FlinkApplicationConfiguration  
_업데이트 필요 사항_: 중단 없음

`SqlApplicationConfiguration`  
SQL 기반 Managed Service for Apache Flink 애플리케이션에 대한 작성 및 업데이트 매개변수입니다.  
*필수 항목 *여부: 아니요  
_형식_: SqlApplicationConfiguration  
_업데이트 필요 사항_: 중단 없음

`VpcConfigurations`  
애플리케이션에서 사용할 수 있는 VPC 구성에 대한 설명의 배열입니다.  
*필수 항목 *여부: 아니요  
_유형_: VpcConfiguration 목록  
_업데이트 필요 사항_: 중단 없음

`ZeppelinApplicationConfiguration`  
Kinesis Data Analytics Studio 노트북의 구성 파라미터입니다.  
*필수 항목 *여부: 아니요  
_형식_: ZeppelinApplicationConfiguration  
_업데이트 필요 사항_: 중단 없음
