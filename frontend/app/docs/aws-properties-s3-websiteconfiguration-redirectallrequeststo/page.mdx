# AWS::S3::버킷 RedirectAllRequestsTo

Amazon S3 버킷의 웹 사이트 엔드포인트에 대한 모든 요청의 리디렉션 동작을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "HostName" : String,
  "Protocol" : String
}
```

## 속성

`HostName`  
요청이 리디렉션되는 호스트의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Protocol`  
요청을 리디렉션할 때 사용할 프로토콜입니다. 기본값은 원래 요청에 사용되는 프로토콜입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `http | https`  
_업데이트 필요 사항_: 중단 없음
