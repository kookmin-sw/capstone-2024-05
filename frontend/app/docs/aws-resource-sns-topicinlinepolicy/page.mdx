# AWS::SNS::TopicInlinePolicy

`AWS::SNS::TopicInlinePolicy` 리소스 속성 설명을 사용할 수 없습니다. SNS용.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::SNS::TopicInlinePolicy",
  "Properties" : {
      "PolicyDocument" : Json,
      "TopicArn" : String
    }
}
```

## 속성

`PolicyDocument`  
속성 설명을 사용할 수 없습니다.  
_필수 항목 여부_: 예  
_형식_: Json  
_업데이트 필요 사항_: 중단 없음

`TopicArn`  
속성 설명을 사용할 수 없습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
