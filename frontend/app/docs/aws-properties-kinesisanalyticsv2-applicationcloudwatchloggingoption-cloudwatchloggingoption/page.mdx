# AWS::KinesisAnalyticsV2::ApplicationCloudWatchLogging옵션 CloudWatchLogging옵션

로그 스트림 Amazon 리소스 이름(ARN)을 포함하여 Amazon CloudWatch 로깅 옵션에 대한 설명을 제공합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "LogStreamARN" : String
}
```

## 속성

`LogStreamARN`  
애플리케이션 메시지를 수신하기 위한 CloudWatch 로그의 ARN입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `2048`  
_패턴_: `arn:.*`  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

- Amazon Kinesis Data Analytics API 참조\* \*의 CloudWatchLoggingOption
