# AWS::ElasticLoadBalancing::LoadBalancer 연결 설정

Classic Load Balancer의 유휴 제한 시간 값을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "IdleTimeout" : Integer
}
```

## 속성

`IdleTimeout`  
부하 분산 장치에 의해 연결이 닫히기 전에 연결이 유휴 상태(연결을 통해 데이터가 전송되지 않음)가 허용되는 시간(초)입니다.  
_필수 항목 여부_: 예  
_유형_: Integer  
_최소_: `1`  
_최대_: `3600`  
_업데이트 필요 사항_: 중단 없음
