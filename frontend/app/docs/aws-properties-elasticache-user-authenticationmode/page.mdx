# AWS::ElastiCache::사용자 인증 모드

사용할 인증 모드를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Passwords" : [ String, ... ],
  "Type" : String
}
```

## 속성

`Passwords`  
로 설정된 경우 인증에 사용할 암호를 지정합니다 `Type` `password`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Type`  
인증 유형을 지정합니다. 가능한 옵션은 IAM 인증, 암호 및 암호 없음입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `iam | no-password-required | password`  
_업데이트 필요 사항_: 중단 없음
