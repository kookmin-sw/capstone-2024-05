# AWS::CloudFront::D istribution 상태 코드

기본 오리진에서 반환될 때 CloudFront가 두 번째 오리진으로 장애 조치하도록 트리거하도록 지정하는 상태 코드에 대한 복잡한 데이터 형식입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Items" : [ Integer, ... ],
  "Quantity" : Integer
}
```

## 속성

`Items`  
원본 그룹에 대한 항목(상태 코드)입니다.  
_필수 항목 여부_: 예  
_유형_: 정수 목록  
_업데이트 필요 사항_: 중단 없음

`Quantity`  
상태 코드의 수입니다.  
_필수 항목 여부_: 예  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
