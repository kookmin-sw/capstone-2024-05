# AWS::KinesisAnalyticsV2::애플리케이션 레코드열

SQL 기반 Managed Service for Apache Flink 애플리케이션의 경우, 스트리밍 소스의 각 데이터 요소를 인애플리케이션 스트림의 해당 컬럼에 맵핑하는 방법을 설명합니다.

참조 데이터 원본의 형식을 설명하는 데도 사용됩니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Mapping" : String,
  "Name" : String,
  "SqlType" : String
}
```

## 속성

`Mapping`  
스트리밍 입력 또는 참조 데이터 원본의 데이터 요소에 대한 참조입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `65535`  
_업데이트 필요 사항_: 중단 없음

`Name`  
인애플리케이션 입력 스트림 또는 참조 테이블에서 작성되는 컬럼의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `256`  
_패턴_: `[^-s<>&]*`  
_업데이트 필요 사항_: 중단 없음

`SqlType`  
인애플리케이션 입력 스트림 또는 참조 테이블에서 생성된 컬럼의 유형입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `100`  
_업데이트 필요 사항_: 중단 없음
