# AWS::CodeDeploy::D eploymentGroup TargetGroupPairInfo

`TargetGroupPairInfo` 속성 유형은 속성 설명을 사용할 수 없음을 지정합니다. AWS::CodeDeploy::D eploymentGroup의 경우.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ProdTrafficRoute" : TrafficRoute,
  "TargetGroups" : [ TargetGroupInfo, ... ],
  "TestTrafficRoute" : TrafficRoute
}
```

## 속성

`ProdTrafficRoute`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: TrafficRoute  
_업데이트 필요 사항_: 중단 없음

`TargetGroups`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: TargetGroupInfo 목록  
_업데이트 필요 사항_: 중단 없음

`TestTrafficRoute`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: TrafficRoute  
_업데이트 필요 사항_: 중단 없음
