# AWS::KinesisFirehose::D eliveryStream DynamicPartitioningConfiguration

`DynamicPartitioningConfiguration` 속성 형식은 파티션 키를 기반으로 분할하여 스트리밍 데이터에서 대상 데이터 집합을 만드는 동적 분할 메커니즘의 구성을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Enabled" : Boolean,
  "RetryOptions" : RetryOptions
}
```

## 속성

`Enabled`  
이 Kinesis Data Firehose 전송 스트림에 대해 동적 파티셔닝을 활성화할지 여부를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`RetryOptions`  
Kinesis Data Firehose가 Amazon S3 접두사로 데이터를 전송할 수 없는 경우의 재시도 동작을 지정합니다.  
*필수 항목 *여부: 아니요  
_형식_: RetryOptions  
_업데이트 필요 사항_: 중단 없음
