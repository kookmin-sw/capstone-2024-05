# AWS::Cognito::UserPool StringAttributeConstraints

`StringAttributeConstraints` 속성 유형은 Amazon Cognito 사용자 풀의 문자열 속성 제약 조건을 정의합니다. `StringAttributeConstraints` 은 SchemaAttribute 속성 형식의 하위 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "MaxLength" : String,
  "MinLength" : String
}
```

## 속성

`MaxLength`  
최대 길이입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `131072`  
_업데이트 필요 사항_: 중단 없음

`MinLength`  
최소 길이입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `131072`  
_업데이트 필요 사항_: 중단 없음
