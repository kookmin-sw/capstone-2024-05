# AWS::CloudFront::OriginRequestPolicy

오리진 요청 정책입니다.

캐시 동작에 연결된 경우 오리진 요청 정책은 CloudFront가 오리진에 보내는 요청에 포함하는 값을 결정합니다. CloudFront에서 오리진에 보내는 각 요청에는 다음이 포함됩니다.

- 최종 사용자 요청의 요청 본문 및 URL 경로(도메인 이름 제외)입니다.
- CloudFront가 모든 오리진 요청에 자동으로 포함하는 헤더(예: `Host`, `User-Agent`및 `X-Amz-Cf-Id`.
- 캐시 정책 또는 오리진 요청 정책에 지정된 모든 HTTP 헤더, 쿠키 및 URL 쿼리 문자열입니다. 여기에는 최종 사용자 요청의 항목이 포함될 수 있으며, 헤더의 경우 CloudFront에서 추가한 추가 항목이 포함될 수 있습니다.

CloudFront는 캐시에서 요청과 일치하는 객체를 찾을 수 없을 때 요청을 보냅니다. 오리진에 값을 전송하고 캐시 키에도 포함 하려면 `CachePolicy`.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::CloudFront::OriginRequestPolicy",
  "Properties" : {
      "OriginRequestPolicyConfig" : OriginRequestPolicyConfig
    }
}
```

## 속성

`OriginRequestPolicyConfig`  
오리진 요청 정책 구성입니다.  
_필수 항목 여부_: 예  
_유형_: OriginRequestPolicyConfig  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 intrinsic 함수에 전달하면 `Ref` `Ref`오리진 요청 정책 ID를 반환합니다. 예를 들면 다음과 같습니다 `befd7079-9bbc-4ebf-8ade-498a3694176c`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Id`  
오리진 요청 정책의 고유 식별자입니다. 예를 들면 다음과 같습니다 `befd7079-9bbc-4ebf-8ade-498a3694176c`.

`LastModifiedTime` 오리진 요청 정책을 마지막으로 수정한 날짜 및 시간입니다.
