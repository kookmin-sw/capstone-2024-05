# AWS::EC2::LaunchTemplate PrivateIpAdd

네트워크 인터페이스에 대한 보조 개인 IPv4 주소를 지정합니다.

`PrivateIpAdd` 는 AWS::EC2::LaunchTemplate NetworkInterface의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Primary" : Boolean,
  "PrivateIpAddress" : String
}
```

## 속성

`Primary`  
개인 IPv4 주소가 기본 개인 IPv4 주소인지 여부를 나타냅니다. 하나의 IPv4 주소만 기본으로 지정할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`PrivateIpAddress`  
프라이빗 IPv4 주소입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
