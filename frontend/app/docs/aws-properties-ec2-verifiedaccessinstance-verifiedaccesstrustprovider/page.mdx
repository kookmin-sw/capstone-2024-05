# AWS::EC2::VerifiedAccessInstance VerifiedAccessTrustProvider

Verified Access 트러스트 공급자에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Description" : String,
  "DeviceTrustProviderType" : String,
  "TrustProviderType" : String,
  "UserTrustProviderType" : String,
  "VerifiedAccessTrustProviderId" : String
}
```

## 속성

`Description`  
AWS Verified Access 트러스트 공급자에 대한 설명입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`DeviceTrustProviderType`  
장치 기반 신뢰 공급자의 유형입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `crowdstrike | jamf`  
_업데이트 필요 사항_: 중단 없음

`TrustProviderType`  
Verified Access 신뢰 공급자의 유형입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `device | user`  
_업데이트 필요 사항_: 중단 없음

`UserTrustProviderType`  
사용자 기반 신뢰 공급자의 유형입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `iam-identity-center | oidc`  
_업데이트 필요 사항_: 중단 없음

`VerifiedAccessTrustProviderId`  
AWS Verified Access 신뢰 공급자의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
