# AWS::EC2::인스턴스 최대 절전 모드 옵션

인스턴스에 대한 최대 절전 모드 옵션을 지정합니다.

`HibernationOptions` 은 AWS::EC2::Instance 리소스의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Configured" : Boolean
}
```

## 속성

`Configured`  
 `true` 인스턴스를 최대 절전 모드로 활성화하려면 로 설정합니다.  
기본값: `false`  
_필수_: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
