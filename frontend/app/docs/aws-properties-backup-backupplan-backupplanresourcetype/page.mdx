# AWS::백업::백업 계획 BackupPlanResourceType

백업 계획을 만드는 데 사용되는 속성이 포함된 개체를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AdvancedBackupSettings" : [ AdvancedBackupSettingResourceType, ... ],
  "BackupPlanName" : String,
  "BackupPlanRule" : [ BackupRuleResourceType, ... ]
}
```

## 속성

`AdvancedBackupSettings`  
각 리소스 종류에 대한 백업 옵션 목록입니다.  
*필수 항목 *여부: 아니요  
_유형_: AdvancedBackupSettingResourceType 목록  
_업데이트 필요 사항_: 중단 없음

`BackupPlanName`  
백업 계획의 표시 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`BackupPlanRule`  
개체의 배열로 `BackupRule` , 각 개체는 선택한 리소스를 백업하는 데 사용되는 예약된 작업을 지정합니다.  
_필수 항목 여부_: 예  
_유형_: BackupRuleResourceType 목록  
_업데이트 필요 사항_: 중단 없음
