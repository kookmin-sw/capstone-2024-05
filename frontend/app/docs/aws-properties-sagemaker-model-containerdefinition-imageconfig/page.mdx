# AWS::SageMaker::모델 이미지 구성

모델 컨테이너가 Amazon ECR에 있는지 아니면 Amazon Virtual Private Cloud(VPC)에서 액세스할 수 있는 프라이빗 Docker 레지스트리에 있는지 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "RepositoryAccessMode" : String,
  "RepositoryAuthConfig" : RepositoryAuthConfig
}
```

## 속성

`RepositoryAccessMode` 이 값을 다음 값 중 하나로 설정합니다.

- `Platform` - 모델 이미지는 Amazon ECR에서 호스팅됩니다.
- `Vpc` - 모델 이미지는 VPC의 프라이빗 Docker 레지스트리에서 호스팅됩니다.
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _허용되는 값_: `Platform | Vpc`  
  _업데이트 필요 사항_: 바꿔 놓음

`RepositoryAuthConfig`  
(선택 사항) 모델 이미지가 호스팅되는 개인 Docker 레지스트리에 대한 인증 구성을 지정합니다. 필드 `Vpc` 의 값으로 `RepositoryAccessMode` 지정 하고 모델 이미지가 호스팅되는 개인용 Docker 레지스트리에 인증이 필요한 경우에만 이 특성의 값을 지정하십시오.  
*필수 항목 *여부: 아니요  
_유형_: RepositoryAuthConfig  
_업데이트 필요 사항_: 바꿔 놓음
