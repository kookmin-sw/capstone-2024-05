# AWS::S3::버킷 복제 대상

S3 복제 시간 제어(S3 RTC) 활성화를 포함하여 복제 대상 및 해당 구성에 대한 정보를 위한 컨테이너입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AccessControlTranslation" : AccessControlTranslation,
  "Account" : String,
  "Bucket" : String,
  "EncryptionConfiguration" : EncryptionConfiguration,
  "Metrics" : Metrics,
  "ReplicationTime" : ReplicationTime,
  "StorageClass" : String
}
```

## 속성

`AccessControlTranslation`  
교차 계정 시나리오(원본 버킷과 대상 버킷 소유자가 동일하지 않은 경우)에서만 이를 지정하고 복제본 소유권을 대상 버킷을 소유한 AWS 계정으로 변경하려는 경우에만 지정합니다. 복제 구성에 지정되지 않은 경우 원본 객체를 소유한 동일한 AWS 계정에서 복제본을 소유합니다.  
*필수 항목 *여부: 아니요  
_형식_: AccessControlTranslation  
_업데이트 필요 사항_: 중단 없음

`Account`  
대상 버킷 소유자 계정 ID입니다. 교차 계정 시나리오에서 속성을 지정하여 복제본 소유권을 대상 버킷을 소유한 AWS 계정으로 변경하도록 Amazon S3에 지시하는 경우 `AccessControlTranslation` 이 ID는 대상 버킷 소유자의 계정 ID입니다. 자세한 내용은 Amazon S3 사용 설명서의 교차 리전 복제 추가 구성: 복제본 소유자 변경을 참조하십시오\**.  
속성을 지정하는 경우 `AccessControlTranslation` `Account` 속성은 필수입니다.  
*필수*: 조건부  
*유형*: 문자열  
*업데이트 필요 사항\*: 중단 없음

`Bucket`  
 Amazon S3에서 결과를 저장할 버킷의 Amazon 리소스 이름(ARN)입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`EncryptionConfiguration`  
암호화 관련 정보를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: EncryptionConfiguration  
_업데이트 필요 사항_: 중단 없음

`Metrics`  
 복제 지표 및 이벤트를 활성화하는 복제 지표 관련 설정을 지정하는 컨테이너입니다.  
*필수 항목 *여부: 아니요  
_유형_: 지표  
_업데이트 필요 사항_: 중단 없음

`ReplicationTime`  
 S3 RTC 활성화 여부 및 모든 객체 및 객체에 대한 작업을 복제해야 하는 시간을 포함하여 S3 복제 시간 제어(S3 RTC)를 지정하는 컨테이너입니다. 블록과 함께 지정해야 합니다 `Metrics` .  
*필수 항목 *여부: 아니요  
_형식_: ReplicationTime  
_업데이트 필요 사항_: 중단 없음

`StorageClass`  
 객체를 복제할 때 사용할 스토리지 클래스(예: S3 표준 또는 축소된 중복성)입니다. 기본적으로 Amazon S3는 원본 객체의 스토리지 클래스를 사용하여 객체 복제본을 생성합니다.  
유효한 값은 `StorageClass`Amazon S3 API Reference*의 PUT Bucket replication action 요소를 * 참조하십시오.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `DEEP_ARCHIVE | GLACIER | GLACIER_IR | INTELLIGENT_TIERING | ONEZONE_IA | OUTPOSTS | REDUCED_REDUNDANCY | SNOW | STANDARD | STANDARD_IA`  
_업데이트 필요 사항_: 중단 없음
