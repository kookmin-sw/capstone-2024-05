# AWS::D ynamoDB::GlobalTable WriteProvisionedThroughputSettings

쓰기 용량에 대한 Auto Scaling 정책을 지정합니다. 이 정책은 모든 복제본에 적용됩니다. 가 로 설정된 경우 이 설정을 지정해야 합니다 `BillingMode` `PROVISIONED`.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "WriteCapacityAutoScalingSettings" : CapacityAutoScalingSettings
}
```

## 속성

`WriteCapacityAutoScalingSettings`  
복제본 테이블 또는 global secondary index에 대한 Auto Scaling 설정을 지정합니다.  
*필수 항목 *여부: 아니요  
_형식_: CapacityAutoScalingSettings  
_업데이트 필요 사항_: 중단 없음
