# AWS::EC2::고객 게이트웨이

고객 게이트웨이를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::CustomerGateway",
  "Properties" : {
      "BgpAsn" : Integer,
      "DeviceName" : String,
      "IpAddress" : String,
      "Tags" : [Tag, ... ],
      "Type" : String
    }
}
```

## 속성

`BgpAsn`  
BGP를 지원하는 디바이스의 경우 고객 게이트웨이의 BGP ASN입니다.  
기본값: 65000  
_필수 항목 여부_: 예  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음

`DeviceName`  
고객 게이트웨이 디바이스의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`IpAddress`  
 고객 게이트웨이 디바이스의 외부 인터페이스에 대한 IPv4 주소입니다. 주소는 정적이어야 합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
고객 게이트웨이에 대한 하나 이상의 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

`Type`  
이 고객 게이트웨이가 지원하는 VPN 연결 유형(`ipsec.1`).  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `ipsec.1`  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`고객 게이트웨이의 ID가 반환됩니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`CustomerGatewayId` 고객 게이트웨이의 ID입니다.
