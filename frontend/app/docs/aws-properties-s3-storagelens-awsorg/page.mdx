# AWS::S3::StorageLens AwsOrg

이 리소스에는 AWS Organization for Amazon S3 Storage Lens의 세부 정보가 포함되어 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Arn" : String
}
```

## 속성

`Arn`  
이 리소스에는 AWS Organization의 ARN이 포함되어 있습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
