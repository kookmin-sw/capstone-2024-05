# AWS::EC2::인스턴스 네트워크 인터페이스

인스턴스에 연결할 네트워크 인터페이스를 지정합니다.

인스턴스를 시작할 때 네트워크 인터페이스를 생성할 수 있습니다. 예제는 AWS::EC2::Instance 예제를 참조하세요.

또는 인스턴스를 시작할 때 기존 네트워크 인터페이스를 연결할 수 있습니다. 예제는 AWS::EC2:NetworkInterface 예제를 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AssociateCarrierIpAddress" : Boolean,
  "AssociatePublicIpAddress" : Boolean,
  "DeleteOnTermination" : Boolean,
  "Description" : String,
  "DeviceIndex" : String,
  "GroupSet" : [ String, ... ],
  "Ipv6AddressCount" : Integer,
  "Ipv6Addresses" : [ InstanceIpv6Address, ... ],
  "NetworkInterfaceId" : String,
  "PrivateIpAddress" : String,
  "PrivateIpAddresses" : [ PrivateIpAddressSpecification, ... ],
  "SecondaryPrivateIpAddressCount" : Integer,
  "SubnetId" : String
}
```

## 속성

`AssociateCarrierIpAddress`  
네트워크 인터페이스에 통신 사업자 IP 주소를 할당할지 여부를 나타냅니다.  
Wavelength Zone의 서브넷에 있는 네트워크 인터페이스에만 통신사 IP 주소를 할당할 수 있습니다. 통신 사업자 IP 주소에 대한 자세한 내용은 * AWS Wavelength 개발자 안내서*의 통신 사업자 IP 주소를 참조하세요.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`AssociatePublicIpAddress`  
인스턴스에 퍼블릭 IPv4 주소를 할당할지 여부를 나타냅니다. 인스턴스를 시작할 때 네트워크 인터페이스를 생성하는 경우에만 적용됩니다. 네트워크 인터페이스는 기본 네트워크 인터페이스여야 합니다. 기본 서브넷으로 시작하는 경우 기본값은 `true`입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`DeleteOnTermination`  
인스턴스가 종료될 때 네트워크 인터페이스가 삭제되는지 여부를 나타냅니다. 인스턴스를 시작할 때 네트워크 인터페이스를 생성하는 경우에만 적용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`Description`  
네트워크 인터페이스에 대한 설명입니다. 인스턴스를 시작할 때 네트워크 인터페이스를 생성하는 경우에만 적용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`DeviceIndex`  
연결 순서에서 네트워크 인터페이스의 위치입니다. 기본 네트워크 인터페이스의 장치 인덱스는 0입니다.  
인스턴스를 시작할 때 네트워크 인터페이스를 생성하는 경우 디바이스 인덱스를 지정해야 합니다.  
_필수_: 조건부  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`GroupSet`  
네트워크 인터페이스에 대한 보안 그룹의 ID입니다. 인스턴스를 시작할 때 네트워크 인터페이스를 생성하는 경우에만 적용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Ipv6AddressCount`  
네트워크 인터페이스에 할당할 IPv6 주소의 수입니다. Amazon EC2는 서브넷 범위에서 IPv6 주소를 선택합니다. 이 옵션과 동일한 요청에서 특정 IPv6 주소를 할당하는 옵션을 지정할 수 없습니다. 시작할 최소 인스턴스 수를 지정한 경우 이 옵션을 지정할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Ipv6Addresses`  
네트워크 인터페이스에 할당할 IPv6 주소입니다. 이 옵션과 동일한 요청에서 여러 IPv6 주소를 할당하는 옵션을 지정할 수 없습니다. 시작할 최소 인스턴스 수를 지정한 경우 이 옵션을 지정할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: InstanceIpv6Address 목록  
_업데이트 필요 사항_: 중단 없음

`NetworkInterfaceId`  
기존 네트워크 인터페이스를 연결할 때 네트워크 인터페이스의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PrivateIpAddress`  
네트워크 인터페이스의 개인 IPv4 주소입니다. 인스턴스를 시작할 때 네트워크 인터페이스를 생성하는 경우에만 적용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PrivateIpAddresses`  
네트워크 인터페이스에 할당할 하나 이상의 개인 IPv4 주소입니다. 하나의 프라이빗 IPv4 주소만 기본으로 지정할 수 있습니다.  
*필수 항목 *여부: 아니요  
_형식_: PrivateIpAddressSpecification 목록  
_업데이트 필요 사항_: 중단 없음

`SecondaryPrivateIpAddressCount`  
보조 프라이빗 IPv4 주소의 수입니다. 이 옵션을 지정할 수 없으며 개인 IP 주소 옵션을 사용하여 둘 이상의 개인 IP 주소를 지정할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`SubnetId`  
네트워크 인터페이스와 연결된 서브넷의 ID입니다. 인스턴스를 시작할 때 네트워크 인터페이스를 생성하는 경우에만 적용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
