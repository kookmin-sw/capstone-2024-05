# AWS::S3Object람다::액세스포인트

리소스는 `AWS::S3ObjectLambda::AccessPoint` 버킷에 액세스하는 데 사용되는 객체 Lambda 액세스 포인트를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::S3ObjectLambda::AccessPoint",
  "Properties" : {
      "Name" : String,
      "ObjectLambdaConfiguration" : ObjectLambdaConfiguration
    }
}
```

## 속성

`Name`  
이 액세스 포인트의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`ObjectLambdaConfiguration`  
객체 Lambda 액세스 포인트를 생성할 때 사용되는 구성입니다.  
_필수 항목 여부_: 예  
_유형_: ObjectLambdaConfiguration  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Alias` 객체 Lambda 액세스 포인트의 별칭입니다.

`Alias.Status`  
객체 Lambda 액세스 포인트 별칭의 상태입니다. 유효한 값 : `PROVISIONING` | `READY`을 클릭합니다.

`Alias.Value`  
객체 Lambda 액세스 포인트의 별칭 이름 값입니다. 예를 들면 다음과 같습니다 `myolap-1a4n8yjrb3kda96f67zwrwiiuse1a--ol-s3`.

`Arn` 객체 Lambda 액세스 포인트의 ARN을 지정합니다.

`CreationDate` 지정된 객체 Lambda 액세스 포인트가 생성된 날짜 및 시간입니다.

`PolicyStatus` 속성 설명을 사용할 수 없습니다.

`PolicyStatus.IsPublic` 속성 설명을 사용할 수 없습니다.

`PublicAccessBlockConfiguration` 속성 설명을 사용할 수 없습니다.

`PublicAccessBlockConfiguration.BlockPublicAcls` 속성 설명을 사용할 수 없습니다.

`PublicAccessBlockConfiguration.BlockPublicPolicy` 속성 설명을 사용할 수 없습니다.

`PublicAccessBlockConfiguration.IgnorePublicAcls` 속성 설명을 사용할 수 없습니다.

`PublicAccessBlockConfiguration.RestrictPublicBuckets` 속성 설명을 사용할 수 없습니다.
