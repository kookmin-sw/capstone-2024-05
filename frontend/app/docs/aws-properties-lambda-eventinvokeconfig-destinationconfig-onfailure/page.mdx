# AWS::Lambda::EventInvokeConfig OnFailure

처리에 실패한 이벤트의 대상입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Destination" : String
}
```

## 속성

`Destination`  
대상 리소스의 Amazon 리소스 이름(ARN)입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `0`  
_최대_: `350`  
_패턴_: `^$|arn:(aws[a-zA-Z0-9-]*):([a-zA-Z0-9-])+:([a-z]{2}(-gov)?-[a-z]+-d{1})?:(d{12})?:(.*)`  
_업데이트 필요 사항_: 중단 없음

## 예제

### On-Failure 대상 구성

실패한 비동기 호출의 레코드를 SQS 대기열로 전송하도록 함수를 구성합니다.
