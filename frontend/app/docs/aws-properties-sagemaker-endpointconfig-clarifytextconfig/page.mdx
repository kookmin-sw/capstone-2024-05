# AWS::SageMaker::EndpointConfig ClarifyTextConfig

`ClarifyTextConfig` 속성 유형은 속성 설명을 사용할 수 없음을 지정합니다. AWS::SageMaker::EndpointConfig의 경우.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Granularity" : String,
  "Language" : String
}
```

## 속성

`Granularity`  
속성 설명을 사용할 수 없습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Language`  
속성 설명을 사용할 수 없습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
