# AWS::KinesisAnalyticsV2::애플리케이션 SqlApplicationConfiguration

SQL 기반 Managed Service for Apache Flink 애플리케이션에 대한 입력, 출력 및 참조 데이터 소스에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Inputs" : [ Input, ... ]
}
```

## 속성

`Inputs`  
응용 프로그램에서 사용하는 입력 스트림을 설명하는 Input 개체의 배열입니다.  
*필수 항목 *여부: 아니요  
_유형_: 입력 목록  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

- Amazon Kinesis Data Analytics API 참조\* \*의 SqlApplicationConfiguration
