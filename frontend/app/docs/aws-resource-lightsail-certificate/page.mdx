# AWS::Lightsail::인증서

리소스는 `AWS::Lightsail::Certificate` 콘텐츠 전송 네트워크(CDN) 배포 및 컨테이너 서비스와 함께 사용할 수 있는 SSL/TLS 인증서를 지정합니다.

**메모**  
로드 밸런서와 함께 사용할 수 있는 인증서에 대한 자세한 내용은 AWS::Lightsail::LoadBalancerTlsCertificate를 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Lightsail::Certificate",
  "Properties" : {
      "CertificateName" : String,
      "DomainName" : String,
      "SubjectAlternativeNames" : [ String, ... ],
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`CertificateName`  
인증서의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`DomainName`  
인증서의 도메인 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 업데이트가 지원되지 않습니다.

`SubjectAlternativeNames`  
인증서의 대체 도메인(예: `example.org`)과 하위 도메인(예: `blog.example.com` )을 지정하는 문자열 배열입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 업데이트가 지원되지 않습니다.

`Tags`  
이 리소스에 적용할 키-값 쌍의 배열입니다.  
자세한 내용은 _AWS CloudFormation 사용 설명서의 태그를 참조하세요_.  
`Value` of `Tags` 는 Lightsail 리소스의 선택 사항입니다.
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`이 `Ref`리소스에 대한 고유 식별자를 반환합니다.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`CertificateArn` 인증서의 Amazon 리소스 이름(ARN)입니다.

`Status` 인증서의 유효성 검사 상태입니다.
