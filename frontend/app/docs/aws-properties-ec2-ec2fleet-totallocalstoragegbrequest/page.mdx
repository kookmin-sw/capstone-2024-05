# AWS::EC2::EC2Fleet TotalLocalStorageGBRequest

총 로컬 저장소의 최소 및 최대 크기(GB)입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Max" : Double,
  "Min" : Double
}
```

## 속성

`Max`  
총 로컬 스토리지의 최대 크기(GB)입니다. 최대 제한을 지정하지 않으려면 이 매개 변수를 생략합니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 바꿔 놓음

`Min`  
총 로컬 저장소의 최소 크기(GB)입니다. 최소 제한을 지정하지 않으려면 이 매개 변수를 생략합니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 바꿔 놓음
