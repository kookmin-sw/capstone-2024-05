# AWS::람다::별칭 버전 가중치

Lambda 함수 별칭의 트래픽 이동 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "FunctionVersion" : String,
  "FunctionWeight" : Double
}
```

## 속성

`FunctionVersion`  
두 번째 버전의 한정자입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`FunctionWeight`  
별칭이 두 번째 버전으로 라우팅하는 트래픽의 백분율입니다.  
_필수 항목 여부_: 예  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음

## 예제

### 라우팅 구성

들어오는 요청의 절반을 두 번째 버전으로 라우팅하는 별칭입니다.
