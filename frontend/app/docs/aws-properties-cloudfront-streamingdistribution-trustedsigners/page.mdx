# AWS::CloudFront::StreamingDistribution 신뢰할 수 있는 서명자

CloudFront에서 서명된 URL 및 서명된 쿠키의 서명을 확인하는 데 퍼블릭 키를 사용할 수 있는 AWS 계정 목록입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AwsAccountNumbers" : [ String, ... ],
  "Enabled" : Boolean
}
```

## 속성

`AwsAccountNumbers`  
CloudFront에서 서명된 URL 및 서명된 쿠키의 서명을 확인하는 데 사용할 수 있는 활성 CloudFront 키 페어가 포함된 AWS 계정 번호입니다. 키 페어를 소유하는 AWS 계정이 CloudFront 배포를 소유하는 계정과 동일한 경우 이 필드의 값은 입니다`self`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Enabled`  
이 필드는 `true` 목록에 있는 AWS 계정 중 하나라도 신뢰할 수 있는 서명자로 구성된 경우입니다. 그렇지 않은 경우 이 필드는 `false`입니다.  
_필수 항목 여부_: 예  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
