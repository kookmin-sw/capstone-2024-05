# AWS::Rekognition::StreamProcessor 커넥티드 홈 설정

스트리밍 비디오에 사용할 커넥티드 홈 설정입니다. 에 대한 요청 매개 변수에서 설정을 정의해야 `CreateStreamProcessor` 합니다. CreateStreamProcessor 요청에 이 설정을 포함하면 커넥티드 홈 기능에 스트림 프로세서를 사용할 수 있습니다. 그런 다음 스트림 프로세서가 감지할 항목(예: 사람 또는 애완동물)을 선택할 수 있습니다.

스트림 프로세서가 시작되면 지정된 각 오브젝트 클래스에 대해 하나의 알림이 전송됩니다. 예를 들어, 패키지와 애완 동물을 선택한 경우 패키지가 처음 감지될 때 하나의 SNS 알림이 게시되고 애완 동물이 처음 감지될 때 하나의 SNS 알림이 게시됩니다. 세션 종료 요약도 게시됩니다. 자세한 내용은 StreamProcessorSettings의 ConnectedHome 섹션을 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Labels" : [ String, ... ],
  "MinConfidence" : Double
}
```

## 속성

`Labels`  
비디오에서 감지할 항목(예: 사람, 패키지 또는 애완 동물)을 지정합니다. 이 목록에 포함할 수 있는 현재 유효한 레이블은 "PERSON", "PET", "PACKAGE" 및 "ALL"입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 바꿔 놓음

`MinConfidence`  
비디오에서 개체에 레이블을 지정하는 데 필요한 최소 신뢰도입니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 바꿔 놓음
