# AWS::Cognito::UserPoolUser 속성 유형

특성이 표준인지 사용자 지정인지를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Value" : String
}
```

## 속성

`Name`  
특성의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `32`  
_패턴_: `[p{L}p{M}p{S}p{N}p{P}]+`  
_업데이트 필요 사항_: 중단 없음

`Value`  
특성의 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2048`  
_업데이트 필요 사항_: 중단 없음
