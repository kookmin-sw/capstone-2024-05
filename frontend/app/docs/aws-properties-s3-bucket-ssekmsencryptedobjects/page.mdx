# AWS::S3::버킷 SseKmsEncryptedObjects

AWS KMS로 암호화된 S3 객체 선택에 대한 필터 정보를 위한 컨테이너입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Status" : String
}
```

## 속성

`Status`  
Amazon S3가 AWS Key Management Service에 저장된 AWS KMS 키를 사용하여 서버 측 암호화로 생성된 객체를 복제할지 여부를 지정합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `Disabled | Enabled`  
_업데이트 필요 사항_: 중단 없음
