# AWS::EC2::인스턴스 ElasticInferenceAccelerator

인스턴스에 대한 Elastic Inference 액셀러레이터를 지정합니다.

`ElasticInferenceAccelerator` 은 AWS::EC2::Instance 리소스의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Count" : Integer,
  "Type" : String
}
```

## 속성

`Count`  
인스턴스에 연결할 탄력적 추론 액셀러레이터의 수입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Type`  
 탄력적 추론 가속기의 유형입니다. 가능한 값은 `eia1.medium`, `eia1.large`, `eia1.xlarge`, `eia2.medium`, `eia2.large`및 `eia2.xlarge`입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
