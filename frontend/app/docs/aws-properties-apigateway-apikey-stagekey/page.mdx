# AWS::ApiGateway::ApiKey 스테이지키

`StageKey` 은 API 키와 연결할 단계를 지정하는 AWS::ApiGateway::ApiKey 리소스의 속성입니다. 이 연결을 사용하면 키가 있는 클라이언트만 해당 단계의 메서드에 요청할 수 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "RestApiId" : String,
  "StageName" : String
}
```

## 속성

`RestApiId`  
연결된 RestApi의 문자열 식별자입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`StageName`  
스테이지 키와 연결된 스테이지 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
