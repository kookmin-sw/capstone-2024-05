# AWS::SageMaker::엔드포인트 구성 서버리스 구성

엔드포인트 변형에 대한 서버리스 구성을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "MaxConcurrency" : Integer,
  "MemorySizeInMB" : Integer,
  "ProvisionedConcurrency" : Integer
}
```

## 속성

`MaxConcurrency`  
서버리스 엔드포인트가 처리할 수 있는 최대 동시 호출 수입니다.  
_필수 항목 여부_: 예  
_유형_: Integer  
_최소_: `1`  
_최대_: `200`  
_업데이트 필요 사항_: 바꿔 놓음

`MemorySizeInMB`  
서버리스 엔드포인트의 메모리 크기입니다. 유효한 값은 1024MB, 2048MB, 3072MB, 4096MB, 5120MB 또는 6144MB와 같이 1GB 단위로 증가합니다.  
_필수 항목 여부_: 예  
_유형_: Integer  
_최소_: `1024`  
_최대_: `6144`  
_업데이트 필요 사항_: 바꿔 놓음

`ProvisionedConcurrency`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음
