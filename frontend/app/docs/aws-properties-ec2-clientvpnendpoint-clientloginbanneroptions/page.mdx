# AWS::EC2::ClientVpnEndpoint ClientLoginBanner옵션

VPN 세션이 설정될 때 AWS 제공 클라이언트에 표시되는 사용자 지정 가능한 텍스트 배너를 활성화하기 위한 옵션입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BannerText" : String,
  "Enabled" : Boolean
}
```

## 속성

`BannerText`  
VPN 세션이 설정될 때 AWS에서 클라이언트에 제공하는 배너에 표시되는 사용자 지정 가능한 텍스트입니다. UTF-8로 인코딩된 문자만 사용할 수 있습니다. 최대 1400자까지 입력할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Enabled`  
VPN 세션이 설정될 때 AWS 제공 클라이언트에 표시되는 사용자 지정 가능한 텍스트 배너를 활성화하거나 비활성화합니다.  
유효한 값: `true | false`  
기본값: `false`  
_필수_: 예  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
