# AWS::S3::버킷 지표 구성

Amazon S3 버킷의 CloudWatch 요청 지표(지표 구성 ID로 지정)에 대한 지표 구성을 지정합니다. 기존 메트릭 구성을 업데이트하는 경우 이는 기존 메트릭 구성을 완전히 대체한다는 점에 유의하세요. 유지하려는 요소를 포함하지 않으면 해당 요소가 지워집니다. 예제는 AWS::S3::Bucket 단원을 참조하십시오. 자세한 내용은 _Amazon S3 API Reference의 PUT Bucket 지표를 참조하세요_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AccessPointArn" : String,
  "Id" : String,
  "Prefix" : String,
  "TagFilters" : [ TagFilter, ... ]
}
```

## 속성

`AccessPointArn`  
개체에 대한 작업을 수행하는 동안 사용된 액세스 포인트입니다. 메트릭 구성에는 필터 기준을 충족하는 개체만 포함됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Id`  
메트릭 구성을 식별하는 데 사용되는 ID입니다. 메트릭 구성을 식별하는 데 도움이 되는 모든 값을 선택할 수 있습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Prefix`  
개체가 메트릭 결과에 포함되어야 하는 접두사입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`TagFilters`  
메트릭 구성 필터로 사용할 태그 필터 목록을 지정합니다. 메트릭 구성에는 필터 기준을 충족하는 개체만 포함됩니다.  
*필수 항목 *여부: 아니요  
_유형_: TagFilter 목록  
_업데이트 필요 사항_: 중단 없음
