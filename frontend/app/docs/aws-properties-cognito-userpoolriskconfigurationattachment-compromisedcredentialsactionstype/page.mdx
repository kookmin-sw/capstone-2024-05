# AWS::Cognito::UserPoolRiskConfigurationAttachment 손상된자격 증명작업유형

손상된 자격 증명 작업 유형입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "EventAction" : String
}
```

## 속성

`EventAction`  
이벤트 작업입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `BLOCK | NO_ACTION`  
_업데이트 필요 사항_: 중단 없음
