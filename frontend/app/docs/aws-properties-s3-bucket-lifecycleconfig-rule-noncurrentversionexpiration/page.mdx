# AWS::S3::버킷 NoncurrentVersionExpiration

최신이 아닌 객체 버전이 만료되는 시기를 지정합니다. 만료 시 Amazon S3는 최신이 아닌 객체 버전을 영구적으로 삭제합니다. 버전 관리가 활성화된(또는 일시 중지된) 버킷에서 이 수명 주기 구성 작업을 설정하여 Amazon S3가 객체 수명 주기의 특정 기간에 최신이 아닌 객체 버전을 삭제하도록 요청합니다. 수명 주기 규칙 구성 설정에 대한 자세한 내용은 AWS::S3::버킷 규칙 단원을 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "NewerNoncurrentVersions" : Integer,
  "NoncurrentDays" : Integer
}
```

## 속성

`NewerNoncurrentVersions`  
Amazon S3가 유지할 최신이 아닌 버전 수를 지정합니다. 최신이 아닌 버전이 이렇게 많으면 Amazon S3가 관련 작업을 수행합니다. 최신이 아닌 버전에 대한 자세한 내용은 _Amazon S3 사용 설명서의 수명 주기 구성 요소를 참조하세요_.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`NoncurrentDays`  
Amazon S3가 관련 작업을 수행할 수 있을 때까지 객체가 최신이 아닌 상태일 수를 지정합니다. 현재 날짜가 아닌 날짜 계산에 대한 자세한 내용은 Amazon S3 사용 설명서의 객체가 최신이 아닌 날짜가 되었을 때 Amazon S3에서 계산하는 방법을 참조하십시오\**.  
*필수 항목 여부*: 예  
*유형*: Integer  
*업데이트 필요 사항\*: 중단 없음
