# AWS::ElasticBeanstalk::환경 티어

AWS::ElasticBeanstalk::Environment 리소스의 환경 티어에 대해 설명합니다. 자세한 내용은 _AWS Elastic Beanstalk 개발자 안내서의 환경 티어를 참조하십시오_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Type" : String,
  "Version" : String
}
```

## 속성

`Name` 이 환경 계층의 이름입니다.  
유효한 값:

- 웹 서버 계층*의 경우 * – `WebServer`
- 작업자 계층의* 경우 * – `Worker`필수 항목 여부\*
  *: 아니요  
  *유형*: 문자열  
  *업데이트 필요 사항\*: 바꿔 놓음

`Type` 이 환경 계층의 유형입니다.  
유효한 값:

- 웹 서버 계층*의 경우 * – `Standard`
- 작업자 계층의* 경우 * – `SQS/HTTP`필수 항목 여부\*
  *: 아니요  
  *유형*: 문자열  
  *업데이트 필요 사항\*: 바꿔 놓음

`Version`  
이 환경 계층의 버전입니다. 값을 설정하지 않으면 Elastic Beanstalk는 호환되는 최신 작업자 티어 버전을 사용합니다.  
이 멤버는 더 이상 사용되지 않습니다. 설정한 특정 버전은 최신 버전이 될 수 있습니다. 지정하지 않은 상태로 두는 것이 좋습니다.
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
