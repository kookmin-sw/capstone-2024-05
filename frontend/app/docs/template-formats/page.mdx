# AWS CloudFormation 템플릿 형식

AWS CloudFormation 템플릿을 JSON 형식으로 작성할 수 있습니다. AWS는 AWS CloudFormation Designer를 포함하여 두 형식 모두에 대해 모든 AWS CloudFormation 기능을 지원합니다.

사용할 형식을 결정할 때 가장 편하게 작업할 수 있는 형식을 선택합니다.

Designer 외부에서 생성한 AWS CloudFormation 템플릿에 주석을 추가할 수 있습니다.

각 형식의 템플릿 구문에 대한 자세한 내용은 템플릿 분석을 참조하십시오.

AWS CloudFormation은 다음과 같은 JSON 사양을 지원합니다.

JSON (영문)  
AWS CloudFormation은 ECMA-404 JSON 표준을 따릅니다. JSON 형식에 대한 자세한 내용은 some-address를 참조하세요
