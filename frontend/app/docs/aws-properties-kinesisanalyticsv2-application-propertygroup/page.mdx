# AWS::KinesisAnalyticsV2::애플리케이션 속성 그룹

응용 프로그램에 전달된 속성 키-값 쌍입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "PropertyGroupId" : String,
  "PropertyMap" : {Key: Value, ...}
}
```

## 속성

`PropertyGroupId`  
응용 프로그램 실행 속성 키-값 쌍의 키를 설명합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `50`  
_패턴_: `[a-zA-Z0-9_.-]+`  
_업데이트 필요 사항_: 중단 없음

`PropertyMap`  
응용 프로그램 실행 속성 키-값 쌍의 값을 설명합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 맵  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

- Amazon Kinesis Data Analytics API 참조\* \*의 PropertyGroup
