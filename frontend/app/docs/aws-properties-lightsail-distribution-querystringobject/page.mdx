# AWS::Lightsail::D istribution QueryStringObject

`QueryStringObject` 은 CacheSettings 속성의 속성입니다. Amazon Lightsail 콘텐츠 전송 네트워크(CDN) 배포가 캐싱의 기반이 되는 쿼리 문자열 파라미터에 대해 설명합니다.

지정한 쿼리 문자열의 경우, 배포는 최종 사용자 요청의 쿼리 문자열 값에 따라 지정된 콘텐츠의 개별 버전을 캐시합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Option" : Boolean,
  "QueryStringsAllowList" : [ String, ... ]
}
```

## 속성

`Option`  
배포가 쿼리 문자열을 기반으로 전달 및 캐시되는지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`QueryStringsAllowList`  
배포가 오리진에 전달하는 특정 쿼리 문자열입니다.  
배포는 지정된 쿼리 문자열을 기반으로 콘텐츠를 캐싱합니다.  
`option` 파라미터가 true인 경우, 배포는 파라미터를 사용하여 무엇을 지정했는지에 관계없이 모든 쿼리 문자열을 전달합니다 `QueryStringsAllowList` .  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음
