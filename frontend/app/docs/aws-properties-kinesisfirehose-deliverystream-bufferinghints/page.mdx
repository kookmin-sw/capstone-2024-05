# AWS::KinesisFirehose::D eliveryStream 버퍼링힌트

`BufferingHints` 속성 유형은 Amazon Kinesis Data Firehose(Kinesis Data Firehose)가 수신 데이터를 대상으로 전송하기 전에 버퍼링하는 방법을 지정합니다. 첫 번째 버퍼 조건이 충족되면 Kinesis Data Firehose가 데이터를 전송하도록 트리거됩니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "IntervalInSeconds" : Integer,
  "SizeInMBs" : Integer
}
```

## 속성

`IntervalInSeconds`  
Kinesis Data Firehose가 수신 데이터를 대상으로 전송하기 전에 버퍼링하는 시간(초)입니다. 유효한 값은 `IntervalInSeconds`Amazon Kinesis Data Firehose API Reference*의 BufferingHints 데이터 형식에 대한 내용을 참조하십시오*.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `60`  
_최대_: `900`  
_업데이트 필요 사항_: 중단 없음

`SizeInMBs`  
Kinesis Data Firehose가 데이터를 대상으로 전송하기 전에 수신 데이터에 사용하는 버퍼의 크기(MB)입니다. 유효한 값은 `SizeInMBs`Amazon Kinesis Data Firehose API Reference*의 BufferingHints 데이터 형식에 대한 내용을 참조하십시오*.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `1`  
_최대_: `128`  
_업데이트 필요 사항_: 중단 없음
