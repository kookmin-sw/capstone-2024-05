# AWS::EC2::LaunchTemplate Ipv6추가

Amazon EC2 시작 템플릿에서 IPv6 주소를 지정합니다.

`Ipv6Add` 는 AWS::EC2::LaunchTemplate NetworkInterface의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Ipv6Address" : String
}
```

## 속성

`Ipv6Address`  
서브넷의 IPv6 CIDR 블록 범위에 있는 하나 이상의 특정 IPv6 주소. 여러 IPv6 주소를 지정하는 경우에는 이 옵션을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
