# AWS::AutoScalingPlans::ScalingPlan 태그 필터

`TagFilter` 는 AWS Auto Scaling(Auto Scaling 계획)과 함께 사용할 애플리케이션 소스의 태그를 지정하는 ApplicationSource의 하위 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Key" : String,
  "Values" : [ String, ... ]
}
```

## 속성

`Key`  
태그 키입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `128`  
_패턴_: `[u0020-uD7FFuE000-uFFFDuD800uDC00-uDBFFuDFFFrnt]*`  
_업데이트 필요 사항_: 중단 없음

`Values`  
태그 값(0 - 20)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음
