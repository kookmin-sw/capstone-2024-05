# AWS::VpcLattice::규칙 PathMatchType

경로 일치 유형을 설명합니다. 각 규칙에는 다음 경로 유형 중 하나만 포함될 수 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Exact" : String,
  "Prefix" : String
}
```

## 속성

`Exact`  
경로와 정확히 일치합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Prefix`  
경로의 접두사 일치입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
