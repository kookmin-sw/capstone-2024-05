# AWS::ECS::TaskDefinition 임시 스토리지

작업에 할당할 임시 저장소의 양입니다. 이 파라미터는 AWS Fargate에서 호스팅되는 태스크에 대해 사용 가능한 임시 스토리지의 총 용량을 기본 용량 이상으로 확장하는 데 사용됩니다. 자세한 내용은 *AWS Fargate용 Amazon ECS 사용 설명서 *의 Fargate 태스크 스토리지를 참조하세요.

**메모**  
Fargate 시작 유형을 사용하는 태스크의 경우 태스크에는 다음 플랫폼이 필요합니다.  
Linux 플랫폼 버전 `1.4.0` 이상.
Windows 플랫폼 버전 `1.0.0` 이상.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "SizeInGiB" : Integer
}
```

## 속성

`SizeInGiB`  
작업에 대해 설정할 임시 스토리지의 총 양(GiB)입니다. 지원되는 최소값은 `21` GiB이고 지원되는 최대 값은 `200` GiB입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음
