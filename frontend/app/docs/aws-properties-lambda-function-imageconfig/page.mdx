# AWS::Lambda::함수 이미지 구성

컨테이너 이미지 Dockerfile 설정을 재정의하는 구성 값입니다. 자세한 내용은 컨테이너 이미지 설정을 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Command" : [ String, ... ],
  "EntryPoint" : [ String, ... ],
  "WorkingDirectory" : String
}
```

## 속성

`Command`  
ENTRYPOINT를 사용하여 전달할 매개 변수를 지정합니다. 목록에서 최대 1,500개의 매개 변수를 지정할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_최대_: `1500`  
_업데이트 필요 사항_: 중단 없음

`EntryPoint`  
응용 프로그램에 대한 진입점을 지정하며, 이 진입점은 일반적으로 런타임 실행 파일의 위치입니다. 목록에 최대 1,500개의 문자열 항목을 지정할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_최대_: `1500`  
_업데이트 필요 사항_: 중단 없음

`WorkingDirectory`  
작업 디렉터리를 지정합니다. 디렉터리 문자열의 길이는 1,000자를 초과할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `1000`  
_업데이트 필요 사항_: 중단 없음
