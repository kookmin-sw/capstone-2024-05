# AWS::VpcLattice::대상 그룹

대상 그룹을 만듭니다. 대상 그룹은 응용 프로그램 또는 서비스를 실행하는 대상 또는 계산 리소스의 모음입니다. 대상 그룹은 단일 서비스에서만 사용할 수 있습니다.

자세한 내용은 Amazon VPC Lattice 사용 설명서*의 대상 그룹을 참조하세요*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::VpcLattice::TargetGroup",
  "Properties" : {
      "Config" : TargetGroupConfig,
      "Name" : String,
      "Tags" : [Tag, ... ],
      "Targets" : [ Target, ... ],
      "Type" : String
    }
}
```

## 속성

`Config`  
대상 그룹 구성입니다.  
*필수 항목 *여부: 아니요  
_유형_: TargetGroupConfig  
_업데이트 필요 사항_: 중단 없음

`Name`  
대상 그룹의 이름입니다. 이름은 계정 내에서 고유해야 합니다. 유효한 문자는 a-z, 0-9 및 하이픈(-)입니다. 하이픈은 첫 번째 또는 마지막 문자로 사용하거나 다른 하이픈 바로 뒤에 사용할 수 없습니다.  
이름을 지정하지 않으면 CloudFormation에서 이름을 생성합니다. 그러나 이름을 지정하고 나중에 자원을 바꾸려면 새 이름을 지정해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
대상 그룹에 대한 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

`Targets`  
대상을 설명합니다.  
*필수 항목 *여부: 아니요  
_유형_: 대상 목록  
_업데이트 필요 사항_: 중단 없음

`Type`  
대상 그룹의 유형입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`대상 `Ref`그룹의 Amazon 리소스 이름(ARN)을 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Arn` 대상 그룹의 Amazon 리소스 이름(ARN)입니다.

`CreatedAt` 대상 그룹이 생성된 날짜 및 시간으로, ISO-8601 형식으로 지정됩니다.

`Id` 대상 그룹의 ID입니다.

`LastUpdatedAt` 대상 그룹이 마지막으로 업데이트된 날짜 및 시간으로, ISO-8601 형식으로 지정됩니다.

`Status`  
작업의 상태입니다. 상태가 인 경우 작업을 다시 시도할 수 있습니다 `CREATE_FAILED`. 그러나 상태가 인 상태에서 다시 시도하면 `CREATE_IN_PROGRESS`상태가 변경되지 않습니다.
