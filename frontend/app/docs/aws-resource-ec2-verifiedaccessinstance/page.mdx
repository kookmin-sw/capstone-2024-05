# AWS::EC2::VerifiedAccessInstance (영문)

Verified Access 인스턴스에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::VerifiedAccessInstance",
  "Properties" : {
      "Description" : String,
      "LoggingConfigurations" : VerifiedAccessLogs,
      "Tags" : [Tag, ... ],
      "VerifiedAccessTrustProviderIds" : [ String, ... ],
      "VerifiedAccessTrustProviders" : [ VerifiedAccessTrustProvider, ... ]
    }
}
```

## 속성

`Description`  
AWS Verified Access 인스턴스에 대한 설명입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`LoggingConfigurations`  
Verified Access 인스턴스에 대한 현재 로깅 구성입니다.  
*필수 항목 *여부: 아니요  
_형식_: VerifiedAccessLogs  
_업데이트 필요 사항_: 중단 없음

`Tags`  
태그.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

`VerifiedAccessTrustProviderIds`  
AWS Verified Access 신뢰 공급자의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`VerifiedAccessTrustProviders`  
AWS Verified Access 신뢰 공급자의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: VerifiedAccessTrustProvider 목록  
_업데이트 필요 사항_: 중단 없음
