# AWS::CloudTrail::채널 대상

대상 수신 이벤트에 대한 정보를 포함합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Location" : String,
  "Type" : String
}
```

## 속성

`Location`  
 CloudTrail Lake 통합에 사용되는 채널의 경우 위치는 채널에서 이벤트를 수신하는 이벤트 데이터 스토어의 ARN입니다. 서비스 연결 채널의 경우 위치는 AWS 서비스의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `3`  
_최대_: `1024`  
_패턴_: `^[a-zA-Z0-9._/-:]+$`  
_업데이트 필요 사항_: 중단 없음

`Type`  
채널에서 도착하는 이벤트의 대상 유형입니다. CloudTrail Lake 통합에 사용되는 채널의 경우 값은 입니다 `EventDataStore`. 서비스 연결 채널의 경우 값은 입니다 `AWS_SERVICE`.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `AWS_SERVICE | EVENT_DATA_STORE`  
_업데이트 필요 사항_: 중단 없음
