# AWS::SageMaker::사용자 프로필 JupyterServerAppSettings

JupyterServer 앱 설정.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DefaultResourceSpec" : ResourceSpec
}
```

## 속성

`DefaultResourceSpec`  
JupyterServer 앱에서 사용하는 기본 SageMaker 이미지의 기본 인스턴스 유형 및 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: ResourceSpec  
_업데이트 필요 사항_: 중단 없음
