# AWS::KinesisAnalytics::애플리케이션 CSVMapping파라미터

레코드 형식에서 CSV와 같은 구분 기호를 사용하는 경우 추가 매핑 정보를 제공합니다. 예를 들어, 다음 샘플 레코드는 CSV 형식을 사용하며, 여기서 레코드는 행 구분 기호로 *'n'*을 사용하고 열 구분 기호로 쉼표(",")를 사용합니다.

`"name1", "address1"`

`"name2", "address2"`

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "RecordColumnDelimiter" : String,
  "RecordRowDelimiter" : String
}
```

## 속성

`RecordColumnDelimiter`  
열 구분 기호입니다. 예를 들어 CSV 형식에서 쉼표(",")는 일반적인 열 구분 기호입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_업데이트 필요 사항_: 중단 없음

`RecordRowDelimiter`  
행 구분 기호입니다. 예를 들어, CSV 형식에서 _'n'_ 은 일반적인 행 구분 기호입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_업데이트 필요 사항_: 중단 없음
