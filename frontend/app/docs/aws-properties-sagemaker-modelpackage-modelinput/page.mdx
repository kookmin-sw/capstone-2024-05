# AWS::SageMaker::ModelPackage 모델 입력

모델의 입력 개체입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DataInputConfig" : String
}
```

## 속성

`DataInputConfig`  
모델에 대한 입력 구성 개체입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `1024`  
_패턴_: `[Ss]+`  
_업데이트 필요 사항_: 일부 중단
