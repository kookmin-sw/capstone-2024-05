# AWS::SageMaker::엔드포인트 경보

엔드포인트의 지표를 모니터링하도록 구성된 Amazon CloudWatch 경보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AlarmName" : String
}
```

## 속성

`AlarmName`  
계정에 있는 CloudWatch 경보의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_패턴_: `^(?!s*$).+`  
_업데이트 필요 사항_: 중단 없음
