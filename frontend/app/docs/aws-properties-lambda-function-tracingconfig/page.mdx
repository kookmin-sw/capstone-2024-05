# AWS::Lambda::함수 추적 구성

함수의 AWS X-Ray 추적 구성입니다. 들어오는 요청을 샘플링하고 기록하려면 `Mode` `Active`로 설정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Mode" : String
}
```

## 속성

`Mode`  
추적 모드입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `Active | PassThrough`  
_업데이트 필요 사항_: 중단 없음
