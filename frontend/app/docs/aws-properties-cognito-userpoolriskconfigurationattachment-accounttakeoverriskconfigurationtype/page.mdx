# AWS::Cognito::UserPoolRiskConfigurationAttachment 계정인수 위험구성 유형

잠재적인 계정 탈취를 위해 감지된 다양한 수준의 위험에 대한 완화 조치 및 알림에 대한 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Actions" : AccountTakeoverActionsType,
  "NotifyConfiguration" : NotifyConfigurationType
}
```

## 속성

`Actions`  
계정 탈취 위험 구성 작업.  
_필수 항목 여부_: 예  
_유형_: AccountTakeoverActionsType  
_업데이트 필요 사항_: 중단 없음

`NotifyConfiguration`  
이메일 알림을 구성하는 데 사용되는 알림 구성입니다.  
*필수 항목 *여부: 아니요  
_형식_: NotifyConfigurationType  
_업데이트 필요 사항_: 중단 없음
