# AWS::ECR::P ullThroughCacheRule

풀스루 캐시 규칙을 만듭니다. 풀스루 캐시 규칙은 Amazon ECR 프라이빗 레지스트리의 외부 퍼블릭 레지스트리에서 이미지를 캐싱하는 방법을 제공합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::ECR::PullThroughCacheRule",
  "Properties" : {
      "EcrRepositoryPrefix" : String,
      "UpstreamRegistryUrl" : String
    }
}
```

## 속성

`EcrRepositoryPrefix`  
풀스루 캐시 규칙과 연결된 Amazon ECR 리포지토리 접두사입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `2`  
_최대_: `20`  
_패턴_: `[a-z0-9]+(?:[._-][a-z0-9]+)*`  
_업데이트 필요 사항_: 바꿔 놓음

`UpstreamRegistryUrl`  
풀스루 캐시 규칙과 연결된 업스트림 레지스트리 URL입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

## 예제

다음 리소스 예제에서는 개인 레지스트리에 대한 풀스루 캐시 규칙을 만드는 방법을 보여 줍니다.

### 개인 레지스트리에 대한 풀스루 캐시 규칙 만들기Create a pullthrough cache rule for a private registry

다음 예제에서는 Amazon ECR 퍼블릭 레지스트리에서 프라이빗 레지스트리로`my-ecr` 이름 접두사가 있는 리포지토리를 캐싱하는 풀스루 캐시 규칙을 생성합니다.

#### JSON (영문)

```
{
    "Resources": {
        "MyECRPullThroughCacheRule": {
            "Type": "AWS::ECR::PullThroughCacheRule",
            "Properties": {
                "EcrRepositoryPrefix": "my-ecr",
                "UpstreamRegistryUrl": "public.ecr.aws"
            }
        }
    }
}
```
