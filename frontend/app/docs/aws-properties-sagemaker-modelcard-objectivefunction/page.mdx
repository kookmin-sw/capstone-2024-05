# AWS::SageMaker::ModelCard 목표 함수

모델 학습 중에 최적화되는 함수입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Function" : Function,
  "Notes" : String
}
```

## 속성

`Function`  
최적화 방향, 메트릭 및 추가 설명을 자세히 설명하는 함수 개체입니다.  
*필수 항목 *여부: 아니요  
_유형_: 기능  
_업데이트 필요 사항_: 중단 없음

`Notes`  
가능한 목적 함수에 대한 다른 고려 사항을 포함하여 객체 함수에 대한 참고 사항.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
