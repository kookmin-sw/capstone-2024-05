# AWS::CloudWatch::MetricStream MetricStreamFilter

이 구조에는 지표 네임스페이스와 선택적으로 지표 스트림에 포함하거나 지표 스트림에서 제외할 지표 이름 목록이 포함되어 있습니다.

지표 스트림의 필터에는 최대 1,000개의 총 이름이 포함될 수 있습니다. 이 제한은 필터에 있는 네임스페이스 이름과 메트릭 이름의 합계에 적용됩니다. 예를 들어 여기에는 각각 99개의 지표가 있는 10개의 지표 네임스페이스 필터 또는 각 필터에 49개의 지표가 지정된 20개의 네임스페이스 필터가 포함될 수 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "MetricNames" : [ String, ... ],
  "Namespace" : String
}
```

## 속성

`MetricNames`  
지표 스트림에 포함하거나 제외할 지표의 이름입니다.  
이 매개 변수를 생략하면 이 필터가 제외 필터로 지정되었는지 또는 포함 필터로 지정되었는지에 따라 네임스페이스의 모든 메트릭이 포함되거나 제외됩니다.  
각 메트릭 이름에는 ASCII 인쇄 가능 문자(ASCII 범위 32 - 126)만 포함될 수 있습니다. 각 메트릭 이름에는 공백이 아닌 문자가 하나 이상 포함되어야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Namespace`  
필터에 있는 메트릭 네임스페이스의 이름입니다.  
네임스페이스는 ASCII 인쇄 가능 문자(ASCII 범위 32 - 126)만 포함할 수 있습니다. 공백이 아닌 문자를 하나 이상 포함해야 합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
