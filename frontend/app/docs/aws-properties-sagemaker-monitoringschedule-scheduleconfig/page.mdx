# AWS::SageMaker::MonitoringSchedule ScheduleConfig

모니터링 일정에 대한 구성 세부 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ScheduleExpression" : String
}
```

## 속성

`ScheduleExpression` 모니터링 일정에 대한 세부 정보를 설명하는 cron 표현식입니다.  
현재 유일하게 지원되는 cron 표현식은 다음과 같습니다.

- 매시간 작업이 시작되도록 설정하려면 다음을 사용하십시오.

  `Hourly: cron(0 * ? * * *)`

- 매일 작업을 시작하려는 경우:

  `cron(0 [00-23] ? * * *)`
  예를 들어, 다음은 유효한 cron 표현식입니다.

- 매일 정오(UTC) `cron(0 12 ? * * *)` :
- 매일 자정(UTC): `cron(0 0 ? * * *)`
  6시간, 12시간마다 실행을 지원하기 위해 다음도 지원됩니다.  
   `cron(0 [00-23]/[01-24] ? * * *)`  
  예를 들어, 다음은 유효한 cron 표현식입니다.
- 오후 5시(UTC)부터 12시간마다: `cron(0 17/12 ? * * *)`
- 자정부터 2시간마다: `cron(0 0/2 ? * * *)`
- cron 표현식이 오후 5시(UTC)에 시작하도록 설정되어 있더라도 실행을 실행하기 위해 요청된 실제 시간에서 0-20분 지연이 있을 수 있습니다.
- 일별 일정을 원하는 경우 이 매개 변수를 제공하지 않는 것이 좋습니다. Amazon SageMaker는 매일 실행할 시간을 선택합니다.
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _최소_: `1`  
  _최대_: `256`  
  _업데이트 필요 사항_: 중단 없음
