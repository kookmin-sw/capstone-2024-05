# AWS::ECS::TaskSet 확장

작업 집합에서 배치하고 계속 실행할 원하는 작업 수의 부동 소수점 백분율입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Unit" : String,
  "Value" : Double
}
```

## 속성

`Unit`  
배율 값의 측정 단위입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `PERCENT`  
_업데이트 필요 사항_: 중단 없음

`Value`  
작업 집합의 크기를 조정하기 위한 서비스 의 합계 백분율로 지정된 값`desiredCount`입니다. 허용되는 값은 0에서 100 사이의 숫자입니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음
