# AWS::CloudFront::Monitoring구독 MonitoringSubscription

모니터링 구독입니다. 이 구조에는 지정된 CloudFront 배포에 대해 추가 CloudWatch 지표가 활성화되었는지 여부에 대한 정보가 포함되어 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "RealtimeMetricsSubscriptionConfig" : RealtimeMetricsSubscriptionConfig
}
```

## 속성

`RealtimeMetricsSubscriptionConfig`  
추가 CloudWatch 지표에 대한 구독 구성입니다.  
*필수 항목 *여부: 아니요  
_유형_: RealtimeMetricsSubscriptionConfig  
_업데이트 필요 사항_: 중단 없음
