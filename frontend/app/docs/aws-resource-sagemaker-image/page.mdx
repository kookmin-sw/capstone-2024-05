# AWS::SageMaker::이미지

사용자 지정 SageMaker 이미지를 생성합니다. SageMaker 이미지는 이미지 버전의 집합입니다. 각 이미지 버전은 Amazon Elastic Container Registry(ECR)에 저장된 컨테이너 이미지를 나타냅니다. 자세한 정보는 자체 SageMaker 이미지 가져오기을 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::SageMaker::Image",
  "Properties" : {
      "ImageDescription" : String,
      "ImageDisplayName" : String,
      "ImageName" : String,
      "ImageRoleArn" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`ImageDescription`  
이미지에 대한 설명입니다.  
_길이 제약_: 최소 길이는 1입니다. 최대 길이는 512입니다.  
_패턴_: `.*`  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ImageDisplayName`  
이미지의 표시 이름입니다.  
_길이 제약_: 최소 길이는 1입니다. 최대 길이는 128입니다.  
_패턴_: `^S(.*S)?$`  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ImageName`  
이미지의 이름입니다. 계정에서 지역별로 고유해야 합니다.  
_길이 제약_: 최소 길이는 1입니다. 최대 길이는 63입니다.  
_패턴_: `^a-zA-Z0-9{0,62}$`  
*필수 항목 *여부: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`ImageRoleArn`  
Amazon SageMaker가 사용자를 대신하여 작업을 수행할 수 있도록 하는 IAM 역할의 Amazon 리소스 이름(ARN)입니다.  
_길이 제약_: 최소 길이는 20입니다. 최대 길이는 2048입니다.  
_패턴_: `^arn:aws[a-z-]*:iam::d{12}:role/?[a-zA-Z_0-9+=,.@-_/]+$`  
*필수 항목 *여부: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Tags`  
이 리소스에 적용할 키-값 쌍 목록입니다.  
_배열 멤버_: 최소 항목 수는 0개입니다. 최대 항목 수는 50개입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`ImageArn이 반환됩니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`ImageArn`  
이미지의 Amazon 리소스 이름(ARN)입니다.  
_유형_: 문자열  
_길이 제약_: 최대 길이는 256입니다.  
_패턴_: `^arn:aws(-[w]+)*:sagemaker:.+:[0-9]{12}:image/a-z0-9*$`
