# AWS::Lightsail::인스턴스 위치

`Location` 은 AWS::Lightsail::Instance 리소스의 속성입니다. 인스턴스의 위치를 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AvailabilityZone" : String,
  "RegionName" : String
}
```

## 속성

`AvailabilityZone`  
인스턴스의 가용 영역입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`RegionName`  
인스턴스에 대한 AWS 리전의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
