# AWS::S3::버킷 LambdaConfiguration

호출할 AWS Lambda 함수와 이를 호출할 이벤트에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Event" : String,
  "Filter" : NotificationFilter,
  "Function" : String
}
```

## 속성

`Event`  
AWS Lambda 함수를 호출할 Amazon S3 버킷 이벤트입니다. 자세한 내용은 _Amazon S3 사용 설명서의 지원되는 이벤트 유형을 참조하세요_.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Filter`  
AWS Lambda 함수를 호출하는 객체를 결정하는 필터링 규칙입니다. 예를 들어, 확장자가 있는 이미지 파일만 `.jpg` Amazon S3 버킷에 추가될 때 함수를 호출하도록 필터를 생성할 수 있습니다.  
*필수 항목 *여부: 아니요  
_형식_: NotificationFilter  
_업데이트 필요 사항_: 중단 없음

`Function`  
지정된 이벤트 유형이 발생할 때 Amazon S3가 호출하는 AWS Lambda 함수의 Amazon 리소스 이름(ARN)입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
