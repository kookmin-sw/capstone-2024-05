# AWS::SageMaker::EndpointConfig 캡처옵션

끝점이 입력 데이터를 캡처할지 또는 출력 데이터를 캡처할지를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CaptureMode" : String
}
```

## 속성

`CaptureMode`  
끝점이 입력 데이터를 캡처할지 또는 출력 데이터를 캡처할지를 지정합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `Input | Output`  
_업데이트 필요 사항_: 바꿔 놓음
