# AWS::KinesisFirehose::D eliveryStream CloudWatchLogging옵션

`CloudWatchLoggingOptions` 속성 유형은 Amazon Kinesis Data Firehose(Kinesis Data Firehose)가 전송 스트림에 사용하는 Amazon CloudWatch Logs(CloudWatch Logs) 로깅 옵션을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Enabled" : Boolean,
  "LogGroupName" : String,
  "LogStreamName" : String
}
```

## 속성

`Enabled`  
CloudWatch Logs 로깅이 활성화되었는지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`LogGroupName`  
Kinesis Data Firehose에서 사용할 로그 스트림이 포함된 CloudWatch Logs 로그 그룹의 이름입니다.  
조건부. 로깅을 사용하는 경우 이 특성을 지정해야 합니다.  
_필수_: 조건부  
_유형_: 문자열  
_최소_: `0`  
_최대_: `512`  
_패턴_: `[.-_/#A-Za-z0-9]*`  
_업데이트 필요 사항_: 중단 없음

`LogStreamName`  
Kinesis Data Firehose가 데이터 전송에 대한 로그를 전송하는 데 사용하는 CloudWatch Logs 로그 스트림의 이름입니다.  
조건부. 로깅을 사용하는 경우 이 특성을 지정해야 합니다.  
_필수_: 조건부  
_유형_: 문자열  
_최소_: `0`  
_최대_: `512`  
_패턴_: `[^:*]*`  
_업데이트 필요 사항_: 중단 없음
