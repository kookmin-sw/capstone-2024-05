# AWS::EC2::LaunchTemplate 모니터링

인스턴스에 대해 자세한 모니터링을 사용할 수 있는지 여부를 지정합니다. 세부 모니터링에 대한 자세한 내용은 Amazon EC2 사용 설명서의 인스턴스에 대한 세부 모니터링 활성화 또는 해제를 참조하세요\*\*.

`Monitoring` 은 AWS::EC2::LaunchTemplate LaunchTemplateData의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Enabled" : Boolean
}
```

## 속성

`Enabled`  
`true` 자세한 모니터링을 사용하도록 지정합니다. 그렇지 않으면 기본 모니터링이 활성화됩니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

- Amazon EC2 API 참조\* \*의 LaunchTemplatesMonitoringRequest
