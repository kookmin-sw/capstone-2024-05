# AWS::Cognito::UserPool 복구 옵션

우선 순위를 키로, 복구 메서드 이름을 값으로 포함하는 맵입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Priority" : Integer
}
```

## 속성

`Name`  
사용자의 복구 방법을 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `admin_only | verified_email | verified_phone_number`  
_업데이트 필요 사항_: 중단 없음

`Priority`  
메서드의 우선 순위를 지정하는 양의 정수로, 1이 가장 높은 우선 순위입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `1`  
_최대_: `2`  
_업데이트 필요 사항_: 중단 없음
