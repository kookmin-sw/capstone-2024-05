# AWS::ECS::태스크 정의 태스크 정의배치 제약 조건

작업 정의의 작업 배치에 대한 제약 조건입니다. 자세한 내용은 _Amazon Elastic Container Service 개발자 안내서의 작업 배치 제약 조건을 참조하세요_.

**메모**  
AWS Fargate에서 실행되는 태스크에는 태스크 배치 제약이 지원되지 않습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Expression" : String,
  "Type" : String
}
```

## 속성

`Expression`  
제약 조건에 적용할 클러스터 쿼리 언어 식입니다. 자세한 내용은 _Amazon Elastic Container Service 개발자 안내서의 클러스터 쿼리 언어를 참조하세요_.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Type`  
제약 조건의 유형입니다. `MemberOf` 제약 조건은 유효한 후보 그룹에서 선택하도록 제한합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `memberOf`  
_업데이트 필요 사항_: 바꿔 놓음
