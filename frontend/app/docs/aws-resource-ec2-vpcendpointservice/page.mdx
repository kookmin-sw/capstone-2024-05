# AWS::EC2::VPCEndpointService

서비스 소비자(AWS 계정, 사용자 및 IAM 역할)가 연결할 수 있는 VPC 엔드포인트 서비스 구성을 생성합니다.

끝점 서비스 구성을 만들려면 먼저 서비스에 대해 다음 중 하나를 만들어야 합니다.

- Network Load Balancer입니다. 서비스 소비자는 인터페이스 끝점을 사용하여 서비스에 연결합니다.
- Gateway Load Balancer입니다. 서비스 소비자는 Gateway Load Balancer 엔드포인트를 사용하여 서비스에 연결합니다.

자세한 내용은 AWS PrivateLink 사용 설명서를 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::VPCEndpointService",
  "Properties" : {
      "AcceptanceRequired" : Boolean,
      "ContributorInsightsEnabled" : Boolean,
      "GatewayLoadBalancerArns" : [ String, ... ],
      "NetworkLoadBalancerArns" : [ String, ... ],
      "PayerResponsibility" : String
    }
}
```

## 속성

`AcceptanceRequired`  
서비스에 대한 끝점을 만들기 위한 서비스 소비자의 요청을 수락해야 하는지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`ContributorInsightsEnabled`  
AWS PrivateLink에서 제공하는 기본 제공 Contributor Insights 규칙을 활성화할지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`GatewayLoadBalancerArns`  
Gateway Load Balancer의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`NetworkLoadBalancerArns`  
Network Load Balancer의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`PayerResponsibility`  
엔드포인트 비용을 담당하는 엔터티입니다. 기본값은 끝점 소유자입니다. 지급인 책임을 서비스 소유자로 설정하면 엔드포인트 소유자에게 다시 설정할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `ServiceOwner`  
_업데이트 필요 사항_: 중단 없음
