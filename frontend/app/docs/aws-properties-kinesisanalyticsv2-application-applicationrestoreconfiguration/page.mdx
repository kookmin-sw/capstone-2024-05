# AWS::KinesisAnalyticsV2::애플리케이션 애플리케이션복원구성

이전에 저장된 응용 프로그램 상태를 사용하여 응용 프로그램을 다시 시작할 때 사용할 메서드와 스냅숏을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ApplicationRestoreType" : String,
  "SnapshotName" : String
}
```

## 속성

`ApplicationRestoreType`  
응용 프로그램을 복원하는 방법을 지정합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `RESTORE_FROM_CUSTOM_SNAPSHOT | RESTORE_FROM_LATEST_SNAPSHOT | SKIP_RESTORE_FROM_SNAPSHOT`  
_업데이트 필요 사항_: 중단 없음

`SnapshotName`  
응용 프로그램을 다시 시작하는 데 사용할 응용 프로그램 상태의 기존 스냅숏 식별자입니다. 응용 프로그램은 에 대해 지정된 경우 이 값을 사용합니다 `RESTORE_FROM_CUSTOM_SNAPSHOT` `ApplicationRestoreType`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `256`  
_패턴_: `[a-zA-Z0-9_.-]+`  
_업데이트 필요 사항_: 중단 없음
