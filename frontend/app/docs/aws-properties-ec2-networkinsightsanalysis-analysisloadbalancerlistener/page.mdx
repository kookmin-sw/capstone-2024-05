# AWS::EC2::NetworkInsights분석 분석LoadBalancer리스너

부하 분산 장치 수신기에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "InstancePort" : Integer,
  "LoadBalancerPort" : Integer
}
```

## 속성

`InstancePort`  
[Classic Load Balancer] 수신기의 백 엔드 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `0`  
_최대_: `65535`  
_업데이트 필요 사항_: 중단 없음

`LoadBalancerPort`  
부하 분산 장치가 수신 대기 중인 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `0`  
_최대_: `65535`  
_업데이트 필요 사항_: 중단 없음
