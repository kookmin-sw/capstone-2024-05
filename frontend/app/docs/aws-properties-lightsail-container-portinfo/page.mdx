# AWS::Lightsail::컨테이너 PortInfo

`PortInfo` 는 Container 속성의 속성입니다. Amazon Lightsail 컨테이너 서비스의 컨테이너에 사용할 포트와 프로토콜에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Port" : String,
  "Protocol" : String
}
```

## 속성

`Port`  
컨테이너의 열린 방화벽 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Protocol`  
열린 포트의 프로토콜 이름입니다.  
_허용되는 값_: `HTTP` | `HTTPS` | `TCP` | `UDP`  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
