# AWS::CloudFront::OriginRequestPolicy 오리진요청정책

오리진 요청 정책 구성입니다.

이 구성은 CloudFront가 오리진에 보내는 요청에 포함하는 값을 결정합니다. CloudFront에서 오리진에 보내는 각 요청에는 다음이 포함됩니다.

- 최종 사용자 요청의 요청 본문 및 URL 경로(도메인 이름 제외)입니다.
- CloudFront가 모든 오리진 요청에 자동으로 포함하는 헤더(예: `Host`, `User-Agent`및 `X-Amz-Cf-Id`.
- 캐시 정책 또는 오리진 요청 정책에 지정된 모든 HTTP 헤더, 쿠키 및 URL 쿼리 문자열입니다. 여기에는 최종 사용자 요청의 항목이 포함될 수 있으며, 헤더의 경우 CloudFront에서 추가한 추가 항목이 포함될 수 있습니다.

CloudFront는 캐시에서 요청과 일치하는 객체를 찾을 수 없을 때 요청을 보냅니다. 오리진에 값을 전송하고 캐시 키에도 포함 하려면 `CachePolicy`.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Comment" : String,
  "CookiesConfig" : CookiesConfig,
  "HeadersConfig" : HeadersConfig,
  "Name" : String,
  "QueryStringsConfig" : QueryStringsConfig
}
```

## 속성

`Comment`  
오리진 요청 정책을 설명하는 주석입니다. 주석은 128자를 초과할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`CookiesConfig`  
오리진 요청에 포함할 최종 사용자 요청의 쿠키입니다.  
_필수 항목 여부_: 예  
_유형_: CookiesConfig  
_업데이트 필요 사항_: 중단 없음

`HeadersConfig`  
오리진 요청에 포함할 HTTP 헤더입니다. 여기에는 최종 사용자 요청의 헤더와 CloudFront에서 추가한 추가 헤더가 포함될 수 있습니다.  
_필수 항목 여부_: 예  
_유형_: HeadersConfig  
_업데이트 필요 사항_: 중단 없음

`Name`  
오리진 요청 정책을 식별하기 위한 고유한 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`QueryStringsConfig`  
오리진 요청에 포함할 최종 사용자 요청의 URL 쿼리 문자열입니다.  
_필수 항목 여부_: 예  
_형식_: QueryStringsConfig  
_업데이트 필요 사항_: 중단 없음
