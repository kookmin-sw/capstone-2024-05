# AWS::EC2::EC2Fleet FleetLaunchTemplateSpecificationRequest

Amazon EC2 인스턴스를 구성하기 위해 EC2 집합에서 사용할 시작 템플릿을 지정합니다.

다음을 지정해야 합니다.

- 시작 템플릿의 ID 또는 이름이지만 둘 다 아님.
- 시작 템플릿의 버전입니다.

`FleetLaunchTemplateSpecificationRequest` 은 FleetLaunchTemplateConfigRequest 속성 유형의 속성입니다.

시작 템플릿 생성에 대한 자세한 내용은 Amazon EC2 사용 설명서*의 AWS::EC2::LaunchTemplate 및 시작 템플릿 생성을 참조하십시오*.

시작 템플릿의 예는 예제를 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "LaunchTemplateId" : String,
  "LaunchTemplateName" : String,
  "Version" : String
}
```

## 속성

`LaunchTemplateId`  
시작 템플릿의 ID입니다.  
또는 를 지정해야 하지만 `LaunchTemplateId` `LaunchTemplateName`둘 다 지정해서는 안 됩니다.  
_필수_: 조건부  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`LaunchTemplateName`  
시작 템플릿의 이름입니다.  
또는 를 지정해야 하지만 `LaunchTemplateName` `LaunchTemplateId`둘 다 지정해서는 안 됩니다.  
_필수_: 조건부  
_유형_: 문자열  
_최소_: `3`  
_최대_: `128`  
_패턴_: `[a-zA-Z0-9().-/_]+`  
_업데이트 필요 사항_: 바꿔 놓음

`Version`  
시작 템플릿 버전 번호, `$Latest`, 또는 `$Default`. 값을 지정해야 하며, 그렇지 않으면 요청이 실패합니다.  
값이 인 경우 `$Latest`Amazon EC2는 최신 버전의 시작 템플릿을 사용합니다.  
값이 인 경우 `$Default`Amazon EC2는 시작 템플릿의 기본 버전을 사용합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
