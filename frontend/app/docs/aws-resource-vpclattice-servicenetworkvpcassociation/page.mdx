# AWS::VpcLattice::ServiceNetworkVpcAssociation

VPC를 서비스 네트워크와 연결합니다. VPC를 서비스 네트워크와 연결하면 해당 VPC 내의 모든 리소스가 클라이언트가 되어 서비스 네트워크의 다른 서비스와 통신할 수 있습니다. 자세한 내용은 _Amazon VPC Lattice 사용 설명서의 VPC 연결 관리를 참조하세요_.

연결 해제가 진행 중인 경우 이 작업을 사용할 수 없습니다. 연결이 실패하면 연결을 삭제하고 다시 만들어 다시 시도합니다.

이 작업의 결과로 서비스 네트워크 계정 및 VPC 소유자 계정에 연결이 생성됩니다.

서비스 네트워크 및 VPC 연결에 보안 그룹을 추가하는 경우 연결에는 항상 하나 이상의 보안 그룹이 있어야 합니다. 언제든지 보안 그룹을 추가하거나 편집할 수 있습니다. 그러나 모든 보안 그룹을 제거하려면 먼저 연결을 삭제하고 보안 그룹 없이 다시 만들어야 합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::VpcLattice::ServiceNetworkVpcAssociation",
  "Properties" : {
      "SecurityGroupIds" : [ String, ... ],
      "ServiceNetworkIdentifier" : String,
      "Tags" : [Tag, ... ],
      "VpcIdentifier" : String
    }
}
```

## 속성

`SecurityGroupIds`  
보안 그룹의 ID입니다. 보안 그룹은 기본적으로 추가되지 않습니다. 보안 그룹을 추가하여 네트워크 수준 제어를 적용하여 VPC에서 서비스 네트워크 및 해당 서비스에 액세스할 수 있는 리소스를 제어할 수 있습니다. 자세한 내용은 Amazon VPC 사용 설명서의 보안 그룹을 사용하여 리소스에 대한 트래픽 제어를 참조하세요\**.  
*필수 항목 *여부: 아니요  
*유형*: 문자열 목록  
*업데이트 필요 사항\*: 중단 없음

`ServiceNetworkIdentifier`  
서비스 네트워크의 ID 또는 Amazon 리소스 이름(ARN)입니다. 작업에 지정된 리소스가 서로 다른 계정에 있는 경우 ARN을 사용해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
연결에 대한 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

`VpcIdentifier`  
VPC의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`연결 `Ref`의 Amazon 리소스 이름(ARN)을 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Arn` 서비스 네트워크와 VPC 간 연결의 Amazon 리소스 이름(ARN)입니다.

`CreatedAt` 연결이 생성된 날짜 및 시간으로, ISO-8601 형식으로 지정됩니다.

`Id` 서비스 네트워크와 VPC 간의 지정된 연결 ID입니다.

`ServiceNetworkArn` 서비스 네트워크의 Amazon 리소스 이름(ARN)입니다.

`ServiceNetworkId` 서비스 네트워크의 ID입니다.

`ServiceNetworkName` 서비스 네트워크의 이름입니다.

`Status` 연결의 상태입니다.

`VpcId` VPC의 ID입니다.
