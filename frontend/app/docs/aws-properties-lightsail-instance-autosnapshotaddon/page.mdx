# AWS::Lightsail::인스턴스 AutoSnapshotAddOn

`AutoSnapshotAddOn` 는 AddOn 속성의 속성입니다. 인스턴스에 대한 자동 스냅샷 추가 기능에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "SnapshotTimeOfDay" : String
}
```

## 속성

`SnapshotTimeOfDay` 자동 스냅샷이 생성되는 일일 시간입니다.  
제약 조건:

- 형식이어야 하며 `HH:00` 시간 단위로 증가해야 합니다.
- UTC(협정 세계시)로 지정됩니다.
- 스냅샷은 지정된 시간과 최대 45분 후 사이에 자동으로 생성됩니다.
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _패턴_: `^(0[0-9]|1[0-9]|2[0-3]):[0-5][0-9]$`  
  _업데이트 필요 사항_: 중단 없음
