# AWS::백업 게이트웨이::하이퍼바이저

게이트웨이가 연결할 하이퍼바이저의 사용 권한을 나타냅니다.

하이퍼바이저는 가상 머신을 생성 및 관리하고 리소스를 할당하는 하드웨어, 소프트웨어 또는 펌웨어입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::BackupGateway::Hypervisor",
  "Properties" : {
      "Host" : String,
      "KmsKeyArn" : String,
      "LogGroupArn" : String,
      "Name" : String,
      "Password" : String,
      "Tags" : [Tag, ... ],
      "Username" : String
    }
}
```

## 속성

`Host`  
하이퍼바이저의 서버 호스트입니다. IP 주소 또는 FQDN(정규화된 도메인 이름)일 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `3`  
_최대_: `128`  
_패턴_: `.+`  
_업데이트 필요 사항_: 중단 없음

`KmsKeyArn`  
하이퍼바이저를 암호화하는 데 사용되는 AWS Key Management Service의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `50`  
_최대_: `500`  
_패턴_: `(^arn:(aws|aws-cn|aws-us-gov):kms:([a-zA-Z0-9-]+):([0-9]+):(key|alias)/(S+)$)|(^alias/(S+)$)`  
_업데이트 필요 사항_: 바꿔 놓음

`LogGroupArn`  
요청된 로그 내에 있는 게이트웨이 그룹의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `2048`  
_패턴_: `$|^arn:(aws|aws-cn|aws-us-gov):logs:([a-zA-Z0-9-]+):([0-9]+):log-group:[a-zA-Z0-9_-/.]+:*`  
_업데이트 필요 사항_: 중단 없음

`Name`  
하이퍼바이저의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `100`  
_패턴_: `[a-zA-Z0-9-]*`  
_업데이트 필요 사항_: 중단 없음

`Password`  
하이퍼바이저의 암호입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `100`  
_패턴_: `[ -~]+`  
_업데이트 필요 사항_: 중단 없음

`Tags`  
가져올 하이퍼바이저 구성의 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 바꿔 놓음

`Username`  
하이퍼바이저의 사용자 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `100`  
_패턴_: `[ -.0-[]-~]*[!-.0-[]-~][ -.0-[]-~]*`  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달 `Ref` 하면 `Ref`가 반환`HypervisorArn`됩니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`HypervisorArn`  
반환 `HypervisorArn`, 하이퍼바이저를 고유하게 식별하는 Amazon 리소스 이름(ARN)입니다. 예를 들어: `arn:aws:backup-gateway:us-east-1:123456789012:hypervisor/hype-1234D67D`
