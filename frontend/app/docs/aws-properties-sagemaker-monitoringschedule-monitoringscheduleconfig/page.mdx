# AWS::SageMaker::MonitoringSchedule MonitoringScheduleConfig

모니터링 일정을 구성하고 모니터링 작업을 정의합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "MonitoringJobDefinition" : MonitoringJobDefinition,
  "MonitoringJobDefinitionName" : String,
  "MonitoringType" : String,
  "ScheduleConfig" : ScheduleConfig
}
```

## 속성

`MonitoringJobDefinition`  
모니터링 작업을 정의합니다.  
*필수 항목 *여부: 아니요  
_형식_: MonitoringJobDefinition  
_업데이트 필요 사항_: 중단 없음

`MonitoringJobDefinitionName`  
스케줄링할 모니터링 작업 정의의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9*$`  
_업데이트 필요 사항_: 중단 없음

`MonitoringType`  
예약할 모니터링 작업 정의의 유형입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `DataQuality | ModelBias | ModelExplainability | ModelQuality`  
_업데이트 필요 사항_: 중단 없음

`ScheduleConfig`  
모니터링 일정을 구성합니다.  
*필수 항목 *여부: 아니요  
_형식_: ScheduleConfig  
_업데이트 필요 사항_: 중단 없음
