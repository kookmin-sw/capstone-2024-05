# AWS::Lightsail::D istribution CacheBehavior

`CacheBehavior` 은 AWS::Lightsail::D istribution 리소스의 속성입니다. Amazon Lightsail 콘텐츠 전송 네트워크(CDN) 배포의 기본 캐시 동작에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Behavior" : String
}
```

## 속성

`Behavior` 배포의 캐시 동작입니다.  
다음 캐시 동작을 지정할 수 있습니다.

- ** `cache` ** - 이 옵션은 정적 사이트에 가장 적합합니다. 지정된 경우 배포는 전체 웹 사이트를 정적 콘텐츠로 캐시하고 제공합니다. 이 동작은 보는 사람에 따라 변경되지 않는 정적 콘텐츠가 있는 웹 사이트 또는 쿠키를 사용하지 않는 웹 사이트에 적합합니다.
- ** `dont-cache` ** - 이 옵션은 정적 콘텐츠와 동적 콘텐츠를 혼합하여 제공하는 사이트에 가장 적합합니다. 지정된 경우, 배포는 배포의 파라미터에 지정된 콘텐츠만 캐싱하고 제공합니다 `CacheBehaviorPerPath` . 이 동작은 쿠키, 헤더 및 쿼리 문자열을 사용하여 개별 사용자에 대한 콘텐츠를 개인화하는 웹 사이트 또는 웹 애플리케이션에 적합합니다.
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _허용되는 값_: `cache | dont-cache`  
  _업데이트 필요 사항_: 중단 없음
