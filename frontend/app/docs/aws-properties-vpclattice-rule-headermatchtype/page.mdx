# AWS::VpcLattice::규칙 헤더 매치 유형

헤더 일치 유형을 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Contains" : String,
  "Exact" : String,
  "Prefix" : String
}
```

## 속성

`Contains`  
A에는 형식 일치가 포함되어 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Exact`  
정확한 형식 일치입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Prefix`  
접두사 유형 일치입니다. 값을 접두사와 일치시킵니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
