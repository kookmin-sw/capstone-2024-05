# AWS::VpcLattice::ServiceNetworkServiceAssociation DnsEntry

DNS 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DomainName" : String,
  "HostedZoneId" : String
}
```

## 속성

`DomainName`  
서비스의 도메인 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`HostedZoneId`  
호스팅 영역의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
