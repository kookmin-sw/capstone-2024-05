# AWS::SageMaker::ModelCard 교육지표

SageMaker 훈련 작업의 결과입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Notes" : String,
  "Value" : Double
}
```

## 속성

`Name`  
SageMaker 훈련 작업의 결과 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Notes`  
학습 작업의 결과를 설명하는 추가 참고 사항입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Value`  
SageMaker 훈련 작업의 결과 값입니다.  
_필수 항목 여부_: 예  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음
