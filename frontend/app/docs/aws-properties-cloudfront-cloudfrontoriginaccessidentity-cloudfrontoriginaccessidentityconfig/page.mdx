# AWS::CloudFront::CloudFrontOriginAccessIdentity CloudFrontOriginAccessIdentityConfig

원본 액세스 ID 구성. 리소스에 `GET` 요청을 보냅니다 `/CloudFront API version/CloudFront/identity ID/config` .

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Comment" : String
}
```

## 속성

`Comment`  
원본 액세스 ID를 설명하는 주석입니다. 주석은 128자를 초과할 수 없습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

## 참고 항목
