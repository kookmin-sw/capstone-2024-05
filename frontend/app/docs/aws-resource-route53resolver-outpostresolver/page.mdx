# AWS::Route53해석자::OutpostResolver

Outposts에 Amazon Route 53 Resolver를 생성합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Route53Resolver::OutpostResolver",
  "Properties" : {
      "InstanceCount" : Integer,
      "Name" : String,
      "OutpostArn" : String,
      "PreferredInstanceType" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`InstanceCount`  
Outposts의 해석기에 대한 Amazon EC2 인스턴스 수입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Name`  
확인자의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_업데이트 필요 사항_: 중단 없음

`OutpostArn`  
Outposts의 ARN(Amazon 리소스 이름)입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_패턴_: `^arn:aws([a-z-]+)?:outposts:[a-zd-]+:d{12}:outpost/op-[a-f0-9]{17}$`  
_업데이트 필요 사항_: 바꿔 놓음

`PreferredInstanceType`  
 Amazon EC2 인스턴스 유형입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_업데이트 필요 사항_: 중단 없음

`Tags`  
Route 53 Resolver를 식별하는 데 도움이 되는 키 값 페어입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref` Outpost Resolver의`Id` 를 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Arn` Outposts의 해석기에 대한 ARN(Amazon 리소스 이름)입니다.

`CreationTime` Outpost Resolver가 생성된 날짜 및 시간(Unix 시간 형식 및 UTC(협정 세계시))입니다.

`CreatorRequestId`  
확인자 엔드포인트를 생성한 요청을 식별하는 고유한 문자열입니다. 이렇게 `CreatorRequestId` 하면 작업을 두 번 실행할 위험 없이 실패한 요청을 다시 시도할 수 있습니다.

`Id` Outposts에 있는 확인자의 ID입니다.

`ModificationTime` Outpost Resolver가 수정된 날짜 및 시간(Unix 시간 형식 및 UTC(협정 세계시))입니다.

`Status` 확인자의 상태입니다.  
유효한 값 : CREATING | 운영 | 업데이트 | 삭제 | ACTION_NEEDED | FAILED_CREATION | FAILED_DELETION.

`StatusMessage` 확인자에 대한 자세한 설명입니다.

## 예제

### Outposts에서 해석기 생성

다음 예제에서는 AWS Outposts에서 Amazon Route 53 Resolver를 생성합니다.

#### JSON (영문)

```
{
"Type": "AWS::Route53Resolver::OutpostResolver",
"Properties": {
    "Name": "SampleOutpostResolver",
    "InstanceCount": 4,
    "OutpostArn": "arn:aws:outposts:us-west-2:123456789012:outpost/op-12345678901234567",
    "PreferredInstanceType": "m5.large",
    "Tags": [
        {
            "Key": "keyname1",
            "Value": "value1"
        },
        {
            "Key": "keyname2",
            "Value": "value2"
        }
      ]
    }
}
```
