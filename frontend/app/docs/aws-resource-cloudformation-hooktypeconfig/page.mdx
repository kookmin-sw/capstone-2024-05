# AWS::CloudFormation::HookTypeConfig

리소스는 `HookTypeConfig` 후크의 구성을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::CloudFormation::HookTypeConfig",
  "Properties" : {
      "Configuration" : String,
      "ConfigurationAlias" : String,
      "TypeArn" : String,
      "TypeName" : String
    }
}
```

## 속성

`Configuration`  
이 AWS 계정 및 AWS 리전에서 활성화된 후크 유형 구성을 지정합니다.  
`TypeName` and `Configuration` 또는 `TypeARN` and 를 `Configuration` 지정해야 합니다.  
_필수_: 조건부  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ConfigurationAlias`  
이 AWS 계정 및 AWS 리전에서 활성화된 후크 유형 구성을 지정합니다.  
기본값은 `default` alias입니다. 후크 유형은 현재 기본 구성 별칭을 지원합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`TypeArn`  
후크를 설정할 Amazon 리소스 번호(ARN)입니다 `Configuration` .  
`TypeName` and `Configuration` 또는 `TypeARN` and 를 `Configuration` 지정해야 합니다.  
_필수_: 조건부  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`TypeName`  
후크의 고유한 이름입니다. 후크에 대한 세 부분으로 구성된 네임스페이스를 지정하며, 권장 패턴은 `Organization::Service::Hook`입니다.  
`TypeName` and `Configuration` 또는 `TypeARN` and 를 `Configuration` 지정해야 합니다.  
_필수_: 조건부  
_유형_: 문자열  
_최소_: `10`  
_최대_: `196`  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`리소스 버전의 ARN을 반환합니다. 예를 들어:

`arn:aws:cloudformation:us-west-2:123456789012:type-configuration/hook/My-Sample-Hook/default`

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`ConfigurationArn` 이 계정 및 리전에서 활성화된 후크 유형 구성의 Amazon 리소스 번호(ARN)입니다.

## 예제

### TypeName을 사용하여 후크 구성 지정

다음 예제에서는 속성 유형을 `TypeName`사용하여 새 후크 구성을 지정하는 방법을 보여 줍니다.

#### JSON (영문)

```
{
    "AWSTemplateFormatVersion": "2010-09-09",
    "Resources": {
        "HookTypeConfig": {
            "Type": "AWS::CloudFormation::HookTypeConfig",
            "Properties": {
                "TypeName": "My::Sample::Hook",
                "Configuration": "{\"CloudFormationConfiguration\":{\"HookConfiguration\":{\"TargetStacks\":\"ALL\",\"FailureMode\":\"WARN\",\"Properties\":{\"limitSize\": \"1\",\"encryptionAlgorithm\": \"aws:kms\"}}}}"
            }
        }
    }
}
```

### TypeArn을 사용하여 후크 구성 지정

다음 예제에서는 속성 유형을 `TypeArn`사용하여 새 후크 구성을 지정하는 방법을 보여 줍니다.

#### JSON (영문)

```
{
    "AWSTemplateFormatVersion": "2010-09-09",
    "Resources": {
        "HookTypeConfig": {
            "Type": "AWS::CloudFormation::HookTypeConfig",
            "Properties": {
                "TypeArn": "arn:aws:cloudformation:us-west-2:123456789012:type/hook/My-Sample-Hook",
                "Configuration": "{\"CloudFormationConfiguration\":{\"HookConfiguration\":{\"TargetStacks\":\"ALL\",\"FailureMode\":\"WARN\",\"Properties\":{\"limitSize\": \"1\",\"encryptionAlgorithm\": \"aws:kms\"}}}}"
            }
        }
    }
}
```
