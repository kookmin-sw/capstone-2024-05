# AWS::SageMaker::모델 MultiModelConfig

다중 모델 끝점을 호스팅하기 위한 추가 구성을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ModelCacheSetting" : String
}
```

## 속성

`ModelCacheSetting`  
다중 모델 엔드포인트에 대한 모델을 캐시할지 여부입니다. 기본적으로 다중 모델 엔드포인트는 모델을 캐시하므로 모델이 호출될 때마다 메모리에 로드할 필요가 없습니다. 일부 사용 사례는 모델 캐싱의 이점을 얻지 못합니다. 예를 들어 엔드포인트가 각각 자주 호출되지 않는 많은 수의 모델을 호스팅하는 경우 모델 캐싱을 비활성화하면 엔드포인트의 성능이 더 향상될 수 있습니다. 모델 캐싱을 비활성화하려면 이 파라미터의 값을 Disabled로 설정합니다.
