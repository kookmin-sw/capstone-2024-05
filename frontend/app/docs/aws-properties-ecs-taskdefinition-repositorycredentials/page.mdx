# AWS::ECS::태스크 정의 리포지토리 자격 증명

개인 레지스트리 인증을 위한 리포지토리 자격 증명입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CredentialsParameter" : String
}
```

## 속성

`CredentialsParameter`  
프라이빗 리포지토리 자격 증명이 포함된 보안 암호의 Amazon 리소스 이름(ARN)입니다.  
Amazon ECS API, AWS CLI 또는 AWS SDK를 사용할 때 보안 암호가 시작하는 작업과 동일한 리전에 있는 경우 전체 ARN 또는 보안 암호의 이름을 사용할 수 있습니다. AWS Management Console을 사용하는 경우 보안 암호의 전체 ARN을 지정해야 합니다.
