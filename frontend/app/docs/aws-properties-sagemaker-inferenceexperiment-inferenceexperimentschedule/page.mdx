# AWS::SageMaker::InferenceExperiment 추론ExperimentSchedule

추론 실험의 시작 및 종료 시간입니다.

추론 실험에 설정할 수 있는 최대 기간은 30일입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "EndTime" : String,
  "StartTime" : String
}
```

## 속성

`EndTime`  
추론 실험이 종료되었거나 종료될 타임스탬프입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`StartTime`  
추론 실험이 시작되었거나 시작될 타임스탬프입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
