# AWS::EC2::ClientVpnEndpoint DirectoryServiceAuthenticationRequest

클라이언트 인증에 사용할 Active Directory에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DirectoryId" : String
}
```

## 속성

`DirectoryId`  
인증에 사용할 Active Directory의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
