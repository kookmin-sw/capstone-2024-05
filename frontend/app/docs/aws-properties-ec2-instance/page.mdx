# AWS::EC2::인스턴스

EC2 인스턴스를 지정합니다.

탄력적 IP 주소가 인스턴스에 연결된 경우 AWS CloudFormation은 인스턴스를 업데이트한 후 탄력적 IP 주소를 다시 연결합니다. 스택 업데이트에 대한 자세한 내용은 AWS CloudFormation 스택 업데이트 단원을 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::Instance",
  "Properties" : {
      "AdditionalInfo" : String,
      "Affinity" : String,
      "AvailabilityZone" : String,
      "BlockDeviceMappings" : [ BlockDeviceMapping, ... ],
      "CpuOptions" : CpuOptions,
      "CreditSpecification" : CreditSpecification,
      "DisableApiTermination" : Boolean,
      "EbsOptimized" : Boolean,
      "ElasticGpuSpecifications" : [ ElasticGpuSpecification, ... ],
      "ElasticInferenceAccelerators" : [ ElasticInferenceAccelerator, ... ],
      "EnclaveOptions" : EnclaveOptions,
      "HibernationOptions" : HibernationOptions,
      "HostId" : String,
      "HostResourceGroupArn" : String,
      "IamInstanceProfile" : String,
      "ImageId" : String,
      "InstanceInitiatedShutdownBehavior" : String,
      "InstanceType" : String,
      "Ipv6AddressCount" : Integer,
      "Ipv6Addresses" : [ InstanceIpv6Address, ... ],
      "KernelId" : String,
      "KeyName" : String,
      "LaunchTemplate" : LaunchTemplateSpecification,
      "LicenseSpecifications" : [ LicenseSpecification, ... ],
      "Monitoring" : Boolean,
      "NetworkInterfaces" : [ NetworkInterface, ... ],
      "PlacementGroupName" : String,
      "PrivateDnsNameOptions" : PrivateDnsNameOptions,
      "PrivateIpAddress" : String,
      "PropagateTagsToVolumeOnCreation" : Boolean,
      "RamdiskId" : String,
      "SecurityGroupIds" : [ String, ... ],
      "SecurityGroups" : [ String, ... ],
      "SourceDestCheck" : Boolean,
      "SsmAssociations" : [ SsmAssociation, ... ],
      "SubnetId" : String,
      "Tags" : [Tag, ... ],
      "Tenancy" : String,
      "UserData" : String,
      "Volumes" : [ Volume, ... ]
    }
}
```

## 속성

`AdditionalInfo`  
이 속성은 내부용으로 예약되어 있습니다. 이 기능을 사용하면 스택이 다음 오류와 함께 실패합니다`Bad property set: [Testing this property] (Service: AmazonEC2; Status Code: 400; Error Code: InvalidParameterCombination; Request ID: 0XXXXXX-49c7-4b40-8bcc-76885dcXXXXX)`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 일부 중단

`Affinity`  
인스턴스가 전용 호스트와 연결되어 있는지 여부를 나타냅니다. 인스턴스가 항상 시작된 호스트와 동일한 호스트에서 다시 시작되도록 하려면 를 지정합니다`host`. 사용 가능한 모든 호스트에서 인스턴스를 다시 시작하되 인스턴스가 실행된 마지막 호스트에서 시작하려고 하면(최선을 다해) 를 지정합니다`default`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 일부 중단

`AvailabilityZone`  
인스턴스의 가용 영역입니다.  
지정하지 않으면 가용 영역이 리전의 로드 밸런싱 기준에 따라 자동으로 선택됩니다.  
이 파라미터는 DescribeImageAttribute에서 지원되지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`BlockDeviceMappings`  
시작 시 인스턴스에 연결할 블록 장치를 정의하는 블록 장치 매핑 항목입니다.  
기본적으로 AMI에 대한 블록 장치 매핑에 지정된 블록 장치가 사용됩니다. 인스턴스 블록 디바이스 매핑을 사용하여 AMI 블록 디바이스 매핑을 재정의할 수 있습니다. 루트 볼륨의 경우 볼륨 크기, 볼륨 유형, 볼륨 암호화 설정 및 설정만 재정의할 수 있습니다 `DeleteOnTermination` .  
인스턴스가 실행된 후에는 `DeleteOnTermination` 인스턴스를 중단하지 않고 연결된 볼륨에 대한 파라미터만 수정할 수 있습니다. 다른 매개 변수를 수정하면 인스턴스가 바뀝니다.
*필수 항목 *여부: 아니요  
_형식_: BlockDeviceMapping 목록  
_업데이트 필요 사항_: 일부 중단

`CpuOptions`  
인스턴스의 CPU 옵션입니다. 자세한 내용은 _Amazon Elastic Compute Cloud 사용 설명서의 CPU 옵션 최적화를 참조하세요_.  
*필수 항목 *여부: 아니요  
_형식_: CpuOptions  
_업데이트 필요 사항_: 바꿔 놓음

`CreditSpecification`  
성능 순간 확장 가능 인스턴스의 CPU 사용량에 대한 크레딧 옵션입니다. 유효한 값은 `standard` 및 `unlimited`입니다. 시작 후 이 속성을 변경하려면 ModifyInstanceCreditSpecification을 사용합니다. 자세한 내용은 _Amazon EC2 사용 설명서의 버스트 가능한 성능 인스턴스를 참조하세요_.  
기본값: `standard` (T2 인스턴스) 또는 `unlimited` (T3/T3a/T4g 인스턴스)  
`host` 테넌시가 있는 T3 인스턴스의 경우에만 `standard` 지원됩니다.  
*필수 항목 *여부: 아니요  
_유형_: CreditSpecification  
_업데이트 필요 사항_: 중단 없음

`DisableApiTermination`  
이 파라미터를 `true`로 설정하면 Amazon EC2 콘솔, CLI 또는 API를 사용하여 인스턴스를 종료할 수 없으며, 그렇지 않으면 종료할 수 있습니다. 시작 후 이 속성을 변경하려면 ModifyInstanceAttribute를 사용합니다. 또는 로 설정하면 `InstanceInitiatedShutdownBehavior` `terminate`인스턴스에서 shutdown 명령을 실행하여 인스턴스를 종료할 수 있습니다.  
기본값: `false`  
_필수_: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`EbsOptimized`  
인스턴스가 Amazon EBS I/O에 최적화되어 있는지 여부를 나타냅니다. 이 최적화는 최적의 Amazon EBS I/O 성능을 제공하기 위해 Amazon EBS에 전용 처리량과 최적화된 구성 스택을 제공합니다. 이 최적화는 모든 인스턴스 유형에서 사용할 수 있는 것은 아닙니다. EBS 최적화 인스턴스를 사용하는 경우 추가 사용 요금이 적용됩니다.  
기본값: `false`  
_필수_: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 일부 중단

`ElasticGpuSpecifications`  
인스턴스와 연결할 탄력적 GPU입니다. Elastic GPU는 애플리케이션의 그래픽 성능을 가속화하기 위해 Windows 인스턴스에 연결할 수 있는 GPU 리소스입니다. 자세한 내용은 _Amazon EC2 사용 설명서의 Amazon EC2_ Elastic GPU를 참조하세요.  
*필수 항목 *여부: 아니요  
_유형_: ElasticGpuSpecification 목록  
_업데이트 필요 사항_: 바꿔 놓음

`ElasticInferenceAccelerators`  
인스턴스와 연결할 탄력적 추론 액셀러레이터입니다. Elastic 추론 액셀러레이터는 딥 러닝(DL) 추론 워크로드를 가속화하기 위해 Amazon EC2 인스턴스에 연결할 수 있는 리소스입니다.  
동일한 요청에서 서로 다른 세대의 액셀러레이터를 지정할 수 없습니다.  
2023년 4월 15일부터 AWS는 신규 고객을 Amazon Elastic Inference(EI)에 온보딩하지 않으며, 기존 고객이 더 나은 가격과 성능을 제공하는 옵션으로 워크로드를 마이그레이션할 수 있도록 지원합니다. 2023년 4월 15일 이후에는 신규 고객이 Amazon SageMaker, Amazon ECS 또는 Amazon EC2에서 Amazon EI 액셀러레이터를 사용하여 인스턴스를 시작할 수 없습니다. 그러나 지난 30일 동안 Amazon EI를 한 번 이상 사용한 고객은 현재 고객으로 간주되며 서비스를 계속 사용할 수 있습니다.
*필수 항목 *여부: 아니요  
_유형_: ElasticInferenceAccelerator 목록  
_업데이트 필요 사항_: 바꿔 놓음

`EnclaveOptions`  
인스턴스가 AWS Nitro Enclaves에 대해 활성화되었는지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_형식_: EnclaveOptions  
_업데이트 필요 사항_: 바꿔 놓음

`HibernationOptions`  
인스턴스가 최대 절전 모드를 사용할 수 있는지 여부를 나타냅니다. 이 파라미터는 인스턴스가 최대 절전 모드 사전 요구 사항을 충족하는 경우에만 유효합니다. 자세한 내용은 *Amazon EC2 사용 설명서*의 인스턴스 하이버네이트를 참조하세요.  
동일한 인스턴스에서 최대 절전 모드와 AWS Nitro Enclaves를 활성화할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: HibernationOptions  
_업데이트 필요 사항_: 바꿔 놓음

`HostId`  
속성에 host를 지정하는 경우 `Affinity` 인스턴스가 연결된 전용 호스트의 ID입니다. ID를 지정하지 않으면 Amazon EC2는 계정에서 사용 가능하고 호환되는 전용 호스트에서 인스턴스를 시작합니다. 이러한 유형의 실행을 대상 지정되지 않은 실행이라고 합니다. 대상이 지정되지 않은 시작의 경우 인스턴스를 성공적으로 시작할 수 있는 호환 가능한 전용 호스트가 있어야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 일부 중단

`HostResourceGroupArn`  
인스턴스를 시작할 호스트 리소스 그룹의 ARN입니다. 호스트 리소스 그룹 ARN을 지정하는 경우 **Tenancy** 파라미터를 생략 하거나 로 설정합니다`host`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`IamInstanceProfile`  
IAM 인스턴스 프로파일의 이름입니다. 새 IAM 인스턴스 프로파일을 생성하려면 AWS::IAM::InstanceProfile 리소스를 사용합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ImageId`  
AMI의 ID입니다. 인스턴스를 시작하려면 AMI ID가 필요하며 여기 또는 시작 템플릿에 지정해야 합니다.  
_필수_: 조건부  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`InstanceInitiatedShutdownBehavior`  
인스턴스에서 종료를 시작할 때 인스턴스가 중지되는지 또는 종료되는지를 나타냅니다(시스템 종료를 위해 운영 체제 명령 사용).  
기본값: `stop`  
_필수_: 아니요  
_유형_: 문자열  
_허용되는 값_: `stop | terminate`  
_업데이트 필요 사항_: 중단 없음

`InstanceType`  
인스턴스 유형입니다. 자세한 내용은 *Amazon EC2 사용 설명서*의 인스턴스 유형을 참조하세요.

소스/대상 검사를 활성화하거나 비활성화하여 인스턴스가 수신하는 모든 트래픽의 소스 또는 대상인지 확인합니다. 값이 `true`이면 소스/대상 확인이 활성화되고, 그렇지 않으면 비활성화됩니다. 기본값은 `true`입니다. 인스턴스가 네트워크 주소 변환, 라우팅 또는 방화벽과 같은 서비스를 실행하는 경우 소스/대상 확인을 비활성화해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`SsmAssociations`  
이 인스턴스와 연결할 AWS Systems Manager의 SSM 문서 및 파라미터 값입니다. 이 속성을 사용하려면 인스턴스에 대한 IAM 인스턴스 프로파일 역할을 지정해야 합니다. 자세한 내용은 AWS Systems Manager 사용 설명서의 Systems Manager에 대한 IAM 인스턴스 프로파일 생성을 참조하세요\**.  
현재 하나의 문서만 인스턴스와 연결할 수 있습니다.
*필수 항목 *여부: 아니요  
*유형*: SsmAssociation 목록

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`인스턴스 `Ref`ID가 반환됩니다. 예를 들면 다음과 같습니다 `i-1234567890abcdef0`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`AvailabilityZone`  
지정된 인스턴스가 시작되는 가용 영역입니다. 예를 들면 다음과 같습니다 `us-east-1b`.  
Fn::GetAZs 내장 함수를 사용하여 리전의 모든 가용 영역 목록을 검색할 수 있습니다.

`PrivateDnsName`  
지정된 인스턴스의 프라이빗 DNS 이름입니다. 예를 들면 다음과 같습니다 `ip-10-24-34-0.ec2.internal`.

`PrivateIp`  
지정된 인스턴스의 개인 IP 주소입니다. 예를 들면 다음과 같습니다 `10.24.34.0`.

`PublicDnsName`  
지정된 인스턴스의 퍼블릭 DNS 이름입니다. 예를 들면 다음과 같습니다 `ec2-107-20-50-45.compute-1.amazonaws.com`.

`PublicIp`  
지정된 인스턴스의 퍼블릭 IP 주소입니다. 예를 들면 다음과 같습니다 `192.0.2.0`.

## 예제

### EBS 블록 디바이스 매핑이 있는 EC2 인스턴스

다음 예제에서는 크기가 20GB인 io1 볼륨을 지정하는 항목과 NoDevice를 사용하여 AMI 블록 디바이스 매핑에 지정된 디바이스를 재정의하는 항목이 있는 블록 디바이스 매핑이 있는 EC2 인스턴스를 생성합니다.

#### JSON (영문)

```
"MyEC2Instance" : {
   "Type" : "AWS::EC2::Instance",
   "Properties" : {
      "ImageId" : "ami-79fd7eee",
      "KeyName" : "testkey",
      "BlockDeviceMappings" : [
         {
            "DeviceName" : "/dev/sdm",
            "Ebs" : {
              "VolumeType" : "io1",
              "Iops" : "200",
              "DeleteOnTermination" : "false",
              "VolumeSize" : "20"
            }
          },
          {
            "DeviceName" : "/dev/sdk",
            "NoDevice" : {}
          }
      ]
   }
}
```

### 공용 IP 주소 자동 할당

공용 IP 주소가 기본 네트워크 인터페이스(디바이스 인덱스는 0)이고 기존 네트워크 인터페이스를 연결하지 않고 새 네트워크 인터페이스를 만드는 경우에만 공용 IP 주소를 네트워크 인터페이스와 연결할 수 있습니다.

#### JSON (영문)

```
"Ec2Instance" : {
  "Type" : "AWS::EC2::Instance",
  "Properties" : {
    "ImageId" : { "Fn::FindInMap" : [ "RegionMap", { "Ref" : "AWS::Region" }, "AMI" ]},
    "KeyName" : { "Ref" : "KeyName" },
    "NetworkInterfaces": [ {
      "AssociatePublicIpAddress": "true",
      "DeviceIndex": "0",
      "GroupSet": [{ "Ref" : "myVPCEC2SecurityGroup" }],
      "SubnetId": { "Ref" : "PublicSubnet" }
    } ]
  }
}
```
