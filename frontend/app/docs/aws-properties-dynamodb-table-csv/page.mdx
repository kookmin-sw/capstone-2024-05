# AWS::D ynamoDB::테이블 csv

CSV 형식으로 가져온 소스 파일에 대한 옵션입니다. 값은 Delimiter 및 HeaderList입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Delimiter" : String,
  "HeaderList" : [ String, ... ]
}
```

## 속성

`Delimiter`  
 가져올 CSV 파일에서 항목을 구분하는 데 사용되는 구분 기호입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`HeaderList`  
 가져오는 모든 소스 CSV 파일에 대한 공통 헤더를 지정하는 데 사용되는 헤더 목록입니다. 이 필드를 지정하면 각 CSV 파일의 첫 번째 줄이 헤더 대신 데이터로 처리됩니다. 이 필드를 지정하지 않으면 각 CSV 파일의 첫 번째 줄이 헤더로 처리됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 바꿔 놓음
