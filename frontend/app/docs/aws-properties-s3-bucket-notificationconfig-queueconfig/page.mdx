# AWS::S3::버킷 대기열 구성

Amazon S3가 지정된 이벤트를 감지할 때 Amazon Simple Queue Service(Amazon SQS) 대기열에 메시지를 게시하기 위한 구성을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Event" : String,
  "Filter" : NotificationFilter,
  "Queue" : String
}
```

## 속성

`Event`  
Amazon SQS에 메시지를 게시하려는 Amazon S3 버킷 이벤트입니다. 자세한 내용은 _Amazon S3 사용 설명서의 지원되는 이벤트 유형을 참조하세요_.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Filter`  
 알림을 트리거하는 개체를 결정하는 필터링 규칙입니다. 예를 들어, 확장자가 있는 이미지 파일이 버킷에 추가될 때만 Amazon S3가 알림을 보내도록 필터를 생성할 수 `.jpg` 있습니다. 자세한 내용은 Amazon S3 사용 설명서의 객체 키 이름 필터링을 사용하여 이벤트 알림 구성을 참조하세요\**.  
*필수 항목 *여부: 아니요  
*형식*: NotificationFilter  
*업데이트 필요 사항\*: 중단 없음

`Queue`  
Amazon S3가 지정된 유형의 이벤트를 감지할 때 메시지를 게시하는 Amazon SQS 대기열의 Amazon 리소스 이름(ARN)입니다. FIFO 대기열은 SQS 대기열을 이벤트 알림 대상으로 활성화할 때 허용되지 않습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
