# AWS::KinesisFirehose::D elivery스트림 암호화구성

`EncryptionConfiguration` 속성 유형은 Amazon Simple Storage Service(Amazon S3)로 데이터를 전송할 때 Amazon Kinesis Data Firehose(Kinesis Data Firehose)가 사용하는 암호화 설정을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "KMSEncryptionConfig" : KMSEncryptionConfig,
  "NoEncryptionConfig" : String
}
```

## 속성

`KMSEncryptionConfig`  
Amazon S3가 데이터를 암호화하는 데 사용하는 AWS Key Management Service(AWS KMS) 암호화 키입니다.  
*필수 항목 *여부: 아니요  
_형식_: KMSEncryptionConfig  
_업데이트 필요 사항_: 중단 없음

`NoEncryptionConfig`  
암호화를 비활성화합니다. 유효한 값은 `NoEncryptionConfig`Amazon Kinesis Data Firehose API Reference*의 EncryptionConfiguration 데이터 형식에 대한 내용을 참조하십시오*.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `NoEncryption`  
_업데이트 필요 사항_: 중단 없음
