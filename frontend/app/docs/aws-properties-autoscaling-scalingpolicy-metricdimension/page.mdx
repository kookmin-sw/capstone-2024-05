# AWS::AutoScaling::ScalingPolicy 지표 차원

`MetricDimension` AWS::AutoScaling::ScalingPolicy CustomizedMetricSpecification 속성 유형의 속성 에 대한 CloudWatch 지표 자격 증명의 일부인 이름/값 페어를 지정합니다 `Dimensions` . 중복 치수는 허용되지 않습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Value" : String
}
```

## 속성

`Name`  
차원의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Value`  
차원의 값입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
