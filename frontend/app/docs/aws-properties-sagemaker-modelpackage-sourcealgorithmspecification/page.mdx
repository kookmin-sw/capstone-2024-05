# AWS::SageMaker::ModelPackage SourceAlgorithm사양

모델 패키지를 만드는 데 사용된 알고리즘 목록입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "SourceAlgorithms" : [ SourceAlgorithm, ... ]
}
```

## 속성

`SourceAlgorithms`  
모델 패키지를 만드는 데 사용된 알고리즘 목록입니다.  
_필수 항목 여부_: 예  
_유형_: SourceAlgorithm 목록  
_최대_: `1`  
_업데이트 필요 사항_: 바꿔 놓음
