# AWS::EC2::NetworkInsightsAnalysis AdditionalDetail

경로 분석에 대한 추가 세부 정보를 설명합니다. 자세한 내용은 Reachability Analyzer 추가 세부 정보 코드를 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AdditionalDetailType" : String,
  "Component" : AnalysisComponent,
  "LoadBalancers" : [ AnalysisComponent, ... ],
  "ServiceName" : String
}
```

## 속성

`AdditionalDetailType`  
추가 세부 정보 코드입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Component`  
경로 구성 요소입니다.  
*필수 항목 *여부: 아니요  
_유형_: AnalysisComponent  
_업데이트 필요 사항_: 중단 없음

`LoadBalancers`  
부하 분산 장치.  
*필수 항목 *여부: 아니요  
_유형_: AnalysisComponent 목록  
_업데이트 필요 사항_: 중단 없음

`ServiceName`  
VPC 엔드포인트 서비스의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
