# AWS::ECS::서비스 AwsVpc구성

작업 또는 서비스에 대한 네트워킹 세부 정보를 나타내는 개체입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AssignPublicIp" : String,
  "SecurityGroups" : [ String, ... ],
  "Subnets" : [ String, ... ]
}
```

## 속성

`AssignPublicIp`  
작업의 탄력적 네트워크 인터페이스가 퍼블릭 IP 주소를 수신하는지 여부입니다. 기본값은 `DISABLED`입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `DISABLED | ENABLED`  
_업데이트 필요 사항_: 중단 없음

`SecurityGroups`  
작업 또는 서비스와 연결된 보안 그룹의 ID입니다. 보안 그룹을 지정하지 않으면 VPC의 기본 보안 그룹이 사용됩니다. 당 지정할 수 있는 보안 그룹은 5개로 제한됩니다`AwsVpcConfiguration`.  
지정된 모든 보안 그룹은 동일한 VPC에 속해야 합니다.
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Subnets`  
작업 또는 서비스와 연결된 서브넷의 ID입니다. 지정할 수 있는 서브넷은 16개로 제한됩니다`AwsVpcConfiguration`.  
지정된 모든 서브넷은 동일한 VPC에 있어야 합니다.
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음
