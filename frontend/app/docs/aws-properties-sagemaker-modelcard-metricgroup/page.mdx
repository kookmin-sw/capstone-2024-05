# AWS::SageMaker::ModelCard 지표 그룹

메트릭 그룹 개체를 초기화하는 데 사용하는 메트릭 데이터 그룹입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "MetricData" : [ MetricDataItems, ... ],
  "Name" : String
}
```

## 속성

`MetricData`  
메트릭 개체 목록입니다. `MetricDataItems` 목록에는 다음 값 중 하나가 포함될 수 있습니다.

- `bar_chart_metric`
- `matrix_metric`
- `simple_metric`
- `linear_graph_metric`
  메트릭 스키마에 대한 자세한 내용은 모델 카드 JSON 스키마의 정의 섹션을 참조하세요.  
  _필수 항목 여부_: 예  
  _유형_: MetricDataItem 목록  
  _업데이트 필요 사항_: 중단 없음

`Name`  
메트릭 그룹 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
