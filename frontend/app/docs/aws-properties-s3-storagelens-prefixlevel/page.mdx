# AWS::S3::StorageLens 접두사 수준

이 리소스에는 Amazon S3 Storage Lens의 접두사 수준에 대한 세부 정보가 포함되어 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "StorageMetrics" : PrefixLevelStorageMetrics
}
```

## 속성

`StorageMetrics`  
Amazon S3 Storage Lens에 대한 접두사 수준 스토리지 지표의 속성입니다.  
_필수 항목 여부_: 예  
_형식_: PrefixLevelStorageMetrics  
_업데이트 필요 사항_: 중단 없음
