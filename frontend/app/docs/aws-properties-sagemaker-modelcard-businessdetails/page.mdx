# AWS::SageMaker::모델 카드 비즈니스 세부 정보

모델이 비즈니스 목표를 지원하는 방법에 대한 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BusinessProblem" : String,
  "BusinessStakeholders" : String,
  "LineOfBusiness" : String
}
```

## 속성

`BusinessProblem`  
모델이 해결하려는 특정 비즈니스 문제입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`BusinessStakeholders`  
모델에 대한 관련 관련자입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`LineOfBusiness`  
모델이 제공하는 광범위한 비즈니스 요구 사항.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
