# AWS::KinesisFirehose::D eliveryStream ParquetSerDe

데이터를 Amazon S3에 저장하기 전에 Parquet 형식으로 변환하는 데 사용할 직렬 변환기입니다. 자세한 내용은 Apache Parquet을 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BlockSizeBytes" : Integer,
  "Compression" : String,
  "EnableDictionaryCompression" : Boolean,
  "MaxPaddingBytes" : Integer,
  "PageSizeBytes" : Integer,
  "WriterVersion" : String
}
```

## 속성

`BlockSizeBytes`  
HDFS(Hadoop Distributed File System) 블록 크기입니다. 이는 쿼리하기 전에 Amazon S3에서 HDFS로 데이터를 복사하려는 경우에 유용합니다. 기본값은 256MiB이고 최소값은 64MiB입니다. Kinesis Data Firehose는 패딩 계산에 이 값을 사용합니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `67108864`  
_업데이트 필요 사항_: 중단 없음

`Compression`  
데이터 블록에 사용할 압축 코드입니다. 가능한 값은 `UNCOMPRESSED`, `SNAPPY`, 및 `GZIP`이며 기본값은 `SNAPPY`입니다. `SNAPPY` 더 높은 감압 속도를 위해 사용합니다. `GZIP` 속도보다 압축비가 더 중요한 경우 사용합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `GZIP | SNAPPY | UNCOMPRESSED`  
_업데이트 필요 사항_: 중단 없음

`EnableDictionaryCompression`  
사전 압축을 사용할지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`MaxPaddingBytes`  
적용할 최대 안쪽 여백입니다. 이는 쿼리하기 전에 Amazon S3에서 HDFS로 데이터를 복사하려는 경우에 유용합니다. 기본값은 0입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `0`  
_업데이트 필요 사항_: 중단 없음

`PageSizeBytes`  
Parquet 페이지 크기입니다. 열 청크는 페이지로 나뉩니다. 페이지는 개념적으로 분할할 수 없는 단위입니다(압축 및 인코딩 측면에서). 최소값은 64KiB이고 기본값은 1MiB입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `65536`  
_업데이트 필요 사항_: 중단 없음

`WriterVersion`  
출력할 행 형식의 버전을 나타냅니다. 가능한 값은 `V1` 및 `V2`입니다. 기본값은 `V1`입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `V1 | V2`  
_업데이트 필요 사항_: 중단 없음
