# AWS::ApiGatewayV2::ApiGatewayManagedOverrides

`AWS::ApiGatewayV2::ApiGatewayManagedOverrides` 리소스는 빠른 생성을 사용할 때 암시적으로 구성된 API Gateway 관리형 리소스의 기본 속성을 재정의합니다. 빨리 만들기를 사용하여 API를 만들 때 `AWS::ApiGatewayV2::Route`, `AWS::ApiGatewayV2::Integration`은(는`AWS::ApiGatewayV2::Stage`) 자동으로 만들어지고 . `AWS::ApiGatewayV2::Api` `AWS::ApiGatewayV2::ApiGatewayManagedOverrides` 이 자원을 사용하면 이러한 암시적 자원의 특성을 설정하거나 대체할 수 있습니다. HTTP API에 대해서만 지원됩니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::ApiGatewayV2::ApiGatewayManagedOverrides",
  "Properties" : {
      "ApiId" : String,
      "Integration" : IntegrationOverrides,
      "Route" : RouteOverrides,
      "Stage" : StageOverrides
    }
}
```

## 속성

`ApiId`  
API Gateway 관리형 리소스의 구성을 재정의할 API의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Integration`  
API Gateway 관리형 통합에 대한 통합 구성을 재정의합니다.  
*필수 항목 *여부: 아니요  
_형식_: IntegrationOverrides  
_업데이트 필요 사항_: 중단 없음

`Route`  
API Gateway 관리형 라우팅에 대한 라우팅 구성을 재정의합니다.  
*필수 항목 *여부: 아니요  
_유형_: RouteOverrides  
_업데이트 필요 사항_: 중단 없음

`Stage`  
API Gateway 관리형 단계에 대한 단계 구성을 재정의합니다.  
*필수 항목 *여부: 아니요  
_유형_: StageOverrides  
_업데이트 필요 사항_: 중단 없음
