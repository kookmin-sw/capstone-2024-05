# AWS::CloudFront::스트리밍 배포

이 리소스는 더 이상 사용되지 않습니다. Amazon CloudFront는 2020년 12월 31일부터 실시간 메시징 프로토콜(RTMP) 배포를 더 이상 사용하지 않습니다. 자세한 내용은 Amazon CloudFront 토론 포럼의 공지 사항을 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::CloudFront::StreamingDistribution",
  "Properties" : {
      "StreamingDistributionConfig" : StreamingDistributionConfig,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`StreamingDistributionConfig`  
RTMP 배포에 대한 현재 구성 정보입니다.  
_필수 항목 여부_: 예  
_형식_: StreamingDistributionConfig  
_업데이트 필요 사항_: 중단 없음

`Tags`  
0개 이상의 요소를 포함하는 복합 형식 `Tag` 입니다.  
_필수 항목 여부_: 예  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`는 `Ref`스트리밍 배포 ID(예: )를 반환합니다 `E1E7FEN9T35R9W`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.
