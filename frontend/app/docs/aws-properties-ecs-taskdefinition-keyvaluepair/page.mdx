# AWS::ECS::태스크 정의 KeyValuePair

키-값 쌍 객체입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Value" : String
}
```

## 속성

`Name`  
키-값 페어의 이름입니다. 환경 변수의 경우 환경 변수의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Value`  
키-값 쌍의 값입니다. 환경 변수의 경우 환경 변수의 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
