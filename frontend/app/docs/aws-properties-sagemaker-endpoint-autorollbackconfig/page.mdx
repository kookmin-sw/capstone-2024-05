# AWS::SageMaker::엔드포인트 AutoRollbackConfig

끝점 배포 실패 및 복구를 처리하기 위한 자동 롤백 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Alarms" : [ Alarm, ... ]
}
```

## 속성

`Alarms`  
엔드포인트의 지표를 모니터링하도록 구성된 계정의 CloudWatch 경보 목록입니다. 배포 중에 경보가 울리면 SageMaker는 배포를 롤백합니다.  
_필수 항목 여부_: 예  
_유형_: 알람 목록  
_최대_: `10`  
_업데이트 필요 사항_: 중단 없음
