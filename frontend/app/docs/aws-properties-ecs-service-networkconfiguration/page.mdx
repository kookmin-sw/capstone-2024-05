# AWS::ECS::서비스 네트워크 구성

이 `NetworkConfiguration` 속성은 작업 또는 서비스에 대한 네트워크 구성을 나타내는 개체를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AwsvpcConfiguration" : AwsVpcConfiguration
}
```

## 속성

`AwsvpcConfiguration`  
작업과 연결된 VPC 서브넷 및 보안 그룹입니다.  
지정된 모든 서브넷 및 보안 그룹은 동일한 VPC에 있어야 합니다.
*필수 항목 *여부: 아니요  
_유형_: AwsVpcConfiguration  
_업데이트 필요 사항_: 중단 없음
