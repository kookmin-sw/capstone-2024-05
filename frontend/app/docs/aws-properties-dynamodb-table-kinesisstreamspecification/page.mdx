# AWS::D ynamoDB::테이블 KinesisStream사양

지정된 테이블에 대한 Kinesis Data Streams 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "StreamArn" : String
}
```

## 속성

`StreamArn`  
특정 Kinesis 데이터 스트림에 대한 ARN입니다.  
길이 제약: 최소 길이는 37입니다. 최대 길이는 1024입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

- DynamoDB를 사용한 Kinesis Data Streams의 변경 데이터 캡처
