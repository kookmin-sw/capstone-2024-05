# AWS::Route53해석자::방화벽 도메인 목록

AWS::Route53Resolver::FirewallRule에서 사용할 방화벽 도메인 목록에 대한 개략적인 정보입니다. GetFirewallDomainList에서 반환됩니다.

이 도메인 목록에 대해 정의된 도메인을 검색하려면 ListFirewallDomains를 호출합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Route53Resolver::FirewallDomainList",
  "Properties" : {
      "DomainFileUrl" : String,
      "Domains" : [ String, ... ],
      "Name" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`DomainFileUrl`  
가져올 도메인 목록이 포함된 Amazon Simple Storage Service(Amazon S3)에 저장된 파일의 정규화된 URL 또는 URI입니다.  
파일은 DNS 방화벽과 동일한 리전에 있는 S3 버킷에 있어야 합니다. 파일은 텍스트 파일이어야 하며 한 줄에 하나의 도메인을 포함해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `1024`  
_업데이트 필요 사항_: 중단 없음

`Domains`  
정의한 도메인 목록 목록입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Name`  
도메인 목록의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `64`  
_패턴_: `(?!^[0-9]+$)([a-zA-Z0-9-_' ']+)`  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
도메인 목록과 연결할 태그 키 및 값의 목록입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_최대_: `200`  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`개체 `Ref`가 반환됩니다 `FirewallDomainList` .

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Arn` 방화벽 도메인 목록의 Amazon 리소스 이름(ARN)입니다.

`CreationTime` 도메인 목록이 생성된 날짜 및 시간(Unix 시간 형식 및 UTC(협정 세계시))입니다.

`CreatorRequestId` 요청을 식별하기 위해 사용자가 정의한 고유한 문자열입니다. 이렇게 하면 작업을 두 번 실행할 위험 없이 실패한 요청을 다시 시도할 수 있습니다. 고유한 문자열(예: 타임스탬프)이 될 수 있습니다.

`DomainCount` 도메인 목록에 지정된 도메인 이름의 수입니다.

`Id` 도메인 목록의 ID입니다.

`ManagedOwnerName`  
내가 관리하지 않는 목록에만 사용되는 목록의 소유자입니다. 예를 들어 관리되는 도메인 목록 `AWSManagedDomainsMalwareDomainList` 에는 관리되는 소유자 이름이 `Route 53 Resolver DNS Firewall`있습니다.

`ModificationTime` 도메인 목록이 마지막으로 수정된 날짜 및 시간(Unix 시간 형식 및 UTC(협정 세계시))입니다.

`Status` 도메인 목록의 상태입니다.

`StatusMessage` 사용 가능한 경우 목록의 상태에 대한 추가 정보입니다.
