# AWS::SageMaker::FeatureGroup 온라인 스토어 구성

이를 사용하여 AWS Key Management Service(KMS) 키 ID를 지정하거나 `KMSKeyId`저장 데이터 암호화의 경우 를 지정합니다. `OnlineStore` General Assembly에서 플래그`EnableOnlineStore`를 지정하여 켜거나 끌 수 있습니다.

기본값은 `False`입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "EnableOnlineStore" : Boolean,
  "SecurityConfig" : OnlineStoreSecurityConfig
}
```

## 속성

`EnableOnlineStore`  
`OnlineStore` 플래그 를 `False` 지정하여 끕니다 `EnableOnlineStore` . 플래그 `OnlineStore` 를 `True` 지정하여 켭 `EnableOnlineStore` 니다 .  
기본값은 `False`입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 바꿔 놓음

`SecurityConfig`  
`KMSKeyId`의 저장 데이터 암호화를 위해 KMS 키 ID()를 지정하는 데 사용합니다`OnlineStore`.  
*필수 항목 *여부: 아니요  
_형식_: OnlineStoreSecurityConfig  
_업데이트 필요 사항_: 바꿔 놓음
