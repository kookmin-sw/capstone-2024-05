# AWS::RDS::D BCluster ServerlessV2ScalingConfiguration

`ServerlessV2ScalingConfiguration` 속성 유형은 Aurora Serverless V2 DB 클러스터의 크기 조정 구성을 지정합니다.

자세한 내용은 Amazon Aurora 사용 설명서*의 Amazon Aurora Serverless v2 사용을 참조하세요*.

Aurora 클러스터가 있는 경우 DB 인스턴스 클래스를 `ScalingConfigurationInfo` 사용하는 DB 인스턴스를 추가하기 전에 속성을 설정해야 합니다 `db.serverless` . 자세한 내용은 Aurora Serverless v2를 사용하는 클러스터에는 _Amazon Aurora 사용 설명서에 지정된 용량 범위가 있어야 합니다_.

이 속성은 Aurora Serverless v2에서만 지원됩니다. Aurora Serverless v1의 경우 `ScalingConfiguration` property를 사용합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "MaxCapacity" : Double,
  "MinCapacity" : Double
}
```

## 속성

`MaxCapacity`  
Aurora Serverless v2 클러스터의 DB 인스턴스에 대한 최대 Aurora 용량 단위(ACU) 수입니다. ACU 값은 40, 40.5, 41 등과 같이 절반 단위로 지정할 수 있습니다. 사용할 수 있는 가장 큰 값은 128입니다.  
최대 용량은 0.5 ACU보다 커야 합니다. 자세한 내용은 Amazon Aurora 사용 설명서의 클러스터에 대한 최대 Aurora Serverless v2 용량 설정 선택을 참조하세요\**.  
*필수 항목 *여부: 아니요  
*유형*: 더블  
*업데이트 필요 사항\*: 중단 없음

`MinCapacity`  
Aurora Serverless v2 클러스터의 DB 인스턴스에 대한 최소 Aurora 용량 단위(ACU) 수입니다. ACU 값은 8, 8.5, 9 등과 같이 반 단계 단위로 지정할 수 있습니다. 사용할 수 있는 가장 작은 값은 0.5입니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음
