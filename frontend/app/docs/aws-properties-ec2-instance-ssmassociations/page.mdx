# AWS::EC2::인스턴스 SsmAssociation

인스턴스와 연결할 AWS Systems Manager의 SSM 문서 및 파라미터 값을 지정합니다.

`SsmAssociations` 은 AWS::EC2::Instance 리소스의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AssociationParameters" : [ AssociationParameter, ... ],
  "DocumentName" : String
}
```

## 속성

`AssociationParameters`  
연결된 SSM 문서와 함께 사용할 입력 파라미터 값입니다.  
*필수 항목 *여부: 아니요  
_형식_: AssociationParameter 목록  
_업데이트 필요 사항_: 중단 없음

`DocumentName`  
인스턴스와 연결할 SSM 문서의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
