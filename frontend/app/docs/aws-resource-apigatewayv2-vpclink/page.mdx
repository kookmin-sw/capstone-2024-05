# AWS::ApiGatewayV2::Vpc링크

`AWS::ApiGatewayV2::VpcLink` 리소스는 VPC 링크를 생성합니다. HTTP API에 대해서만 지원됩니다. VPC 링크를 성공적으로 생성하려면 VPC 링크 상태가 `PENDING` 에서 로 `AVAILABLE` 전환되어야 하며 , 최대 10분이 걸릴 수 있습니다. 자세한 내용은 API Gateway 개발자 안내서의 HTTP API용 VPC 링크 작업을 참조하십시오\*\*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::ApiGatewayV2::VpcLink",
  "Properties" : {
      "Name" : String,
      "SecurityGroupIds" : [ String, ... ],
      "SubnetIds" : [ String, ... ],
      "Tags" : {Key: Value, ...}
    }
}
```

## 속성

`Name`  
VPC 링크의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`SecurityGroupIds`  
VPC 링크에 대한 보안 그룹 ID 목록입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 바꿔 놓음

`SubnetIds`  
VPC 링크에 포함할 서브넷 ID 목록입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
태그 모음입니다. 각 태그 요소는 지정된 리소스와 연결됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 맵  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`VPC `Ref`링크의 ID(예: )를 반환합니다 `abcde1`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`VpcLinkId` VPC 링크 ID입니다.
