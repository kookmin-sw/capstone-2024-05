# AWS::SageMaker::모델 카드

Amazon SageMaker 모델 카드를 생성합니다.

모델 카드 사용 방법에 대한 자세한 내용은 Amazon SageMaker 모델 카드 단원을 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::SageMaker::ModelCard",
  "Properties" : {
      "Content" : Content,
      "CreatedBy" : UserContext,
      "LastModifiedBy" : UserContext,
      "ModelCardName" : String,
      "ModelCardStatus" : String,
      "SecurityConfig" : SecurityConfig,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`Content`  
모델 카드의 내용입니다. 콘텐츠는 모델 카드 JSON 스키마를 사용합니다.  
_필수 항목 여부_: 예  
_유형_: 콘텐츠  
_업데이트 필요 사항_: 중단 없음

`CreatedBy` 다음 중 하나 이상을 만들거나 수정한 사용자에 대한 정보입니다.

- 실험
- 재판
- 평가판 구성 요소
- 계보 그룹
- 프로젝트
- 모델 카드
  *필수 항목 *여부: 아니요  
  _형식_: UserContext  
  _업데이트 필요 사항_: 중단 없음

`LastModifiedBy`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: UserContext  
_업데이트 필요 사항_: 중단 없음

`ModelCardName`  
모델 카드의 고유 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}$`  
_업데이트 필요 사항_: 바꿔 놓음

`ModelCardStatus` 조직 내 모델 카드의 승인 상태입니다. 조직마다 모델 카드 검토 및 승인에 대한 기준이 다를 수 있습니다.

- `Draft`: 모델 카드는 진행중인 작업입니다.
- `PendingReview`: 모델 카드가 검토 대기 중입니다.
- `Approved`: 모델 카드가 승인되었습니다.
- `Archived`: 모델 카드가 보관됩니다. 모델 카드를 더 이상 업데이트하지 않아도 되지만 내보낼 수는 있습니다.
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _허용되는 값_: `Approved | Archived | Draft | PendingReview`  
  _업데이트 필요 사항_: 중단 없음

`SecurityConfig`  
모델 카드 데이터를 보호하는 데 사용되는 보안 구성입니다.  
*필수 항목 *여부: 아니요  
_유형_: SecurityConfig  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
모델 카드의 메타데이터를 관리하는 데 사용되는 키-값 쌍입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_최대_: `50`  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`모델 카드 이름이 반환됩니다.

Ref 함수 사용에 대한 자세한 내용은 Ref를 참조하십시오.

### Fn::GetAtt

`Fn::GetAtt` 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다. Fn::GetAtt 내장 함수 사용에 대한 자세한 내용은 Fn::GetAtt 단원을 참조하십시오.

####

`CreatedBy.DomainId` 속성 설명을 사용할 수 없습니다.

`CreatedBy.UserProfileArn` 속성 설명을 사용할 수 없습니다.

`CreationTime` 속성 설명을 사용할 수 없습니다.

`LastModifiedBy.DomainId` 속성 설명을 사용할 수 없습니다.

`LastModifiedBy.UserProfileArn` 속성 설명을 사용할 수 없습니다.

`ModelCardArn`  
모델 카드의 Amazon 리소스 번호(ARN)입니다. 예를 들어, `arn:aws:sagemaker:us-west-2:012345678901:modelcard/examplemodelcard`.

`ModelCardProcessingStatus` 속성 설명을 사용할 수 없습니다.
