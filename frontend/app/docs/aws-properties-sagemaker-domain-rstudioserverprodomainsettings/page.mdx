# AWS::SageMaker::D omain RStudioServerProDomainSettings

도메인 수준 앱을 `RStudioServerPro`구성하는 설정 컬렉션입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DefaultResourceSpec" : ResourceSpec,
  "DomainExecutionRoleArn" : String,
  "RStudioConnectUrl" : String,
  "RStudioPackageManagerUrl" : String
}
```

## 속성

`DefaultResourceSpec`  
기본값 `InstanceType`, `SageMakerImageArn`및 `SageMakerImageVersionArn` 도메인을 정의하는 컬렉션입니다.  
*필수 항목 *여부: 아니요  
_유형_: ResourceSpec  
_업데이트 필요 사항_: 바꿔 놓음

`DomainExecutionRoleArn`  
도메인 수준 앱에 대한 실행 역할의 ARN입니다 `RStudioServerPro` .  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `20`  
_최대_: `2048`  
_패턴_: `^arn:aws[a-z-]*:iam::d{12}:role/?[a-zA-Z_0-9+=,.@-_/]+$`  
_업데이트 필요 사항_: 중단 없음

`RStudioConnectUrl`  
RStudio Connect 서버를 가리키는 URL입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`RStudioPackageManagerUrl`  
RStudio Package Manager 서버를 가리키는 URL입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
