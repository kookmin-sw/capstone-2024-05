# AWS::ECS::태스크 정의 환경 파일

컨테이너에 전달할 환경 변수를 포함하는 파일 목록입니다. 최대 10개의 환경 파일을 지정할 수 있습니다. 파일에는 `.env` 파일 확장자가 있어야 합니다. 환경 파일의 각 줄에는 형식의 환경 변수가 포함되어야 합니다 `VARIABLE=VALUE` . 로 시작하는 줄 `#` 은 주석으로 처리되고 무시됩니다. 환경 변수 파일 구문에 대한 자세한 내용은 파일에서 기본 환경 변수 선언을 참조하세요.

컨테이너 정의에서 매개 변수를 사용하여 지정된 환경 변수가 있는 경우 해당 환경 변수가 `environment` 환경 파일에 포함된 변수보다 우선합니다. 동일한 변수를 포함하는 여러 환경 파일이 지정된 경우 위에서 아래로 처리됩니다. 고유한 변수 이름을 사용하는 것이 좋습니다. 자세한 내용은 _Amazon Elastic Container Service 개발자 안내서의 환경 변수 지정을 참조하세요_.

Fargate 시작 유형에는 다음 플랫폼을 사용해야 합니다.

- Linux 플랫폼 버전 `1.4.0` 이상.
- Windows 플랫폼 버전 `1.0.0` 이상.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : String,
  "Value" : String
}
```

## 속성

`Type`  
사용할 파일 형식입니다. 유일하게 지원되는 값은 `s3`입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `s3`  
_업데이트 필요 사항_: 바꿔 놓음

`Value`  
환경 변수 파일을 포함하는 Amazon S3 객체의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
