# AWS::SageMaker::P ipeline 파이프라인 정의

`PipelineDefinition` 속성 유형은 속성 설명을 사용할 수 없음을 지정합니다. AWS::SageMaker::P ipeline입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "PipelineDefinitionBody" : String,
  "PipelineDefinitionS3Location" : S3Location
}
```

## 속성

`PipelineDefinitionBody`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PipelineDefinitionS3Location`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: S3Location  
_업데이트 필요 사항_: 중단 없음
