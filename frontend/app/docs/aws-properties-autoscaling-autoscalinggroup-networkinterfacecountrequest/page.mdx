# AWS::AutoScaling::AutoScalingGroup NetworkInterfaceCountRequest

`NetworkInterfaceCountRequest` 은 `InstanceRequirements` 인스턴스 유형의 최소 및 최대 네트워크 인터페이스 수를 설명하는 AWS::AutoScaling::AutoScalingGroup LaunchTemplateOverrides 속성 유형의 속성 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Max" : Integer,
  "Min" : Integer
}
```

## 속성

`Max`  
네트워크 인터페이스의 최대 수입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `0`  
_업데이트 필요 사항_: 중단 없음

`Min`  
네트워크 인터페이스의 최소 수입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `0`  
_업데이트 필요 사항_: 중단 없음
