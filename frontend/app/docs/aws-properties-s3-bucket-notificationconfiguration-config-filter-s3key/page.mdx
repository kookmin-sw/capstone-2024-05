# AWS::S3::버킷 S3KeyFilter

객체 키 이름, 접두사 및 접미사 필터링 규칙의 컨테이너입니다. 객체 키 이름 필터링에 대한 자세한 내용은 Amazon S3 사용 설명서의 객체 키 이름 필터링을 사용하여 이벤트 알림 구성을 참조하세요\*\*.

**메모**  
동일한 유형의 필터 규칙은 두 번 이상 사용할 수 없습니다. 예를 들어, 두 개의 접두사 규칙을 지정할 수 없습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Rules" : [ FilterRule, ... ]
}
```

## 속성

`Rules`  
필터 규칙에 대한 기준을 정의하는 키-값 쌍에 대한 컨테이너 목록입니다.  
_필수 항목 여부_: 예  
_유형_: FilterRule 목록  
_업데이트 필요 사항_: 중단 없음
