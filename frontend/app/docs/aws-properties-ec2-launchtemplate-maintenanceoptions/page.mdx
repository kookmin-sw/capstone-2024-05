# AWS::EC2::LaunchTemplate 유지 관리 옵션

인스턴스의 유지 관리 옵션.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AutoRecovery" : String
}
```

## 속성

`AutoRecovery`  
인스턴스의 자동 복구 동작을 비활성화하거나 기본값으로 설정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `default | disabled`  
_업데이트 필요 사항_: 중단 없음
