# AWS::KinesisAnalyticsV2::ApplicationOutput 대상 스키마

SQL 기반 Managed Service for Apache Flink 애플리케이션에서 레코드가 대상에 기록될 때의 데이터 형식을 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "RecordFormatType" : String
}
```

## 속성

`RecordFormatType`  
출력 스트림의 레코드 형식을 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `CSV | JSON`  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

- Amazon Kinesis Data Analytics API 참조\* \*의 DestinationSchema
