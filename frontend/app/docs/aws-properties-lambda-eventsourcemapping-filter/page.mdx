# AWS::Lambda::EventSourceMapping 필터

이벤트 필터링 패턴을 정의하는 개체 `FilterCriteria` 내의 구조체입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Pattern" : String
}
```

## 속성

`Pattern`  
 필터 패턴입니다. 필터 패턴의 구문에 대한 자세한 내용은 필터 규칙 구문을 참조하십시오.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `4096`  
_패턴_: `.*`  
_업데이트 필요 사항_: 중단 없음
