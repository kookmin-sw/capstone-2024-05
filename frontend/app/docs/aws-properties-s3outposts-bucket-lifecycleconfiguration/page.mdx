# AWS::S3Outposts::버킷 수명 주기 구성

S3 on Outposts 버킷에 저장된 객체의 수명 주기 구성을 위한 컨테이너입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Rules" : [ Rule, ... ]
}
```

## 속성

`Rules`  
S3 on Outposts 버킷에 저장된 객체에 대한 수명 주기 구성 규칙의 컨테이너입니다.  
_필수 항목 여부_: 예  
_유형_: 규칙 목록  
_업데이트 필요 사항_: 중단 없음
