# AWS::SageMaker::InferenceExperiment ModelInfrastructureConfig

모델이 배포될 인프라에 대한 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "InfrastructureType" : String,
  "RealTimeInferenceConfig" : RealTimeInferenceConfig
}
```

## 속성

`InfrastructureType` 모델을 배포할 추론 옵션입니다. 가능한 값은 다음과 같습니다.

- `RealTime`: 실시간 추론에 배포합니다.
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _허용되는 값_: `RealTimeInference`  
  _업데이트 필요 사항_: 중단 없음

`RealTimeInferenceConfig`  
실시간 유추에 모델을 배포하기 위한 인프라 구성입니다.  
_필수 항목 여부_: 예  
_형식_: RealTimeInferenceConfig  
_업데이트 필요 사항_: 중단 없음
