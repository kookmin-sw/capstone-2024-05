# AWS::SageMaker::P roject 프로비저닝 파라미터

프로젝트를 서비스 카탈로그 제품으로 프로비저닝할 때 사용되는 키 값 쌍입니다. 자세한 내용은 AWS Service Catalog란 무엇인가를 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Key" : String,
  "Value" : String
}
```

## 속성

`Key`  
프로비저닝 매개 변수를 식별하는 키입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `1000`  
_패턴_: `.*`  
_업데이트 필요 사항_: 바꿔 놓음

`Value`  
프로비저닝 매개 변수의 값입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `4096`  
_패턴_: `.*`  
_업데이트 필요 사항_: 바꿔 놓음
