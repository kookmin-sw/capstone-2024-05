# AWS::EC2::네트워크 인터페이스 Ipv6접두사 사양

IPv6 접두사에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Ipv6Prefix" : String
}
```

## 속성

`Ipv6Prefix`  
IPv6 접두사입니다. 자세한 내용은 _Amazon Elastic Compute Cloud 사용 설명서의 Amazon EC2 네트워크 인터페이스에 접두사 할당을 참조하세요_.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
