# AWS::Lambda::함수 환경

함수의 환경 변수 설정입니다. 환경 변수를 사용하여 코드를 업데이트하지 않고 함수의 동작을 조정할 수 있습니다. 환경 변수는 함수의 버전별 구성에 저장되는 문자열 쌍입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Variables" : {Key: Value, ...}
}
```

## 속성

`Variables`  
환경 변수 키-값 쌍. 자세한 내용은 Lambda 환경 변수 사용을 참조하세요.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 맵  
_업데이트 필요 사항_: 중단 없음

## 예제

### 환경 변수

함수에 환경 변수를 추가합니다. 각 변수는 키-값 쌍입니다. 이 예제에서는 a `databaseName` 및 a `databaseUser`에 대한 값을 지정합니다.
