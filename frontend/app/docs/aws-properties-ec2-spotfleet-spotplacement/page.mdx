# AWS::EC2::SpotFleet 스팟 배치

스팟 인스턴스 배치에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AvailabilityZone" : String,
  "GroupName" : String,
  "Tenancy" : String
}
```

## 속성

`AvailabilityZone`  
가용 영역입니다.  
여러 가용 영역을 지정하려면 쉼표를 사용하여 구분합니다. 예: "us-west-2a, us-west-2b".  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`GroupName`  
배치 그룹의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tenancy`  
인스턴스의 테넌시입니다(인스턴스가 VPC에서 실행 중인 경우). 테넌시가 있는 인스턴스는 `dedicated` 단일 테넌트 하드웨어에서 실행됩니다. `host` 테넌시는 스팟 인스턴스에 대해 지원되지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `dedicated | default | host`  
_업데이트 필요 사항_: 바꿔 놓음
