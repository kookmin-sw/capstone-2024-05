# AWS::CloudFormation::TypeActivation (영문)

공용 타사 확장을 활성화하여 스택 템플릿에서 사용할 수 있도록 합니다. 자세한 내용은 * AWS CloudFormation 사용 설명서*의 퍼블릭 확장 프로그램 사용을 참조하세요.

계정 및 리전에서 퍼블릭 타사 확장을 활성화한 후에는 SetTypeConfiguration을 사용하여 확장에 대한 구성 속성을 지정합니다. 자세한 내용은 _CloudFormation 사용 설명서의 계정 수준에서 확장 구성을 참조하세요_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::CloudFormation::TypeActivation",
  "Properties" : {
      "AutoUpdate" : Boolean,
      "ExecutionRoleArn" : String,
      "LoggingConfig" : LoggingConfig,
      "MajorVersion" : String,
      "PublicTypeArn" : String,
      "PublisherId" : String,
      "Type" : String,
      "TypeName" : String,
      "TypeNameAlias" : String,
      "VersionBump" : String
    }
}
```

## 속성

`AutoUpdate`  
확장 게시자가 새 부 버전을 게시할 때 이 계정 및 리전에서 확장을 자동으로 업데이트할지 여부 \** 입니다. 게시자가 릴리스한 주 버전은 수동으로 업데이트해야 합니다.  
기본값은 `true`입니다.  
*필수 항목 *여부: 아니요  
*유형*: Boolean  
*업데이트 필요 사항\*: 중단 없음

`ExecutionRoleArn`  
익스텐션을 활성화하는 데 사용할 IAM 실행 역할의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `256`  
_패턴_: `arn:.+:iam::[0-9]{12}:role/.+`  
_업데이트 필요 사항_: 중단 없음

`LoggingConfig`  
확장에 대한 로깅 구성 정보를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: LoggingConfig  
_업데이트 필요 사항_: 바꿔 놓음

`MajorVersion`  
여러 주 버전을 사용할 수 있는 경우 활성화하려는 이 확장의 주 버전입니다. 기본값은 최신 주 버전입니다. CloudFormation은 선택한 메이저 버전의 사용 가능한 최신 _마이너_ 버전을 사용합니다.  
`MajorVersion` 또는 를 지정할 수 `VersionBump`있지만 둘 다 지정할 수는 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PublicTypeArn`  
퍼블릭 확장의 Amazon 리소스 번호(ARN)입니다.  
조건부: `PublicTypeArn`, 또는 `TypeName`, , `Type`및 를 지정해야 합니다`PublisherId`.  
_필수_: 조건부  
_유형_: 문자열  
_최대_: `1024`  
_패턴_: `arn:aws[A-Za-z0-9-]{0,64}:cloudformation:[A-Za-z0-9-]{1,64}::type/.+/[0-9a-zA-Z]{12,40}/.+`  
_업데이트 필요 사항_: 중단 없음

`PublisherId`  
확장 게시자의 ID입니다.  
조건부: `PublicTypeArn`, 또는 `TypeName`, , `Type`및 를 지정해야 합니다`PublisherId`.  
_필수_: 조건부  
_유형_: 문자열  
_최소_: `1`  
_최대_: `40`  
_패턴_: `[0-9a-zA-Z]{12,40}`  
_업데이트 필요 사항_: 중단 없음

`Type`  
확장 유형입니다.  
조건부: `PublicTypeArn`, 또는 `TypeName`, , `Type`및 를 지정해야 합니다`PublisherId`.  
_필수_: 조건부  
_유형_: 문자열  
_허용되는 값_: `HOOK | MODULE | RESOURCE`  
_업데이트 필요 사항_: 중단 없음

`TypeName`  
확장의 이름입니다.  
조건부: `PublicTypeArn`, 또는 `TypeName`, , `Type`및 를 지정해야 합니다`PublisherId`.  
_필수_: 조건부  
_유형_: 문자열  
_최소_: `10`  
_최대_: `204`  
_패턴_: `[A-Za-z0-9]{2,64}::[A-Za-z0-9]{2,64}::[A-Za-z0-9]{2,64}(::MODULE){0,1}`  
_업데이트 필요 사항_: 중단 없음

`TypeNameAlias`  
이 계정 및 리전에서 퍼블릭 확장에 할당할 별칭입니다. 익스텐션에 대한 별칭을 지정하는 경우 CloudFormation은 별칭을 이 계정 및 리전 내의 익스텐션 유형 이름으로 취급합니다. 별칭을 사용하여 템플릿, API 호출 및 CloudFormation 콘솔에서 익스텐션을 참조해야 합니다.  
확장 별칭은 지정된 계정 및 리전 내에서 고유해야 합니다. 동일한 계정 및 리전에서 동일한 유형 이름 별칭을 사용하여 동일한 퍼블릭 리소스를 여러 번 활성화할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `10`  
_최대_: `204`  
_패턴_: `[A-Za-z0-9]{2,64}::[A-Za-z0-9]{2,64}::[A-Za-z0-9]{2,64}(::MODULE){0,1}`  
_업데이트 필요 사항_: 중단 없음

`VersionBump`  
사용 가능한 경우 이전에 활성화된 유형을 새 주 버전 또는 부 버전으로 수동으로 업데이트합니다. 이 매개 변수를 사용하여 의 값을 업데이트할 수도 있습니다 `AutoUpdate`.

- `MAJOR`: CloudFormation은 확장 프로그램을 최신 메이저 버전(사용 가능한 경우)으로 업데이트합니다.
- `MINOR`: CloudFormation은 확장 프로그램을 최신 마이너 버전(사용 가능한 경우)으로 업데이트합니다.
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _허용되는 값_: `MAJOR | MINOR`  
  _업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`이 계정 및 리전에서 활성화된 익스텐션의 Amazon 리소스 번호(ARN)를 반환합니다.

`{ "Ref": "arn:aws:cloudformation:us-east-1:123456789013:type/resource/My-Example" }`

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Arn` 이 계정 및 리전에서 활성화된 익스텐션의 Amazon 리소스 번호(ARN)입니다.
