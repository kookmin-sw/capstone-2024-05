# AWS::CloudWatch::AnomalyDetector 지표

특정 메트릭을 나타냅니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Dimensions" : [ Dimension, ... ],
  "MetricName" : String,
  "Namespace" : String
}
```

## 속성

`Dimensions`  
지표의 차원입니다.  
*필수 항목 *여부: 아니요  
_유형_: 차원 목록  
_최대_: `30`  
_업데이트 필요 사항_: 바꿔 놓음

`MetricName`  
메트릭의 이름입니다. 이것은 필수 필드입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_업데이트 필요 사항_: 바꿔 놓음

`Namespace`  
메트릭의 네임스페이스입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_패턴_: `[^:].*`  
_업데이트 필요 사항_: 바꿔 놓음
