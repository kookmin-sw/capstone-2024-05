# AWS::EC2::NetworkInsights액세스 범위 분석

네트워크 액세스 범위 분석에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::NetworkInsightsAccessScopeAnalysis",
  "Properties" : {
      "NetworkInsightsAccessScopeId" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`NetworkInsightsAccessScopeId`  
네트워크 액세스 범위의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
태그.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`네트워크 인사이트 분석의 ID를 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`AnalyzedEniCount` 분석된 네트워크 인터페이스 수입니다.

`EndDate` 분석의 종료 날짜입니다.

`FindingsFound` 결과가 있는지 여부를 나타냅니다(true | false | unknown).

`NetworkInsightsAccessScopeAnalysisArn` 네트워크 액세스 범위 분석의 ARN입니다.

`NetworkInsightsAccessScopeAnalysisId` 네트워크 액세스 범위 분석의 ID입니다.

`StartDate` 분석의 시작 날짜입니다.

`Status` 분석의 상태(실행 중 | 성공 | 실패)입니다.

`StatusMessage` 상태 메시지입니다.
