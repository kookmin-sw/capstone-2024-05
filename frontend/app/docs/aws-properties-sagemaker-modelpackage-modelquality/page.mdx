# AWS::SageMaker::모델 패키지 모델 품질

모델 품질 통계 및 제약 조건.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Constraints" : MetricsSource,
  "Statistics" : MetricsSource
}
```

## 속성

`Constraints`  
모델 품질 구속조건.  
*필수 항목 *여부: 아니요  
_유형_: MetricsSource  
_업데이트 필요 사항_: 바꿔 놓음

`Statistics`  
모델 품질 통계.  
*필수 항목 *여부: 아니요  
_유형_: MetricsSource  
_업데이트 필요 사항_: 바꿔 놓음
