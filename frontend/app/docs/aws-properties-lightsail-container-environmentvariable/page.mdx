# AWS::Lightsail::컨테이너 환경 변수

`EnvironmentVariable` 는 Container 속성의 속성입니다. 컨테이너 서비스에서 실행되는 애플리케이션 또는 스크립트의 동적 구성을 제공하는 키-값 매개 변수인 컨테이너 서비스의 컨테이너 환경 변수를 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Value" : String,
  "Variable" : String
}
```

## 속성

`Value`  
환경 변수 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Variable`  
환경 변수 키입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
