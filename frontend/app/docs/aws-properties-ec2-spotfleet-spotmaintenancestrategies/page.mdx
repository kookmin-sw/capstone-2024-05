# AWS::EC2::SpotFleet SpotMaintenance전략

중단될 위험이 높은 스팟 인스턴스를 관리하기 위한 전략입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CapacityRebalance" : SpotCapacityRebalance
}
```

## 속성

`CapacityRebalance`  
Amazon EC2가 스팟 인스턴스가 중단될 위험이 높다는 신호를 방출할 때 사용할 스팟 인스턴스 교체 전략입니다. 자세한 내용은 _Linux 인스턴스용 Amazon EC2 사용 설명서의 용량 리밸런싱을 참조하세요_.  
*필수 항목 *여부: 아니요  
_유형_: SpotCapacityRebalance  
_업데이트 필요 사항_: 바꿔 놓음
