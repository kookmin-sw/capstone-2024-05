# AWS::CloudFront::응답 헤더 정책

응답 헤더 정책입니다.

응답 헤더 정책에는 HTTP 응답 헤더 집합에 대한 정보가 포함되어 있습니다.

응답 헤더 정책을 생성한 후 해당 ID를 사용하여 CloudFront 배포에서 하나 이상의 캐시 동작에 연결할 수 있습니다. 캐시 동작에 연결된 경우 응답 헤더 정책은 CloudFront가 캐시 동작과 일치하는 요청에 대한 HTTP 응답에 포함하는 HTTP 헤더에 영향을 줍니다. CloudFront는 응답 헤더 정책의 구성에 따라 응답 헤더를 추가하거나 제거합니다.

자세한 내용은 Amazon CloudFront 개발자 안내서*의 CloudFront 응답에서 HTTP 헤더 추가 또는 제거를 참조하세요*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::CloudFront::ResponseHeadersPolicy",
  "Properties" : {
      "ResponseHeadersPolicyConfig" : ResponseHeadersPolicyConfig
    }
}
```

## 속성

`ResponseHeadersPolicyConfig`  
응답 헤더 정책 구성입니다.  
_필수 항목 여부_: 예  
_유형_: ResponseHeadersPolicyConfig  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`응답 헤더 정책 ID를 반환합니다. 예를 들면 다음과 같습니다 `57f99797-3b20-4e1b-a728-27972a74082a`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Id`  
응답 헤더 정책의 고유 식별자입니다. 예를 들면 다음과 같습니다 `57f99797-3b20-4e1b-a728-27972a74082a`.

`LastModifiedTime` 응답 헤더 정책을 마지막으로 수정한 날짜 및 시간입니다.
