# AWS::D ynamoDB::테이블 InputFormatOptions

대상 테이블로 가져온 데이터에 대한 형식 옵션입니다. CsvOption이라는 하나의 값이 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Csv" : Csv
}
```

## 속성

`Csv`  
 CSV 형식으로 가져온 소스 파일에 대한 옵션입니다. 값은 Delimiter 및 HeaderList입니다.  
*필수 항목 *여부: 아니요  
_유형_: Csv  
_업데이트 필요 사항_: 바꿔 놓음
