# AWS::ApiGatewayV2::D

`AWS::ApiGatewayV2::Deployment` 리소스는 API에 대한 배포를 만듭니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::ApiGatewayV2::Deployment",
  "Properties" : {
      "ApiId" : String,
      "Description" : String,
      "StageName" : String
    }
}
```

## 속성

`ApiId`  
API 식별자입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Description`  
배포 리소스에 대한 설명입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`StageName`  
배포와 연결할 기존 단계의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`배포 `Ref`ID(예: )를 반환합니다 `123abc`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`DeploymentId` 배포 ID입니다.

## 예제

### 배포 생성 예제

다음 예제에서는 경로가 정의된 `deployment` API `MyApi` 에 대한 리소스를 `MyRoute` 만듭니다 .

#### JSON (영문)

```
{
    "Deployment": {
        "Type": "AWS::ApiGatewayV2::Deployment",
        "DependsOn": [
            "MyRoute"
        ],
        "Properties": {
            "Description": "My deployment",
            "ApiId": {
                "Ref": "MyApi"
            },
            "StageName": "Beta"
        }
    }
}
```
