# AWS::RedshiftServerless::Workgroup 작업 그룹

끝점이 만들어지는 컴퓨팅 리소스의 컬렉션입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BaseCapacity" : Integer,
  "ConfigParameters" : [ ConfigParameter, ... ],
  "CreationDate" : String,
  "Endpoint" : Endpoint,
  "EnhancedVpcRouting" : Boolean,
  "NamespaceName" : String,
  "PubliclyAccessible" : Boolean,
  "SecurityGroupIds" : [ String, ... ],
  "Status" : String,
  "SubnetIds" : [ String, ... ],
  "WorkgroupArn" : String,
  "WorkgroupId" : String,
  "WorkgroupName" : String
}
```

## 속성

`BaseCapacity`  
Redshift 처리 장치(RPU) 단위의 작업 그룹 기본 데이터 웨어하우스 용량입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`ConfigParameters`  
데이터베이스에 대한 고급 제어를 위해 설정할 매개 변수 배열입니다. 옵션은 `auto_mv`, `datestyle`, `enable_case_sensitivity_identifier`, `enable_user_activity_logging`, `query_group`, , 및 `search_path`성능 경계를 정의할 수 있는 쿼리 모니터링 메트릭입니다. 쿼리 모니터링 규칙 및 사용 가능한 지표에 대한 자세한 내용은 Amazon Redshift Serverless에 대한 쿼리 모니터링 지표 섹션을 참조하세요.  
*필수 항목 *여부: 아니요  
_유형_: ConfigParameter 목록  
_업데이트 필요 사항_: 중단 없음

`CreationDate`  
작업 그룹을 만든 날짜입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Endpoint`  
작업 그룹에서 만든 끝점입니다.  
*필수 항목 *여부: 아니요  
_유형_: 끝점  
_업데이트 필요 사항_: 중단 없음

`EnhancedVpcRouting`  
Amazon Redshift Serverless가 VPC를 통해 트래픽을 라우팅하도록 강제하는 향상된 Virtual Private Cloud(VPC) 라우팅을 활성화할지 여부를 지정하는 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`NamespaceName`  
작업 그룹이 연결된 네임스페이스입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PubliclyAccessible`  
공용 네트워크에서 작업 그룹에 액세스할 수 있는지 여부를 지정하는 값입니다  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`SecurityGroupIds`  
작업 그룹과 연결할 보안 그룹 ID의 배열입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Status`  
작업 그룹의 상태입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`SubnetIds`  
작업 그룹이 연결된 서브넷 ID의 배열입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`WorkgroupArn`  
작업 그룹에 연결되는 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`WorkgroupId`  
작업 그룹의 고유 식별자입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`WorkgroupName`  
작업 그룹의 이름입니다.  
*필수 항목 *여부: 아니요
