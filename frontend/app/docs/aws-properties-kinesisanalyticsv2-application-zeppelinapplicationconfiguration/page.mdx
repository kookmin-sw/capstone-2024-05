# AWS::KinesisAnalyticsV2::애플리케이션 ZeppelinApplicationConfiguration

Kinesis Data Analytics Studio 노트북의 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CatalogConfiguration" : CatalogConfiguration,
  "CustomArtifactsConfiguration" : [ CustomArtifactConfiguration, ... ],
  "DeployAsApplicationConfiguration" : DeployAsApplicationConfiguration,
  "MonitoringConfiguration" : ZeppelinMonitoringConfiguration
}
```

## 속성

`CatalogConfiguration`  
Kinesis Data Analytics Studio 노트북의 쿼리에 사용하는 Amazon Glue Data Catalog입니다.  
*필수 항목 *여부: 아니요  
_형식_: CatalogConfiguration  
_업데이트 필요 사항_: 중단 없음

`CustomArtifactsConfiguration`  
개체 목록입니다 `CustomArtifactConfiguration` .  
*필수 항목 *여부: 아니요  
_형식_: CustomArtifactConfiguration 목록  
_최대_: `50`  
_업데이트 필요 사항_: 중단 없음

`DeployAsApplicationConfiguration`  
Kinesis Data Analytics Studio 노트북을 내구성 상태의 애플리케이션으로 배포하는 데 필요한 정보입니다.  
*필수 항목 *여부: 아니요  
_형식_: DeployAsApplicationConfiguration  
_업데이트 필요 사항_: 중단 없음

`MonitoringConfiguration`  
Kinesis Data Analytics Studio 노트북의 모니터링 구성입니다.  
*필수 항목 *여부: 아니요  
_유형_: ZeppelinMonitoringConfiguration  
_업데이트 필요 사항_: 중단 없음
