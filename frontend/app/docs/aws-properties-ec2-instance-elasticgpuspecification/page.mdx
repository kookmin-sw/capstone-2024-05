# AWS::EC2::인스턴스 ElasticGpu사양

탄력적 GPU의 유형을 지정합니다. Elastic GPU는 애플리케이션의 그래픽 성능을 가속화하기 위해 Amazon EC2 인스턴스에 연결할 수 있는 GPU 리소스입니다. 자세한 내용은 _Windows 인스턴스용 Amazon EC2 사용 설명서의 Amazon EC2 Elastic GPU를 참조하십시오_.

`ElasticGpuSpecification` 은 AWS::EC2::Instance 리소스의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : String
}
```

## 속성

`Type`  
Elastic Graphics 액셀러레이터의 유형입니다. 에 지정할 값에 대한 자세한 내용은 `Type`Windows 인스턴스용 Amazon Elastic Compute Cloud 사용 설명서*의 Elastic Graphics 기본 사항, 특히 Elastic Graphics 액셀러레이터 열을 참조하십시오*.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
