# AWS::람다::별칭

리소스 `AWS::Lambda::Alias` 는 Lambda 함수 버전에 대한 별칭을 생성합니다. 별칭을 사용하여 다른 버전을 호출하기 위해 업데이트할 수 있는 함수 ID를 클라이언트에 제공합니다.

별칭을 매핑하여 호출 요청을 두 버전 간에 분할할 수도 있습니다. 이 `RoutingConfig` 파라미터를 사용하여 두 번째 버전과 수신하는 호출 요청의 백분율을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Lambda::Alias",
  "Properties" : {
      "Description" : String,
      "FunctionName" : String,
      "FunctionVersion" : String,
      "Name" : String,
      "ProvisionedConcurrencyConfig" : ProvisionedConcurrencyConfiguration,
      "RoutingConfig" : AliasRoutingConfiguration
    }
}
```

## 속성

`Description`  
별칭에 대한 설명입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `256`  
_업데이트 필요 사항_: 중단 없음

`FunctionName` Lambda 함수의 이름입니다.

**이름 형식**

- **함수 이름** - `MyFunction`.
- **함수 ARN** - `arn:aws:lambda:us-west-2:123456789012:function:MyFunction`.
- **부분 ARN** - `123456789012:function:MyFunction`.
  길이 제약은 전체 ARN에만 적용됩니다. 함수 이름만 지정하면 길이가 64자로 제한됩니다.  
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _최소_: `1`  
  _최대_: `140`  
  _패턴_: `(arn:(aws[a-zA-Z-]*)?:lambda:)?([a-z]{2}(-gov)?-[a-z]+-d{1}:)?(d{12}:)?(function:)?([a-zA-Z0-9-_]+)(:($LATEST|[a-zA-Z0-9-_]+))?`  
  _업데이트 필요 사항_: 바꿔 놓음

`FunctionVersion`  
별칭이 호출하는 함수 버전입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `1024`  
_패턴_: `($LATEST|[0-9]+)`  
_업데이트 필요 사항_: 중단 없음

`Name`  
별칭의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `128`  
_패턴_: `(?!^[0-9]+$)([a-zA-Z0-9-_]+)`  
_업데이트 필요 사항_: 바꿔 놓음

`ProvisionedConcurrencyConfig`  
함수의 별칭에 대해 프로비저닝된 동시성 구성을 지정합니다.  
*필수 항목 *여부: 아니요  
_형식_: ProvisionedConcurrencyConfiguration  
_업데이트 필요 사항_: 중단 없음

`RoutingConfig`  
별칭의 라우팅 구성입니다.  
*필수 항목 *여부: 아니요  
_유형_: AliasRoutingConfiguration  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`리소스 ARN을 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.
