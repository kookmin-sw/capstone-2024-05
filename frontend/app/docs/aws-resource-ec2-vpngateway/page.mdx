# AWS::EC2::VPNGateway

가상 프라이빗 게이트웨이를 지정합니다. 가상 프라이빗 게이트웨이는 VPN 연결의 VPC 측 엔드포인트입니다. VPC 자체를 생성하기 전에 가상 프라이빗 게이트웨이를 생성할 수 있습니다.

자세한 내용은 AWS Site-to-Site VPN 사용 설명서의 AWS Site-to-Site* VPN을 참조하세요*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::VPNGateway",
  "Properties" : {
      "AmazonSideAsn" : Integer,
      "Tags" : [Tag, ... ],
      "Type" : String
    }
}
```

## 속성

`AmazonSideAsn`  
BGP 세션의 Amazon 측에 대한 프라이빗 자율 시스템 번호(ASN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
가상 프라이빗 게이트웨이에 할당된 모든 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

`Type`  
가상 프라이빗 게이트웨이가 지원하는 VPN 연결 유형입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `ipsec.1`  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`VPN 게이트웨이의 ID를 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`VPNGatewayId` VPN 게이트웨이의 ID입니다.

## 예제

### VPN 게이트웨이

다음 예에서는 IPSec 1을 사용하는 VPN 게이트웨이를 선언합니다.

#### JSON (영문)

```
"myVPNGateway" : {
   "Type" : "AWS::EC2::VPNGateway",
   "Properties" : {
      "Type" : "ipsec.1",
      "Tags" : [ { "Key" : "Use", "Value" : "Test" } ]
  }
}
```
