# AWS::S3::버킷 알림 필터

객체 키 이름 필터링 규칙을 지정합니다. 키 이름 필터링에 대한 자세한 내용은 Amazon S3 사용 설명서의 객체 키 이름 필터링을 사용하여 이벤트 알림 구성을 참조하세요\*\*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "S3Key" : S3KeyFilter
}
```

## 속성

`S3Key`  
객체 키 이름, 접두사 및 접미사 필터링 규칙의 컨테이너입니다.  
_필수 항목 여부_: 예  
_형식_: S3KeyFilter  
_업데이트 필요 사항_: 중단 없음
