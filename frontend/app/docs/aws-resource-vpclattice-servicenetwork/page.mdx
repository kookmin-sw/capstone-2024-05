# AWS::VpcLattice::서비스 네트워크

서비스 네트워크를 생성합니다. 서비스 네트워크는 서비스 모음에 대한 논리적 경계입니다. 서비스 및 VPC를 서비스 네트워크와 연결할 수 있습니다.

자세한 내용은 Amazon VPC Lattice 사용 설명서*의 서비스 네트워크를 참조하세요*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::VpcLattice::ServiceNetwork",
  "Properties" : {
      "AuthType" : String,
      "Name" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`AuthType` IAM 정책의 유형입니다.

- `NONE`: 리소스가 IAM 정책을 사용하지 않습니다. 이것이 기본값입니다.
- `AWS_IAM`: 리소스가 IAM 정책을 사용합니다. 이 유형을 사용하면 인증이 활성화되고 인증 정책이 필요합니다.
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _업데이트 필요 사항_: 중단 없음

`Name`  
서비스 네트워크의 이름입니다. 이름은 계정에 고유해야 합니다. 유효한 문자는 a-z, 0-9 및 하이픈(-)입니다. 하이픈은 첫 번째 또는 마지막 문자로 사용하거나 다른 하이픈 바로 뒤에 사용할 수 없습니다.  
이름을 지정하지 않으면 CloudFormation에서 이름을 생성합니다. 그러나 이름을 지정하고 나중에 자원을 바꾸려면 새 이름을 지정해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
서비스 네트워크에 대한 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`서비스 `Ref`네트워크의 Amazon 리소스 이름(ARN)을 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Arn` 서비스 네트워크의 Amazon 리소스 이름(ARN)입니다.

`CreatedAt` 서비스 네트워크가 생성된 날짜 및 시간으로, ISO-8601 형식으로 지정됩니다.

`Id` 서비스 네트워크의 ID입니다.

`LastUpdatedAt` ISO-8601 형식으로 지정된 마지막 업데이트 날짜 및 시간입니다.
