# AWS::EC2::SpotFleet 액셀러레이터 수 요청

인스턴스에 있는 액셀러레이터(GPU, FPGA 또는 AWS Inferentia 칩)의 최소 및 최대 수입니다. 액셀러레이터를 사용하는 인스턴스 유형을 제외하려면 `Max` 로 설정합니다`0`.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Max" : Integer,
  "Min" : Integer
}
```

## 속성

`Max`  
액셀러레이터의 최대 수입니다. 최대 제한을 지정하지 않으려면 이 매개 변수를 생략합니다. 액셀러레이터를 사용하는 인스턴스 유형을 제외하려면 `Max` 로 설정합니다`0`.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음

`Min`  
액셀러레이터의 최소 수입니다. 최소 제한을 지정하지 않으려면 이 매개 변수를 생략합니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음
