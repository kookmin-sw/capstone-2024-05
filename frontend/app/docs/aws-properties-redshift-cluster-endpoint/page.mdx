# AWS::Redshift::클러스터 엔드포인트

연결 끝점에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Address" : String,
  "Port" : String
}
```

## 속성

`Address`  
클러스터의 DNS 주소입니다. 이 속성은 읽기 전용입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 중단 없음

`Port`  
데이터베이스 엔진이 수신 대기 중인 포트입니다. 이 속성은 읽기 전용입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
