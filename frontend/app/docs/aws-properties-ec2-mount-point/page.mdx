# AWS::EC2::인스턴스 볼륨

인스턴스에 연결할 볼륨을 지정합니다.

`Volume` 는 AWS::EC2::Instance 리소스의 임베디드 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Device" : String,
  "VolumeId" : String
}
```

## 속성

`Device`  
장치 이름(예: `/dev/sdh` 또는 `xvdh`).  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`VolumeId`  
EBS 볼륨의 ID입니다. 볼륨과 인스턴스는 동일한 가용 영역 내에 있어야 합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
