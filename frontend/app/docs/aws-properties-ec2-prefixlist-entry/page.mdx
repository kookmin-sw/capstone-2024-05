# AWS::EC2::P refixList 항목

접두사 목록에 대한 항목입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Cidr" : String,
  "Description" : String
}
```

## 속성

`Cidr`  
CIDR 블록입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Description`  
항목에 대한 설명입니다.  
제약 조건: 최대 255자까지 입력할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
