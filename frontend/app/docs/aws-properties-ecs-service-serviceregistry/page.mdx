# AWS::ECS::서비스 서비스 레지스트리

이 특성은 `ServiceRegistry` 서비스 레지스트리의 세부사항을 지정합니다. 자세한 내용은 _Amazon Elastic Container Service 개발자 안내서의 서비스 검색을 참조하세요_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ContainerName" : String,
  "ContainerPort" : Integer,
  "Port" : Integer,
  "RegistryArn" : String
}
```

## 속성

`ContainerName`  
서비스 검색 서비스에 사용할 컨테이너 이름 값입니다. 작업 정의에 이미 지정되어 있습니다. 서비스 작업이 지정하는 작업 정의가 `bridge` or 네트워크 모드를 사용하는 경우 작업 정의에서 `host` and 조합을 `containerName` 지정해야 합니다 `containerPort` . 서비스 작업에서 지정하는 작업 정의가 `awsvpc` 네트워크 모드를 사용하고 유형 SRV DNS 레코드가 사용되는 경우 `containerName` 및 `containerPort` 조합 또는 `port` 값을 지정해야 합니다 . 그러나 둘 다 지정할 수는 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ContainerPort`  
서비스 검색 서비스에 사용할 포트 값입니다. 작업 정의에 이미 지정되어 있습니다. 서비스 작업이 지정하는 작업 정의가 `bridge` or 네트워크 모드를 사용하는 경우 작업 정의에서 `host` and 조합을 `containerName` 지정해야 합니다 `containerPort` . 서비스 작업이 지정하는 작업 정의가 `awsvpc` 네트워크 모드를 사용하고 유형 SRV DNS 레코드가 사용되는 경우 `containerName` 및 `containerPort` 조합 또는 `port` 값을 지정해야 합니다 . 그러나 둘 다 지정할 수는 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Port`  
서비스 검색 서비스에서 SRV 레코드를 지정한 경우 사용되는 포트 값입니다. `awsvpc` 네트워크 모드와 SRV 레코드를 모두 사용하는 경우 이 필드를 사용할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`RegistryArn`  
서비스 레지스트리의 Amazon 리소스 이름(ARN)입니다. 현재 지원되는 서비스 레지스트리는 AWS Cloud Map입니다. 자세한 내용은 CreateService를 참조하세요.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
