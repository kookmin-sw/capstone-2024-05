# AWS::Route53RecoveryControl::제어판

Amazon Route 53 Application Recovery Controller에 새 제어판을 생성합니다. 제어판은 단일 트랜잭션에서 함께 변경할 수 있는 라우팅 제어 그룹을 나타냅니다. 제어판을 사용하여 조직 전체의 애플리케이션 운영 상태를 중앙에서 확인하고, 단일 트랜잭션에서 다중 앱 장애 조치를 트리거할 수 있습니다(예: 한 AWS 리전(셀)에서 다른 리전(셀)으로 장애 조치).

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Route53RecoveryControl::ControlPanel",
  "Properties" : {
      "ClusterArn" : String,
      "Name" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`ClusterArn`  
제어판에 대한 클러스터의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Name`  
제어판의 이름입니다. 이름에 공백이 아닌 문자를 사용할 수 있습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Tags`  
태그의 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`개체 `Ref`가 반환됩니다 `ControlPanelArn` .

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`ControlPanelArn` 제어판의 Amazon 리소스 이름(ARN)입니다.

`DefaultControlPanel` 클러스터의 기본 제어판에 대해 true로 설정된 부울 플래그입니다.

`RoutingControlCount` 제어판에 있는 라우팅 제어의 수입니다.

`Status` 제어판의 배포 상태입니다. 상태는 PENDING, DEPLOYED PENDING_DELETION 중 하나일 수 있습니다.
