# AWS::SageMaker::ModelPackage 지표소스

메트릭 소스에 대한 세부 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ContentDigest" : String,
  "ContentType" : String,
  "S3Uri" : String
}
```

## 속성

`ContentDigest`  
메트릭 소스에 사용되는 해시 키입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `72`  
_패턴_: `^[Ss][Hh][Aa]256:[0-9a-fA-F]{64}$`  
_업데이트 필요 사항_: 바꿔 놓음

`ContentType`  
메트릭 원본 콘텐츠 형식입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `256`  
_패턴_: `.*`  
_업데이트 필요 사항_: 바꿔 놓음

`S3Uri`  
지표 소스의 S3 URI입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `1024`  
_패턴_: `^(https|s3)://([^/]+)/?(.*)$`  
_업데이트 필요 사항_: 바꿔 놓음
