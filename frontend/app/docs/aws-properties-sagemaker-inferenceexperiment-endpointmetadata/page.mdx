# AWS::SageMaker::InferenceExperiment 엔드포인트메타데이터

엔드포인트의 메타데이터입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "EndpointConfigName" : String,
  "EndpointName" : String,
  "EndpointStatus" : String
}
```

## 속성

`EndpointConfigName`  
끝점 구성의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}`  
_업데이트 필요 사항_: 중단 없음

`EndpointName`  
끝점의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}`  
_업데이트 필요 사항_: 중단 없음

`EndpointStatus`  
 엔드포인트의 상태입니다. 엔드포인트 상태의 가능한 값은 some-address를 참조하세요.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `Creating | Deleting | Failed | InService | OutOfService | RollingBack | SystemUpdating | UpdateRollbackFailed | Updating`  
_업데이트 필요 사항_: 중단 없음
