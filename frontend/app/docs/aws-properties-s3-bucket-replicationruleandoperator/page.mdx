# AWS::S3::버킷 ReplicationRuleAnd연산자

규칙 필터를 지정하기 위한 컨테이너입니다. 필터는 규칙이 적용되는 개체의 하위 집합을 결정합니다. 이 요소는 둘 이상의 필터를 지정하는 경우에만 필요합니다.

예를 들어:

- a와 a 를 `Prefix`모두 지정하는 경우 `TagFilter`이러한 필터를 태그로 래핑합니다 `And` .
- 여러 태그를 기반으로 필터를 지정하는 경우 태그의 요소를 래핑 `TagFilter` 합니다 `And`

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Prefix" : String,
  "TagFilters" : [ TagFilter, ... ]
}
```

## 속성

`Prefix`  
규칙이 적용되는 객체의 하위 집합을 식별하는 객체 키 이름 접두사입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`TagFilters`  
키 및 값 쌍을 포함하는 태그의 배열입니다.  
*필수 항목 *여부: 아니요  
_유형_: TagFilter 목록  
_업데이트 필요 사항_: 중단 없음
