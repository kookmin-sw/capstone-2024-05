# AWS::SageMaker::ModelCard 함수

기능 세부 정보.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Condition" : String,
  "Facet" : String,
  "Function" : String
}
```

## 속성

`Condition`  
목적 함수 지표의 조건에 대한 설명(선택 사항)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Facet`  
모델의 목적 함수에 대한 메트릭입니다. 예를 들어, _loss_ 또는 *rmse*입니다. 다음 목록에는 메트릭에 지정할 수 있는 값의 예가 나와 있습니다.

- `ACCURACY`
- `AUC`
- `LOSS`
- `MAE`
- `RMSE`
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _업데이트 필요 사항_: 중단 없음

`Function` 모델 목적 함수의 최적화 방향입니다. 다음 값 중 하나를 지정해야 합니다.

- `Maximize`
- `Minimize`
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _업데이트 필요 사항_: 중단 없음
