# AWS::ElasticBeanstalk::애플리케이션 MaxAgeRule

지정된 일 수 후에 애플리케이션 버전을 삭제하는 수명 주기 규칙입니다.

`MaxAgeRule` 는 ApplicationVersionLifecycleConfig 속성 형식의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DeleteSourceFromS3" : Boolean,
  "Enabled" : Boolean,
  "MaxAgeInDays" : Integer
}
```

## 속성

`DeleteSourceFromS3`  
`true` Elastic Beanstalk가 애플리케이션 버전을 삭제할 때 Amazon S3에서 버전의 소스 번들을 삭제하려면 로 설정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`Enabled`  
`true` 규칙을 적용하거나 `false` 사용하지 않도록 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`MaxAgeInDays`  
응용 프로그램 버전을 보존할 일 수를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
