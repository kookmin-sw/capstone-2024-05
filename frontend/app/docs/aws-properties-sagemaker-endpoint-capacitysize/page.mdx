# AWS::SageMaker::엔드포인트 용량 크기

블루/그린 배포, 롤링 배포 또는 롤백 전략에 대해 활성화할 엔드포인트 용량의 유형과 크기를 지정합니다. 배치를 인스턴스 수, 전체 백분율 또는 플릿으로 지정할 수 있습니다.

롤백 전략의 경우 이 객체의 필드를 지정하지 않거나 `Value` 100%로 설정하면 SageMaker는 블루/그린 롤백 전략을 사용하고 모든 트래픽을 블루 플릿으로 롤백합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : String,
  "Value" : Integer
}
```

## 속성

`Type` 끝점 용량 유형을 지정합니다.

- `INSTANCE_COUNT`: 엔드포인트는 인스턴스 수에 따라 활성화됩니다.
- `CAPACITY_PERCENT`: 엔드포인트는 지정된 용량 비율에 따라 활성화됩니다.
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _허용되는 값_: `CAPACITY_PERCENT | INSTANCE_COUNT`  
  _업데이트 필요 사항_: 중단 없음

`Value`  
용량 크기를 인스턴스 수 또는 용량 백분율로 정의합니다.  
_필수 항목 여부_: 예  
_유형_: Integer  
_최소_: `1`  
_업데이트 필요 사항_: 중단 없음
