# AWS::SageMaker::스페이스

도메인에서 실시간 공동 작업에 사용되는 공간을 만듭니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::SageMaker::Space",
  "Properties" : {
      "DomainId" : String,
      "SpaceName" : String,
      "SpaceSettings" : SpaceSettings,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`DomainId`  
연결된 도메인의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `63`  
_업데이트 필요 사항_: 바꿔 놓음

`SpaceName`  
공간의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}`  
_업데이트 필요 사항_: 바꿔 놓음

`SpaceSettings`  
공간 설정 모음입니다.  
*필수 항목 *여부: 아니요  
_유형_: SpaceSettings  
_업데이트 필요 사항_: 중단 없음

`Tags`  
이 리소스에 적용할 키-값 쌍의 배열입니다.  
자세한 내용은 태그를 참조하십시오.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_최대_: `50`  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`SpaceArn` 공간의 Amazon 리소스 이름(ARN)입니다.
