# AWS::VpcLattice::리스너 전달

순방향 동작입니다. 규칙과 일치하는 트래픽은 지정된 대상 그룹으로 전달됩니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "TargetGroups" : [ WeightedTargetGroup, ... ]
}
```

## 속성

`TargetGroups`  
대상 그룹입니다. 규칙과 일치하는 트래픽은 지정된 대상 그룹으로 전달됩니다. 순방향 작업을 사용하면 각 대상 그룹의 우선 순위 지정 및 선택을 제어하는 가중치를 할당할 수 있습니다. 즉, 요청은 가중치에 따라 개별 대상 그룹에 배포됩니다. 예를 들어, 두 대상 그룹의 가중치가 동일한 경우 각 대상 그룹은 트래픽의 절반을 수신합니다.  
기본값은 1입니다. 즉, 하나의 대상 그룹만 제공되는 경우 가중치를 설정할 필요가 없습니다. 트래픽의 100%가 해당 대상 그룹으로 이동합니다.  
_필수 항목 여부_: 예  
_유형_: WeightedTargetGroup 목록  
_업데이트 필요 사항_: 중단 없음
