# AWS::SageMaker::InferenceExperiment DataStorageConfig

추론 요청 및 응답 데이터를 저장하기 위한 Amazon S3 위치 및 구성입니다.

데이터 캡처에 사용할 수 있는 선택적 매개 변수입니다. 자세한 내용은 데이터 캡처를 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ContentType" : CaptureContentTypeHeader,
  "Destination" : String,
  "KmsKey" : String
}
```

## 속성

`ContentType`  
다른 헤더를 처리하는 방법을 지정하는 구성입니다. 헤더가 지정되지 않은 경우 SageMaker는 데이터를 캡처할 때 기본적으로 base64로 인코딩합니다.  
*필수 항목 *여부: 아니요  
_형식_: CaptureContentTypeHeader  
_업데이트 필요 사항_: 중단 없음

`Destination`  
추론 요청 및 응답 데이터가 저장되는 Amazon S3 버킷입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `512`  
_패턴_: `^(https|s3)://([^/])/?(.*)$`  
_업데이트 필요 사항_: 중단 없음

`KmsKey`  
 Amazon SageMaker가 Amazon S3 서버 측 암호화를 사용하여 저장된 캡처된 데이터를 암호화하는 데 사용하는 AWS Key Management Service 키입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2048`  
_패턴_: `.*`  
_업데이트 필요 사항_: 중단 없음
