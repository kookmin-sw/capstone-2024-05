# AWS::Redshift::ClusterSecurityGroupIngress

Amazon Redshift 보안 그룹에 인바운드(수신) 규칙을 추가합니다. 클러스터에 액세스하는 애플리케이션이 인터넷에서 실행되는지 아니면 Amazon EC2 인스턴스에서 실행되는지에 따라 CIDR(Classless Interdomain Routing)/IP(Internet Protocol) 범위 또는 Amazon EC2 보안 그룹에 대한 인바운드 액세스 권한을 부여할 수 있습니다. Amazon Redshift 보안 그룹에 최대 20개의 수신 규칙을 추가할 수 있습니다.

Amazon EC2 보안 그룹에 대한 액세스 권한을 부여하는 경우 _EC2SecurityGroupName_ 및 *EC2SecurityGroupOwnerId*를 지정합니다. Amazon EC2 보안 그룹과 Amazon Redshift 클러스터는 동일한 AWS 리전에 있어야 합니다.

CIDR/IP 주소 범위에 대한 액세스 권한을 부여하는 경우 *CIDRIP*를 지정합니다. CIDR 블록에 대한 개요는 Classless Inter-Domain Routing에 대한 Wikipedia 문서를 참조하세요.

또한 이러한 IP 주소 또는 EC2 인스턴스에서 실행 중인 클라이언트가 클러스터에 연결할 수 있는 권한을 부여받을 수 있도록 보안 그룹을 클러스터와 연결해야 합니다. 보안 그룹 관리에 대한 자세한 내용은 _Amazon Redshift Cluster Management Guide의 보안 그룹 작업 단원을 참조하십시오_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Redshift::ClusterSecurityGroupIngress",
  "Properties" : {
      "CIDRIP" : String,
      "ClusterSecurityGroupName" : String,
      "EC2SecurityGroupName" : String,
      "EC2SecurityGroupOwnerId" : String
    }
}
```

## 속성

`CIDRIP`  
Amazon Redshift 보안 그룹에 추가할 IP 범위입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 바꿔 놓음

`ClusterSecurityGroupName`  
수신 규칙이 추가되는 보안 그룹의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 바꿔 놓음

`EC2SecurityGroupName`  
추가할 EC2 보안 그룹, Amazon Redshift 보안 그룹입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 바꿔 놓음

`EC2SecurityGroupOwnerId`  
_EC2SecurityGroupName_ 파라미터로 지정된 보안 그룹 소유자의 AWS 계정 번호입니다 . AWS 액세스 키 ID는 허용되는 값이 아닙니다.  
본보기: `111122223333`  
조건부. 특성을 지정하는 경우 `EC2SecurityGroupName` 이 특성을 지정해야 합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 바꿔 놓음

## 예제

### Ingress 규칙

다음 코드 조각은 Amazon Redshift 클러스터 보안 그룹에 대한 수신 규칙을 설명합니다.

#### JSON (영문)

```
"myClusterSecurityGroupIngressIP" : {
  "Type": "AWS::Redshift::ClusterSecurityGroupIngress",
  "Properties": {
    "ClusterSecurityGroupName" : {"Ref":"myClusterSecurityGroup"},
    "CIDRIP" : "10.0.0.0/16"
  }
}
```
