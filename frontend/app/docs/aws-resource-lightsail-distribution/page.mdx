# AWS::Lightsail::D istribution

리소스는 `AWS::Lightsail::Distribution` CDN(콘텐츠 배달 네트워크) 배포를 지정합니다. AWS 리전에서만 배포를 생성할 수 있습니다 `us-east-1` .

배포는 전 세계에 분산된 캐싱 서버 네트워크로, Lightsail 인스턴스에서 호스팅되는 웹 사이트 또는 웹 애플리케이션, Lightsail 버킷에서 호스팅되는 정적 콘텐츠 또는 Lightsail 로드 밸런서를 통해 호스팅되는 웹 사이트의 성능을 개선합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Lightsail::Distribution",
  "Properties" : {
      "BundleId" : String,
      "CacheBehaviors" : [ CacheBehaviorPerPath, ... ],
      "CacheBehaviorSettings" : CacheSettings,
      "CertificateName" : String,
      "DefaultCacheBehavior" : CacheBehavior,
      "DistributionName" : String,
      "IpAddressType" : String,
      "IsEnabled" : Boolean,
      "Origin" : InputOrigin,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`BundleId`  
배포에 적용된 번들의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`CacheBehaviors`  
배포의 경로별 캐시 동작을 설명하는 객체의 배열입니다.  
*필수 항목 *여부: 아니요  
_유형_: CacheBehaviorPerPath 목록  
_업데이트 필요 사항_: 중단 없음

`CacheBehaviorSettings`  
배포의 캐시 동작 설정을 설명하는 객체입니다.  
*필수 항목 *여부: 아니요  
_형식_: CacheSettings  
_업데이트 필요 사항_: 중단 없음

`CertificateName`  
배포에 연결된 SSL/TLS 인증서의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_패턴_: `w[w-]*w`  
_업데이트 필요 사항_: 중단 없음

`DefaultCacheBehavior`  
배포의 기본 캐시 동작을 설명하는 객체입니다.  
_필수 항목 여부_: 예  
_형식_: CacheBehavior  
_업데이트 필요 사항_: 중단 없음

`DistributionName`  
배포의 이름  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`IpAddressType`  
배포의 IP 주소 유형입니다.  
가능한 값은 `ipv4` IPv4 전용과 `dualstack` IPv4 및 IPv6입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `dualstack | ipv4`  
_업데이트 필요_: 업데이트가 지원되지 않습니다.

`IsEnabled`  
배포를 사용할 수 있는지 여부를 나타내는 부울 값입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`Origin`  
배포의 오리진 리소스(예: Lightsail 인스턴스, 버킷 또는 로드 밸런서)를 설명하는 객체입니다.  
배포는 오리진에서 콘텐츠를 가져오고, 캐시하고, 제공합니다.  
_필수 항목 여부_: 예  
_형식_: InputOrigin  
_업데이트 필요 사항_: 중단 없음

`Tags`  
이 리소스에 적용할 키-값 쌍의 배열입니다.  
자세한 내용은 _AWS CloudFormation 사용 설명서의 태그를 참조하세요_.  
`Value` of `Tags` 는 Lightsail 리소스의 선택 사항입니다.
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`이 `Ref`리소스에 대한 고유 식별자를 반환합니다.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`AbleToUpdateBundle` 배포의 현재 번들을 다른 번들로 업데이트할 수 있는지 여부를 나타냅니다.

`DistributionArn` 배포의 Amazon 리소스 이름(ARN)입니다.

`Status` 배포의 상태입니다.

## 발언

_캐시 동작 설정 구성_

`CacheBehaviorSettings` 매개 변수가 로 설정되어 `DefaultCacheBehavior`있거나 `cache` 매개 변수에 동작이 있는 경로가 있는 `CacheBehaviors` 경우에만 매개 변수를 설정할 수 `cache` 있습니다 . 이러한 조건 중 어느 것도 충족되지 `CacheBehaviorSettings` 않으면 배포에 대해 설정되지 않고 스택이 드리프트됩니다.
