# AWS::ECR::리포지토리 ImageScanningConfiguration

리포지토리에 대한 이미지 스캔 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ScanOnPush" : Boolean
}
```

## 속성

`ScanOnPush`  
이미지를 저장소로 푸시한 후 스캔할지 여부를 결정하는 설정입니다. 로 설정하면 `true`이미지가 푸시된 후 스캔됩니다. 이 매개 변수를 지정하지 않으면 기본적으로 스캔이 시작 `false` 되고 수동으로 스캔을 시작하지 않는 한 이미지가 스캔되지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
