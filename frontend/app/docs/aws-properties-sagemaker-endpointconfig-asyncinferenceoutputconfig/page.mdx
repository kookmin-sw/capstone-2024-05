# AWS::SageMaker::엔드포인트 구성 AsyncInferenceOutputConfig

비동기 유추 호출 출력에 대한 구성을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "KmsKeyId" : String,
  "NotificationConfig" : AsyncInferenceNotificationConfig,
  "S3FailurePath" : String,
  "S3OutputPath" : String
}
```

## 속성

`KmsKeyId`  
Amazon SageMaker가 Amazon S3의 비동기 추론 출력을 암호화하는 데 사용하는 AWS Key Management Service(AWS KMS) 키입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`NotificationConfig`  
비동기 유추에 대한 유추 결과 알림에 대한 구성을 지정합니다.  
*필수 항목 *여부: 아니요  
_형식_: AsyncInferenceNotificationConfig  
_업데이트 필요 사항_: 바꿔 놓음

`S3FailurePath`  
실패 추론 응답을 업로드할 Amazon S3 위치입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `512`  
_패턴_: `^(https|s3)://([^/])/?(.*)$`  
_업데이트 필요 사항_: 바꿔 놓음

`S3OutputPath`  
추론 응답을 업로드할 Amazon S3 위치입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
