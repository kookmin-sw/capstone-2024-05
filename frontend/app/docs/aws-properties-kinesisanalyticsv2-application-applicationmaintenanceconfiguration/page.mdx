# AWS::KinesisAnalyticsV2::애플리케이션 애플리케이션유지 관리 구성

Kinesis Data Analytics 애플리케이션에 대한 유지 관리 기간 파라미터를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ApplicationMaintenanceWindowStartTime" : String
}
```

## 속성

`ApplicationMaintenanceWindowStartTime`  
유지 관리 기간의 시작 시간을 지정합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
