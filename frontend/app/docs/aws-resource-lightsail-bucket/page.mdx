# AWS::Lightsail::버킷

리소스는 `AWS::Lightsail::Bucket` 버킷을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Lightsail::Bucket",
  "Properties" : {
      "AccessRules" : AccessRules,
      "BucketName" : String,
      "BundleId" : String,
      "ObjectVersioning" : Boolean,
      "ReadOnlyAccessAccounts" : [ String, ... ],
      "ResourcesReceivingAccess" : [ String, ... ],
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`AccessRules`  
버킷에 대한 액세스 규칙을 설명하는 객체입니다.  
*필수 항목 *여부: 아니요  
_형식_: AccessRules  
_업데이트 필요 사항_: 중단 없음

`BucketName`  
버킷의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`BundleId`  
버킷의 번들 ID(예: `small_1_0`).  
버킷 번들은 버킷의 월별 비용, 스토리지 공간 및 데이터 전송 할당량을 지정합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_패턴_: `.*S.*`  
_업데이트 필요 사항_: 중단 없음

`ObjectVersioning` 버킷에 대해 객체 버전 관리가 활성화되었는지 여부를 나타냅니다.  
다음 옵션을 구성할 수 있습니다.

- `Enabled` - 객체 버전 관리가 활성화되었습니다.
- `Suspended` - 객체 버전 관리는 이전에 사용 설정되었지만 현재 일시 중단되었습니다. 기존 객체 버전은 유지됩니다.
- `NeverEnabled` - 객체 버전 관리가 활성화된 적이 없습니다.
  *필수 항목 *여부: 아니요  
  _유형_: Boolean  
  _패턴_: `.*S.*`  
  _업데이트 필요 사항_: 중단 없음

`ReadOnlyAccessAccounts`  
버킷에 대한 읽기 전용 액세스 권한이 있는 AWS 계정 ID의 배열입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_최대_: `10`  
_업데이트 필요 사항_: 중단 없음

`ResourcesReceivingAccess`  
버킷에 액세스할 수 있는 Lightsail 인스턴스의 배열입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Tags`  
이 리소스에 적용할 키-값 쌍의 배열입니다.  
자세한 내용은 _AWS CloudFormation 사용 설명서의 태그를 참조하세요_.  
`Value` of `Tags` 는 Lightsail 리소스의 선택 사항입니다.
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`이 `Ref`리소스에 대한 고유 식별자를 반환합니다.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`AbleToUpdateBundle` 현재 배포에 적용된 번들을 다른 번들로 변경할 수 있는지 여부를 나타내는 부울 값입니다.

`BucketArn` 버킷의 Amazon 리소스 이름(ARN)입니다.

`Url` 버킷의 URL입니다.
