# AWS::EC2::NetworkInsights분석 포트 범위

포트 범위를 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "From" : Integer,
  "To" : Integer
}
```

## 속성

`From`  
범위의 첫 번째 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`To`  
범위의 마지막 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
