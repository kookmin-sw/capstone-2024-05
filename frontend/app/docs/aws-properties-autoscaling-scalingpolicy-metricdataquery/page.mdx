# AWS::AutoScaling::ScalingPolicy 지표 데이터 쿼리

반환할 메트릭 데이터입니다. 또한 이 호출이 하나의 지표에 대한 데이터만 반환하는지 또는 반환된 지표 통계의 값에 대해 수학 표현식을 수행하여 새 시계열을 생성하는지 여부를 정의합니다. 시계열은 일련의 데이터 포인트로, 각 포인트는 타임스탬프와 연결됩니다.

`MetricDataQuery` 은 다음 속성 유형의 속성입니다.

- AWS::AutoScaling::ScalingPolicy PredictiveScalingCustomizedScalingMetric
- AWS::AutoScaling::ScalingPolicy PredictiveScalingCustomizedLoadMetric
- AWS::AutoScaling::ScalingPolicy PredictiveScalingCustomizedCapacityMetric

예측 조정은 CloudWatch에서 수신한 시계열 데이터를 사용하여 과거 워크로드 패턴을 기반으로 용량을 예약하는 방법을 이해합니다.

단일 지표를 호출하거나 여러 지표에 대해 수학 표현식을 수행할 수 있습니다. 메트릭 사양에 사용되는 모든 표현식은 결국 단일 시계열을 반환해야 합니다.

자세한 내용과 예제는 Amazon EC2 Auto Scaling 사용 설명서*의 사용자 지정 지표를 사용한 고급 예측 조정 정책 구성을 참조하세요*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Expression" : String,
  "Id" : String,
  "Label" : String,
  "MetricStat" : MetricStat,
  "ReturnData" : Boolean
}
```

## 속성

`Expression`  
이 개체가 수학 식을 수행하는 경우 반환된 데이터에 대해 수행할 수학 식입니다. 이 표현식 `Id` 은 다른 지표의 를 사용하여 해당 지표를 참조할 수 있으며, `Id` 다른 표현식의 결과를 사용할 수도 있습니다.  
조건부: 각 객체 내에서 `MetricDataQuery` 또는 둘 중 하나를 지정해야 하지만 둘 다 지정해서는 안 됩니다 `Expression` `MetricStat`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `1023`  
_패턴_: `[u0020-uD7FFuE000-uFFFDuD800uDC00-uDBFFuDFFFrnt]*`  
_업데이트 필요 사항_: 중단 없음

`Id`  
응답에서 개체의 결과를 식별하는 짧은 이름입니다. 이 이름은 `MetricDataQuery` 단일 조정 정책에 대해 지정된 모든 객체에서 고유해야 합니다. 이 데이터 집합에 대해 수학 식을 수행하는 경우 이 이름은 해당 데이터를 나타내며 수학 식에서 변수로 사용할 수 있습니다. 유효한 문자는 문자, 숫자 및 밑줄입니다. 첫 번째 문자는 소문자여야 합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_패턴_: `[u0020-uD7FFuE000-uFFFDuD800uDC00-uDBFFuDFFFrnt]*`  
_업데이트 필요 사항_: 중단 없음

`Label`  
이 지표 또는 표현식에 대한 사람이 읽을 수 있는 레이블입니다. 이는 값이 무엇을 나타내는지 알 수 있도록 수학 표현식인 경우에 특히 유용합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2047`  
_패턴_: `[u0020-uD7FFuE000-uFFFDuD800uDC00-uDBFFuDFFFrnt]*`  
_업데이트 필요 사항_: 중단 없음

`MetricStat`  
반환할 메트릭 데이터에 대한 정보입니다.  
조건부: 각 객체 내에서 `MetricDataQuery` 또는 둘 중 하나를 지정해야 하지만 둘 다 지정해서는 안 됩니다 `Expression` `MetricStat`.  
*필수 항목 *여부: 아니요  
_유형_: MetricStat  
_업데이트 필요 사항_: 중단 없음

`ReturnData`  
이 메트릭의 타임스탬프 및 원시 데이터 값을 반환할지 여부를 나타냅니다.  
수학 표현식을 사용하는 경우 `true` 메트릭 사양의 기반이 되는 최종 수학 표현식에 대해서만 이 값을 지정합니다. `false` `ReturnData` 메트릭 사양에 사용되는 다른 모든 메트릭 및 표현식에 대해 for를 지정해야 합니다.  
지표만 검색하고 수학 표현식을 수행하지 않는 경우 에 대해 아무 것도 지정하지 마십시오`ReturnData`. 이렇게 하면 기본값()으로 설정됩니다`true`.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
