# AWS::EC2::인스턴스 엔클레이브 옵션

인스턴스가 AWS Nitro Enclaves에 대해 활성화되었는지 여부를 나타냅니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Enabled" : Boolean
}
```

## 속성

`Enabled`  
이 파라미터를 `true`로 설정하면 인스턴스가 AWS Nitro Enclaves에 대해 활성화되고, 그렇지 않으면 AWS Nitro Enclaves에 대해 활성화되지 않습니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
