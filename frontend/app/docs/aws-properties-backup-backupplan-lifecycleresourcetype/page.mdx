# AWS::백업::BackupPlan 수명 주기 리소스 유형

복구 지점이 콜드 스토리지로 전환되거나 삭제되기까지 걸리는 시간(일)을 결정하는 개체 배열이 `Transition` 포함된 개체를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DeleteAfterDays" : Double,
  "MoveToColdStorageAfterDays" : Double
}
```

## 속성

`DeleteAfterDays`  
복구 지점을 만든 후 삭제되는 기간(일)을 지정합니다. 보다 커야 합니다`MoveToColdStorageAfterDays`.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음

`MoveToColdStorageAfterDays`  
생성 후 복구 지점이 콜드 스토리지로 이동되는 기간(일)을 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음
