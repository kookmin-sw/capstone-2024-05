# AWS::S3ObjectLambda::액세스 포인트 정책 상태

이 액세스 포인트 정책이 공용인지 여부를 나타냅니다. Amazon S3가 정책을 평가하여 퍼블릭인지 여부를 결정하는 방법에 대한 자세한 내용은 _Amazon S3 사용 설명서의 "퍼블릭"의 의미를 참조하십시오_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "IsPublic" : Boolean
}
```

## 속성

`IsPublic`

*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
