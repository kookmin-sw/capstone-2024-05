# AWS::S3::버킷 데이터 내보내기

Amazon S3 버킷에 대한 스토리지 클래스 분석과 관련된 데이터를 내보내는 방법을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Destination" : Destination,
  "OutputSchemaVersion" : String
}
```

## 속성

`Destination`  
분석을 위해 데이터를 저장할 위치입니다.  
_필수 항목 여부_: 예  
_유형_: 대상  
_업데이트 필요 사항_: 중단 없음

`OutputSchemaVersion`  
데이터를 내보낼 때 사용할 출력 스키마의 버전입니다. 여야 합니다 `V_1`.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `V_1`  
_업데이트 필요 사항_: 중단 없음
