# AWS::CloudFront::P ublicKey PublicKeyConfig

서명된 URL 및 서명된 쿠키 또는 필드 레벨 암호화와 함께 사용할 수 있는 공개 키에 대한 구성 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CallerReference" : String,
  "Comment" : String,
  "EncodedKey" : String,
  "Name" : String
}
```

## 속성

`CallerReference`  
요청을 재생할 수 없도록 요청에 포함된 문자열입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Comment`  
공개 키를 설명하는 주석입니다. 주석은 128자를 초과할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`EncodedKey`  
서명된 URL 및 서명된 쿠키 또는 필드 레벨 암호화와 함께 사용할 수 있는 공개 키입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Name`  
공개 키를 식별하는 데 도움이 되는 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
