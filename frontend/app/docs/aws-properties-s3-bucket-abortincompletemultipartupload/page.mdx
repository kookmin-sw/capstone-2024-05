# AWS::S3::버킷 AbortIncompleteMultipartUpload

불완전한 멀티파트 업로드가 시작된 후 Amazon S3가 업로드의 모든 부분을 영구적으로 제거하기 전에 대기할 일 수를 지정합니다. 자세한 내용은 Amazon S3 사용 설명서의 버킷 수명 주기 정책을 사용하여 불완전한 멀티파트 업로드 중지를 참조하십시오\*\*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DaysAfterInitiation" : Integer
}
```

## 속성

`DaysAfterInitiation`  
Amazon S3가 불완전한 멀티파트 업로드를 중지할 때까지의 기간(일)을 지정합니다.  
_필수 항목 여부_: 예  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
