# AWS::SageMaker::D evice 디바이스

특정 장치의 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Description" : String,
  "DeviceName" : String,
  "IotThingName" : String
}
```

## 속성

`Description`  
장치에 대한 설명입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `40`  
_패턴_: `[Ss]+`  
_업데이트 필요 사항_: 중단 없음

`DeviceName`  
장치의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}$`  
_업데이트 필요 사항_: 바꿔 놓음

`IotThingName`  
AWS 사물 인터넷(IoT) 객체 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `128`  
_패턴_: `[a-zA-Z0-9:_-]+`  
_업데이트 필요 사항_: 중단 없음
