# AWS::KinesisFirehose::D eliveryStream OutputFormatConfiguration

Kinesis Data Firehose가 Amazon S3에 데이터를 쓰기 전에 데이터 형식을 변환하는 데 사용할 직렬 변환기를 지정합니다. true로 설정된 경우 이 매개 변수가 필요합니다 `Enabled` .

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Serializer" : Serializer
}
```

## 속성

`Serializer`  
사용할 serializer를 지정합니다. ORC SerDe 또는 Parquet SerDe를 선택할 수 있습니다. 둘 다 null이 아닌 경우 서버는 요청을 거부합니다.  
*필수 항목 *여부: 아니요  
_유형_: Serializer  
_업데이트 필요 사항_: 중단 없음
