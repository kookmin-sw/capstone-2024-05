# AWS::CloudFront::Monitoring구독

모니터링 구독입니다. 이 구조에는 지정된 CloudFront 배포에 대해 추가 CloudWatch 지표가 활성화되었는지 여부에 대한 정보가 포함되어 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::CloudFront::MonitoringSubscription",
  "Properties" : {
      "DistributionId" : String,
      "MonitoringSubscription" : MonitoringSubscription
    }
}
```

## 속성

`DistributionId`  
지표를 활성화하려는 배포의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`MonitoringSubscription`  
추가 CloudWatch 지표에 대한 구독 구성입니다.  
_필수 항목 여부_: 예  
_형식_: MonitoringSubscription  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`모니터링 구독에 대한 고유 식별자를 반환하며, 이 식별자는 구독이 속한 배포의 배포 ID와 동일합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.
