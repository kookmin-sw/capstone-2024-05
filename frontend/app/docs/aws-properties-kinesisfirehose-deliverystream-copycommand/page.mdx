# AWS::KinesisFirehose::D eliveryStream CopyCommand

`CopyCommand` 속성 유형은 `COPY` Amazon Kinesis Data Firehose(Kinesis Data Firehose)가 Amazon S3 버킷에서 Amazon Redshift 클러스터로 데이터를 로드하는 데 사용하는 Amazon Redshift 명령을 구성합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CopyOptions" : String,
  "DataTableColumns" : String,
  "DataTableName" : String
}
```

## 속성

`CopyOptions`  
Amazon Redshift 명령과 함께 사용할 파라미터입니다 `COPY` . 예제 는 `CopyOptions`Amazon Kinesis Data Firehose API Reference*의 CopyCommand 데이터 형식에 대한 내용을 참조하십시오*.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `204800`  
_패턴_: `.*`  
_업데이트 필요 사항_: 중단 없음

`DataTableColumns`  
쉼표로 구분된 열 이름 목록입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `204800`  
_패턴_: `.*`  
_업데이트 필요 사항_: 중단 없음

`DataTableName`  
대상 테이블의 이름입니다. 테이블이 데이터베이스에 이미 있어야 합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `512`  
_패턴_: `.*`  
_업데이트 필요 사항_: 중단 없음
