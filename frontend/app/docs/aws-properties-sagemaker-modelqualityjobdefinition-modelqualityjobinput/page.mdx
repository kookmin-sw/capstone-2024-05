# AWS::SageMaker::ModelQualityJobDefinition 모델 품질 작업 입력

모델 품질 모니터링 작업에 대한 입력입니다. 현재 엔드폰트는 모델 품질 모니터링 작업에 대한 입력이 지원됩니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BatchTransformInput" : BatchTransformInput,
  "EndpointInput" : EndpointInput,
  "GroundTruthS3Input" : MonitoringGroundTruthS3Input
}
```

## 속성

`BatchTransformInput`  
속성 설명을 사용할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: BatchTransformInput  
_업데이트 필요 사항_: 바꿔 놓음

`EndpointInput`  
끝점에 대한 입력 개체  
*필수 항목 *여부: 아니요  
_형식_: EndpointInput  
_업데이트 필요 사항_: 바꿔 놓음

`GroundTruthS3Input`  
모델에 제공된 실측 자료 레이블입니다.  
_필수 항목 여부_: 예  
_유형_: MonitoringGroundTruthS3Input  
_업데이트 필요 사항_: 바꿔 놓음
