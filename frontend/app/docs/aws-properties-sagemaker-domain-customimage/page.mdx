# AWS::SageMaker::D omain 사용자 지정 이미지

사용자 지정 SageMaker 이미지입니다. 자세한 정보는 자체 SageMaker 이미지 가져오기을 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AppImageConfigName" : String,
  "ImageName" : String,
  "ImageVersionNumber" : Integer
}
```

## 속성

`AppImageConfigName`  
AppImageConfig의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}`  
_업데이트 필요 사항_: 중단 없음

`ImageName`  
CustomImage의 이름입니다. 계정에 고유해야 합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}$`  
_업데이트 필요 사항_: 중단 없음

`ImageVersionNumber`  
CustomImage의 버전 번호입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `0`  
_업데이트 필요 사항_: 중단 없음
