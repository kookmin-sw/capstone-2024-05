# AWS::ECS::태스크 정의 호스트 항목

이 특성은 `HostEntry` 해당 자원`/etc/hosts`의 매개변수 를 `extraHosts` 통해 컨테이너의 파일에 `ContainerDefinition` 추가되는 호스트 이름 및 IP 주소를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Hostname" : String,
  "IpAddress" : String
}
```

## 속성

`Hostname`  
항목에 사용할 호스트 이름입니다 `/etc/hosts` .  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`IpAddress`  
항목에 사용할 IP 주소입니다 `/etc/hosts` .  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
