# AWS::EC2::VPNConnection VpnTunnelOptions사양

단일 VPN 터널에 대한 터널 옵션입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "PreSharedKey" : String,
  "TunnelInsideCidr" : String
}
```

## 속성

`PreSharedKey`  
가상 프라이빗 게이트웨이와 고객 게이트웨이 간에 초기 인증을 설정하기 위한 사전 공유 키(PSK)입니다.  
제약 조건: 허용되는 문자는 영숫자, 마침표(.) 및 밑줄(\_)입니다. 길이는 8자에서 64자 사이여야 하며 영(0)으로 시작할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`TunnelInsideCidr`  
터널의 내부 IP 주소 범위입니다. 지정된 CIDR 블록은 동일한 가상 프라이빗 게이트웨이를 사용하는 모든 VPN 연결에서 고유해야 합니다.  
제약 조건: 범위의 크기 /30 CIDR 블록입니다 `169.254.0.0/16` . 다음 CIDR 블록은 예약되어 있으며 사용할 수 없습니다.

- `169.254.0.0/30`
- `169.254.1.0/30`
- `169.254.2.0/30`
- `169.254.3.0/30`
- `169.254.4.0/30`
- `169.254.5.0/30`
- `169.254.169.252/30`
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _업데이트 필요 사항_: 바꿔 놓음
