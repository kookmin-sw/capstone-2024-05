# AWS::CloudFront::ResponseHeadersPolicy RemoveHeader

CloudFront가 이 응답 헤더 정책이 연결된 캐시 동작과 일치하는 요청에 대한 HTTP 응답에서 제거하는 HTTP 헤더의 이름입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Header" : String
}
```

## 속성

`Header`  
HTTP 헤더 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
