# AWS::D ynamoDB::GlobalTable ContributorInsights사양

복제본 또는 해당 인덱스 중 하나에 대한 Contributor Insights 설정을 구성합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Enabled" : Boolean
}
```

## 속성

`Enabled`  
CloudWatch Contributor Insights를 활성화할지(true) 또는 비활성화할지(false)를 나타냅니다.  
_필수 항목 여부_: 예  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
