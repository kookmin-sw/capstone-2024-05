# AWS::S3::Bucket EventBridgeConfiguration

Amazon S3는 버킷에서 특정 이벤트가 발생할 때마다 Amazon EventBridge로 이벤트를 전송할 수 있습니다(_Amazon S3 사용 설명서의 EventBridge 사용 참조_).

다른 대상과 달리 EventBridge로의 이벤트 전송은 버킷에 대해 활성화하거나 비활성화할 수 있습니다. 활성화하면 모든 이벤트가 EventBridge로 전송되며 EventBridge 규칙을 사용하여 이벤트를 추가 대상으로 라우팅할 수 있습니다. 자세한 내용은 _Amazon EventBridge 사용 설명서의 What Is Amazon EventBridge를 참조하세요_

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "EventBridgeEnabled" : Boolean
}
```

## 속성

`EventBridgeEnabled`  
Amazon EventBridge로 이벤트를 전송할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

## 예제

### EventBridgeConfiguration 활성화

다음 예제 템플릿은 EventBridge가 활성화된 알림 구성이 있는 Amazon S3 버킷을 보여줍니다.

#### JSON (영문)

```
{
"Resources": {
  "S3Bucket": {
    "Type": "AWS::S3::Bucket",
    "Properties": {
      "NotificationConfiguration": {
        "EventBridgeConfiguration": {
          "EventBridgeEnabled": true
          }
        }
      }
    }
  }
}
```
