# AWS::KinesisAnalyticsV2::애플리케이션 ZeppelinMonitoringConfiguration

Kinesis Data Analytics Studio 노트북에 대한 Amazon CloudWatch 로깅의 구성 파라미터에 대해 설명합니다. CloudWatch 로깅에 대한 자세한 내용은 모니터링 단원을 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "LogLevel" : String
}
```

## 속성

`LogLevel`  
애플리케이션에 대한 CloudWatch Logs의 세부 정보 표시 정도입니다. `INFO`, `WARN`, `ERROR`또는 로 설정할 수 있습니다`DEBUG`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_허용되는 값_: `DEBUG | ERROR | INFO | WARN`  
_업데이트 필요 사항_: 중단 없음
