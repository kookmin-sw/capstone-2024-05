# AWS::EC2::TransitGatewayRouteTablePropagation

지정된 연결이 지정된 전파 경로 테이블로 경로를 전파할 수 있도록 합니다.

전송 게이트웨이 경로 전파 활성화에 대한 자세한 내용은 Amazon EC2 API Reference의 EnableTransitGatewayRouteTablePropagation을 참조하세요\*\*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::TransitGatewayRouteTablePropagation",
  "Properties" : {
      "TransitGatewayAttachmentId" : String,
      "TransitGatewayRouteTableId" : String
    }
}
```

## 속성

`TransitGatewayAttachmentId`  
첨부 파일의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`TransitGatewayRouteTableId`  
전파 경로 테이블의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`전파된 전송 게이트웨이 라우팅 테이블의 ID를 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.
