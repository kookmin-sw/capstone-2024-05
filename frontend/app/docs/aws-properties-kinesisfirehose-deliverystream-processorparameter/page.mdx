# AWS::KinesisFirehose::D eliveryStream ProcessorParameter

이 `ProcessorParameter` 속성은 Amazon Kinesis Data Firehose 전송 스트림에 대한 데이터 프로세서의 프로세서 파라미터를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ParameterName" : String,
  "ParameterValue" : String
}
```

## 속성

`ParameterName`  
매개 변수의 이름입니다. 현재 지원되는 기본값은 3, `NumberOfRetries` 60 입니다`BufferIntervalInSeconds`. `BufferSizeInMBs` 범위는 0.2MB에서 최대 3MB 사이입니다. 기본 버퍼링 힌트는 Splunk를 제외한 모든 대상에 대해 1MB입니다. Splunk의 경우 기본 버퍼링 힌트는 256KB입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `BufferIntervalInSeconds | BufferSizeInMBs | Delimiter | JsonParsingEngine | LambdaArn | MetadataExtractionQuery | NumberOfRetries | RoleArn | SubRecordType`  
_업데이트 필요 사항_: 중단 없음

`ParameterValue`  
매개 변수 값입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `5120`  
_패턴_: `^(?!s*$).+`  
_업데이트 필요 사항_: 중단 없음
