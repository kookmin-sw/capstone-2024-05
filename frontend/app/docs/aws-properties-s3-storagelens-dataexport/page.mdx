# AWS::S3::StorageLens 데이터 내보내기

이 리소스에는 Amazon S3 Storage Lens 지표 내보내기에 대한 세부 정보가 포함되어 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CloudWatchMetrics" : CloudWatchMetrics,
  "S3BucketDestination" : S3BucketDestination
}
```

## 속성

`CloudWatchMetrics`  
이 속성은 S3 Storage Lens 지표에 대한 Amazon CloudWatch 게시 옵션을 활성화합니다.  
*필수 항목 *여부: 아니요  
_유형_: CloudWatchMetrics  
_업데이트 필요 사항_: 중단 없음

`S3BucketDestination`  
이 속성에는 S3 Storage Lens 지표 내보내기가 배치될 버킷의 세부 정보가 포함되어 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: S3BucketDestination  
_업데이트 필요 사항_: 중단 없음
