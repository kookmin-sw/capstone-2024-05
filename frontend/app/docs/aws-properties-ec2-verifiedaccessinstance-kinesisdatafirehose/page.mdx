# AWS::EC2::VerifiedAccessInstance KinesisDataFirehose

Kinesis를 로깅 대상으로 사용하는 옵션.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DeliveryStream" : String,
  "Enabled" : Boolean
}
```

## 속성

`DeliveryStream`  
전송 스트림의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Enabled`  
로깅을 사용할 수 있는지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
