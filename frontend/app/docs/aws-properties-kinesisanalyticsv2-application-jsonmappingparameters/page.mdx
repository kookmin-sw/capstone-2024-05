# AWS::KinesisAnalyticsV2::애플리케이션 JSONMapping파라미터

SQL 기반 Managed Service for Apache Flink 애플리케이션의 경우, JSON이 스트리밍 소스의 레코드 형식일 때 추가 맵핑 정보를 제공합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "RecordRowPath" : String
}
```

## 속성

`RecordRowPath`  
레코드를 포함하는 최상위 부모의 경로입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `65535`  
_패턴_: `^(?=^$)(?=^S+$).*$`  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

- Amazon Kinesis Data Analytics API 참조\* \*의 JSONMappingParameters
