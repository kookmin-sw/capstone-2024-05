# AWS::VpcLattice::대상 그룹 TargetGroupConfig

대상 그룹의 구성을 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "HealthCheck" : HealthCheckConfig,
  "IpAddressType" : String,
  "LambdaEventStructureVersion" : String,
  "Port" : Integer,
  "Protocol" : String,
  "ProtocolVersion" : String,
  "VpcIdentifier" : String
}
```

## 속성

`HealthCheck`  
상태 확인 구성입니다. 대상 그룹 유형이 또는 인 경우 지원되지 않습니다 `LAMBDA` `ALB`.  
*필수 항목 *여부: 아니요  
_유형_: HealthCheckConfig  
_업데이트 필요 사항_: 중단 없음

`IpAddressType`  
대상 그룹에 사용되는 IP 주소의 유형입니다. 대상 그룹 유형이 인 경우에만 지원됩니다`IP`. 기본값은 `IPV4`입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`LambdaEventStructureVersion`  
Lambda 함수가 수신하는 이벤트 구조의 버전입니다. 대상 그룹 유형이 인 경우에만 지원됩니다`LAMBDA`. 기본값은 `V1`입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Port`  
대상이 수신 대기 중인 포트입니다. HTTP의 경우 기본값은 80입니다. HTTPS의 경우 기본값은 443입니다. 대상 그룹 유형이 인 경우 지원되지 않습니다`LAMBDA`.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음

`Protocol`  
트래픽을 대상으로 라우팅하는 데 사용할 프로토콜입니다. 기본값은 대상 그룹의 프로토콜입니다. 대상 그룹 유형이 인 경우 지원되지 않습니다`LAMBDA`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`ProtocolVersion`  
프로토콜 버전입니다. 기본값은 `HTTP1`입니다. 대상 그룹 유형이 인 경우 지원되지 않습니다`LAMBDA`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`VpcIdentifier`  
VPC의 ID입니다. 대상 그룹 유형이 인 경우 지원되지 않습니다`LAMBDA`.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
