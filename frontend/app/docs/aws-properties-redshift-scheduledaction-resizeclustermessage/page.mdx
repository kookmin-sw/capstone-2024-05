# AWS::Redshift::ScheduledAction 크기 조정 클러스터 메시지

클러스터 크기 조정 작업에 대해 설명합니다. 예를 들어 API 작업을 실행하기 위한 예약된 작업이 있습니다 `ResizeCluster` .

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Classic" : Boolean,
  "ClusterIdentifier" : String,
  "ClusterType" : String,
  "NodeType" : String,
  "NumberOfNodes" : Integer
}
```

## 속성

`Classic`  
크기 조정 작업이 클래식 크기 조정 프로세스를 사용하는지 여부를 나타내는 부울 값입니다. 이 매개 변수를 제공하지 않거나 값을 로 설정하면 `false`크기 조정 유형이 탄력적입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`ClusterIdentifier`  
크기를 조정할 클러스터의 고유 식별자입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 중단 없음

`ClusterType`  
지정된 클러스터에 대한 새 클러스터 유형입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 중단 없음

`NodeType`  
추가할 노드의 새 노드 유형입니다. 지정하지 않으면 클러스터의 현재 노드 유형이 사용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 중단 없음

`NumberOfNodes`  
클러스터의 새 노드 수입니다. 지정하지 않으면 클러스터의 현재 노드 수가 사용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
