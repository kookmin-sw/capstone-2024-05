# AWS::EC2::인스턴스 연결 끝점

EC2 인스턴스 연결 엔드포인트를 생성합니다.

EC2 인스턴스 연결 엔드포인트를 사용하면 인스턴스에 퍼블릭 IPv4 주소가 없어도 인스턴스에 연결할 수 있습니다. 자세한 내용은 Amazon EC2 사용 설명서의 EC2 인스턴스 연결 엔드포인트를 사용하여 퍼블릭 IPv4 주소를 요구하지 않고 인스턴스에 연결을 참조하세요\*\*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::InstanceConnectEndpoint",
  "Properties" : {
      "ClientToken" : String,
      "PreserveClientIp" : Boolean,
      "SecurityGroupIds" : [ String, ... ],
      "SubnetId" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`ClientToken`  
요청의 멱등성을 보장하기 위해 제공하는 대/소문자를 구분하는 고유한 식별자입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`PreserveClientIp`  
클라이언트의 IP 주소가 원본으로 유지되는지 여부를 나타냅니다. 값은 `true` 또는 `false`입니다.

- 인 경우 `true`리소스에 연결할 때 클라이언트의 IP 주소가 사용됩니다.
- 인 경우 `false`리소스에 연결할 때 탄력적 네트워크 인터페이스 IP 주소가 사용됩니다.
  기본값: `true`  
  _필수_: 아니요  
  _유형_: Boolean  
  _업데이트 필요 사항_: 바꿔 놓음

`SecurityGroupIds`  
엔드포인트와 연결할 하나 이상의 보안 그룹입니다. 보안 그룹을 지정하지 않으면 VPC의 기본 보안 그룹이 엔드포인트와 연결됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_최대_: `16`  
_업데이트 필요 사항_: 바꿔 놓음

`SubnetId`  
EC2 인스턴스 연결 엔드포인트를 생성할 서브넷의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
생성 중에 EC2 인스턴스 연결 엔드포인트에 적용할 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`EC2 인스턴스 연결 엔드포인트의 ID가 반환됩니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Id` EC2 인스턴스 연결 엔드포인트의 ID입니다.
