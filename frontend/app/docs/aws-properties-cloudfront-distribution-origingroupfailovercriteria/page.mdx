# AWS::CloudFront::D istribution OriginGroupFailoverCriteria

CloudFront가 기본 오리진에서 두 번째 오리진으로 장애 조치할 상태 코드를 포함하여 오리진 그룹의 장애 조치 기준에 대한 정보를 포함하는 복잡한 데이터 형식입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "StatusCodes" : StatusCodes
}
```

## 속성

`StatusCodes`  
기본 오리진에서 반환될 때 CloudFront가 두 번째 오리진으로 장애 조치하도록 트리거하는 상태 코드입니다.  
_필수 항목 여부_: 예  
_형식_: StatusCodes  
_업데이트 필요 사항_: 중단 없음
