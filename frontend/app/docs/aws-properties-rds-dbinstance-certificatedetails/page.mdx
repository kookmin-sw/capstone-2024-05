# AWS::RDS::D BInstance 인증서 세부 정보

DB 인스턴스의 서버 인증서에 대한 세부 정보를 반환합니다.

자세한 내용은 Amazon RDS 사용 설명서*의 SSL/TLS를 사용하여 DB 인스턴스에 대한 연결 암호화 * 및 Amazon Aurora 사용 설명서*의 SSL/TLS를 사용하여 DB 클러스터에 대한 연결 암호화를 * 참조하세요.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CAIdentifier" : String,
  "ValidTill" : String
}
```

## 속성

`CAIdentifier`  
DB 인스턴스의 서버 인증서에 사용되는 CA 인증서의 CA 식별자입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ValidTill`  
DB 인스턴스 서버 인증서의 만료 날짜입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
