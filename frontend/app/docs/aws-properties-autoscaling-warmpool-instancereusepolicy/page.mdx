# AWS::AutoScaling::WarmPool 인스턴스 재사용 정책

AWS::AutoScaling::WarmPool 리소스의 속성`InstanceReusePolicy`에 대한 인스턴스 재사용 정책을 지정하는 구조 입니다.

자세한 내용은 Amazon EC2 Auto Scaling 사용 설명서*의 Amazon EC2 Auto Scaling용 웜 풀을 참조하세요*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ReuseOnScaleIn" : Boolean
}
```

## 속성

`ReuseOnScaleIn`  
Auto Scaling 그룹의 인스턴스를 축소 시 웜 풀로 반환할 수 있는지 여부를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
