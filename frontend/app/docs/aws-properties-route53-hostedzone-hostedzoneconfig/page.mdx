# AWS::Route53::HostedZone HostedZoneConfig

호스팅 영역에 대한 선택적 설명이 포함된 복합 유형입니다. 주석을 지정하지 않으려면 `HostedZoneConfig` 및 `Comment` 요소를 모두 생략합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Comment" : String
}
```

## 속성

`Comment`  
호스팅 영역에 대해 포함하려는 모든 설명입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `256`  
_업데이트 필요 사항_: 중단 없음
