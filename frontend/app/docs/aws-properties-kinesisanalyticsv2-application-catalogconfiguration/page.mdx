# AWS::KinesisAnalyticsV2::애플리케이션 카탈로그구성

기본 Amazon Glue 데이터베이스에 대한 구성 파라미터입니다. Kinesis Data Analytics Studio 노트북에서 작성하는 SQL 쿼리에 이 데이터베이스를 사용합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "GlueDataCatalogConfiguration" : GlueDataCatalogConfiguration
}
```

## 속성

`GlueDataCatalogConfiguration`  
기본 Amazon Glue 데이터베이스에 대한 구성 파라미터입니다. Kinesis Data Analytics Studio 노트북에 작성하는 Apache Flink SQL 쿼리 및 테이블 API 변환에 이 데이터베이스를 사용합니다.  
*필수 항목 *여부: 아니요  
_유형_: GlueDataCatalogConfiguration  
_업데이트 필요 사항_: 중단 없음
