# AWS::SageMaker::ModelPackage 검증 사양

SageMaker가 모델 패키지를 검증하기 위해 실행하는 배치 변환 작업을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ValidationProfiles" : [ ValidationProfile, ... ],
  "ValidationRole" : String
}
```

## 속성

`ValidationProfiles`  
객체 배열로 `ModelPackageValidationProfile` , 각 객체는 SageMaker가 모델 패키지를 검증하기 위해 실행하는 배치 변환 작업을 지정합니다.  
_필수 항목 여부_: 예  
_유형_: ValidationProfile 목록  
_최대_: `1`  
_업데이트 필요 사항_: 바꿔 놓음

`ValidationRole`  
모델 패키지의 검증에 사용할 IAM 역할입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `20`  
_최대_: `2048`  
_패턴_: `^arn:aws[a-z-]*:iam::d{12}:role/?[a-zA-Z_0-9+=,.@-_/]+$`  
_업데이트 필요 사항_: 바꿔 놓음
