# AWS::SageMaker::ModelCard TrainingHyperParameter

모델 학습에서 구성된 하이퍼 매개 변수입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Value" : String
}
```

## 속성

`Name`  
하이퍼 매개 변수의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Value`  
hyper 매개 변수에 대해 지정된 값입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
