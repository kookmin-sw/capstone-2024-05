# AWS::KinesisAnalytics::애플리케이션 입력병렬 처리

지정된 스트리밍 소스에 대해 만들 응용 프로그램 내 스트림의 수를 설명합니다. 병렬 처리에 대한 자세한 내용은 응용 프로그램 입력 구성을 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Count" : Integer
}
```

## 속성

`Count`  
생성할 인애플리케이션 스트림의 수입니다. 자세한 내용은 제한을 참조하십시오.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `1`  
_최대_: `64`  
_업데이트 필요 사항_: 중단 없음
