# AWS::D ynamoDB::테이블 PointInTimeRecovery사양

특정 시점 복구를 사용하도록 설정하는 데 사용되는 설정입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "PointInTimeRecoveryEnabled" : Boolean
}
```

## 속성

`PointInTimeRecoveryEnabled`  
테이블에서 특정 시점 복구가 활성화되었는지(true) 또는 비활성화되었는지(false)를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

PointInTimeRecoverySpecification을 참조하십시오.
