# AWS::SageMaker::FeatureGroup S3StorageConfig

에 대한 Amazon Simple Storage(Amazon S3) 위치 및 보안 구성입니다 `OfflineStore`.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "KmsKeyId" : String,
  "S3Uri" : String
}
```

## 속성

`KmsKeyId`  
S3 위치에 기록된 모든 객체를 암호화하는 데 사용되는 키의 AWS Key Management Service(KMS) 키 ARN입니다 `OfflineStore` .  
`roleARN` 매개 변수로 전달되는 IAM에는 `CreateFeatureGroup` 다음과 같은 권한이 있어야 합니다. `KmsKeyId`

- `"kms:GenerateDataKey"`
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _최대_: `2048`  
  _패턴_: `.*`  
  _업데이트 필요 사항_: 바꿔 놓음

`S3Uri`  
S3 URI 또는 Amazon S3의 위치 `OfflineStore`입니다.  
S3 URI의 형식은 다음과 유사합니다 `s3://example-bucket/prefix/`.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `1024`  
_패턴_: `^(https|s3)://([^/]+)/?(.*)$`  
_업데이트 필요 사항_: 바꿔 놓음
