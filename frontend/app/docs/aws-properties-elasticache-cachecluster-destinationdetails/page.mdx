# AWS::ElastiCache::CacheCluster 대상 세부 정보

CloudWatch Logs 대상 또는 Kinesis Data Firehose 대상의 구성 세부 정보입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CloudWatchLogsDetails" : CloudWatchLogsDestinationDetails,
  "KinesisFirehoseDetails" : KinesisFirehoseDestinationDetails
}
```

## 속성

`CloudWatchLogsDetails`  
CloudWatch Logs 대상의 구성 세부 정보입니다. 이 필드는 필수로 표시되지만 CloudWatch Logs가 대상으로 선택된 경우에만 표시됩니다.  
*필수 항목 *여부: 아니요  
_유형_: CloudWatchLogsDestinationDetails  
_업데이트 필요 사항_: 중단 없음

`KinesisFirehoseDetails`  
Kinesis Data Firehose 대상의 구성 세부 정보입니다. 이 필드는 Kinesis Data Firehose가 대상으로 선택된 경우에만 필수로 표시됩니다.  
*필수 항목 *여부: 아니요  
_유형_: KinesisFirehoseDestinationDetails  
_업데이트 필요 사항_: 중단 없음
