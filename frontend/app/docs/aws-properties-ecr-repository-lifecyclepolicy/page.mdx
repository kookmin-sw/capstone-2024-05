# AWS::ECR::리포지토리 수명 주기 정책

`LifecyclePolicy` 속성 유형은 수명 주기 정책을 지정합니다. 수명 주기 정책 구문에 대한 자세한 내용은 _Amazon ECR 사용 설명서의 수명 주기 정책 템플릿을 참조하세요_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "LifecyclePolicyText" : String,
  "RegistryId" : String
}
```

## 속성

`LifecyclePolicyText`  
리포지토리에 적용할 JSON 리포지토리 정책 텍스트입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `100`  
_최대_: `30720`  
_업데이트 필요 사항_: 중단 없음

`RegistryId`  
리포지토리가 포함된 레지스트리와 연결된 AWS 계정 ID입니다. 레지스트리를 지정하지 않으면 기본 레지스트리가 사용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_패턴_: `[0-9]{12}`  
_업데이트 필요 사항_: 중단 없음
