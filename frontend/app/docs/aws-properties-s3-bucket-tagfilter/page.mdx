# AWS::S3::버킷 태그 필터

Amazon S3 버킷에 대한 객체의 하위 집합을 식별하는 데 사용할 태그를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Key" : String,
  "Value" : String
}
```

## 속성

`Key`  
태그 키입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Value`  
태그 값입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
