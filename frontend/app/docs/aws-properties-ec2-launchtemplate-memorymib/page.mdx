# AWS::EC2::LaunchTemplate 메모리MiB

최소 및 최대 메모리 양(MiB)입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Max" : Integer,
  "Min" : Integer
}
```

## 속성

`Max`  
최대 메모리 양(MiB)입니다. 최대 제한을 지정하지 않으려면 이 매개 변수를 생략합니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`Min`  
최소 메모리 양(MiB)입니다. 최소 제한을 지정하지 않으려면 `0`을 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
