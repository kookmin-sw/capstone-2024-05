# AWS::Redshift::EndpointAccess Vpc보안 그룹

엔드포인트와 연결된 보안 그룹입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Status" : String,
  "VpcSecurityGroupId" : String
}
```

## 속성

`Status`  
엔드포인트의 상태입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 중단 없음

`VpcSecurityGroupId`  
VPC 보안 그룹의 식별자입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 중단 없음
