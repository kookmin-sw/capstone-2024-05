# AWS::ApiGatewayV2::ApiGatewayManagedOverrides 경로 설정

이 `RouteSettings` 속성은 API Gateway 관리형 라우팅에 대한 라우팅 설정을 재정의합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DataTraceEnabled" : Boolean,
  "DetailedMetricsEnabled" : Boolean,
  "LoggingLevel" : String,
  "ThrottlingBurstLimit" : Integer,
  "ThrottlingRateLimit" : Double
}
```

## 속성

`DataTraceEnabled`  
`true``false`이 경로에 대해 () 또는 () 데이터 추적 로깅을 사용할 수 있는지 여부를 지정합니다. 이 속성은 Amazon CloudWatch Logs에 푸시된 로그 항목에 영향을 줍니다. WebSocket API에 대해서만 지원됩니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`DetailedMetricsEnabled`  
세부 메트릭을 사용할 수 있는지 여부를 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`LoggingLevel`  
이 경로에 대한 로깅 수준을 지정합니다: `INFO`, `ERROR`, 또는 `OFF`. 이 속성은 Amazon CloudWatch Logs에 푸시된 로그 항목에 영향을 줍니다. WebSocket API에 대해서만 지원됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ThrottlingBurstLimit`  
제한 버스트 제한을 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`ThrottlingRateLimit`  
제한 속도 제한을 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음
