# AWS::SageMaker::ModelBiasJobDefinition ModelBiasBaselineConfig

기준 모델 바이어스 작업에 대한 구성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BaseliningJobName" : String,
  "ConstraintsResource" : ConstraintsResource
}
```

## 속성

`BaseliningJobName`  
기준 모델 편향 작업의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `1`  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}`  
_업데이트 필요 사항_: 바꿔 놓음

`ConstraintsResource`  
모니터링 작업에 대한 constraints 리소스입니다.  
*필수 항목 *여부: 아니요  
_유형_: ConstraintsResource  
_업데이트 필요 사항_: 바꿔 놓음
