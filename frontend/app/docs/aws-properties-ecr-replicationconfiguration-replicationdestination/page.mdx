# AWS::ECR::복제 구성 복제 대상

복제 규칙의 대상을 나타내는 개체의 배열입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Region" : String,
  "RegistryId" : String
}
```

## 속성

`Region`  
복제할 리전입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `2`  
_최대_: `25`  
_패턴_: `[0-9a-z-]{2,25}`  
_업데이트 필요 사항_: 중단 없음

`RegistryId`  
복제할 Amazon ECR 프라이빗 레지스트리의 AWS 계정 ID입니다. 자체 레지스트리 내에서 교차 리전 복제를 구성할 때 자체 계정 ID를 지정합니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_패턴_: `[0-9]{12}`  
_업데이트 필요 사항_: 중단 없음
