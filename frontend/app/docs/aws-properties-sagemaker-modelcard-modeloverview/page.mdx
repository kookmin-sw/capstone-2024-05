# AWS::SageMaker::ModelCard 모델 개요

모델에 대한 개요입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AlgorithmType" : String,
  "InferenceEnvironment" : InferenceEnvironment,
  "ModelArtifact" : [ String, ... ],
  "ModelCreator" : String,
  "ModelDescription" : String,
  "ModelId" : String,
  "ModelName" : String,
  "ModelOwner" : String,
  "ModelVersion" : Double,
  "ProblemType" : String
}
```

## 속성

`AlgorithmType`  
문제를 해결하는 데 사용되는 알고리즘입니다.  
*필수 항목 *여부: 아니요

_업데이트 필요 사항_: 중단 없음
