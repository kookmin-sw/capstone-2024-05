# AWS::CloudFront::ResponseHeadersPolicy 사용자 지정 헤더 구성

HTTP 응답 헤더 이름 및 해당 값의 목록입니다. CloudFront는 이 응답 헤더 정책과 연결된 캐시 동작과 일치하는 요청에 대해 보내는 HTTP 응답에 이러한 헤더를 포함합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Items" : [ CustomHeader, ... ]
}
```

## 속성

`Items`  
HTTP 응답 헤더 및 해당 값의 목록입니다.  
_필수 항목 여부_: 예  
_형식_: CustomHeader 목록  
_업데이트 필요 사항_: 중단 없음
