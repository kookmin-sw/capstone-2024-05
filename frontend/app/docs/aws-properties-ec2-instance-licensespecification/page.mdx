# AWS::EC2::인스턴스 라이선스 사양

사용할 라이센스 구성을 지정합니다.

`LicenseSpecification` 은 AWS::EC2::Instance 리소스의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "LicenseConfigurationArn" : String
}
```

## 속성

`LicenseConfigurationArn`  
라이선스 구성의 Amazon 리소스 이름(ARN)입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
