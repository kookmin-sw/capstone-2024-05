# AWS::CloudWatch::AnomalyDetector 범위

각각 `Range` 은 변칙 검색 모델을 학습하거나 업데이트하는 데 사용하지 않도록 제외할 일 또는 시간 범위를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "EndTime" : String,
  "StartTime" : String
}
```

## 속성

`EndTime`  
제외할 범위의 종료 시간입니다. 형식은 입니다 `yyyy-MM-dd'T'HH:mm:ss`. 예를 들어, `2019-07-01T23:59:59`.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`StartTime`  
제외할 범위의 시작 시간입니다. 형식은 입니다 `yyyy-MM-dd'T'HH:mm:ss`. 예를 들어, `2019-07-01T23:59:59`.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
