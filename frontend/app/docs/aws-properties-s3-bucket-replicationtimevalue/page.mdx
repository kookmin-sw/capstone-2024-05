# AWS::S3::버킷 ReplicationTimeValue

S3 복제 시간 제어(S3 RTC) 및 복제 지표에 대한 시간 값을 지정하는 컨테이너입니다 `EventThreshold`.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Minutes" : Integer
}
```

## 속성

`Minutes`  
 분 단위의 시간을 지정하는 정수를 포함합니다.  
 유효한 값: 15  
_필수 항목 여부_: 예  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
