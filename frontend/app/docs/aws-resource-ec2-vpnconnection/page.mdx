# AWS::EC2::VPNConnection

가상 프라이빗 게이트웨이와 VPN 고객 게이트웨이 또는 전송 게이트웨이와 VPN 고객 게이트웨이 간의 VPN 연결을 지정합니다.

Transit Gateway와 고객 게이트웨이 간의 VPN 연결을 지정하려면 `TransitGatewayId` and `CustomerGatewayId` 속성을 사용합니다.

가상 프라이빗 게이트웨이와 고객 게이트웨이 간의 VPN 연결을 지정하려면 `VpnGatewayId` and `CustomerGatewayId` 속성을 사용합니다.

자세한 내용은 AWS Site-to-Site VPN 사용 설명서의 AWS Site-to-Site* VPN을 참조하세요*.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::VPNConnection",
  "Properties" : {
      "CustomerGatewayId" : String,
      "StaticRoutesOnly" : Boolean,
      "Tags" : [Tag, ... ],
      "TransitGatewayId" : String,
      "Type" : String,
      "VpnGatewayId" : String,
      "VpnTunnelOptionsSpecifications" : [ VpnTunnelOptionsSpecification, ... ]
    }
}
```

## 속성

`CustomerGatewayId`  
VPN 연결 끝에 있는 고객 게이트웨이의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`StaticRoutesOnly`  
VPN 연결에서 고정 경로만 사용하는지 여부를 나타냅니다. BGP를 지원하지 않는 디바이스에는 고정 경로를 사용해야 합니다.  
BGP(Border Gateway Protocol)를 지원하지 않는 디바이스에 대한 VPN 연결을 생성하는 경우 를 지정해야 합니다`true`.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
VPN 연결에 할당된 모든 태그입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

`TransitGatewayId`  
VPN 연결과 연결된 전송 게이트웨이의 ID입니다.  
둘 중 하나를 지정해야 하지만 `TransitGatewayId` `VpnGatewayId`둘 다 지정해서는 안 됩니다.  
_필수_: 조건부  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Type`  
VPN 연결의 유형입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `ipsec.1`  
_업데이트 필요 사항_: 바꿔 놓음

`VpnGatewayId`  
VPN 연결의 AWS 측에 있는 가상 프라이빗 게이트웨이의 ID입니다.  
둘 중 하나를 지정해야 하지만 `TransitGatewayId` `VpnGatewayId`둘 다 지정해서는 안 됩니다.  
_필수_: 조건부  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`VpnTunnelOptionsSpecifications`  
VPN 연결을 위한 터널 옵션입니다.  
*필수 항목 *여부: 아니요  
_유형_: VpnTunnelOptionsSpecification 목록  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`VPN 연결의 ID가 반환됩니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`VpnConnectionId` VPN 연결의 ID입니다.

## 예제

### VPN 연결

다음 예제에서는 myVPNGateway와 MyCustomerGateway 간의 VPN 연결을 지정합니다.

#### JSON (영문)

```
"myVPNConnection" : {
   "Type" : "AWS::EC2::VPNConnection",
   "Properties" : {
      "Type" : "ipsec.1",
      "StaticRoutesOnly" : "true",
      "CustomerGatewayId" : {"Ref" : "myCustomerGateway"},
      "VpnGatewayId" : {"Ref" : "myVPNGateway"}
   }
}
```
