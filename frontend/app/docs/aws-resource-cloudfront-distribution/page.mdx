# AWS::CloudFront::D배포

배포는 CloudFront에 콘텐츠를 전송할 위치와 콘텐츠 전송을 추적 및 관리하는 방법에 대한 세부 정보를 알려줍니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::CloudFront::Distribution",
  "Properties" : {
      "DistributionConfig" : DistributionConfig,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`DistributionConfig`  
배포의 구성입니다.  
_필수 항목 여부_: 예  
_형식_: DistributionConfig  
_업데이트 필요 사항_: 중단 없음

`Tags`  
0개 이상의 요소를 포함하는 복합 형식 `Tag` 입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`CloudFront `Ref`배포 ID를 반환합니다. 예를 들면 다음과 같습니다 `E27LVI50CSW06W`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`DomainName`  
리소스의 도메인 이름(예: `d111111abcdef8.cloudfront.net`.

`Id`  
배포의 식별자(예: `EDFDVBD632BHDS5`.

## 예제

### 배포 생성

다음 예제에서는 배포를 지정하고 단일 태그를 할당합니다.

#### JSON (영문)

```
{
    "AWSTemplateFormatVersion": "2010-09-09",
    "Resources": {
        "cloudfrontdistribution": {
            "Type": "AWS::CloudFront::Distribution",
            "Properties": {
                "DistributionConfig": {
                    "CacheBehaviors": [
                        {
                            "LambdaFunctionAssociations": [
                                {
                                    "EventType": "string-value",
                                    "LambdaFunctionARN": "string-value"
                                }
                            ]
                        }
                    ],
                    "DefaultCacheBehavior": {
                        "LambdaFunctionAssociations": [
                            {
                                "EventType": "string-value",
                                "LambdaFunctionARN": "string-value"
                            }
                        ]
                    },
                    "IPV6Enabled": "boolean-value",
                    "Origins": [
                        {
                            "CustomOriginConfig": {
                                "OriginKeepaliveTimeout": "integer-value",
                                "OriginReadTimeout": "integer-value"
                            }
                        }
                    ]
                },
                "Tags": [
                    {
                        "Key": "string-value",
                        "Value": "string-value"
                    }
                ]
            }
        }
    }
}
```
