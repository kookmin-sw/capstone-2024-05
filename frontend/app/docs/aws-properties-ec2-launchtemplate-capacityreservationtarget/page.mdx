# AWS::EC2::LaunchTemplate CapacityReservationTarget

대상 용량 예약을 지정합니다.

`CapacityReservationTarget` 은 Amazon EC2 LaunchTemplate LaunchTemplateData 속성 유형의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CapacityReservationId" : String,
  "CapacityReservationResourceGroupArn" : String
}
```

## 속성

`CapacityReservationId`  
인스턴스를 실행할 용량 예약의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`CapacityReservationResourceGroupArn`  
인스턴스를 실행할 용량 예약 리소스 그룹의 ARN입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
