# AWS::SageMaker::기능 그룹

새 `FeatureGroup`. A `FeatureGroup` 는 를 설명하기 위해 `Features` 정의된 `FeatureStore` 그룹입니다`Record`.

`FeatureGroup` FeatureGroup에 포함된 스키마와 기능을 정의합니다. `FeatureGroup` 정의는 `Features`, a`RecordIdentifierFeatureName`, 및 `EventTimeFeatureName` 및 및 `OnlineStore` 의 목록 `OfflineStore` 으로 구성됩니다. AWS 서비스 할당량을 확인하여 `FeatureGroup`AWS 계정에 대한 할당량을 확인합니다.

**중요하다**  
를 만들려면 and 중 하나 이상을 포함 `OnlineStoreConfig` `OfflineStoreConfig` 해야 합니다 `FeatureGroup`.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::SageMaker::FeatureGroup",
  "Properties" : {
      "Description" : String,
      "EventTimeFeatureName" : String,
      "FeatureDefinitions" : [ FeatureDefinition, ... ],
      "FeatureGroupName" : String,
      "OfflineStoreConfig" : OfflineStoreConfig,
      "OnlineStoreConfig" : OnlineStoreConfig,
      "RecordIdentifierFeatureName" : String,
      "RoleArn" : String,
      "Tags" : [Tag, ... ]
    }
}
```
