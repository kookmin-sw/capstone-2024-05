# AWS::ElasticLoadBalancing::LoadBalancer 연결드레이닝 정책

Classic Load Balancer에 대한 연결 드레이닝 설정을 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Enabled" : Boolean,
  "Timeout" : Integer
}
```

## 속성

`Enabled`  
부하 분산 장치에 대해 연결 드레이닝을 사용할 수 있는지 여부를 지정합니다.  
_필수 항목 여부_: 예  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`Timeout`  
인스턴스를 등록 취소하기 전에 기존 연결을 열어 두는 최대 시간(초)입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음
