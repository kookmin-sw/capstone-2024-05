# AWS::EC2::CapacityReservation 태그 사양

이 리소스에 적용할 키-값 쌍의 배열입니다.

자세한 내용은 태그를 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ResourceType" : String,
  "Tags" : [Tag, ... ]
}
```
