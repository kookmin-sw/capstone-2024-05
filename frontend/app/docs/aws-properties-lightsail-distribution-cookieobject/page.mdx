# AWS::Lightsail::D istribution 쿠키 객체

`CookieObject` 은 CacheSettings 속성의 속성입니다. Amazon Lightsail 콘텐츠 전송 네트워크(CDN) 배포가 오리진에 쿠키를 전달하는지 여부와 전달한다면 어떤 쿠키를 오리진에 전달하는지 설명합니다.

지정한 쿠키의 경우, 배포는 최종 사용자 요청의 쿠키 값에 따라 지정된 콘텐츠의 개별 버전을 캐시합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CookiesAllowList" : [ String, ... ],
  "Option" : String
}
```

## 속성

`CookiesAllowList`  
배포의 오리진으로 전달할 특정 쿠키입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Option` 캐시 동작을 위해 배포의 오리진에 전달할 쿠키를 지정합니다.  
배포에 대해 다음 구성 중 하나를 사용합니다.

- ** `all` ** - 모든 쿠키를 오리진에 전달합니다.
- ** `none` ** - 오리진에 쿠키를 전달하지 않습니다.
- ** `allow-list` ** - 매개 변수를 사용하여 지정한 쿠키만 전달합니다 `CookiesAllowList` .
  *필수 항목 *여부: 아니요  
  _유형_: 문자열  
  _허용되는 값_: `all | allow-list | none`  
  _업데이트 필요 사항_: 중단 없음
