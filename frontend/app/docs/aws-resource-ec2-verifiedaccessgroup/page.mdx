# AWS::EC2::VerifiedAccessGroup

Verified Access 그룹에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::EC2::VerifiedAccessGroup",
  "Properties" : {
      "Description" : String,
      "PolicyDocument" : String,
      "PolicyEnabled" : Boolean,
      "Tags" : [Tag, ... ],
      "VerifiedAccessInstanceId" : String
    }
}
```

## 속성

`Description`  
AWS Verified Access 그룹에 대한 설명입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PolicyDocument`  
Verified Access 정책 문서입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PolicyEnabled`  
Verified Access 정책의 상태입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`Tags`  
태그.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 중단 없음

`VerifiedAccessInstanceId`  
AWS Verified Access 인스턴스의 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`Verified Access 그룹의 ID를 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`CreationTime` 생성 시간입니다.

`LastUpdatedTime` 마지막 업데이트 시간입니다.

`VerifiedAccessGroupId` Verified Access 그룹의 ID입니다.
