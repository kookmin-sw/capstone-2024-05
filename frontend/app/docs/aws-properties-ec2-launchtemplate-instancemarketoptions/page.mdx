# AWS::EC2::LaunchTemplate 인스턴스 마켓옵션

인스턴스에 대한 지역/국가(구매) 옵션을 지정합니다.

`InstanceMarketOptions` 는 AWS::EC2::LaunchTemplate LaunchTemplateData의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "MarketType" : String,
  "SpotOptions" : SpotOptions
}
```

## 속성

`MarketType`  
시장 유형입니다.  
_필수_: 조건부  
_유형_: 문자열  
_허용되는 값_: `spot`  
_업데이트 필요 사항_: 중단 없음

`SpotOptions`  
스팟 인스턴스에 대한 옵션.  
*필수 항목 *여부: 아니요  
_형식_: SpotOptions  
_업데이트 필요 사항_: 중단 없음

## 참고 항목
