# AWS::Lightsail::컨테이너 PublicEndpoint

`PublicEndpoint` 는 ContainerServiceDeployment 속성의 속성입니다. 컨테이너 서비스에서 컨테이너의 퍼블릭 엔드포인트 설정에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ContainerName" : String,
  "ContainerPort" : Integer,
  "HealthCheckConfig" : HealthCheckConfig
}
```

## 속성

`ContainerName`  
끝점 구성이 적용되는 배포의 컨테이너 항목 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ContainerPort`  
트래픽이 전달되는 지정된 컨테이너의 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`HealthCheckConfig`  
컨테이너의 상태 확인 구성을 설명하는 객체입니다.  
*필수 항목 *여부: 아니요  
_유형_: HealthCheckConfig  
_업데이트 필요 사항_: 중단 없음
