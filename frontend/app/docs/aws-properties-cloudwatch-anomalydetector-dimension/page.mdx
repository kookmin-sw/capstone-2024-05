# AWS::CloudWatch::AnomalyDetector 차원

차원은 지표 ID의 일부인 이름/값 쌍입니다. 차원은 지표에 대한 고유 식별자의 일부이기 때문에 지표 중 하나에 고유한 이름/값 쌍을 추가할 때마다 해당 지표의 새 변형을 생성하게 됩니다. 예를 들어, 많은 Amazon EC2 지표 는 `InstanceId` 차원 이름으로 게시되고 실제 인스턴스 ID는 해당 차원의 값으로 게시됩니다.

지표에 최대 30개의 차원을 할당할 수 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Value" : String
}
```

## 속성

`Name`  
차원의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `255`  
_업데이트 필요 사항_: 바꿔 놓음

`Value`  
차원의 값입니다. 차원 값은 ASCII 문자만 포함해야 하며 공백이 아닌 문자를 하나 이상 포함해야 합니다. ASCII 제어 문자는 차원 값의 일부로 지원되지 않습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `1024`  
_업데이트 필요 사항_: 바꿔 놓음
