# AWS Lambda 지원 사용자 지정 리소스

Lambda 함수를 사용자 지정 리소스와 연결하면 사용자 지정 리소스가 생성, 업데이트 또는 삭제될 때마다 함수가 호출됩니다. AWS CloudFormation에서는 Lambda API를 호출하여 함수를 호출하고 모든 요청 데이터(예: 요청 유형 및 리소스 속성)를 함수에 전달합니다. Lambda 함수와 AWS CloudFormation의 강력한 기능 및 사용자 지정 기능을 함께 사용하면 스택 생성 중에 AMI ID를 동적으로 조회하거나 문자열 반전 함수와 같은 유틸리티 함수를 구현 및 사용하는 등 다양한 시나리오를 사용할 수 있습니다.

**항목**

- 연습: Amazon Machine Image ID 조회
- `cfn-response` 모듈
