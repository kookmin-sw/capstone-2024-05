# AWS::S3::MultiRegionAccessPoint정책 정책 상태

버킷의 정책 상태에 대한 컨테이너 요소입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "IsPublic" : String
}
```

## 속성

`IsPublic`  
이 버킷의 정책 상태입니다. `TRUE` 이 버킷이 공용임을 나타냅니다. `FALSE` 은 버킷이 퍼블릭이 아님을 나타냅니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
