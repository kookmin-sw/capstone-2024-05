# AWS::KinesisAnalyticsV2::ApplicationOutput 람다출력

SQL 기반 Managed Service for Apache Flink 애플리케이션의 출력을 구성할 때 Amazon Lambda 함수를 대상으로 식별합니다. Lambda 함수의 Amazon 리소스 이름(ARN) 함수를 제공합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ResourceARN" : String
}
```

## 속성

`ResourceARN`  
쓸 대상 Lambda 함수의 Amazon 리소스 이름(ARN)입니다.  
최신 버전보다 이전 버전의 Lambda 함수를 지정하려면 Lambda 함수 ARN에 Lambda 함수 버전을 포함합니다. Lambda ARN에 대한 자세한 내용은 ARN 예제: Amazon Lambda 단원을 참조하십시오
_필수 항목 여부_: 예  
_유형_: 문자열  
_최소_: `1`  
_최대_: `2048`  
_패턴_: `arn:.*`  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

- Amazon Kinesis Data Analytics API 참조\* \*의 LambdaOutput
