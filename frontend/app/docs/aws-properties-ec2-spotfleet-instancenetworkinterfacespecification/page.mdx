# AWS::EC2::SpotFleet 인스턴스네트워크 인터페이스 사양

네트워크 인터페이스에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "AssociatePublicIpAddress" : Boolean,
  "DeleteOnTermination" : Boolean,
  "Description" : String,
  "DeviceIndex" : Integer,
  "Groups" : [ String, ... ],
  "Ipv6AddressCount" : Integer,
  "Ipv6Addresses" : [ InstanceIpv6Address, ... ],
  "NetworkInterfaceId" : String,
  "PrivateIpAddresses" : [ PrivateIpAddressSpecification, ... ],
  "SecondaryPrivateIpAddressCount" : Integer,
  "SubnetId" : String
}
```

## 속성

`AssociatePublicIpAddress`  
VPC에서 시작하는 인스턴스에 퍼블릭 IPv4 주소를 할당할지 여부를 나타냅니다. 공용 IP 주소는 eth0의 네트워크 인터페이스에만 할당할 수 있으며 기존 네트워크 인터페이스가 아닌 새 네트워크 인터페이스에만 할당할 수 있습니다. 요청에서 네트워크 인터페이스를 두 개 이상 지정할 수 없습니다. 기본 서브넷으로 시작하는 경우 기본값은 `true`입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 바꿔 놓음

`DeleteOnTermination`  
인스턴스가 종료될 때 네트워크 인터페이스가 삭제되는지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 바꿔 놓음

`Description`  
네트워크 인터페이스에 대한 설명입니다. 인스턴스를 시작할 때 네트워크 인터페이스를 생성하는 경우에만 적용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`DeviceIndex`  
연결 순서에서 네트워크 인터페이스의 위치입니다. 기본 네트워크 인터페이스의 장치 인덱스는 0입니다.  
인스턴스를 시작할 때 네트워크 인터페이스를 지정하는 경우 디바이스 인덱스를 지정해야 합니다.  
_필수_: 조건부  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음

`Groups`  
네트워크 인터페이스에 대한 보안 그룹의 ID입니다. 인스턴스를 시작할 때 네트워크 인터페이스를 생성하는 경우에만 적용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 바꿔 놓음

`Ipv6AddressCount`  
네트워크 인터페이스에 할당할 IPv6 주소의 수입니다. Amazon EC2는 서브넷 범위에서 IPv6 주소를 선택합니다. 이 옵션과 동일한 요청에서 특정 IPv6 주소를 할당하는 옵션을 지정할 수 없습니다. 시작할 최소 인스턴스 수를 지정한 경우 이 옵션을 지정할 수 있습니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음

`Ipv6Addresses`  
네트워크 인터페이스에 할당할 IPv6 주소입니다. 이 옵션과 동일한 요청에서 여러 IPv6 주소를 할당하는 옵션을 지정할 수 없습니다. 시작할 최소 인스턴스 수를 지정한 경우 이 옵션을 지정할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: InstanceIpv6Address 목록  
_업데이트 필요 사항_: 바꿔 놓음

`NetworkInterfaceId`  
네트워크 인터페이스의 ID입니다.  
스팟 집합을 생성하는 경우 시작 사양에서 네트워크 인터페이스 ID를 지정할 수 없으므로 이 파라미터를 생략합니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`PrivateIpAddresses`  
네트워크 인터페이스에 할당할 개인 IPv4 주소입니다. 하나의 프라이빗 IPv4 주소만 기본으로 지정할 수 있습니다. RunInstances 요청에서 둘 이상의 인스턴스를 시작하는 경우 이 옵션을 지정할 수 없습니다.  
*필수 항목 *여부: 아니요  
_형식_: PrivateIpAddressSpecification 목록  
_업데이트 필요 사항_: 바꿔 놓음

`SecondaryPrivateIpAddressCount`  
보조 프라이빗 IPv4 주소의 수입니다. 이 옵션을 지정할 수 없으며 개인 IP 주소 옵션을 사용하여 둘 이상의 개인 IP 주소를 지정할 수 없습니다. RunInstances 요청에서 둘 이상의 인스턴스를 시작하는 경우 이 옵션을 지정할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 바꿔 놓음

`SubnetId`  
네트워크 인터페이스와 연결된 서브넷의 ID입니다. 인스턴스를 시작할 때 네트워크 인터페이스를 생성하는 경우에만 적용됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
