# AWS::백업::백업 계획 AdvancedBackupSettingResourceType

자원 종류와 백업 옵션을 포함하는 개체를 지정합니다. 이는 Windows VSS 백업에 대해서만 지원됩니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BackupOptions" : Json,
  "ResourceType" : String
}
```

## 속성

`BackupOptions`  
리소스에 대한 백업 옵션입니다. 각 옵션은 키-값 쌍입니다. 이 옵션은 Windows VSS 백업 작업에만 사용할 수 있습니다.  
유효한 값:  
 `"WindowsVSS":"enabled"` `WindowsVSS` 백업 옵션을 활성화하고 Windows VSS 백업을 만들려면 로 설정합니다.  
정기 `"WindowsVSS":"disabled"` 백업을 만들려면 로 설정합니다. 이 `WindowsVSS` 옵션은 기본적으로 활성화되어 있지 않습니다.  
잘못된 옵션을 지정하면 예외가 발생합니다 `InvalidParameterValueException` .  
Windows VSS 백업에 대한 자세한 내용은 VSS 사용 Windows 백업 만들기를 참조하십시오.  
_필수 항목 여부_: 예  
_형식_: Json  
_업데이트 필요 사항_: 중단 없음

`ResourceType`  
리소스 종류의 이름입니다. 유일하게 지원되는 리소스 유형은 EC2입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
