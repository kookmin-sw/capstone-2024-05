# AWS::Route53RecoveryControl::클러스터 ClusterEndpoint

클러스터 엔드포인트입니다. 클러스터에서 라우팅 제어 상태를 가져오거나 업데이트하려는 경우 엔드포인트 URL과 AWS 리전으로 구성된 5개의 클러스터 엔드포인트 중 하나를 지정합니다.

자세한 내용은 Amazon Route 53 Application Recovery Controller 개발자 안내서의 코드 예제를 참조하십시오.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Endpoint" : String,
  "Region" : String
}
```

## 속성

`Endpoint`  
라우팅 제어 상태를 설정하거나 검색하기 위해 지정하는 5개의 중복 클러스터 중 하나에 대한 클러스터 끝점 URL입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Region`  
클러스터 엔드포인트의 AWS 리전입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
