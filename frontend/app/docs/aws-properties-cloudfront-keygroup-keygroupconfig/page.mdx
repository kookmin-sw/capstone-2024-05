# AWS::CloudFront::KeyGroup KeyGroupConfig

키 그룹 구성입니다.

키 그룹에는 CloudFront 서명된 URL 및 서명된 쿠키와 함께 사용할 수 있는 퍼블릭 키 목록이 포함되어 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Comment" : String,
  "Items" : [ String, ... ],
  "Name" : String
}
```

## 속성

`Comment`  
키 그룹을 설명하는 주석입니다. 주석은 128자를 초과할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Items`  
키 그룹에 있는 퍼블릭 키의 식별자 목록입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`Name`  
키 그룹을 식별하기 위한 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
