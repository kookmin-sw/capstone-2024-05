# AWS::SageMaker::이미지 버전

로 지정된 SageMaker 이미지의 버전을 생성합니다 `ImageName`. 버전은 로 지정된 Amazon Container Registry(ECR) 컨테이너 이미지를 나타냅니다 `BaseImage`.

**메모**  
 `DependsOn` 이 속성을 사용하여 특정 자원의 작성이 다른 자원의 작성 뒤에 오도록 지정할 수 있습니다. 다음과 같은 사용 사례에 사용할 수 있습니다. 자세한 내용은 attribute를 참조하세요 `DependsOn` .

1. `DependsOn` `ImageVersion` 와 `Image` . `ImageVersion` `DependsOn` `Image`
2. `DependsOn`동일한 네임스페이스 내에서 s `ImageVersion` 간의 순서를 설정하는 데 사용할 수 있습니다 `Image` . 예를 들어 ImageVersionB ImageVersionA `DependsOn` 와 둘 다 동일한 부모를 공유하는 경우 `Image`ImageVersionA는 버전 N이고 ImageVersionB는 N+1입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::SageMaker::ImageVersion",
  "Properties" : {
      "Alias" : String,
      "Aliases" : [ String, ... ],
      "BaseImage" : String,
      "Horovod" : Boolean,
      "ImageName" : String,
      "JobType" : String,
      "MLFramework" : String,
      "Processor" : String,
      "ProgrammingLang" : String,
      "ReleaseNotes" : String,
      "VendorGuidance" : String
    }
}
```
