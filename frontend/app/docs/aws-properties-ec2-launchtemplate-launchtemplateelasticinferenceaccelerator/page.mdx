# AWS::EC2::LaunchTemplate LaunchTemplate탄력적 추론액셀러레이터

탄력적 추론 가속기를 지정합니다.

`LaunchTemplateElasticInferenceAccelerator` 은 AWS::EC2::LaunchTemplate LaunchTemplateData의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Count" : Integer,
  "Type" : String
}
```

## 속성

`Count`  
 인스턴스에 연결할 탄력적 추론 액셀러레이터의 수입니다.  
기본값 : 1  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `1`  
_업데이트 필요 사항_: 중단 없음

`Type`  
 탄력적 추론 가속기의 유형입니다. 가능한 값은 eia1.medium, eia1.large 및 eia1.xlarge입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
