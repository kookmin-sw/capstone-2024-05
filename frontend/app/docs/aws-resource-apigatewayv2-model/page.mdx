# AWS::ApiGatewayV2::모델

리소스는 `AWS::ApiGatewayV2::Model` WebSocket API에 대한 데이터 모델을 업데이트합니다. 자세한 내용은 _API Gateway 개발자 안내서의 모델 선택 표현식을 참조하십시오_.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::ApiGatewayV2::Model",
  "Properties" : {
      "ApiId" : String,
      "ContentType" : String,
      "Description" : String,
      "Name" : String,
      "Schema" : Json
    }
}
```

## 속성

`ApiId`  
API 식별자입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`ContentType`  
모델의 콘텐츠 형식(예: "application/json")입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Description`  
모델에 대한 설명입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Name`  
모델의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Schema`  
모델의 스키마입니다. application/json 모델의 경우 JSON 스키마 초안 4 모델이어야 합니다.  
_필수 항목 여부_: 예  
_형식_: Json  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`모델 `Ref`ID(예: )가 반환 됩니다`abc123`.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`ModelId` 모델 ID입니다.

## 예제

### 모델 생성 예제

다음 예제에서는 `model` 라는 API에 대해 `MyModel` 호출되는 리소스를 만듭니다`MyApi`.

#### JSON (영문)

```
{
    "MyModel": {
        "Type": "AWS::ApiGatewayV2::Model",
        "Properties": {
            "Name": "ModelName",
            "ApiId": {
                "Ref": "MyApi"
            },
            "ContentType": "application/json",
            "Schema": {
                "$schema": "http://json-schema.org/draft-04/schema#",
                "title": "DummySchema",
                "type": "object",
                "properties": {
                    "id": {
                        "type": "string"
                    }
                }
            }
        }
    }
}
```
