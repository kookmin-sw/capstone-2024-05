# AWS::SageMaker::P ipeline S3위치

`S3Location` 속성 유형은 속성 설명을 사용할 수 없음을 지정합니다. AWS::SageMaker::P ipeline입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Bucket" : String,
  "ETag" : String,
  "Key" : String,
  "Version" : String
}
```

## 속성

`Bucket`  
속성 설명을 사용할 수 없습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ETag`  
속성 설명을 사용할 수 없습니다.
