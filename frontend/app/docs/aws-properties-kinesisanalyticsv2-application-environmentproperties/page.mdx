# AWS::KinesisAnalyticsV2::애플리케이션 환경 속성

Managed Service for Apache Flink 애플리케이션의 실행 특성에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "PropertyGroups" : [ PropertyGroup, ... ]
}
```

## 속성

`PropertyGroups`  
실행 속성 그룹에 대해 설명합니다.  
*필수 항목 *여부: 아니요  
_형식_: PropertyGroup 목록  
_최대_: `50`  
_업데이트 필요 사항_: 중단 없음

## 참고 항목

- Amazon Kinesis Data Analytics API 참조\* \*의 EnvironmentProperties
