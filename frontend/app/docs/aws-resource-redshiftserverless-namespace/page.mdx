# AWS::RedshiftServerless::네임스페이스

데이터베이스 개체 및 사용자의 컬렉션입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::RedshiftServerless::Namespace",
  "Properties" : {
      "AdminUsername" : String,
      "AdminUserPassword" : String,
      "DbName" : String,
      "DefaultIamRoleArn" : String,
      "FinalSnapshotName" : String,
      "FinalSnapshotRetentionPeriod" : Integer,
      "IamRoles" : [ String, ... ],
      "KmsKeyId" : String,
      "LogExports" : [ String, ... ],
      "NamespaceName" : String,
      "Tags" : [Tag, ... ]
    }
}
```

## 속성

`AdminUsername`  
네임스페이스에서 생성된 기본 데이터베이스에 대한 관리자의 사용자 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`AdminUserPassword`  
네임스페이스에 작성된 기본 데이터베이스에 대한 관리자의 비밀번호입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`DbName`  
네임스페이스에서 만든 기본 데이터베이스의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`DefaultIamRoleArn`  
네임스페이스에서 기본값으로 설정할 IAM 역할의 Amazon 리소스 이름(ARN)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`FinalSnapshotName`  
네임스페이스가 삭제되기 전에 만들 스냅샷의 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`FinalSnapshotRetentionPeriod`  
최종 스냅샷을 보존할 기간입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 중단 없음

`IamRoles`  
네임스페이스와 연결할 IAM 역할 목록입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`KmsKeyId`  
데이터를 암호화하는 데 사용되는 AWS Key Management Service 키의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`LogExports`  
네임스페이스에서 내보낼 수 있는 로그의 유형입니다. 사용 가능한 내보내기 유형은 `userlog`, `connectionlog`, 및 `useractivitylog`입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

`NamespaceName`  
네임스페이스의 이름입니다. 소문자로 된 3-64자의 영숫자여야 하며 예약어일 수 없습니다. 예약어 목록은 Amazon Redshift 데이터베이스 개발자 안내서의 예약어에서 확인할 수 있습니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
네임스페이스에 태그를 지정하는 데 사용되는 키-값 쌍의 맵입니다.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID가 Ref 내장 함수에 제공되면 Ref는 NamespaceName을 반환합니다(예: `sample-namespace.` Ref 함수 사용에 대한 자세한 내용은 Ref를 참조하세요.

### Fn::GetAtt

GetAtt는 이 형식의 지정된 특성에 대한 값을 반환합니다. 자세한 내용은 Fn::GetAtt 단원을 참조하십시오. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

####

`Namespace` 끝점이 만들어지는 컴퓨팅 리소스의 컬렉션입니다.

`Namespace.AdminUsername` 네임스페이스에서 만든 첫 번째 데이터베이스에 대한 관리자의 사용자 이름입니다.

`Namespace.CreationDate` 네임스페이스를 만든 날짜입니다.

`Namespace.DbName` 네임스페이스에서 만든 첫 번째 데이터베이스의 이름입니다.

`Namespace.DefaultIamRoleArn` 네임스페이스에서 기본값으로 설정할 IAM 역할의 Amazon 리소스 이름(ARN)입니다.

`Namespace.IamRoles` 네임스페이스와 연결할 IAM 역할 목록입니다.

`Namespace.KmsKeyId` 데이터를 암호화하는 데 사용되는 AWS Key Management Service 키의 ID입니다.

`Namespace.LogExports`  
네임스페이스에서 내보낼 수 있는 로그의 유형입니다. 사용 가능한 내보내기 유형은 `User log`, `Connection log`, 및 `User activity log`입니다.

`Namespace.NamespaceArn` 네임스페이스와 연결된 Amazon 리소스 이름(ARN)입니다.

`Namespace.NamespaceId` 네임스페이스의 고유 식별자입니다.

`Namespace.NamespaceName` 네임스페이스의 이름입니다. 소문자로 된 3-64자의 영숫자여야 하며 예약어일 수 없습니다. 예약어 목록은 Amazon Redshift 데이터베이스 개발자 안내서의 예약어에서 확인할 수 있습니다.

`Namespace.Status` 네임스페이스의 상태입니다.
