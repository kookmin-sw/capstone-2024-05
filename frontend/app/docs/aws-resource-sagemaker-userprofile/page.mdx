# AWS::SageMaker::사용자 프로필

사용자 프로필을 만듭니다. 사용자 프로필은 도메인 내의 단일 사용자를 나타내며 공유, 보고 및 기타 사용자 지향 기능을 위해 "개인"을 참조하는 주요 방법입니다. 이 엔터티는 사용자가 Amazon SageMaker Studio에 온보딩할 때 생성됩니다. 관리자가 이메일로 사용자를 초대하거나 IAM Identity Center에서 사용자를 가져오면 사용자 프로필이 자동으로 생성됩니다. 사용자 프로필은 개별 사용자에 대한 설정의 기본 보유자이며 사용자의 프라이빗 Amazon Elastic File System(EFS) 홈 디렉터리에 대한 참조가 있습니다.

**메모**  
IAM Identity Center 인증을 사용하는 경우 IAM Identity Center의 사용자 또는 해당 사용자가 포함된 IAM Identity Center의 그룹을 IAM Identity Center 콘솔에서 Amazon SageMaker Studio 애플리케이션에 할당하여 사용자 프로필을 생성해야 합니다. 응용 프로그램 할당에 대한 자세한 내용은 사용자 액세스 할당을 참조하세요. 할당이 완료되면 AWS CloudFormation을 사용하여 IAM Identity Center에서 해당 사용자에 대한 사용자 프로필을 생성할 수 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::SageMaker::UserProfile",
  "Properties" : {
      "DomainId" : String,
      "SingleSignOnUserIdentifier" : String,
      "SingleSignOnUserValue" : String,
      "Tags" : [Tag, ... ],
      "UserProfileName" : String,
      "UserSettings" : UserSettings
    }
}
```

## 속성

`DomainId`  
도메인 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `63`  
_업데이트 필요 사항_: 바꿔 놓음

`SingleSignOnUserIdentifier`  
SingleSignOnUserValue에 지정된 값 형식에 대한 지정자입니다. 현재 유일하게 지원되는 값은 "UserName"입니다. 도메인의 AuthMode가 IAM Identity Center인 경우 이 필드는 필수입니다. 도메인의 AuthMode가 IAM Identity Center가 아닌 경우 이 필드를 지정할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`SingleSignOnUserValue`  
이 UserProfile에 연결된 AWS Single Sign-On 사용자의 사용자 이름입니다. 도메인의 AuthMode가 IAM Identity Center인 경우 이 필드는 필수이며 디렉터리에 있는 사용자의 유효한 사용자 이름과 일치해야 합니다. 도메인의 AuthMode가 IAM Identity Center가 아닌 경우 이 필드를 지정할 수 없습니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
이 리소스에 적용할 키-값 쌍의 배열입니다.  
사용자 프로필에 대해 지정하는 태그는 사용자 프로필이 시작하는 모든 앱에도 추가됩니다.  
자세한 내용은 태그를 참조하십시오.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 바꿔 놓음

`UserProfileName`  
사용자 프로필 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}`  
_업데이트 필요 사항_: 바꿔 놓음

`UserSettings`  
Amazon SageMaker Studio 사용자에게 적용되는 설정 모음입니다.  
*필수 항목 *여부: 아니요  
_유형_: UserSettings  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref`도메인 `Ref`ID와 사용자 프로필 이름(예: `d-xxxxxxxxxxxx` 각각 및 `my-user-profile`)을 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`UserProfileArn`  
사용자 프로필의 Amazon 리소스 이름(ARN)입니다 `arn:aws:sagemaker:us-west-2:account-id:user-profile/my-user-profile`.
