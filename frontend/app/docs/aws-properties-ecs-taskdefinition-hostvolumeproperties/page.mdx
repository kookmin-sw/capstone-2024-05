# AWS::ECS::태스크 정의 HostVolumeProperties

이 `HostVolumeProperties` 속성은 컨테이너 인스턴스 바인드 탑재 호스트 볼륨에 대한 세부 정보를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "SourcePath" : String
}
```

## 속성

`SourcePath`  
`host` 매개 변수를 사용하는 `sourcePath` 경우 컨테이너에 제공되는 호스트 컨테이너 인스턴스의 경로를 선언하도록 지정합니다. 이 파라미터가 비어 있으면 Docker 데몬이 호스트 경로를 할당한 것입니다. `host` 파라미터 `sourcePath` 에 파일 위치가 포함된 경우 데이터 볼륨은 수동으로 삭제할 때까지 호스트 컨테이너 인스턴스의 지정된 위치에 유지됩니다. `sourcePath` 호스트 컨테이너 인스턴스에 값이 없으면 Docker 데몬이 이 값을 생성합니다. 위치가 있는 경우 원본 경로 폴더의 내용이 내보내집니다.  
Fargate 시작 유형을 사용하는 경우 `sourcePath` 파라미터가 지원되지 않습니다.
