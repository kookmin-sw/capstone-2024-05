# AWS::KinesisFirehose::D eliveryStream 프로세서

이 `Processor` 속성은 Amazon Kinesis Data Firehose 전송 스트림에 대한 데이터 프로세서를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Parameters" : [ ProcessorParameter, ... ],
  "Type" : String
}
```

## 속성

`Parameters`  
프로세서 매개 변수입니다.  
*필수 항목 *여부: 아니요  
_형식_: ProcessorParameter 목록  
_업데이트 필요 사항_: 중단 없음

`Type`  
프로세서의 유형입니다. 유효한 값: `Lambda`.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `AppendDelimiterToRecord | Lambda | MetadataExtraction | RecordDeAggregation`  
_업데이트 필요 사항_: 중단 없음
