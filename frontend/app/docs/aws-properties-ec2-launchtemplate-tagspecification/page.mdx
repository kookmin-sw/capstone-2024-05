# AWS::EC2::LaunchTemplate 태그 사양

시작 템플릿에 대해 리소스가 생성될 때 리소스에 적용할 태그를 지정합니다.

`TagSpecification` 은 some-address의 속성 유형이고, some-address는 AWS::EC2::LaunchTemplate, LaunchTemplateData의 속성입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ResourceType" : String,
  "Tags" : [Tag, ... ]
}
```

## 속성

`ResourceType`  
태그를 지정할 리소스의 유형입니다.  
`Valid Values` 태그를 지정할 수 있는 모든 리소스 종류입니다. 그러나 시작 템플릿을 생성할 때 다음 리소스 유형에 대해서만 태그를 지정할 수 있습니다. `instance` `volume` | `elastic-gpu` | `network-interface` | `spot-instances-request`  
리소스를 만든 후 태그를 지정하려면 CreateTags 단원을 참조하십시오.
