# AWS::SageMaker::앱

지정된 UserProfile에 대해 실행 중인 앱을 만듭니다. 이 작업은 연결된 도메인에 액세스할 때와 사용자가 새 커널 구성을 선택할 때 Amazon SageMaker Studio에 의해 자동으로 호출됩니다. 사용자는 여러 앱을 동시에 활성화할 수 있습니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::SageMaker::App",
  "Properties" : {
      "AppName" : String,
      "AppType" : String,
      "DomainId" : String,
      "ResourceSpec" : ResourceSpec,
      "Tags" : [Tag, ... ],
      "UserProfileName" : String
    }
}
```

## 속성

`AppName`  
앱의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}`  
_업데이트 필요 사항_: 바꿔 놓음

`AppType`  
앱의 유형입니다.  
_허용되는 값_: `JupyterServer | KernelGateway | RSessionGateway | RStudioServerPro | TensorBoard | Canvas`  
_필수_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`DomainId`  
도메인 ID입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `63`  
_업데이트 필요 사항_: 바꿔 놓음

`ResourceSpec`  
SageMaker 이미지 및 SageMaker 이미지 버전의 ARN과 버전이 실행되는 인스턴스 유형을 지정합니다.  
*필수 항목 *여부: 아니요  
_유형_: ResourceSpec  
_업데이트 필요 사항_: 바꿔 놓음

`Tags`  
이 리소스에 적용할 키-값 쌍의 배열입니다.  
자세한 내용은 태그를 참조하십시오.  
*필수 항목 *여부: 아니요  
_유형_: 태그 목록  
_업데이트 필요 사항_: 바꿔 놓음

`UserProfileName`  
사용자 프로필 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `63`  
_패턴_: `^a-zA-Z0-9{0,62}`  
_업데이트 필요 사항_: 바꿔 놓음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`앱 유형, 앱 이름, 도메인 ID 및 사용자 프로필 이름을 반환합니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`AppArn`  
앱의 Amazon 리소스 이름(ARN)입니다 `arn:aws:sagemaker:us-west-2:account-id:app/my-app-name`.
