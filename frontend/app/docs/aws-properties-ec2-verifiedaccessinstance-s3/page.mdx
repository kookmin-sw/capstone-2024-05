# AWS::EC2::VerifiedAccess인스턴스 S3

Amazon S3를 로깅 대상으로 사용하는 옵션.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "BucketName" : String,
  "BucketOwner" : String,
  "Enabled" : Boolean,
  "Prefix" : String
}
```

## 속성

`BucketName`  
버킷 이름입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`BucketOwner`  
버킷을 소유한 AWS 계정 번호입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Enabled`  
로깅을 사용할 수 있는지 여부를 나타냅니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`Prefix`  
버킷 접두사입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
