# AWS::Redshift::엔드포인트액세스

Redshift 관리형 VPC 엔드포인트를 생성합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Type" : "AWS::Redshift::EndpointAccess",
  "Properties" : {
      "ClusterIdentifier" : String,
      "EndpointName" : String,
      "ResourceOwner" : String,
      "SubnetGroupName" : String,
      "VpcSecurityGroupIds" : [ String, ... ]
    }
}
```

## 속성

`ClusterIdentifier`  
끝점과 연결된 클러스터의 클러스터 식별자입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 바꿔 놓음

`EndpointName`  
끝점의 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 바꿔 놓음

`ResourceOwner`  
클러스터 소유자의 AWS 계정 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 바꿔 놓음

`SubnetGroupName`  
Amazon Redshift가 엔드포인트를 배포하도록 선택하는 서브넷 그룹 이름입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 바꿔 놓음

`VpcSecurityGroupIds`  
엔드포인트에 권한을 부여하는 인바운드 트래픽에 대한 포트, 프로토콜 및 소스를 정의하는 보안 그룹입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열 목록  
_업데이트 필요 사항_: 중단 없음

## 반환 값

### 참고문헌

이 리소스의 논리적 ID를 내장 함수에 전달하면 `Ref` `Ref`리소스 이름이 반환됩니다.

함수 사용에 대한 자세한 `Ref`내용은 Ref를 참조하세요.

### Fn::GetAtt

`Fn::GetAtt`내장 함수는 이 형식의 지정된 특성에 대한 값을 반환합니다. 다음은 사용 가능한 특성 및 샘플 반환 값입니다.

내장 함수 사용에 대한 자세한 `Fn::GetAtt`내용은 Fn::GetAtt 단원을 참조하십시오.

####

`Address` 엔드포인트의 DNS 주소입니다.

`EndpointCreateTime` 엔드포인트가 생성된 시간(UTC)입니다.

`EndpointStatus` 엔드포인트의 상태입니다.

`Port` 클러스터가 들어오는 연결을 수락하는 포트 번호입니다.

`VpcEndpoint` 프록시를 통해 Amazon Redshift 클러스터에 연결하기 위한 연결 엔드포인트입니다.

`VpcEndpoint.NetworkInterfaces` 속성 설명을 사용할 수 없습니다.

`VpcEndpoint.VpcEndpointId` 프록시를 통해 Amazon Redshift 클러스터를 연결하기 위한 연결 엔드포인트 ID입니다.

`VpcEndpoint.VpcId` 엔드포인트가 연결된 VPC 식별자입니다.

`VpcSecurityGroups` 엔드포인트와 연결된 보안 그룹입니다.
