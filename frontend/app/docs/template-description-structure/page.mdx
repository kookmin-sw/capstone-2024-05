# 묘사

이 `Description` 섹션(선택 사항)을 사용하면 템플릿에 대한 설명을 포함할 수 있습니다.

description 선언의 값은 길이가 0바이트에서 1024바이트 사이인 리터럴 문자열이어야 합니다. 매개 변수 또는 함수를 사용하여 설명을 지정할 수 없습니다. 다음 코드 조각은 설명 선언의 예입니다.

**중요하다**  
스택 업데이트 중에는 섹션을 단독으로 업데이트할 수 없습니다 `Description` . 리소스를 추가, 수정 또는 삭제하는 변경 내용을 포함하는 경우에만 업데이트할 수 있습니다.

## JSON (영문)

```
"Description" : "Here are some details about the template."
```
