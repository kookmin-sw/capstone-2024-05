# AWS::EFS::파일 시스템 백업 정책

백업 정책은 파일 시스템에 대한 자동 백업을 켜거나 끕니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Status" : String
}
```

## 속성

`Status` 파일 시스템에 대한 백업 정책 상태를 설정합니다.

- ** `ENABLED`** - 파일 시스템에 대한 자동 백업을 설정합니다.
- ** `DISABLED`** - 파일 시스템에 대한 자동 백업을 해제합니다.
  _필수 항목 여부_: 예  
  _유형_: 문자열  
  _허용되는 값_: `DISABLED | DISABLING | ENABLED | ENABLING`  
  _업데이트 필요 사항_: 중단 없음
