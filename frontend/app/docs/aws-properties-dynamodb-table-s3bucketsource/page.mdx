# AWS::D ynamoDB::테이블 S3버킷소스

가져올 S3 버킷입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "S3Bucket" : String,
  "S3BucketOwner" : String,
  "S3KeyPrefix" : String
}
```

## 속성

`S3Bucket`  
 가져올 S3 버킷입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`S3BucketOwner`  
 가져올 S3 버킷의 계정 번호입니다. 요청자가 버킷을 소유하는 경우 이는 선택 사항입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음

`S3KeyPrefix`  
 가져오는 모든 S3 객체가 공유하는 키 접두사입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 바꿔 놓음
