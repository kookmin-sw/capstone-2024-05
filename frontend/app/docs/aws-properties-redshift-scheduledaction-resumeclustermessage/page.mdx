# AWS::Redshift::Scheduled작업 재개클러스터메시지

클러스터 다시 시작 작업에 대해 설명합니다. 예를 들어 API 작업을 실행하기 위한 예약된 작업이 있습니다 `ResumeCluster` .

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ClusterIdentifier" : String
}
```

## 속성

`ClusterIdentifier`  
재개할 클러스터의 식별자입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_최대_: `2147483647`  
_업데이트 필요 사항_: 중단 없음
