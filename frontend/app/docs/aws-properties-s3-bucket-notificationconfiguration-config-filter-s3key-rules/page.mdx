# AWS::S3::버킷 필터 규칙

필터링할 Amazon S3 객체 키 이름과 키 이름의 접미사 또는 접두사를 기준으로 필터링할지 여부를 지정합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Name" : String,
  "Value" : String
}
```

## 속성

`Name`  
필터링 규칙이 적용되는 하나 이상의 객체를 식별하는 객체 키 이름 접두사 또는 접미사입니다. 최대 길이는 1,024자입니다. 겹치는 접두사와 접미사는 지원되지 않습니다. 자세한 내용은 _Amazon S3 사용 설명서의 이벤트 알림 구성을 참조하십시오_.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_허용되는 값_: `prefix | suffix`  
_업데이트 필요 사항_: 중단 없음

`Value`  
필터가 객체 키 이름에서 검색하는 값입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
