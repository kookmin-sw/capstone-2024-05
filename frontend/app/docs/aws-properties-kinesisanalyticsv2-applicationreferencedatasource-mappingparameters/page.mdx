# AWS::KinesisAnalyticsV2::ApplicationReferenceDataSource 매핑파라미터

애플리케이션을 작성하거나 업데이트할 때 Apache Flink 애플리케이션의 입력에 대해 SQL 기반 Managed Service를 구성하는 경우, 스트리밍 소스의 레코드 형식(예: JSON, CSV 또는 일부 구분 기호로 구분된 레코드 필드)에 특정한 추가 맵핑 정보를 제공합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CSVMappingParameters" : CSVMappingParameters,
  "JSONMappingParameters" : JSONMappingParameters
}
```

## 속성

`CSVMappingParameters`  
레코드 형식에서 구분 기호(예: CSV)를 사용하는 경우 추가 매핑 정보를 제공합니다.  
*필수 항목 *여부: 아니요  
_형식_: CSVMappingParameters  
_업데이트 필요 사항_: 중단 없음

`JSONMappingParameters`  
JSON이 스트리밍 소스의 레코드 형식인 경우 추가 매핑 정보를 제공합니다.  
*필수 항목 *여부: 아니요  
_형식_: JSONMappingParameters  
_업데이트 필요 사항_: 중단 없음
