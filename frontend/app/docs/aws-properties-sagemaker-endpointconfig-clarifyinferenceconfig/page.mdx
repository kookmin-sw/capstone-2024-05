# AWS::SageMaker::EndpointConfig ClarifyInferenceConfig

`ClarifyInferenceConfig` 속성 유형은 속성 설명을 사용할 수 없음을 지정합니다. AWS::SageMaker::EndpointConfig의 경우.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ContentTemplate" : String,
  "FeatureHeaders" : [ ClarifyHeader, ... ],
  "FeaturesAttribute" : String,
  "FeatureTypes" : [ ClarifyFeatureType, ... ],
  "LabelAttribute" : String,
  "LabelHeaders" : [ ClarifyHeader, ... ],
  "LabelIndex" : Integer,
  "MaxPayloadInMB" : Integer,
  "MaxRecordCount" : Integer,
  "ProbabilityAttribute" : String,
  "ProbabilityIndex" : Integer
}
```
