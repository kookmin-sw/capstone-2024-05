# AWS::Lightsail::인스턴스 하드웨어

`Hardware` 은 AWS::Lightsail::Instance 리소스의 속성입니다. 인스턴스의 하드웨어 속성(예: vCPU 수, 연결된 디스크 및 RAM 용량)을 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "CpuCount" : Integer,
  "Disks" : [ Disk, ... ],
  "RamSizeInGb" : Integer
}
```

## 속성

`CpuCount`  
인스턴스에 있는 vCPU 수입니다.  
이 `CpuCount` 속성은 읽기 전용이며 인스턴스 만들기 또는 인스턴스 업데이트 요청에 지정해서는 안 됩니다.
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 업데이트가 지원되지 않습니다.

`Disks`  
인스턴스에 연결된 디스크입니다.  
디스크 연결 또는 디스크 분리 요청을 수행할 때 인스턴스가 다시 시작됩니다. 이렇게 하면 고정 IP가 연결되지 않은 경우 인스턴스의 퍼블릭 IP 주소가 재설정됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 디스크 목록  
_업데이트 필요 사항_: 일부 중단

`RamSizeInGb`  
인스턴스의 RAM 용량(GB)입니다(예: `1.0`).  
이 `RamSizeInGb` 속성은 읽기 전용이며 인스턴스 만들기 또는 인스턴스 업데이트 요청에 지정해서는 안 됩니다.
*필수 항목 *여부: 아니요  
_유형_: Integer  
_업데이트 필요 사항_: 업데이트가 지원되지 않습니다.
