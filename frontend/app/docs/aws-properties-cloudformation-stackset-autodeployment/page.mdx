# AWS::CloudFormation::StackSet 자동 배포

[`Service-managed` 권한을] StackSets가 대상 조직 단위(OU)에 추가된 AWS Organizations 계정에 자동으로 배포되는지 여부를 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Enabled" : Boolean,
  "RetainStacksOnAccountRemoval" : Boolean
}
```

## 속성

`Enabled`  
로 설정하면 `true`StackSets는 지정된 리전의 대상 조직 또는 조직 단위(OU)에 추가되는 AWS Organizations 계정에 추가 스택 인스턴스를 자동으로 배포합니다. 대상 조직 또는 OU에서 계정이 제거되면 StackSets는 지정된 리전의 계정에서 스택 인스턴스를 삭제합니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음

`RetainStacksOnAccountRemoval`  
로 설정하면 `true`대상 조직 또는 OU에서 계정이 제거될 때 스택 리소스가 유지됩니다. 로 설정하면 `false`스택 리소스가 삭제됩니다. `Enabled` 로 설정된 경우에만 지정합니다`True`.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
