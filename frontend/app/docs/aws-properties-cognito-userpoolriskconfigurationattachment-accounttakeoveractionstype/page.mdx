# AWS::Cognito::UserPoolRiskConfigurationAttachment AccountTakeoverActionsType

계정 인수 작업 유형입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "HighAction" : AccountTakeoverActionType,
  "LowAction" : AccountTakeoverActionType,
  "MediumAction" : AccountTakeoverActionType
}
```

## 속성

`HighAction`  
높은 위험에 대해 취해야 할 조치입니다.  
*필수 항목 *여부: 아니요  
_유형_: AccountTakeoverActionType  
_업데이트 필요 사항_: 중단 없음

`LowAction`  
낮은 위험을 위해 취해야 할 조치입니다.  
*필수 항목 *여부: 아니요  
_유형_: AccountTakeoverActionType  
_업데이트 필요 사항_: 중단 없음

`MediumAction`  
중간 위험에 대해 취해야 할 조치입니다.  
*필수 항목 *여부: 아니요  
_유형_: AccountTakeoverActionType  
_업데이트 필요 사항_: 중단 없음
