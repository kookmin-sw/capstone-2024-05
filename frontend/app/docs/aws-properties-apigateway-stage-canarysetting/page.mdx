# AWS::ApiGateway::스테이지 카나리아 설정

카나리아 배포의 구성 설정입니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "DeploymentId" : String,
  "PercentTraffic" : Double,
  "StageVariableOverrides" : {Key: Value, ...},
  "UseStageCache" : Boolean
}
```

## 속성

`DeploymentId`  
카나리아 배포의 ID입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`PercentTraffic`  
카나리아 배포로 전환된 트래픽의 백분율(0-100)입니다.  
*필수 항목 *여부: 아니요  
_유형_: 더블  
_업데이트 필요 사항_: 중단 없음

`StageVariableOverrides`  
canary에 도입된 새 단계 변수를 포함하여 canary 릴리스 배포를 위해 재정의된 단계 변수. 이러한 단계 변수는 단계 변수 이름과 해당 값 사이의 문자열 간 맵으로 표현됩니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열 맵  
_업데이트 필요 사항_: 중단 없음

`UseStageCache`  
canary 배포가 단계 캐시를 사용하는지 여부를 나타내는 부울 플래그입니다.  
*필수 항목 *여부: 아니요  
_유형_: Boolean  
_업데이트 필요 사항_: 중단 없음
