# AWS::백업::프레임워크 프레임워크 컨트롤

프레임워크의 모든 컨트롤에 대한 자세한 정보를 포함합니다. 각 프레임워크에는 하나 이상의 컨트롤이 포함되어야 합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ControlInputParameters" : [ ControlInputParameter, ... ],
  "ControlName" : String,
  "ControlScope" : ControlScope
}
```

## 속성

`ControlInputParameters`  
`ParameterName` 및 쌍의 목록 `ParameterValue` 입니다.  
*필수 항목 *여부: 아니요  
_형식_: ControlInputParameter 목록  
_업데이트 필요 사항_: 중단 없음

`ControlName`  
컨트롤의 이름입니다. 이 이름은 1자에서 256자 사이입니다.  
_필수 항목 여부_: 예  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`ControlScope`  
컨트롤의 범위입니다. 컨트롤 범위는 컨트롤이 평가할 항목을 정의합니다. 제어 범위의 세 가지 예로는 특정 백업 계획, 특정 태그가 있는 모든 백업 계획 또는 모든 백업 계획이 있습니다.  
*필수 항목 *여부: 아니요  
_형식_: ControlScope  
_업데이트 필요 사항_: 중단 없음
