# AWS::EC2::NetworkInsights분석 분석로드 밸런서 대상

부하 분산 장치 대상에 대해 설명합니다.

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "Address" : String,
  "AvailabilityZone" : String,
  "Instance" : AnalysisComponent,
  "Port" : Integer
}
```

## 속성

`Address`  
IP 주소입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_최소_: `0`  
_최대_: `15`  
_패턴_: `^([0-9]{1,3}.){3}[0-9]{1,3}$`  
_업데이트 필요 사항_: 중단 없음

`AvailabilityZone`  
가용 영역입니다.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음

`Instance`  
인스턴스에 대한 정보입니다.  
*필수 항목 *여부: 아니요  
_유형_: AnalysisComponent  
_업데이트 필요 사항_: 중단 없음

`Port`  
대상이 수신 대기 중인 포트입니다.  
*필수 항목 *여부: 아니요  
_유형_: Integer  
_최소_: `0`  
_최대_: `65535`  
_업데이트 필요 사항_: 중단 없음
