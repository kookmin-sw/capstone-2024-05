# AWS::SageMaker::ModelCard 교육 세부 정보

모델의 학습 세부 정보

## 통사론

AWS CloudFormation 템플릿에서 이 엔터티를 선언하려면 다음 구문을 사용합니다.

### JSON (영문)

```
{
  "ObjectiveFunction" : ObjectiveFunction,
  "TrainingJobDetails" : TrainingJobDetails,
  "TrainingObservations" : String
}
```

## 속성

`ObjectiveFunction`  
모델 학습 중에 최적화되는 함수입니다.  
*필수 항목 *여부: 아니요  
_형식_: ObjectiveFunction  
_업데이트 필요 사항_: 중단 없음

`TrainingJobDetails`  
연결된 학습 작업에 대한 세부 정보입니다.  
*필수 항목 *여부: 아니요  
_형식_: TrainingJobDetails  
_업데이트 필요 사항_: 중단 없음

`TrainingObservations`  
훈련에 대한 모든 관찰.  
*필수 항목 *여부: 아니요  
_유형_: 문자열  
_업데이트 필요 사항_: 중단 없음
